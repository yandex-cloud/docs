Потоки данных создаются на основе {{ ydb-full-name }} и хранятся внутри баз данных.

`Идентификатор потока данных` является составным и состоит из **идентификатора каталога**, **идентификатора базы данных** {{ ydb-short-name }} и **имени потока**.

Например, `aoeu1kuj2dhtaupdb5es/cc8028jgtuabcqutgtbv/stream1` определяет каталог `aoeu1kuj2dhtaupdb5es`, базу данных {{ ydb-short-name }} `cc8028jgtuabcqutgtbv` и поток `stream1`.
