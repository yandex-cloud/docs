Чтобы создать кластер {{ k8s }}:
1. На странице каталога в [консоли управления](https://console.cloud.yandex.ru) в списке сервисов выберите **{{ managed-k8s-name }}**.
1. Нажмите кнопку **Создать кластер**.
1. Введите имя и описание кластера {{ k8s }}.
1. В блоке **Конфигурация мастера**:
    - Выберите [зону доступности](../../overview/concepts/geo-scope.md), в которой будет создан [мастер](../../managed-kubernetes/concepts/index.md#master).
    - Выберите **Сервисный аккаунт для ресурсов**, который будет использоваться для создания ресурсов.
    - Выберите **Сервисный аккаунт для узлов**, который будет использоваться узлами для доступа к реестру Docker-образов.
    - В поле **Публичный адрес** выберите способ назначения адреса **Автоматически**.
    - В поле **Формат сети** выберите:
        - **Список** — чтобы указать сеть и подсеть из списков. Если сети еще не созданы, вы можете создать их прямо на странице создания кластера.
        - **ID** — чтобы указать идентификаторы сети и подсети. 
1. В блоке **Сетевые настройки кластера**:
    - Укажите **CIDR кластера** — диапазон IP-адресов, из которого будут выделяться IP-адреса для подов.
    - Укажите **CIDR сервисов** — диапазон IP-адресов, из которого будут выделяться IP-адреса для сервисов.
1. Нажмите кнопку **Создать кластер**.
