Сервисы {{ yandex-cloud }} делятся на следующие категории:
- _Infrastructure_ — инфраструктурные сервисы для обработки данных, безопасного доступа и обмена трафиком.
- _Data storage & Analytics_ — сервисы для надежного хранения, обработки и визуализации данных.
- _Serverless_ — сервисы для хранения данных и разработки приложений без создания {% if lang == "ru" and audience != "internal" %}[виртуальных машин](../../glossary/vm.md){% else %}виртуальных машин{% endif %}.
- _Machine Learning & Artificial Intelligence (ML & AI)_ — речевые технологии, анализ изображений и машинный перевод.
