#### {{ roles-datasphere-project-admin }} {#projects-admin}

Роль `{{ roles-datasphere-project-admin }}` позволяет управлять правами доступа к проекту.

Пользователь с ролью `{{ roles-datasphere-project-admin }}` может поделиться ресурсами этого проекта с сообществом, на которое имеет права `Developer` (роль `{{ roles-datasphere-communities-developer }}`).

Помимо этого роль `{{ roles-datasphere-project-admin }}` включает в себя все разрешения роли `{{ roles-datasphere-project-editor }}`.
