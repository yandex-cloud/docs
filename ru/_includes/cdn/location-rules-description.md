_Локационные правила_ — это механизм точечного переопределения настроек [CDN-ресурса](../../cdn/concepts/resource.md) для запросов, пути которых соответствуют заданному шаблону.

> Например, в настройках CDN-ресурса включено [кеширование контента](../../cdn/concepts/caching.md), а время жизни кеша составляет 4 дня. Для большей части контента этого достаточно, однако для некоторых специфичных файлов нужно уменьшить время жизни кеша до нескольких секунд. Для решения подобных задач применяются локационные правила.
