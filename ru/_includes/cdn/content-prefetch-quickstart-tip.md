{% note tip %}

Рекомендуется [принудительно загружать](../../cdn/operations/resources/prefetch-files.md) в [кеш](../../cdn/concepts/caching.md) CDN-серверов объемный контент (например, файлы более 200 МБ). Это опциональный шаг: если его не выполнить, то контент будет загружен в кеш [CDN-серверов](../../cdn/concepts/points-of-presence.md) автоматически при первом обращении к нему конечного пользователя.

{% endnote %}
