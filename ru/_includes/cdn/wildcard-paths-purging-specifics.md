Для частичной очистки кеша можно указывать пути к конкретным файлам и каталогам. Каждый путь должен начинаться с символа `/` или (только при использовании [провайдера](../../cdn/concepts/providers.md) `EdgeCDN`) `*`. Символ подстановки `*` может заменять любое количество символов. 

{% note info %}

При использовании провайдера {{ cdn-full-name }} символ подстановки `*` может быть указан только в конце пути. Если указать `*` в начале или середине пути, кеш для соответствующих этому пути файлов не будет очищен.

{% endnote %}
