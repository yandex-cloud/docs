Модуль [{{ td-full-name }} ({{ td-name }})](../../security-deck/concepts/threat-detector.md) представляет собой первую линию защиты облачной инфраструктуры пользователя в {{ sd-full-name }} и позволяет в автоматическом режиме обнаруживать подозрительную активность и уведомлять пользователя об обнаруженных угрозах.

Модуль {{ td-full-name }} анализирует события аудита, фиксируемые в инфраструктуре клиента и регистрируемые с помощью сервиса [{{ at-full-name }}](../../audit-trails/index.yaml). Модуль не требует настройки правил или привлечения экспертов в сфере безопасности и активируется автоматически при настройке [окружения](../../security-deck/concepts/workspace.md) {{ sd-name }}.

При каждом срабатывании [правила](../../security-deck/concepts/threat-detector.md#rules) на потенциальную угрозу безопасности {{ sd-name }} формирует [алерт](../../security-deck/concepts/alerts.md) с подробной информацией о событии и советами по устранению выявленной угрозы. К анализу алерта и поиску решения по устранению вы можете привлечь AI-ассистента, который более подробно изложит суть проблемы и предложит оптимальные способы ее решения.
