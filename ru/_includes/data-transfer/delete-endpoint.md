{% note warning %}

Перед удалением эндпоинта [удалите](../../data-transfer/operations/transfer.md#delete) все трансферы, в которые он входит.

{% endnote %}

Чтобы удалить эндпоинт:

{% list tabs %}

- Консоль управления

    1. Перейдите на страницу каталога и выберите сервис **{{ data-transfer-name }}**.
    1. Выберите эндпоинт, который нужно удалить.
    1. Нажмите кнопку **Удалить** на панели сверху.
    1. В открывшемся окне нажмите кнопку **Удалить**.

{% endlist %}
