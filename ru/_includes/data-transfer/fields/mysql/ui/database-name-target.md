**Имя базы данных** — укажите имя базы данных в выбранном кластере. Оставьте поле пустым, если хотите создать таблицы в базах данных с теми же именами, что и на источнике. В этом случае явно задайте схему БД для служебных таблиц в [дополнительных настройках](../../../../../data-transfer/operations/endpoint/target/mysql.md#additional-settings).
