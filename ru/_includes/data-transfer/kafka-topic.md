* **Настройки топика Apache Kafka**:

    * **Полное имя топика** — укажите имя топика, в который будут отправляться сообщения.

    * **Префикс топика** — укажите префикс топика, аналог настройки `Debezium database.server.name`. Сообщения будут отправляться в топик c именем `<префикс_топика>.<схема>.<имя_таблицы>`.
