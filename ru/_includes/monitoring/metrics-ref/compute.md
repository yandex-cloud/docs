## Сервис {{ compute-full-name }} {#compute}

Все метрики сервиса {{ compute-name }} имеют общую метку `service=compute`.

{% note info %}

Если у диска задано имя, оно будет возвращаться в ответе сервиса. В случае отсутсвия имени в ответе вернется идентификатор диска.

{% endnote %}
