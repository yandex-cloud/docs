### {{ roles.k8s.cluster-api.editor }} {#k8s-clusters-api-editor}

Пользователь с ролью `{{ roles.k8s.cluster-api.editor }}` получает группу `yc:edit` и роль `edit` в {{ k8s }} RBAC для всех [пространств имен](../managed-kubernetes/concepts/index.md#namespace) в [кластере](../managed-kubernetes/concepts/index.md#kubernetes-cluster).
