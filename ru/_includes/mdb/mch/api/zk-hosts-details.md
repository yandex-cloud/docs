Если вы создаете кластер из двух и более хостов {{ CH }} без сервиса координации и в [сети](../../../../vpc/concepts/network.md) кластера есть [подсети](../../../../vpc/concepts/network.md#subnet) в каждой [зоне доступности](../../../../overview/concepts/geo-scope.md), то три хоста {{ ZK }} будут добавлены автоматически, по одному в каждую подсеть.

Если подсети есть только в некоторых зонах доступности или вы хотите использовать другой сервис координации, вручную укажите сервис координации и настройки его хостов.
