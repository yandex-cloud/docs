{% note warning %}

При изменении типа диска хосты кластера пересоздаются. Данные реплицированных таблиц сохраняются автоматически. Данные нереплицированных таблиц будут потеряны. Перед изменением типа диска [преобразуйте](../../../managed-clickhouse/operations/zk-hosts.md#replicated-tables) нереплицированные таблицы в реплицированные или [создайте резервную копию данных](../../../managed-clickhouse/operations/cluster-backups.md).

{% endnote %}
