#### Квоты {#mmg-quotas}

| Вид ограничения                                                                | Значение |
|--------------------------------------------------------------------------------|----------|
| Количество кластеров в одном облаке                                            | 16       |
| Суммарное количество ядер процессора для всех хостов баз данных в одном облаке | 64       |
| Суммарный объем виртуальной памяти для всех хостов баз данных в одном облаке   | 512 ГБ   |
| Суммарный объем хранилищ для всех кластеров в одном облаке                     | 4096 ГБ  |

| Вид ограничения | Минимальное значение | Максимальное значение |

| Количество шардов в кластере | 1 | 10 |
| Количество хостов в шарде при использовании сетевого хранилища | 1 | 7 |

| Количество хостов в одном кластере | 1 хост (кластер [создается](../../managed-mongodb/operations/cluster-create.md) нешардированным) | 70 (10 шардов × 7 хостов) |
| Объем данных на хосте при использовании сетевого хранилища | 10 ГБ | 605 ГБ |
| Объем данных на хосте при использовании локального хранилища | 100 ГБ | 600 ГБ |
