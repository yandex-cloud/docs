Типы конфигураций:

* **s2** — стандартные конфигурации, с соотношением количества vCPU к количеству гигабайт RAM 1:4.
* **m2** — конфигурации с уменьшенным соотношением количества vCPU к количеству гигабайт RAM (1:8). Такие конфигурации могут быть полезны для кластеров с повышенными требованиями к кэшу.
* **b2** — конфигурации с гарантированной долей vCPU ниже 100%. Этот класс хостов предназначен для тестовой нагрузки, минимальная рекомендуемая конфигурация хоста для продакшн-решений — 2 vCPU с гарантированной долей 50%.
