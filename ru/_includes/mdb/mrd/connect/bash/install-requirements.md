**Перед подключением установите зависимости:**

```bash
sudo apt update && sudo apt install -y redis-tools
```

{% note info %}

Для подключения к кластеру с поддержкой TLS [скачайте](https://redis.io/download) архив с исходным кодом утилиты и выполните ее сборку с TLS командой `make BUILD_TLS=yes`.

{% endnote %}
