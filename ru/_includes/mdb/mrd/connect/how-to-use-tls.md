{% note warning %}

Чтобы подключаться с использованием шифрования, [создайте кластер](../../../../managed-redis/operations/cluster-create.md) с {{ RD }} версии не ниже 6 и включенной настройкой **Поддержка TLS**.

{{ RD }} версии 6.0 и ниже вместо FQDN хоста возвращает его IP-адрес. Из-за этого при подключении к кластеру с SSL проверка имени хоста в сертификате завершается ошибкой и подключения не происходит. Поэтому в примерах строк подключения такая проверка отключена.

{% endnote %}
