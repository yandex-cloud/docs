* **Менеджер соединений** — укажите параметры [менеджера подключений](../concepts/pooling.md):
   * **Режим** — выберите один из режимов работы.
   * **Размер** — укажите размер буфера.
   * **Тайм-аут ожидания клиента** — укажите время неактивности клиентского соединения (в миллисекундах), после которого соединение разрывается.
