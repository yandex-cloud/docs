Тип хранилища выбирается при создании кластера:

* Стандартное сетевое хранилище (`network-hdd`) — самый экономичный вариант. Подходит для кластеров, не требовательных к скорости записи и чтения.
* Быстрое локальное хранилище (`local-ssd`) — самые быстрые диски. Объем локального хранилища можно увеличивать только с шагом 100 ГБ.
* Быстрое сетевое хранилище (`network-ssd`) — медленнее, чем локальное хранилище, но, в отличие от него, обеспечивает сохранность данных при выходе из строя оборудования {{ yandex-cloud }}.
