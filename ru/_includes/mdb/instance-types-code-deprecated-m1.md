Типы конфигураций:

* **s1, s2** — стандартные конфигурации, с соотношением количества vCPU к количеству гигабайт RAM 1:4.
* **m1, m2** — конфигурации с уменьшеным соотношением количества vCPU к количеству гигабайт RAM (1:8 в отличие от 1:4 для стандартных классов). Такие конфигурации могут быть полезны для кластеров с повышенными требованиями к кэшу.
* **b1, b2** — конфигурации с гарантированной долей vCPU ниже 100%. Этот класс хостов предназначен для тестовой нагрузки, минимальная рекомендуемая конфигурация хоста для продакшн-решений — 2 vCPU с гарантированной долей 50%.

  {% note info %}

  Хосты с гарантированной долей vCPU ниже 100% не могут использоваться в многохостовых кластерах.  

  {% endnote %}
