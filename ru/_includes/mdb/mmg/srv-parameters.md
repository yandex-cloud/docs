При необходимости вы можете добавить к команде параметры подключения, например:
* `tls` — включение TLS-шифрования соединения: `true` или `false`. По умолчанию шифрование включено.
* `tlsCAFile` — путь к файлу `.pem`, содержащему корневые сертификаты центра сертификации.
* `readPreference` — с каких хостов выполнять чтение:
  * `primary` — только с первичной реплики (мастера);
  * `primaryPreferred` — с мастера, а если он недоступен — с вторичных реплик;
  * `secondary` — только с вторичных реплик;
  * `secondaryPreferred` — с вторичных реплик, а если они отсутствуют, то с мастера;
  * `nearest` — с хоста с наименьшей задержкой.
* `authSource` — имя базы данных, в которой хранятся учетные данные пользователя. По умолчанию совпадает со значением параметра `<имя_БД>`.
* `appName` — имя приложения, отображаемое в логах.

Описание других доступных параметров подключения см. в [официальной документации](https://mongodb.prakticum-team.ru/docs/manual/reference/connection-string-options/).

Строка параметров начинается со знака `?`, разделителем является знак `&`.
