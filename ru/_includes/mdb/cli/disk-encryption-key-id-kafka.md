`--disk-encryption-key-id` — идентификатор [пользовательского ключа KMS](../../../kms/concepts/key.md). Чтобы зашифровать диск, передайте идентификатор ключа KMS в параметре. Подробнее о шифровании дисков см. в разделе [Хранилище](../../../managed-kafka/concepts/storage.md#disk-encryption).

{% note warning %}
   
Включить шифрование дисков можно только при создании кластера.
   
{% endnote %}
