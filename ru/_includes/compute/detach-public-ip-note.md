{% note info %}

Если отвязываемый IP-адрес был назначен автоматически (динамический IP-адрес), то вы не сможете повторно привязать его к облачному ресурсу. Он будет удален. Чтобы сохранить динамический IP-адрес для дальнейшего использования, [сделайте](../../vpc/operations/set-static-ip.md) его статическим.

Статический IP-адрес остается в зарезервированных IP-адресах в [{{ vpc-short-name }}](../../vpc/index.yaml), и его можно использовать для доступа к облачным ресурсам позднее.

{% endnote %}
