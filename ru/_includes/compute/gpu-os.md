Для виртуальных машин с GPU доступны специальные образы операционных систем Windows — 2016 Datacenter GPU (`windows-2016-gvlk-gpu`) и Ubuntu — 16.04 lts GPU (`ubuntu-1604-lts-gpu`), на которых установлены драйверы NVIDIA. Чтобы использовать другие образы, [установите необходимые драйверы](../../compute/operations/vm-operate/install-nvidia-drivers.md) самостоятельно.
