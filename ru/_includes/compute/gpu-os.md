Для виртуальных машин с GPU доступны специальные образы операционных систем Windows (`windows-2016-gvlk-gpu`) и Ubuntu (`ubuntu-1604-lts-gpu`), на которых установлены драйверы NVIDIA. Чтобы использовать другие образы, [установите необходимые драйверы](../../compute/operations/image-create/install-nvidia-drivers.md) самостоятельно.
