{% note warning %}

Для загрузочного диска вспомогательной ВМ используйте образ операционной системы, отличный от того, что установлен на восстанавливаемой ВМ. Например, если на восстанавливаемой ВМ установлена Ubuntu 22.04, для вспомогательной ВМ выберите Ubuntu 20.04, Ubuntu 24.04 или другой дистрибутив. Иначе из-за совпадения UUID файловых систем вспомогательная ВМ может загрузиться с подключенного диска восстанавливаемой ВМ, а не со своего собственного.

{% endnote %}
