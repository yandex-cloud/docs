{% note tip %}

Прежде чем ввести `yes`, убедитесь, что отпечаток (`fingerprint`) ключа, отображаемый в терминале, совпадает с отпечатком SSH-ключа на ВМ. Это защитит соединение от подмены сервера.


{% cut "Посмотреть отпечаток SSH-ключа на ВМ" %}

1. [Получите](../../compute/operations/vm-info/get-serial-port-output.md) вывод последовательного порта ВМ.
1. В выводе найдите блок, который начинается со строки `BEGIN SSH HOST KEY FINGERPRINTS` и заканчивается строкой `END SSH HOST KEY FINGERPRINTS`. В этом блоке для каждого алгоритма шифрования (`ED25519`, `ECDSA`, `RSA`) приведен отпечаток вида `SHA256:...`.
1. Сравните отпечаток для нужного алгоритма с отпечатком, который SSH-клиент показал в терминале. Алгоритм указан в сообщении SSH-клиента.

{% endcut %}

Если отпечатки совпадают, продолжайте подключение. Если отпечатки различаются, отмените подключение и проверьте параметры ВМ и сети.

{% endnote %}
