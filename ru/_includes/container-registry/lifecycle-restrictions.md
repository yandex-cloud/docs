{% note warning %}

Задать политику удаления можно только для [репозитория](../../container-registry/concepts/repository.md). Политика применяется к Docker-образам, имена которых в точности совпадают с именем репозитория. Совпадение по префиксу не поддерживается. Нельзя задать политику для группы репозиториев, реестра, каталога или облака.

{% endnote %}
