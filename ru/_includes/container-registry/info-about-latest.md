{% note important %}

Тег `latest` **не означает**, что помеченная им версия Docker-образа была загружена последней. 

{% endnote %}

Рекомендуется не перезаписывать теги, а использовать для каждой версии Docker-образа уникальный тег. Это позволяет использовать одну версию Docker-образа на всех виртуальных машинах с одинаковой спецификацией и облегчает поиск причины проблемы.
