Доступ к Биллингу предоставляется пользователям с зарегистрированным аккаунтом на Яндексе или в Яндекс 360: 

* Если у вас или вашего сотрудника еще нет аккаунта, создайте его на [Яндексе]{% if lang == "ru" %}(https://passport.yandex.ru/registration){% endif %}{% if lang == "en" %}(https://passport.yandex.com/registration){% endif %} или в [Яндекс 360]{% if lang == "ru" %}(https://yandex.ru/support/business/add-users.html){% endif %}{% if lang == "en" %}(https://yandex.com/support/business/add-users.html){% endif %}.
* Если для авторизации на Яндексе вы используете профиль в социальной сети, [заведите логин и пароль]{% if lang == "ru" %}(https://passport.yandex.ru/passport?mode=postregistration&create_login=1){% endif %}{% if lang == "en" %}(https://passport.yandex.com/passport?mode=postregistration&create_login=1){% endif %}.

К аккаунту обязательно должен быть привязан ваш номер телефона. Подробные инструкции смотрите в Справке [Яндекс ID]{% if lang == "ru" %}(https://yandex.ru/support/passport/authorization/registration.html){% endif %}{% if lang == "en" %}(https://yandex.com/support/passport/authorization/registration.html){% endif %} и [Яндекс 360]{% if lang == "ru" %}(https://yandex.ru/support/business/add-users.html){% endif %}{% if lang == "en" %}(https://yandex.com/support/business/add-users.html){% endif %}.
