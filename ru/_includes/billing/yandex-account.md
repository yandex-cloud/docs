Доступ к Биллингу предоставляется пользователям с зарегистрированным аккаунтом на Яндексе или в Яндекс.Коннекте: 
 
 * Если у вас или вашего сотрудника еще нет аккаунта, создайте его на [Яндексе]{% if lang == "ru" %}(https://passport.yandex.ru/registration){% endif %}{% if lang == "en" %}(https://passport.yandex.com/registration){% endif %} или в [Яндекс.Коннекте]{% if lang == "ru" %}(https://yandex.ru/support/connect/org-structure.html#org-structure__manage-user){% endif %}{% if lang == "en" %}(https://yandex.com/support/connect/org-structure.html#org-structure__manage-user){% endif %}. 
 * Если для авторизации на Яндексе вы используете профиль в социальной сети, [заведите логин и пароль]{% if lang == "ru" %}(https://passport.yandex.ru/passport?mode=postregistration&create_login=1){% endif %}{% if lang == "en" %}(https://passport.yandex.com/passport?mode=postregistration&create_login=1){% endif %}. 
 
К аккаунту обязательно должен быть привязан ваш номер телефона. Подробные инструкции смотрите в Справке [Яндекс ID]{% if lang == "ru" %}(https://yandex.ru/support/passport/authorization/registration.html){% endif %}{% if lang == "en" %}(https://yandex.com/support/passport/authorization/registration.html){% endif %} и [Яндекс.Коннекта]{% if lang == "ru" %}(https://yandex.ru/support/connect/org-structure.html){% endif %}{% if lang == "en" %}(https://yandex.com/support/connect/org-structure.html){% endif %}.
