Изнутри виртуальной машины сервис [метаданных](../compute/concepts/vm-metadata.md) доступен по IP-адресу `169.254.169.254`.

На данный момент сервис метаданных Яндекс.Облака возвращает метаданные в форматах [Google Compute Engine](../compute/operations/vm-info/get-info.md#gce-metadata) и [Amazon EC2](../compute/operations/vm-info/get-info.md#ec2-metadata).
