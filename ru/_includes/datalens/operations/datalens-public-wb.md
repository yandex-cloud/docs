1. На панели слева выберите ![collections](../../../_assets/console-icons/rectangles-4.svg) **Коллекции и воркбуки**.
1. Откройте нужный воркбук и выберите дашборд или чарт.
1. В верхней части объекта нажмите значок ![image](../../../_assets/console-icons/ellipsis.svg) и выберите ![image](../../../_assets/console-icons/arrow-shape-turn-up-right.svg) **Поделиться**.
1. В открывшемся окне перейдите на вкладку ![image](../../../_assets/console-icons/globe.svg) **Публично** и настройте публичный доступ:
        
   {% list tabs group=datalens_public %}

   - Для дашборда {#dashboard}

      1. Включите опцию ![image](../../../_assets/console-icons/plus.svg) **Публичный доступ**.        
      1. (опционально) Нажмите ![image](../../../_assets/console-icons/layers-3-diagonal.svg) **Добавить в Галерею {{ datalens-short-name }}** и заполните форму для [добавления своей работы](../../../datalens/concepts/gallery.md#suggest) в Галерею.
      1. (опционально) Включите опцию **Автор дашборда** и укажите ссылку на источник или автора дашборда и ее текст. Ссылка будет отображаться внизу дашборда.
      1. (опционально) В блоке **Связанные объекты** выберите объекты, доступ к которым станет публичным. По умолчанию {{ datalens-short-name }} автоматически предоставит доступ ко всем связанным объектам.
      1. Внизу справа нажмите ![image](../../../_assets/console-icons/link.svg) **Скопировать ссылку**.

         {% note info %}

         * По ссылке для публичного доступа дашборд и все включенные связанные объекты будут доступны любому пользователю в интернете. Чтобы скопировать ее, нажмите **Скопировать ссылку**.
         * По ссылке из адресной строки браузера дашборд и связанные с ним объекты доступны только авторизованным пользователям с учетом прав доступа к ним.

         {% endnote %}

      1. (опционально) В блоке **Связанные объекты** выберите объекты, доступ к которым станет публичным. По умолчанию {{ datalens-short-name }} автоматически предоставит доступ ко всем связанным объектам.
