## Config {#config}

### Общая информация {#common}

В результате выполнения вкладки Config должны быть экспортированы данные, отвечающие за дополнительную конфигурацию отображения.

### Пример {#example}

```js
module.exports = {
    // форматирование отображаемых значений (по умолчанию: null)
    format: {
        // количество десятичных знаков (по умолчанию: 0)
        precision: 1,
        // умножать на 100 и добавлять знак процентов (по умолчанию: true)
        percent: true
    },
    // переопределение информации, выводимой в тултипе (по умолчанию: null)
    // * point - объект - данные точки, на которую наведен курсор
    manageTooltipConfig: (point) => {
        // возвращаем переопределенную информацию
        return {
            // значения в тултипе
            tooltipValues: [
                {
                    // заголовок (по умолчанию: null)
                    title: 'доля',
                    // значение (по умолчанию: 0)
                    value: point.value
                },
                {
                    title: 'делимое',
                    value: point.dividend
                },
                {
                    title: 'делитель',
                    value: point.divider
                }
            ]
        };
    }
};
```
