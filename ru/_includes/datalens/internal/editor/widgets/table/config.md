## Config {#config}

### Общая информация {#common}

В результате выполнения вкладки Config должны быть экспортированы данные, отвечающие за дополнительную конфигурацию.

### Пример {#example}

Вкладка **Config**
```js
module.exports = {
    // заголовок таблицы, короткая запись
    // title: 'Заголовок таблицы',
    // заголовок таблицы со стилями
    title: {
        // текст заголовока
        text: 'Заголовок таблицы',
        // CSS-стили заголовка
        style: {
            'text-align': 'center',
            'font-size': '20px'
        }
    },
    // id колонки из шапки таблицы, по которой будет производится сортировка
    sort: 'date',
    // порядок сортировки: 'acs' - прямой, 'desc' - обратный
    order: 'asc'
};
```
