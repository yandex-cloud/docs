## Использование {#usage}

### Общая информация {#common}

Использовать модуль можно на любой вкладке.

По клику на подсвеченный пунктиром путь модуля доступны следующие возможности:
* `CMD + клик ` - открыть в новой вкладке исходный код модуля в редакторе.
* `ALT + CMD + клик` - показать документацию модуля.

### Пример {#example}

```js
// запрашиваем модуль по пути my/module
const myModule = require('my/module');
const piDivideByRandom = myModule.PI / myModule.getRandom();
```
