## Документирование {#docs}

### Общая информация {#common}

На вкладках Doc Ru и Doc En с помощью вики-текста можно задокументировать модуль.
Узнать о возможностях вики текста можно в [документации](https://wiki.yandex-team.ru/wiki/Vodstvo/).
Посмотреть форматированный результат можно кликнув на ![eye-outline](../../../../../../_assets/datalens/internal/editor/widgets/module/eye-outline.png) в правом верхнем углу редактора.

### Пример {#example}

```
Модуль предоставляет следующий функционал:
* %%PI%% - константа Пи с двумя знаками после запятой
* %%getRandom()%% - функция без аргументов, которая возвращает случайное число между 0 и 1
```
