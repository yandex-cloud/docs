[**Yandex Charts**](https://charts.yandex-team.ru) — это универсальный сервис рисования графиков,
создаваемых поверх данных из десятков различных источников, с возможностью встраивания этих графиков в интерфейсы других сервисов.

**Charts API** позволяет создавать и обновлять чарты. Через него вы можете скриптом сгенерировать произвольное количество чартов и затем использовать их на дашбордах, снимать скриншоты, выгружать данные и т.д. — на выходе у вас получатся обычные конфиги, которые затем можно редактировать в [Editor](https://charts.yandex-team.ru/editor) или [Wizard](https://charts.yandex-team.ru/wizard). При генерации через API учитывайте, что зачастую вместо создания N графиков проще использовать [параметризацию](../../../../datalens/editor/params.md).
