`<IP-адрес_сервера_в_дополнительной_подсети>` — IPv4-адрес, который будет использоваться сервером в дополнительной подсети, с префиксом [CIDR](https://ru.wikipedia.org/wiki/Бесклассовая_адресация). Например: `192.168.1.10/24`.

Адрес должен принадлежать диапазону настраиваемой дополнительной приватной подсети и быть свободным.
