На аппаратном уровне серверы {{ baremetal-name }} управляются через интерфейс [IPMI](https://ru.wikipedia.org/wiki/Intelligent_Platform_Management_Interface) с помощью контроллера Baseboard Management Controller (BMC).

Изменение пользователем любых настроек контроллера BMC вызовет:
* недоступность [KVM-консоли](../../baremetal/operations/servers/server-kvm.md) для управления сервером;
* невозможность остановить, запустить или перезапустить сервер через [консоль управления]({{ link-console-main }}), {{ yandex-cloud }} [CLI](../../cli/cli-ref/baremetal/cli-ref/index.md) или [API](../../api-design-guide/index.yaml);
* невозможность переустановить операционную систему сервера;
* невозможность просматривать аппаратные характеристики работы сервера (температура процессоров, скорость вращения вентиляторов и т.п.)

{% note alert %}

При изменении пользователем настроек контроллера BMC сервис {{ baremetal-full-name }} не гарантирует работоспособность KVM-консоли и других инструментов управления сервером.

{% endnote %}
