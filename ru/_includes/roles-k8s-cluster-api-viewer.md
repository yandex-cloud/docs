### {{ roles.k8s.cluster-api.viewer }} {#k8s-clusters-api-viewer}

Пользователь с ролью `{{ roles.k8s.cluster-api.viewer }}` получает группу `yc:view` и роль `view` в {{ k8s }} RBAC для всех [пространств имен](../managed-kubernetes/concepts/index.md#namespace) в [кластере](../managed-kubernetes/concepts/index.md#kubernetes-cluster).
