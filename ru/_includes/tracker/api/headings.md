#### Заголовки {#req-headers}

- **Host**

    Адрес узла, предоставляющего API:
    ```
    https://api.tracker.yandex.net
    ```

- **Authorization**

    OAuth-токен в формате `OAuth <значение токена>`, например:
    ```
    OAuth 0c4181a7c2cf4521964a72ff57a34a07
    ```

- **X-Org-ID**

    Идентификатор организации.
