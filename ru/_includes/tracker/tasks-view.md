* Чтобы выбрать поля, которые вы хотите видеть в таблице списка задач слева от диаграммы, нажмите на значок ![](../../_assets/tracker/svg/settings-old.svg). Отметьте нужные поля и нажмите **Применить**.

* Чтобы найти задачи в списке, в поле **Поиск по названию задачи** введите текст.

* Чтобы настроить порядок отображения, нажмите кнопку ![](../../_assets/tracker/svg/sorting.svg)&nbsp;**Сортировка**, укажите параметр и нажмите кнопку ![](../../_assets/tracker/svg/new-first.svg). В этом случае ручное перемещение задач станет недоступно.

* Чтобы сгруппировать задачи по статусу, исполнителю или другим параметрам, нажмите кнопку ![](../../_assets/tracker/svg/group.svg) и выберите подходящий вариант из списка.

* Чтобы отфильтровать список задач, используйте конструктор фильтров. Для этого на панели ![](../../_assets/tracker/svg/filter.svg)&nbsp;**Параметры** выберите параметры и укажите значения, по которым будет сформирован список задач.

* Чтобы изменить размер столбца в блоке, зажмите его край и измените ширину.

* Чтобы скрыть блок со списком задач, нажмите на значок ![](../../_assets/tracker/svg/close-block.svg).
