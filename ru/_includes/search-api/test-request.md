1. В рабочем каталоге откройте сервис {{ search-api }}.
1. Перейдите на вкладку **Тестирование запросов**. 
1. Выберите **Метод запроса** (поддерживаются методы [GET](../../search-api/concepts/get-request.md) и [POST](../../search-api/concepts/post-request.md)) и заполните поля формы.

   {% note info %}

   В тестовом запросе нельзя изменить **Текст поискового запроса**. Запросы отправляются с IP-адреса Яндекса от имени сервиса.

   {% endnote %}
1. Нажмите **Запустить тест**. Результат выполнения запроса появится ниже.
1. При необходимости измените параметры, чтобы скорректировать результаты выдачи, и повторите отправку запроса.
