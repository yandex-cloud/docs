#### Идентификатор каталога и API-ключ {#user-key}

Поле `folderid` содержит [идентификатор каталога](../../resource-manager/operations/folder/get-id.md) сервисного аккаунта, от имени которого вы будете отправлять запросы. Вместе с параметром `apikey` — значением API-ключа сервисного аккаунте — необходим для аутентификации пользователя.
