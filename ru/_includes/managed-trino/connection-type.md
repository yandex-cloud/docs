Задайте настройки для нужного типа подключения — [{{ connection-manager-name }}](../../metadata-hub/concepts/connection-manager.md) или On-premise (пользовательская инсталляция).

{{ connection-manager-name }} позволяет создавать подключения к кластеру с управляемой базой данных и к пользовательской инсталляции базы данных.

При подключении через {{ connection-manager-name }} любые изменения в параметрах подключения автоматически учитываются в {{ TR }}. Для On-Premise изменения в параметрах подключения не отслеживаются в {{ TR }}. Их необходимо отслеживать и вносить вручную.
