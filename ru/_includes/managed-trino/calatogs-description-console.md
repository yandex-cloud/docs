Выберите, в каком формате указать каталоги:
* **{{ ui-key.yacloud.trino.rbac-catalog-match-by-name }}** — выберите имена каталогов. Можно выбрать только каталоги, добавленные в блоке **{{ ui-key.yacloud.trino.title_catalogs }}**.
* **{{ ui-key.yacloud.trino.rbac-catalog-match-by-name-regexp }}** — введите регулярное выражение для имен каталогов.
* **{{ ui-key.yacloud.trino.rbac-catalog-match-by-empty }}** — в поле `<имя_каталога>` может быть любое значение.
