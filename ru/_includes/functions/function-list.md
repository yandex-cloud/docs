Для обращения к функции используйте ее имя или уникальный идентификатор, которые можно узнать с помощью команды: 

```
$ yc serverless function list
+----------------------+--------------------+----------------------+
|          ID          |        NAME        |      FOLDER ID       |
+----------------------+--------------------+----------------------+
| b097d9ous3gep99khe83 | my-beta-function   | aoek49ghmknnpj1ll45e |
+----------------------+--------------------+----------------------+
```
