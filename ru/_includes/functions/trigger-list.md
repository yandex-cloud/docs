Для обращения к триггеру используйте его имя или уникальный идентификатор, которые можно узнать с помощью команды: 

```
$ yc serverless trigger list
+----------------------+------------+----------------------+
|          ID          |    NAME    |      FOLDER ID       |
+----------------------+------------+----------------------+
| dd0gj5tsj2pq9at8ja8i | my-trigger | aoek49ghmknnpj1ll45e |
+----------------------+------------+----------------------+
```
