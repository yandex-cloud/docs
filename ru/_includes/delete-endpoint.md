{% note warning %}

Перед удалением эндпоинта [удалите](../data-transfer/operations/transfer/delete-transfer.md) все трансферы, в которые он входит.

{% endnote %}

Чтобы удалить эндпоинт:

{% list tabs %}

- Консоль управления

  1. В [консоли управления]({{ link-console-main }}) выберите каталог, из которого нужно удалить эндпоинт.
  1. В списке сервисов выберите **{{ data-transfer-name }}**.
  1. Выберите эндпоинт, который нужно удалить.
  1. В верхней правой части окна нажмите кнопку **Удалить**.
  1. В открывшемся окне нажмите кнопку **Удалить**.
  
{% endlist %}
