### Идентификатор вопроса {#id}

Используйте идентификатор вопроса для [предзаполнения формы](../../forms/pre-fill.md).
Идентификатор можно редактировать. Все вопросы в одной форме должны иметь уникальные идентификаторы.

### Обязательный вопрос {#required}

Включите эту опцию, чтобы пометить вопрос <q>звездочкой</q> как обязательный. Без ответа на этот вопрос пользователь не сможет отправить заполненную форму.

### Скрытый вопрос {#hidden}

Включите эту опцию, чтобы не отображать вопрос на форме. Скрытые вопросы можно использовать для [передачи служебных или вспомогательных параметров](../../forms/pre-fill.md#hidden-query).

{% note warning %}

Не включайте опцию **Скрытый вопрос** одновременно с опцией **Обязательный вопрос**, иначе пользователи не смогут отправить заполненную форму.

{% endnote %}
