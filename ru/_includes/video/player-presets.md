_Шаблоны плеера_ позволяют настраивать внешний вид и функциональность видеоплеера для [видео](../../video/concepts/videos.md) и [плейлистов](../../video/concepts/playlists.md) в канале. С помощью шаблонов вы можете:

* Настроить виджеты: плейлист, описание и суммаризацию. Включить нужные виджеты и задать их расположение в интерфейсе плеера.
* Задать стили плеера и виджетов — настроить цветовую схему, размеры элементов и другие визуальные параметры.

### Шаблоны по умолчанию {#default-presets}

При [создании канала](../../video/operations/channels/create.md) автоматически создаются два шаблона:

* `Default Style Preset` — шаблон по умолчанию с облачными стилями.
* `Summarization Style Preset` — шаблон, включающий виджет суммаризации для отображения краткого содержания видео.

Свойства шаблона по умолчанию:

* Стили шаблона по умолчанию распространяются на все видео и плейлисты канала, для которых не переопределен шаблон.
* Шаблон по умолчанию нельзя [удалить](../../video/operations/style-presets/delete.md), но можно [переназначить](../../video/operations/style-presets/set-default.md).

Для конкретного видео или плейлиста можно [переопределить шаблон](../../video/operations/style-presets/assign-to-video.md) в настройках.

Также внешний вид можно задавать через [SDK видеоплеера для IFrame](../../video/iframe-sdk.md). В SDK содержится больше настроек плеера, например, можно включить автовоспроизведение при загрузке плеера, запускать видео без звука, скрыть элементы управления плеером и другие.
