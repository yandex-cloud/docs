Доступны версии 3.1, 4.0 и 4.2.

{% note warning %}

Для интеграции кластера {{ metastore-name }} с сервисом [{{ msp-full-name }}](../../managed-spark/index.yaml) требуется соответствие версий:

* версия {{ SPRK }} 3.5.7 работает с версией {{ metastore-name }} 3.1;
* версия {{ SPRK }} 4.0.0 работает с версиями {{ metastore-name }} 3.1 и выше.

При необходимости версию {{ metastore-name }} 3.1 можно повысить последовательно: 3.1 → 4.0 → 4.2, но понизить версию 4.2 до 3.1 невозможно.

{% endnote %}
