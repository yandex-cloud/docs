Получите список [устройств](../../iot-core/concepts/index.md#device) в реестре:

```
$ yc iot device list --registry-name my-registry
```

```

|          ID          |   NAME    |

| b9135goeh1uc1s2i07nm | my-device |
+----------------------+-----------+
```
