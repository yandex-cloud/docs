Вы можете следить за статусами машин в списке виртуальных машин вашего каталога.

{% note info %}

Обычно запуск виртуальной машины и инициализация RDP занимает до 6 минут. В редких случаях загрузка может значительно превышать указанное время. Например, при использовании:
* [Виртуальных машин с гарантированной долей vCPU меньше 100%](../compute/concepts/performance-levels).
* Виртуальных машин, которые производят нагрузку свыше [лимита IOPS](../compute/concepts/performance-levels) на загрузочный диск.
* Сетевых HDD-дисков малого объема.

{% endnote %}
