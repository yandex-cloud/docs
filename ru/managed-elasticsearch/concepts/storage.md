# Типы хранилища

{{ mes-name }} позволяет использовать для кластеров сетевое и локальное хранилища. Сетевое хранилище реализовано на базе сетевых блоков — виртуальных дисков в инфраструктуре {{ yandex-cloud }}. Локальное хранилище организуется на дисках, которые физически размещаются в серверах кластера.

## Особенности локального хранилища {#local-storage-features}

* Локальное хранилище в кластере из одного хоста не обеспечивает отказоустойчивости: при отказе локального диска данные теряются безвозвратно. Чтобы обеспечить отказоустойчивость, создавайте кластеры из двух и более хостов, настраивайте [шардирование и репликацию](scalability-and-resilience.md) индексов.
* Кластер с локальным хранилищем тарифицируется, даже если он остановлен. Подробнее — в [правилах тарификации](../pricing.md).
