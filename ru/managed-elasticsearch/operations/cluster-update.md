# Изменение настроек кластера

После создания кластера {{ ES }} вы можете:
- [Изменить конфигурацию кластера](#update-cluster).
- [Переместить кластер](#move-cluster) из текущего каталога в другой каталог.

## Изменить конфигурацию {#update-cluster}

{% list tabs %}

- API
  
  Воспользуйтесь методом `update` и передайте в запросе:
  - Идентификатор кластера в параметре `clusterId`. Чтобы узнать идентификатор, [получите список кластеров в каталоге](cluster-list.md#list-clusters).
  - Список полей конфигурации кластера подлежащих изменению в параметре `updateMask`. 
  - Новую конфигурацию в параметре `configSpec`.

  {% note warning %}
  
  Этот метод API сбросит все настройки кластера, которые не были явно указаны в параметре `config_spec`, на значения по умолчанию.
  Чтобы избежать этого, перечислите настройки, которые вы хотите изменить, в параметре `updateMask` (одной строкой через запятую).
  
  {% endnote %}

  - Новое имя кластера в параметре `name` (необязательный параметр).
  
{% endlist %}

## Переместить кластер {#move-cluster}

{% list tabs %}

- API
  
  Чтобы переместить кластер из текущего каталога в другой, воспользуйтесь методом API `move` и передайте в запросе:
  - Идентификатор кластера в параметре `clusterId`. Чтобы узнать идентификатор, [получите список кластеров в каталоге](cluster-list.md#list-clusters).
  - Идентификатор каталога назначения в параметре `destinationFolderId`.

{% endlist %}
