# Подключение домена


К API-шлюзу можно подключить только домен, который создан в том же каталоге. При этом к одному API-шлюзу можно подключить несколько доменов. В таком случае домен будет идентифицироваться по заголовку `Host`.

Чтобы подключить домен к API-шлюзу:

{% list tabs %}

- Консоль управления

	1. Разместите у своего DNS-провайдера или на собственном DNS-сервере запись:
	    ```
	    Тип записи: CNAME
		Хост: <домен>
		Значение: <служебный домен API-шлюза>
	    ```
		Чтобы узнать служебный домен API-шлюза:
		1. Перейдите в [консоль управления](https://console.cloud.yandex.ru/).
		2. Выберите каталог, в котором находится API-шлюз, и в списке сервисов — **API Gateway**
		3. Нажмите на API-шлюз.
		4. В открывшемся окне перейдите в раздел **Обзор**. Служебный домен будет в блоке **Общая информация**.

	2. [Создайте](../../certificate-manager/operations/managed/cert-create.md) сертификат от Let's Encrypt<sup>®</sup>.

	3. [Подтвердите](../../certificate-manager/operations/managed/cert-validate.md) права на домен.

	3. Когда сертификат перейдет в статус `Issued`, [создайте](../../certificate-manager/operations/domain/domain-create.md) домен. В поле **Сертификат** выберите сертификат, который создали на Шаге 2.

	4. В [консоли управления](https://console.cloud.yandex.ru/) выберите каталог, в котором находится API-шлюз, и в списке сервисов — **API Gateway**.

	5. Нажмите на API-шлюз.

	6. В открывшемся окне перейдите в раздел **Домены**, нажмите **Подключить** и выберите домен.

{% endlist %}

{% note info %}

Срок действия сертификата от Let's Encrypt<sup>®</sup> — 90 дней. Его нужно своевременно [обновлять](../../certificate-manager/concepts/managed-certificate.md#renew). В некоторых случаях процедура проверки прав на домен проходит [автоматически](../../certificate-manager/concepts/challenges.md#auto).

{% endnote %}
