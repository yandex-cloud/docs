# Cloud DNS API, gRPC: DnsFirewallService

A set of methods for managing DNS firewalls.

## Methods

#|
||Method | Description ||
|| [Get](get.md) | Returns the specified DNS firewall. ||
|| [List](list.md) | Retrieves the list of DNS firewalls in the specified folder. ||
|| [Create](create.md) | Creates a DNS firewall in the specified folder. ||
|| [Update](update.md) | Updates the specified DNS firewall. ||
|| [Move](move.md) | Moves the specified DNS firewall to another folder. ||
|| [Delete](delete.md) | Deletes the specified DNS firewall. ||
|| [ListOperations](listOperations.md) | Lists operations for the specified DNS firewall. ||
|| [ListAccessBindings](listAccessBindings.md) | Lists existing access bindings for the specified DNS firewall. ||
|| [SetAccessBindings](setAccessBindings.md) | Sets access bindings for the specified DNS firewall. ||
|| [UpdateAccessBindings](updateAccessBindings.md) | Updates access bindings for the specified DNS firewall. ||
|#
