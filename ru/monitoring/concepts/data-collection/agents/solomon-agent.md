# Solomon Agent

Solomon Agent — это программа для сбора, хранения, преобразования и доставки метрик в Solomon. Предоставлялась как основной агент для поставки метрик до появления [Unified Agent](unified-agent.md).

В настоящее время рекомендуется использовать Unified Agent. В документации Unified Agent доступны [инструкции по миграции с Solomon Agent на Unified Agent](https://docs.yandex-team.ru/unified_agent/migration/migration_from_solomon_agent).

Документация Solomon Agent расположена по адресу [https://wiki.yandex-team.ru/solomon/agent](https://wiki.yandex-team.ru/solomon/agent).
