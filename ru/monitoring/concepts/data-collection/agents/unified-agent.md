# Unified Agent

Unified Agent — это программа для сбора, хранения, преобразования и доставки метрик в Solomon. Unified Agent также поддерживает работу с логами. Поддерживается multishard pull для сервис-провайдеров Яндекс.Облака.

Unified Agent является рекомендуемым агентом для поставки метрик в Solomon.

Документация Unified Agent расположена по адресу [https://docs.yandex-team.ru/unified_agent](https://docs.yandex-team.ru/unified_agent).
