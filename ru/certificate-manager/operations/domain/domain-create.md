# Создание домена

{% include [preview-stage](../../../_includes/certificate-manager/preview-stage.md) %}

Чтобы создать домен:

{% list tabs %}

- Консоль управления

    1. В [консоли управления]({{ link-console-main }}) выберите каталог, в котором будет создан домен.
    1. В списке сервисов выберите **{{ certificate-manager-name }}**.
    1. Перейдите во вкладку **Домены**.
    1. Нажмите кнопку **Добавить домен**.
    1. В открывшемся окне в поле **Домен** введите доменное имя.
    1. (опционально) В поле **Описание** введите описание домена.
    1. В поле **Сертификат** выберите сертификат из списка подтвержденных сертификатов от Let's Encrypt<sup>®</sup> с подходящим именем домена.
    1. Нажмите кнопку **Создать**.

    В списке доменов появится новый домен.

{% endlist %}
