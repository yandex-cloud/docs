# Поддерживаемые типы оптических трансиверов

Для подключения к оборудованию {{ yandex-cloud }} требуются оптические модули — трансиверы. При заказе сервиса {{ interconnect-name }} на оборудовании {{ yandex-cloud }} за клиентом резервируется порт с выбранным клиентом типом трансивера. С другой стороны подключения — на оборудовании клиента или оператора связи должен быть установлен трансивер, совместимый с трансивером {{ yandex-cloud }}.

Список поддерживаемых со стороны оборудования {{ yandex-cloud }} трансиверов приведен в таблице ниже. Другие типы трансиверов не поддерживаются. 

{% include [transceivers](../../_includes/interconnect/transceivers.md) %}

Все трансиверы имеют коннекторы Duplex LC с полировкой UPC. 

Для подключения каждой оптической кроссировки должны использоваться только оптические волокна типа Single Mode (SM). Использование оптических волокон типа Multi-Mode (MM) не допускается.

Не поддерживаются следующие типы трансиверов:

* Оптические одноволоконные трансиверы.
* Медные трансиверы, работающие по витой паре (UTP5, UTP6).
