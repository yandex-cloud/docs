# Доступные команды {{ mrd-name }}

{{ mrd-name }} позволяет выполнять [любые команды Redis](https://redis.io/commands), кроме следующих:

- `ACL`
- `BGREWRITEAOF`
- `BGSAVE`
- `COMMAND`
- `CONFIG`
- `DEBUG`
- `LASTSAVE`
- `MIGRATE`
- `MODULE`
- `MONITOR`
- `MOVE`
- `OBJECT`
- `REPLICAOF`
- `SAVE`
- `SHUTDOWN`
- `SLAVEOF`
