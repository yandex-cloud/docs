# Сменить мастер

Если в кластере есть несколько хостов, то при необходимости можно сделать мастером реплику. Переключение в среднем занимает около двух минут, кластер в это время доступен.

{% note info %}

Переключение мастера можно выполнять только в нешардированном кластере.

{% endnote %}

{% list tabs %}

- Консоль управления

    1. Перейдите на страницу каталога и выберите сервис **{{ mrd-name }}**.
    1. Выберите кластер и нажмите кнопку **Переключить мастер** на панели сверху.

        Также можно переключить мастер с помощью кнопки **Переключить мастер** на вкладке **Хосты**.

    1. В открывшемся окне выберите опцию **Я хочу переключить мастер** и нажмите кнопку **Переключить**.

- CLI

    {% include [cli-install](../../_includes/cli-install.md) %}

    {% include [default-catalogue](../../_includes/default-catalogue.md) %}

    Чтобы переключить мастер, выполните команду:

    ```bash
    {{ yc-mdb-rd }} cluster start-failover \
       --name <имя кластера>
    ```

    Имя кластера можно запросить со [списком кластеров в каталоге](cluster-list.md).

- API

    Переключить мастер в кластере можно с помощью метода [startFailover](../api-ref/Cluster/startFailover.md).

{% endlist %}
