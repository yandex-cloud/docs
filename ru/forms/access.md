# Настроить доступ

## Кто может редактировать форму {#access}

По умолчанию редактировать форму может только автор.

{% if audience == "external" %}Если вы используете [{{ forms-full-name }} для бизнеса](forms-for-org.md), вы можете дать доступ к редактированию формы сотрудникам вашей организации:

{% else %}

Чтобы открыть другим доступ к редактированию формы:{% endif %}

1. На верхней панели {{ forms-name }} выберите вкладку **Мои формы**.

1. Выберите форму и перейдите на вкладку **Настройки** → **Доступы**.

1. Выберите, кто может редактировать форму:

   {% if audience == "external" %}
   - **Все сотрудники** — форму смогут редактировать все сотрудники вашей организации.
   {% else %} 
   - **Все сотрудники** — форму смогут редактировать все сотрудники Яндекса.
   {% endif %} 
   - **Только автор**.
   - **Некоторые сотрудники** — форму смогут редактировать только сотрудники, перечисленные в списке. Чтобы добавить в список нового пользователя, введите в поле его логин или имя. {% if audience == "internal" %} Вы также можете указать название команды на Стаффе или в ABC, чтобы выдать доступ сразу на несколько человек.{% endif %}

1. Нажмите кнопку **Сохранить**.

## Кто может редактировать группу форм {#access_groups}

{% if audience == "external" %}

   {% else %}

   {% endif %} 
   - **Только автор**.
   - **Некоторые сотрудники** — группу смогут редактировать только сотрудники, перечисленные в списке. Чтобы добавить в список нового пользователя, введите в поле его логин или имя. {% if audience == "internal" %} Вы также можете указать название команды на Стаффе или в ABC, чтобы выдать доступ сразу на несколько человек.{% endif %}

## Запросить доступ к форме или группе форм через IDM {#idm}

Доступ к любой форме Яндекса удобно настраивать с помощью сервиса **IDM**, даже если владелец формы больше не работает в Яндексе. Права управления формой или группой форм можно выдать одному или нескольким сотрудникам, командам на Стаффе или в ABC.

Если вы настроили права управления формой в [интерфейсе {{ forms-full-name }}](#access), они также отобразятся в **IDM**.

Чтобы настроить доступ через **IDM**:

1. Перейдите в [сервис **IDM**](https://idm.yandex-team.ru/).

1. Нажмите **Запросить роль**.

1. Выберите сервис:
   - **Forms Int** для форм, доступных только сотрудникам Яндекса.
   - **Forms Ext** для форм Яндекса для внешних пользователей.

1. Выберите роль: **Менеджер группы форм** или **Менеджер формы**. 
   Роли **Суперпользователь** и **Техподдержка** принадлежат группе разработки {{ forms-full-name }}, такие роли нельзя запросить.

1. Введите имена сотрудников или название команды на Стаффе или в ABC.

1. Укажите код формы или код группы форм. Код можно скопировать из URL.

   {% cut "Как получить код" %}

   - В формах, доступных сотрудникам Яндекса:
   `{{ link-forms }}surveys/80550/` код `80550`.
   - В формах Яндекса для внешних пользователей:
   `https://forms.yandex.ru/surveys/10034284.07057b6e2f312c17aa8da44a4146b5cfa7115ed7/` код `10034284`.
   - В группе форм:
   `{{ link-forms }}admin/groups/1639/settings` код `1639`.
   
   {% endcut %}

1. Нажмите **Запросить**. Запросы на доступ придут всем, у кого есть права управления формой. Доступ появится, когда один из них одобрит запрос.

{% note info %}

После увольнения сотрудника его руководитель автоматически получит право управления  всеми формами, к которым был доступ у сотрудника. Список таких форм придет руководителю на почту.

{% endnote %}
