# Личные формы в {{ forms-full-name }}

Личные формы в {{ forms-full-name }} могут создавать только пользователи с аккаунтами на Яндексе.

Чтобы создать личную форму, перейдите на страницу [{{ link-forms-admin }}]({{ link-forms-admin }}). Подробнее читайте в разделе [{#T}](quick-guide.md).

О возможностях {{ forms-full-name }} читайте в разделе [{#T}](overview.md).

## Ограничения {#restrictions}

* Пользователям личных форм недоступны [возможности для бизнеса](forms-for-org.md#business-features), такие как интеграция с сервисами [{{ tracker-name }}](create-task.md) и [{{ wiki-name }}](send-wiki.md).

* Редактировать и просматривать ответы в личной форме может только владелец: для такой формы нельзя настроить доступ других пользователей, а также открыть ее на [странице управления {{ forms-full-name }} для бизнеса]({{ link-forms-b2b }}).
