# {{ forms-full-name }} для бизнеса

Сотрудники компаний могут использовать формы в своей работе: принимать через формы заказы или обращения, проводить опросы среди коллег или клиентов. 

Пользователям организаций, зарегистрированных на платформе {{ yandex-cloud }}, доступны расширенные возможности {{ forms-full-name }} для совместной работы. 

## Доступ к формам для бизнеса {#org-access}

- {{ forms-full-name }} для бизнеса можно подключить к существующей организации или создать новую. Подробнее о подключении {{ forms-full-name }} для бизнеса читайте в разделе [{#T}](enable-forms.md).

  Об управлении организациями читайте в [документации {{ org-full-name }}](../organization/).

- Доступ к формам для бизнеса могут получить пользователи организации, зарегистрированные в сервисе {{ org-full-name }}. 

  Если в компании есть система управления пользователями и доступом (например, Active Directory или Google Workspace), сотрудники могут использовать свои корпоративные аккаунты, чтобы войти в организацию {{ yandex-cloud }} и получить доступ к формам через систему единого входа (Single Sign-On, SSO). Для этого администратору организации нужно настроить федерацию удостоверений. 
  
  Подробнее читайте в [документации {{ org-full-name }}](../organization/concepts/add-federation.md).

- Чтобы создавать и редактировать формы для бизнеса, перейдите на страницу [{{ link-forms-b2b }}]({{ link-forms-b2b }}) и [войдите в аккаунт](login.md), который вы используете для работы в организации.
  
  Если вы зарегистрированы в нескольких организациях, в списке форм можно будет увидеть, к какой организации относится каждая из форм.

{% note info %}

Форму для бизнеса нельзя открыть на [странице управления личными формами]({{ link-forms-admin }}). 

{% endnote %}

## Возможности {{ forms-full-name }} для бизнеса {#business-features}

Пользователям организаций доступны расширенные функции:

- Интеграция с сервисами [{{ tracker-name }}](create-task.md) и [{{ wiki-name }}](send-wiki.md).

- [Общий доступ к редактированию форм](forms-settings.md#access) для сотрудников организации.

- Возможность [разрешить доступ к заполнению формы](restrictions.md#sec_access) только для сотрудников организации.

- Возможность [создавать группы форм](forms-settings.md#sec_groups).

- Вопросы, доступные только в формах для бизнеса: [Люди](blocks-ref/people.md), [Отделы](blocks-ref/departments.md), [Команды](blocks-ref/teams.md), [Вики](blocks-ref/wiki.md).
