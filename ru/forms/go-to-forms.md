# Как выбрать версию {{ forms-full-name }}

{% if audience == "external" %}

{{ forms-full-name }} можно использовать в личных делах, например опросить друзей и составить меню для новогодней вечеринки. Либо использовать формы для бизнеса, например принимать заказы для интернет-магазина.

- Чтобы создать [личную форму](personal-forms.md), перейдите на страницу [{{ link-forms-admin }}]({{ link-forms-admin }}) и [войдите в аккаунт на Яндексе]({{ link-passport }}).

- Чтобы создать [форму для бизнеса](forms-for-org.md), перейдите на страницу [{{ link-forms-b2b }}]({{ link-forms-b2b }}) и [войдите в аккаунт сотрудника организации](login.md).

Чтобы переключаться между личными формами и формами для бизнеса, в сервисе {{ forms-full-name }} в правом верхнем углу страницы нажмите кнопку **К личным формам** или **К формам для бизнеса**.

{% else %}

Существует три админки {{ forms-full-name }}. Для создания формы выберите подходящую админку в зависимости от аудитории вашего опроса:

- Если вы создаете форму для внешних пользователей, например форму обратной связи для сервиса, используйте внешнюю админку [{{ link-forms-ext }}]({{ link-forms-ext }}). 
  
  Такая форма будет доступна на домене `forms.yandex.ru` для всех пользователей.

- Если вы создаете форму для сотрудников Яндекса, используйте внутреннюю админку [{{ link-forms-admin }}]({{ link-forms-admin }}). 
  
  Такая форма будет доступна на домене `forms.yandex-team.ru` только для сотрудников Яндекса. Форму, созданную в этой админке, можно использовать для анонимного опроса. Для этого [опубликуйте форму по анонимной ссылке](publish.md#section_link).

- Если вы хотите создать форму для личных целей, используйте админку для внешних пользователей [{{ link-forms-admin }}]({{ link-forms-admin }}).

  Такая форма будет доступна на домене `forms.yandex.ru` для всех пользователей. Подробнее о {{ forms-full-name }} для внешних пользователей читайте в [документации {{ yandex-cloud }}](https://cloud.yandex.ru/docs/forms/). 

{% endif %}
