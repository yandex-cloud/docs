# Опубликовать форму
{% if audience == "external" %}
Сразу после создания форма становится доступной для заполнения. Вы можете опубликовать форму по прямой ссылке, встроить ее в веб-сайт или поделиться ссылкой на форму в соцсетях. Если вы используете [{{ forms-full-name }} для бизнеса](forms-for-org.md), вы можете [вставить форму на страницу {{ wiki-full-name }}](../wiki/actions/forms.md).
{% else %}
Чтобы форма стала доступна для заполнения, нажмите кнопку **Опубликовать** над списком вопросов. После этого вы можете пригласить пользователей заполнить форму по прямой ссылке, встроить форму в веб-сайт, разместить на {{ wiki-name }} или в {{ tracker-name }}.
{% endif %}

При публикации формы вы можете [задать ограничения](restrictions.md) по дате и общему количеству ответов {% if audience == "external" %} или разрешить заполнять форму только сотрудникам организации {{ org-full-name }}{% endif %}.

{% if audience == "external" %}
Если с помощью формы вы будете собирать персональные данные респондентов, пожалуйста, соблюдайте [Условия использования сервиса]({{ link-forms-legal }}).
{% endif %}

{% note info %}

Вы можете вносить изменения в форму и ее настройки даже после публикации.

{% endnote %}


## Опубликовать по ссылке {#section_link}

Форма доступна по прямой ссылке сразу после создания. Чтобы получить ссылку на форму:

1. Выберите форму и нажмите кнопку **Поделиться** над списком вопросов.

1. Чтобы скопировать адрес формы, в поле **Ссылка** нажмите значок ![](../_assets/forms/icon-copy.png).
   {% if audience == "internal" %}Если вы создали форму во [внутренней админке](new-form.md#instance), вы можете использовать ее для анонимного опроса. Для этого скопируйте адрес формы из поля **Ссылка на анонимную форму**.{% endif %}

1. {% if audience == "internal" %}Отправьте ссылку пользователям или разместите ее на общедоступных ресурсах.{% else %}Чтобы поделиться ссылкой в соцсетях или мессенджерах, нажмите значок справа от поля **Ссылка**.{% endif %}

Все опубликованные формы доступны для заполнения по адресу, указанному в поле **Ссылка**.

{% if audience == "external" %}

{% note info %}

Если требуется, чтобы стандартные сообщения {{ forms-full-name }}, такие как предупреждения о неправильно заполненных полях, отображались на английском языке, в адресе ссылки замените домен `.ru` на `.com`.

{% endnote %}

{% endif %}

## Опубликовать на сайте {#section_c21_gdb_42b}

Чтобы опубликовать форму на вашем сайте:

1. Выберите форму и нажмите кнопку **Поделиться** над списком вопросов.

1. Скопируйте код формы из поля **Код для вставки iframe**.

1. Вставьте его в HTML-код страницы, на которой нужно разместить форму.
    Чтобы настроить размер формы, измените значение параметра `width`. Размер можно указать в процентах или пикселях. Например, чтобы форма растягивалась по ширине страницы, укажите `width="100%"`.

## Добавить в {{ tracker-name }} {#section_znn_4yv_lgb}

Во [внутренней админке форм](new-form.md#instance) можно [настроить форму для создания задач](create-task.md#setup) и встроить ее в интерфейс {{ tracker-name }}. Такая форма будет отображаться на странице создания задачи рядом со стандартной. Она поможет пользователям создавать задачи по определенному шаблону, не отвлекаясь на лишние поля и параметры. Подробнее о формах в {{ tracker-name }} читайте в [Справке {{ tracker-full-name }}](https://docs.yandex-team.ru/cloud/tracker/manager/attach-form).

{% endif %}

## Снять с публикации {#section_lvx_52c_tbb}

Чтобы закончить прием ответов и снять форму с публикации:

1. Выберите форму и нажмите кнопку **Поделиться** над списком вопросов.

1. Нажмите кнопку **Снять с публикации**.
