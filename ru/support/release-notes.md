# История изменений описания технической поддержки

## Январь 2026 {#january-2026}

Уточнены ограничения поддержки. Добавлен пункт, связанный с сервисом {{ managed-k8s-name }}.

## Декабрь 2025 {#декабрь-2025}

1. Обновлена таблица времени ответа поддержки:
   * изменена структура отображения приоритетов для обращений типа **Проблема** в таблице;
   * добавлено время ответа на обращения типа **Вопрос**, **Предложение** и **Квоты** через центр поддержки;
   * ответ на обращения типа **Квоты**, заведенный из консоли управления, вынесен в отдельную строку;
   * ответ на обращения, заведенные через электронную почту, вынесен в отдельную строку.
1. Расширена таблица доступных видов обращений для тарифов «Бизнес» и «Премиум».
   * «Бизнес» и «Премиум»: 
      * помощь в восстановлении работоспособности сервисов в случае критических обращений;
      * базовые рекомендации по построению отказоустойчивой инфраструктуры в облаке;
      * подготовка RCA по обращениям с приоритетом **Критичный**.
   * «Премиум»:
      * проведение HA/DR-тестов;
      * инфраструктурный чекап;
      * проведение видеоконференции с экспертами поддержки по запросу;
      * поддержка обслуживания и изменений архитектуры облачных ресурсов.
1. Уточнены ограничения поддержки. Добавлены пункты, связанные с сервисами и услугами:
   * отладка программного обеспечения стороннего производителя;
   * сопровождение продуктов {{ marketplace-full-name }}, где издателем является не {{ yandex-cloud }};
   * использование {{ adv-ddos-protection }};
   * Yandex {{ tracker-short-name }}, Yandex {{ forms-short-name }} и Yandex {{ wiki-short-name }};
   * работа с сервисами управляемых баз данных (MDB) в рамках зоны контроля клиентов {{ yandex-cloud }}.
