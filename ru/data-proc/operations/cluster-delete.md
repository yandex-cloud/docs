# Удаление кластера

Вы можете удалить кластер вместе со всеми хранимыми данными. Помните, что резервные копии данных не создаются автоматически: если вы хотите сохранить данные для восстановления кластера или последующего использования, используйте, например, сервис [{{ objstorage-full-name }}](https://cloud.yandex.ru/services/storage).

## Перед удалением кластера {#before-you-delete}

[Отключите защиту от удаления](./cluster-update.md) для кластера, если она включена.

## Удалить кластер {#delete}

{% list tabs %}

- Консоль управления

    1. Перейдите на [страницу каталога]({{ link-console-main }}) и выберите сервис **{{ dataproc-name }}**.
    1. Нажмите значок ![image](../../_assets/options.svg) для нужного кластера и выберите пункт **Удалить**.
	1. (опционально) Укажите таймаут [декомиссии](../concepts/decommission.md).
    1. Подтвердите удаление кластера.

    {{ dataproc-name }} запустит операцию удаления кластера.
