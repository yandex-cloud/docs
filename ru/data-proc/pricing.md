## Из чего складывается стоимость использования {{ dataproc-name }} {#rules}

Расчет стоимости использования {{ dataproc-name }} учитывает использование вычислительных ресурсов на хостах, суммарный объем хранилища в кластере и исходящий трафик:

* время использования vCPU или GPU;

* количество используемой оперативной памяти в единицу времени (ГБ×час);

* платформа {{ compute-full-name }}, которая используется при создании хостов;

* тип и объем хранилища для каждого хоста (дискового пространства);

* объем исходящего трафика.


{% include [pricing-gb-size](../_includes/pricing-gb-size.md) %}

### Использование вычислительных ресурсов {#rules-compute}

Стоимость начисляется за каждый час работы хоста в соответствии с запрошенными для него вычислительными ресурсами. Цены вычислительных ресурсов рассчитываются на основе [цен за аналогичные ресурсы {{ compute-name }}](../compute/pricing.md#prices), с наценкой за использование образов ВМ {{ dataproc-name }}.

Минимальная единица тарификации — час (например, стоимость 1,5 часов работы vCPU равна стоимости 2 часов). Время, когда хост {{ dataproc-name }} не может выполнять свои основные функции, не тарифицируется.

### Использование дискового пространства {#rules-storage}

Оплачивается объем хранилища, выделенный для каждого из хостов кластера.

Цена указывается за 1 месяц использования. Минимальная единица тарификации — 1 ГБ в час (например, стоимость хранения 1 ГБ в течение 1,5 часов равна стоимости хранения в течение 2 часов).
