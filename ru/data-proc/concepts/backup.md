# Резервные копии

{{ dataproc-name }} не автоматизирует резервное копирование данных кластеров. Чтобы защититься от порчи или потери данных, вы можете использовать стандартные средства Hadoop для создания резервных копий и хранить их с помощью сервиса [{{ objstorage-full-name }}]({{ link-cloud-services }}/storage).

 Услуги сервиса {{ objstorage-name }} [тарифицируются отдельно](../../storage/pricing.md).
