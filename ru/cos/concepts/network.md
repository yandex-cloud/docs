# Особенности работы с сетью

В {{ coi }} {% if lang == "ru" %}[Docker-контейнеры](/blog/posts/2022/03/docker-containers){% else %}Docker-контейнеры{% endif %} запускаются с использованием сетевого интерфейса хоста. Все порты, открытые в Docker-контейнере, будут открыты и на хосте.

Все порты хоста открыты в интернет, что позволяет автоматически получать доступ к портам запущенного Docker-контейнера.
