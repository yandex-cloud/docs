- macOS

  Используйте один из способов:

    * Для установки {{ TF }} с помощью пакетного менеджера [Homebrew](https://brew.sh) выполните команду:

      ```
      brew install terraform
      ```

    * Для установки вручную [скачайте дистрибутив {{ TF }}](https://www.terraform.io/downloads.html) и установите его по [инструкции](https://learn.hashicorp.com/tutorials/terraform/install-cli?in=terraform/aws-get-started).
