# Статические ключи доступа, совместимые с AWS API

_Статические ключи доступа_ — секретный ключ и идентификатор ключа, используемые для аутентификации в сервисах с AWS-совместимым API, например в {{ objstorage-name }}, {{ message-queue-name }} и {{ ydb-name }} при использовании Document API. Ключи доступа используются только для [сервисного аккаунта](../users/service-accounts.md).

Пользователь использует секретный ключ для подписи запросов, а по идентификатору {{ yandex-cloud }} находит этот ключ и с его помощью сверяет полученную подпись. Так как секретный ключ привязан к сервисному аккаунту, это позволяет убедиться, что операция выполняется именно тем, кто владеет этим аккаунтом.
