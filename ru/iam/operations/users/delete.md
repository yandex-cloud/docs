# Удаление пользователя из облака

{% note warning %}

Удаление пользователя из облака — неотменяемая операция. Если вы случайно удалили пользователя, вы можете заново [добавить](create.md) его в облако.

{% endnote %}

{% list tabs %}

- Консоль управления
  
  1. {% include [grant-role-console-first-steps](../../../_includes/iam/grant-role-console-first-steps.md) %}
  2. На странице **Пользователи и роли** в строке с нужным пользователем нажмите значок ![image](../../../_assets/options.svg) и выберите в меню команду **Удалить**.
  
{% endlist %}
