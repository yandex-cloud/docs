## Февраль 2026 {#february-2026}

* Добавлена возможность управления [политиками авторизации](concepts/access-control/access-policies.md). {{ tag-con }}
* Добавлены команды для управления политиками авторизации на уровне организации, облака и каталога:
  * `yc organization-manager organization list-access-policy-bindings`;
  * `yc organization-manager organization bind-access-policy`;
  * `yc organization-manager organization unbind-access-policy`. {{ tag-cli }}
* Добавлены [роли](roles-reference.md):

   {% cut "Yandex Cloud Backup" %}

     Роль | Описание
     --- | ---
     `backup.auditor` | Позволяет просматривать информацию о виртуальных машинах и серверах BareMetal, подключенных к сервису Cloud Backup, о политиках резервного копирования и квотах сервиса, а также об облаке и каталоге.

   {% endcut %}

## Декабрь 2025 {#december-2025}

* Добавлена команда `yc iam access-key issue-ephemeral` для выпуска [эфемерных ключей](concepts/authorization/ephemeral-keys.md). {{ tag-cli }}
* Добавлена поддержка ресурса `yandex_iam_oauth_client_secret` для управления секретами OAuth-клиентов. {{ tag-tf }}
* В ресурс `yandex_iam_service_account` добавлено поле `labels` для работы с метками. {{ tag-tf }}
* В ресурсе `yandex_iam_oauth_client` поля `scopes` и `redirect_uris` изменены на тип set для предотвращения коллизий при сравнении. {{ tag-tf }}
