# Язык поисковых запросов и поисковые операторы

При поиске Яндекс использует только значимые слова во введенной поисковой фразе, опуская _стоп-слова_ — предлоги, союзы и местоимения, которые не меняют смысла ключевой фразы. Например, во фразе «как и когда путешествовать на Марс» учитываются только «путешествовать» и «Марс». Однако если стоп-слова меняют смысл фразы, например «это по дружбе», Яндекс автоматически зафиксирует их и включит в искомую фразу. Результаты поиска будут содержать всю фразу целиком, а не все результаты со словом «дружба». При этом в статистике такие показы учитываются как пословное соответствие, так как задействованы все слова из заданной фразы.

Также на результаты поиска влияют _минус-фразы_ — слова или словосочетания, при наличии которых в поисковом запросе пользователя результат не включается в выдачу. Они используются для исключения нецелевого трафика. Если минус-фраза содержит стоп-слова, система автоматически фиксирует его.

Кроме стоп-слов и минус-фраз для формулирования поисковых запросов и ключевых фраз вы можете использовать _поисковые операторы_ — специальные символы, которые влияют на смысл искомой фразы. В таблице перечислены все операторы, приведены примеры ключевых фраз и результаты поиска с использованием операторов.


| **Оператор** | **Что делает** | **Пример ключевой фразы** | **Когда покажем** |
|---|---|---|---|
| `-` | Исключает слова из запроса. | `доставка -цветы` | ![ok](../../_assets/common/yes.svg) доставка еды</br>![ok](../../_assets/common/yes.svg) доставка на дом</br>![no](../../_assets/common/no.svg) доставка цветов |
| `!` | Фиксирует форму слова (число, падеж, время). | `купить !собаку` | ![ok](../../_assets/common/yes.svg) купить собаку</br>![no](../../_assets/common/no.svg) купить корм для собак</br>![no](../../_assets/common/no.svg) купить собак |
| `+` | Фиксирует стоп-слова (любые слова, не несущие дополнительного смысла). | `работа +на дому` | ![ok](../../_assets/common/yes.svg) работа на дому</br>![no](../../_assets/common/no.svg) работа по дому</br>![no](../../_assets/common/no.svg) работа дома |
| `" "` | Фиксирует количество слов. Показ по запросам, содержащим фразу без дополнительных слов. | `"купить автомобиль"` | ![ok](../../_assets/common/yes.svg) купить автомобиль</br>![ok](../../_assets/common/yes.svg) автомобиль купить</br>![no](../../_assets/common/no.svg) купить красный автомобиль |
| `[]` | Фиксирует порядок слов. При этом учитываются все словоформы и стоп-слова. | `билеты [из москвы в париж]` | ![ok](../../_assets/common/yes.svg) билеты из москвы в париж</br>![ok](../../_assets/common/yes.svg) билеты на самолет из москвы в париж</br>![no](../../_assets/common/no.svg) билеты из парижа в москву</br>![no](../../_assets/common/no.svg) билеты москва париж</br>![no](../../_assets/common/no.svg) билеты из москвы недорого в париж |
| `()` и ```\|``` | Группируют слова при сложных запросах. | ```заказать (роллы\|пицца)``` | ![ok](../../_assets/common/yes.svg) заказать роллы</br>![ok](../../_assets/common/yes.svg) заказать пицца |

[{{ wordstat-name }}](./wordstat.md) поддерживает все операторы в методах `GetRegionsDistribution` и `GetTop`. Метод `GetDynamics` поддерживает все операторы при детализации по дням, при детализации по неделям и месяцам — только оператор `+`.
