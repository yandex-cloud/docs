# Регионы поиска

Регион, которому необходимо отдавать предпочтение при формировании результатов поиска, определяется значением поля `lr` поискового запроса. В качестве регионов могут быть указаны страны, субъекты государств и города.

Список идентификаторов часто используемых стран представлен в таблице ниже.
