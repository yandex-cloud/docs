# Текстовый поиск с учетом геопозиции

По умолчанию результаты поисковой выдачи {{ search-api-name }} не привязаны к местоположению пользователя, поскольку запросы отправляются с серверов {{ yandex-cloud }}. Чтобы результаты поиска зависели от геопозиции как в браузере, в параметрах запроса указывайте IP-адрес пользователя, от имени которого вы хотите получить поисковую выдачу.

{% note info %}

Указанный IP-адрес должен быть валидным и публичным. Адреса приватных подсетей не влияют на результат выдачи.

{% endnote %}

{% list tabs group=instructions %}

- API {#api}

  1. Добавьте в тело запроса поле `X-Forwarded-For-Y` в секции `metadata.fields`:

      **body.json**

      ```json
      ...
      "metadata": {
        "fields": {
          "X-Forwarded-For-Y": "<IP-адрес_пользователя>"
        }
      },
      ...
      ```

      Где `X-Forwarded-For-Y` — IP-адрес пользователя.

  1. Остальные параметры тела запроса задайте в соответствии со [спецификацией](../api-ref/WebSearch/search.md).
  1. Выполните запрос и получите результат. Подробнее см. в разделе [{#T}](./web-search.md).

{% endlist %}
