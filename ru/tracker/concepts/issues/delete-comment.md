# Удалить комментарий

Запрос позволяет удалить комментарий к задаче.

## Формат запроса {#section_rnm_x4j_p1b}

Чтобы удалить комментарий, используйте HTTP-запрос с методом `DELETE`:

```
DELETE /{{ ver }}/issues/<issue-id>/comments/<comment-id>
Host: {{ host }}
Authorization: OAuth <OAuth-токен>
{{ org-id }}
```

{% include [headings](../../../_includes/tracker/api/headings.md) %}

#### Ресурс {#resource}

- **\<issue-id\>**
 Идентификатор или ключ задачи.
 
- **\<comment-id\>**
 [Уникальный идентификатор комментария](get-comments.md#section_xc3_53j_p1b) в числовом формате (id) или формате строки (longId).

> Запрос на удаление комментария, прикрепленного к задаче `JUNE-2`:
> 
> - Используется HTTP-метод DELETE.
>  
> ```
> DELETE /v2/issues/JUNE-2/comments/53a170e8e4b07a4c3358d60a
> Host: {{ host }}
> Authorization: OAuth <OAuth-токен>
> {{ org-id }}
> ```

## Формат ответа {#section_xc3_53j_p1b}

В случае успешного выполнения запроса API возвращает ответ с кодом 204. Тело ответа отсутствует.

## Возможные коды ответа {#section_otf_jrj_p1b}

204
:   Запрос с методом DELETE успешно выполнен, объект удален.

404
:   Запрошенный объект не был найден. Возможно, вы указали неверное значение идентификатора или ключа объекта.
