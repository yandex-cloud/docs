# Просмотреть затраченное время

Чтобы просмотреть количество времени, которое вы затратили на задачу, на странице задачи перейдите на вкладку **Параметры**. Затраченное время отображается в блоке **Учёт времени**.  

Чтобы отредактировать оценку задачи, справа от параметра **Оценка** нажмите на значок ![](../../_assets/tracker/mobile-edit-param.png). Введите значение и нажмите кнопку **Сохранить**. 

{% note info  %}

Если вы не видите блок **Учёт времени**, перейдите в веб-интерфейс и на панели справа добавьте поля **Затрачено времени** и **Оценка** с помощью кнопки ![](../../_assets/tracker/task-params-btn.png) **Выбрать поля**. 

{% endnote %}
