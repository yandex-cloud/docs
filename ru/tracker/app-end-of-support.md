# Прекращение поддержки мобильного приложения {{ tracker-name }}

С августа 2020 года прекращена поддержка мобильного приложения {{ tracker-name }} для iOS и Android.

Мы приняли решение прекратить поддержку мобильного приложения, потому что его возможности сильно ограничены по сравнению с веб-версией {{ tracker-name }} и не отвечают запросам пользователей. Мы считаем, что полезнее будет направить усилия на адаптацию веб-интерфейса {{ tracker-name }} для мобильных устройств. Благодаря этому полные возможности сервиса станут доступны на всех платформах.

Если приложение {{ tracker-name }} уже установлено на ваше устройство, оно остается доступным для использования в текущем виде. При этом приложение перестанет обновляться и будет удалено из магазинов AppStore и Google Play.

Чтобы избежать возможных неудобств, рекомендуем вам перейти на мобильную версию {{ tracker-name }}.
