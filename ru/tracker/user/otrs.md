# Сансара

Чтобы связать задачу в {{ tracker-name }} с тикетом в Сансаре:

1. Откройте страницу задачи.

1. Выберите **Действия** → **Связать с Сансарой**.

1. Выберите тип связи и укажите номер тикета.

Связанные тикеты Сансары отображаются на странице задачи в блоке связей. Первый из привязанных тикетов считается основным. Его номер отображается в поле **Тикет ОТРС** на панели справа.

## Синхронизировать тикет Сансары с задачей {#section_rdm_4hx_3bb}

{{ tracker-name }} может отправлять в Сансару информацию о добавлении комментариев к задаче, смене статусов и исполнителей. Изменения отправляются в основной тикет Сансары.

Чтобы включить синхронизацию, напишите на [tools@](mailto:tools@yandex-team.ru).

{% note warning %}

Синхронизация работает только для тикетов Сансары, прикрепленных после ее включения.

{% endnote %}
