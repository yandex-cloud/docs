# Кондуктор

Чтобы связать тикет в Кондукторе с задачей:

{% list tabs %}

- Вручную

    1. Откройте страницу задачи.

    1. Выберите **Действия** → **Связать с кондуктором**.

    1. Укажите идентификатор тикета.

- Автоматически

    Укажите ключ задачи в комментариях при создании тикета в Кондукторе.

{% endlist %}

Привязанные тикеты отображаются на странице задачи в блоке связей.
