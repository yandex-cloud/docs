# Найти задачу по ключу или названию

Если вы знаете название задачи или ее ключ, вы можете найти задачу с помощью {% if audience == "internal" %}строки поиска на верхней панели {{ tracker-name }}{% else %}языка запросов{% endif %}.

{% list tabs %}

- Веб-интерфейс

    {% if audience == "internal" %}

    В строке поиска в правом верхнем углу введите ключ задачи и нажмите клавишу **Enter**. {{ tracker-name }} сформирует простой запрос, в результатах которого будут отображены все подходящие задачи. Чтобы уточнить результат, [используйте язык запросов](query-filter.md#query-text).
    
    {% else %}

    Чтобы найти задачу по ключу или названию, используйте параметры [языка запросов](query-filter.md):
    * `Key:` для поиска по точному значению ключа.
    * `Summary:` для поиска по [названию](query-filter.md#query-text).
    
    {% endif %}

- Мобильное приложение

    В верхней части экрана в поле **Поиск по всем задачам** введите название или ключ задачи. В результате появится список задач, которые удовлетворяют запросу. В списке отображаются первые 20 записей. Если вы не нашли нужную задачу, уточните запрос.

{% endlist %}
