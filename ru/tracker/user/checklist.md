# Создать чеклист

Чеклист — это перечень пунктов, которые необходимо выполнить для решения задачи. Выполнив очередной пункт, отметьте его в чеклисте, и тогда вы не упустите ни одной важной детали при работе над задачей.

## Создать чеклист {#checklist-create}

1. Откройте страницу задачи.

1. Нажмите кнопку ![](../../_assets/tracker/checklist-create.png).

1. Введите название первого пункта списка. Чтобы сохранить пункт, нажмите ![](../../_assets/tracker/checklist-save.png). Если вы хотите отменить пункт, нажмите ![](../../_assets/tracker/checklist-cancel.png).

1. Чтобы добавить очередной пункт, нажмите **Добавить пункт**.

## Работать с чеклистом {#checklist-operations}

- Чтобы изменить название пункта, нажмите значок **Редактировать пункт**![](../../_assets/tracker/checklist-edit.png).
- Чтобы удалить пункт, нажмите значок **Удалить пункт**![](../../_assets/tracker/checklist-delete.png).
- Чтобы отметить пункт как выполненный, нажмите поле для отметки ![](../../_assets/tracker/checklist-checkbox.png).
- Чтобы снять отметку о выполнении пункта, нажмите поле с отметкой ![](../../_assets/tracker/checklist-checkmark.png).
- Чтобы переместить пункт выше или ниже, нажмите значок ![](../../_assets/tracker/checklist-replace.png) и перетащите пункт в нужное место списка.
- Чтобы скрыть или показать пункты чеклиста, используйте значки ![](../../_assets/tracker/checklist-close.png) и ![](../../_assets/tracker/checklist-open.png).
- Чтобы добавить пункт в чеклист, нажмите **Добавить пункт** внизу списка.
- Чтобы удалить чеклист, нажмите значок **Удалить чеклист**![](../../_assets/tracker/checklist-delete.png) внизу списка.
