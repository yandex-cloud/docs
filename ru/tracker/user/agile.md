# Как работать с доской {{ tracker-name }}

Задачи отображаются на доске в виде карточек с основными параметрами задач. Внешний вид карточки на доске [можно настроить](../manager/edit-agile-board.md#sec_layout): выбрать отображаемые поля и задать цвета для выделения тегов и компонентов.

{% note warning %}

Ограничить доступ к доске нельзя, но можно ограничить доступ к задачам, расположенным на доске. 

На доске отображаются задачи только из тех очередей, к которым у пользователя есть доступ. Чтобы ограничить доступ к отдельным задачам, [добавьте компоненты с ограниченным доступом](../manager/queue-access.md#access-components).  

{% endnote %}

## Управлять задачами {#manage-tasks}

Работа с задачами на доске в {{ tracker-name }} почти не отличается от работы со стикерами на обычной доске:

* Карточки задач распределены по колонкам в соответствии с их статусом. Чтобы изменить статус задачи, перетащите ее в нужную колонку.
* Чтобы изменить положение карточки в колонке, перетащите ее в нужное место (только для досок, у которых включена опция [Разрешить менять порядок задач](../manager/edit-agile-board.md#board-settings)).
* Чтобы изменить параметры или прокомментировать задачу, нажмите на ее ключ и внесите изменения в карточке задачи справа.

## Создать задачу {#create-task}

{% note warning %}

Убедитесь, что в [настройках доски](../manager/edit-agile-board.md#board-settings) указана очередь для создания задач. Новая задача будет создана в этой очереди.

{% endnote %}

{{ tracker-name }} позволяет быстро создавать задачи на доске. Задачу можно создать сразу в спринте или поместить ее в бэклог. Для задачи нужно указать только ее название и тип — остальные поля вы сможете заполнить позже.

#### Создать задачу на доске

1. Откройте доску задач.

1. В правом нижнем углу доски нажмите на кнопку **{{ ui-key.startrek.ui_components_agile_common_AddIssueButton.add-issue-button }}**.

1. Введите название задачи и выберите ее тип.

1. Нажмите кнопку **{{ ui-key.startrek.blocks-desktop_agile-create-issue.create }}**.

Задача будет создана в очереди, которую вы указали в [настройках](../manager/edit-agile-board.md#board-settings). Чтобы увидеть задачу, обновите страницу доски.

#### Создать задачу в спринте

1. В правом верхнем углу доски нажмите кнопку **{{ ui-key.startrek.blocks-desktop_b-page-agile-board.sprints }}**.

1. Выберите спринт, в котором нужно создать задачу.

1. Введите название задачи в поле под списком задач спринта.

1. Выберите тип задачи и нажмите кнопку **{{ ui-key.startrek.blocks-desktop_agile-create-issue.create }}**.

Задача будет создана в очереди, которую вы указали в [настройках](../manager/edit-agile-board.md#board-settings). Чтобы увидеть задачу, обновите страницу доски.

#### Создать задачу в бэклоге

1. В правом верхнем углу доски нажмите кнопку **{{ ui-key.startrek.blocks-desktop_b-page-agile-board.sprints }}**.

1. Введите название задачи в поле под списком задач бэклога.

1. Выберите тип задачи и нажмите кнопку **{{ ui-key.startrek.blocks-desktop_agile-create-issue.create }}**.

Задача будет создана в очереди, которую вы указали в [настройках](../manager/edit-agile-board.md#board-settings). Чтобы увидеть задачу, обновите страницу доски.

## Добавить задачи на доску {#add-tasks}

{% note warning %}

На доске отображаются только те задачи, статусы которых предусмотрены настройками доски.

{% endnote %}

Вы можете автоматически добавлять на доску задачи с заданными параметрами с помощью [фильтра доски](../manager/edit-agile-board.md). Чтобы добавить задачу на доску вручную, создайте новую задачу на доске, поместите задачу в бэклог или добавьте в спринт.

#### Добавить задачу в бэклог

1. Откройте страницу задачи.

1. На панели справа нажмите заголовок поля **{{ ui-key.startrek-backend.fields.issue.boards }}**. Если поля **{{ ui-key.startrek-backend.fields.issue.boards }}** нет, добавьте его с помощью кнопки **{{ ui-key.startrek.blocks-desktop_b-ticket-sidebar.action--select-fields }}**.

1. Начните вводить название доски в поле **{{ ui-key.startrek-backend.fields.issue.boards }}** и выберите подходящий вариант из списка.

Задача попадет на выбранную доску.

#### Добавить задачу в спринт

1. Откройте страницу задачи.

1. На панели справа в блоке **{{ ui-key.startrek-backend.fields.issue.fields.agile }}** нажмите заголовок поля **{{ ui-key.startrek-backend.fields.issue.sprint }}**. Если поля **{{ ui-key.startrek-backend.fields.issue.sprint }}** нет, добавьте его с помощью кнопки **{{ ui-key.startrek.blocks-desktop_b-ticket-sidebar.action--select-fields }}**.

1. Начните вводить название спринта в поле и выберите подходящий вариант из списка. Поле доски при этом заполнится автоматически.

Задача попадет на доску сразу в выбранный спринт.

#### Добавить несколько задач

Чтобы добавить на доску сразу несколько задач, воспользуйтесь [массовым редактированием](../manager/bulk-change.md):

1. Отберите нужные задачи с помощью [фильтра](create-filter.md).

1. Выберите задачи, которые вы хотите добавить на доску, и внизу страницы нажмите кнопку **{{ ui-key.startrek.blocks-desktop_b-bulk-change.edit-fields }}**.

1. Чтобы добавить задачи в бэклог, начните вводить название доски в поле **{{ ui-key.startrek.ui_components_agile_common_AgileTabs.board }}** и выберите подходящий вариант из списка. Если поля **{{ ui-key.startrek.ui_components_agile_common_AgileTabs.board }}** нет, добавьте его с помощью кнопки **{{ ui-key.startrek.blocks-desktop_b-ticket-sidebar.action--select-fields }}**.

1. Чтобы добавить задачи в спринт, начните вводить название спринта в поле **{{ ui-key.startrek-backend.fields.issue.sprint }}** и выберите подходящий вариант из списка. Поле доски при этом заполнится автоматически.

    Если поля **{{ ui-key.startrek-backend.fields.issue.sprint }}** нет, добавьте его с помощью кнопки **{{ ui-key.startrek.blocks-desktop_b-ticket-sidebar.action--select-fields }}**.

1. Нажмите кнопку **{{ ui-key.startrek.ui_components_TransitionScreen.screen-next-button }}** и дождитесь окончания обработки задач.

## Убрать задачи с доски {#remove-tasks}

Вы можете убрать задачи с доски одним из способов:

* Если задача была добавлена вручную, очистите значение поля **{{ ui-key.startrek-backend.fields.issue.boards }}** на странице задачи.

* Если задача была добавлена на доску автоматически с помощью фильтра, уточните параметры фильтра в [настройках доски](../manager/edit-agile-board.md#board-settings) и затем очистите значение поля **{{ ui-key.startrek-backend.fields.issue.boards }}** на странице задачи. 

    {% note alert %}

    Если задача попадает под параметры фильтра, она будет добавлена на доску снова.

    {% endnote %}

Убранные задачи перестанут отображаться на доске, но останутся доступны в {{ tracker-name }}.

## Добавить заметки на доску {#notes_add}

Вы можете прикреплять к колонкам доски свои заметки и комментарии, чтобы другим пользователям было проще на ней ориентироваться. Такие заметки не защищены от редактирования — изменить их сможет любой пользователь.

Чтобы прикрепить заметку к колонке:

1. Откройте страницу доски.

1. Выберите ![](../../_assets/tracker/icon-dots.png) → **{{ ui-key.startrek.blocks-desktop_b-agile-board.add-note }}** вверху колонки.

1. Введите текст заметки. Чтобы оформить текст, используйте [разметку YFM](markup.md).

1. Сохраните изменения.


К каждой колонке можно прикрепить только одну заметку. Она появится в верхней части колонки и будет видна всем пользователям.

## Фильтровать задачи {#filter-tasks}

Если задач на доске слишком много, скройте лишние с помощью фильтров. Вы можете применить несколько фильтров одновременно:

* По спринту. На панели сверху выберите спринт, задачи которого вы хотите просмотреть.

* По резолюции. На панели сверху раскройте меню **{{ ui-key.startrek.blocks-desktop_page-admin-tab_type_fields.field-type--resolution }}** и укажите резолюции задач, которые нужно показать на доске.

* По статусу. На панели сверху раскройте меню **{{ ui-key.startrek.blocks-desktop_page-admin-tab_type_fields.field-type--status }}** и укажите статусы задач, которые нужно показать на доске.

* Ключ или название задачи. Начните вводить текст в поле на панели сверху. На доске останутся только те задачи, ключ или название которых содержит этот текст.

* Быстрые фильтры. По умолчанию на доске есть только два быстрых фильтра: **{{ ui-key.startrek.ui_components_agile_common_QuickFiltersItem.quick-filter--assignee-me }}** и **{{ ui-key.startrek.ui_components_agile_common_QuickFiltersItem.quick-filter--author-me }}**. Чтобы добавить новые фильтры, на панели с фильтрами нажмите кнопку ![](../../_assets/tracker/edit-agile.png).

## Полноэкранный режим {#fullscreen}

Вы можете переключить доску в полноэкранный режим отображения. В полноэкранном режиме карточки задач отображаются более компактно и на доску их помещается больше. Такой режим удобно использовать, например, при планировании или разборе спринта.

* Чтобы переключить доску в полноэкранный режим, откройте страницу доски и нажмите кнопку ![](../../_assets/tracker/fullscreen-agile.png) возле названия доски.

* Чтобы установить периодичность автообновления доски, в полноэкранном режиме в правом нижнем углу страницы в поле **{{ ui-key.startrek.blocks-desktop_b-page-agile-board.auto-refresh-text }}** выберите значение: **{{ ui-key.startrek.blocks-desktop_b-page-agile-board.auto-refresh-period }}**, **Каждые 3 минуты**, **Каждые 5 минут** или **{{ ui-key.startrek.blocks-desktop_b-page-agile-board.auto-refresh-off }}**. 

* Чтобы выйти из полноэкранного режима, нажмите кнопку **{{ ui-key.startrek.blocks-desktop_b-page-agile-board.full-screen-off }}** в правом нижнем углу страницы.

## Добавить доску в избранное {#board-fav}

Чтобы у вас всегда был быстрый доступ к доске, добавьте ее в избранное. Для этого перейдите на страницу нужной доски и нажмите на значок ![](../../_assets/tracker/add-to-favorites.png) справа от названия доски. Избранные доски доступны в меню **{{ ui-key.startrek.blocks-desktop_m-head.menu-agile }}** в верхней панели {{ tracker-name }}.
