# Начальная страница

{% if audience == "external" %}

Чтобы попасть на начальную страницу, нажмите на логотип {{ tracker-name }}. По умолчанию на странице отображается ваш личный [дашборд](dashboard.md). 

{% else %}

Чтобы попасть на начальную страницу, на верхней панели {{ tracker-name }} нажмите **Моя страница**.

По умолчанию на странице отображается список задач из фильтра [<q>Все задачи</q>](my-tickets.md): задачи, для которых вы являетесь автором, исполнителем или наблюдателем.

{% endif %}

Вы можете настроить вашу начальную страницу так, чтобы на ней отображался любой [фильтр](create-filter.md), [дашборд](dashboard.md) или [доска задач](../manager/create-agile-board.md):

1. {% include [open-personal-settings](../../_includes/tracker/open-personal-settings.md) %}

1. В блоке **Моя страница** укажите какой фильтр, дашборд или доску задач вы хотите видеть на главной странице.

1. Нажмите кнопку **Сохранить**.
