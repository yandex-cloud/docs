# Добавить поле задачи

## Стандартные поля задач {#section_ymd_ycj_1gb}

По умолчанию в {{ tracker-name }} доступны наиболее популярные поля (параметры) задач. Список существующих полей можно посмотреть на странице [Настройки {{ tracker-name }}]{% if audience == "external" %}({{ link-admin-fields }}){% else %}({{ link-admin-fields-ya }}){% endif %}.

{% if audience == "external" %}Если нужного вам параметра среди них нет, администратор организации может [создать новое поле](create-param.md#section_pxn_fp4_xgb).{% else %}Если нужного вам параметра среди них нет, отправьте заявку на создание новых полей с помощью [формы](create-param.md#section_bjz_3p4_xgb).{% endif %}

**Учёт времени**|
Первоначальная оценка | Первоначальная оценка времени, необходимого для решения задачи. Время указывается в формате `1w2d3h`.
**Agile**|
Спринт | Название спринта, к которому относится задача.
Story Points | Трудоемкость задачи в баллах Story Points.

## Добавить поле задачи {#section_pxn_fp4_xgb}

{% note warning %}

Добавлять новые поля задач может только администратор организации.

{% endnote %}

По умолчанию в {{ tracker-name }} доступны наиболее популярные поля (параметры) задач. Чтобы добавить новое поле:

1. Убедитесь, что вы авторизованы в {{ tracker-full-name }} как администратор.

1. На верхней панели {{ tracker-name }} выберите ![](../../_assets/tracker/icon-settings.png) → **Настроить {{ tracker-name }}**.

1. Перейдите на вкладку **Поля**.

1. Нажмите кнопку **Создать поле**.

1. Выберите тип поля и нажмите кнопку **Дальше**.

1. Задайте параметры нового поля:

    - **Категория**. Все поля в {{ tracker-name }} сгруппированы по категориям. Выберите категорию, которая лучше всего подходит для нового поля.
    - **Название**. Старайтесь давать полям короткие и емкие названия.
    - **Название на английском языке**. Это название видно в английском интерфейсе {{ tracker-name }}.
    - **Тип чисел** (только для полей типа <q>число</q>).
    - **Значения списка** (только для полей типа <q>выпадающий список</q>) — укажите возможные значения и их порядок.
    - **Выбор значений из списка** (только для полей типа <q>выпадающий список</q>).
    - **Количество сотрудников** (только для полей типа <q>выбор сотрудников</q>).

1. Нажмите кнопку **Создать**.

## Добавить поле задачи {#section_bjz_3p4_xgb}

По умолчанию в {{ tracker-name }} доступны наиболее популярные поля (параметры) задач. Чтобы добавить новое поле задачи, закажите его через форму ниже.

В комментарии к обращению, пожалуйста, напишите, какое поле нужно создать: глобальное (доступное во всех очередях {{ tracker-name }}) или локальное (привязанное к конкретной очереди). Рекомендуем заказывать локальные поля, если они подходят вам с учетом текущих ограничений.

{% note warning %}

Локальные поля временно не поддерживаются в некоторых сценариях работы с задачами. Поддержка этих сценариев будет добавлена позже. Подробнее читайте в разделе [{#T}](../local-fields.md#restrictions).

{% endnote %}

Обращения обрабатываются в течение 2-3 рабочих дней. Если задача срочная, поднимите ее приоритет до <q>критичного</q> и напишите в комментариях, почему это срочно.

{% endif %}
