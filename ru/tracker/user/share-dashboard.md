# Как поделиться дашбордом

Вы можете поделиться своим дашбордом с коллегами:

1. Откройте страницу дашборда.

1. Скопируйте адрес дашборда и отправьте его пользователю, с которым хотите поделиться.


По умолчанию любой дашборд доступен для чтения всем пользователям, но вы можете ограничить доступ к дашборду.

{% note tip %}

Если с вами поделились дашбордом, [добавьте его в избранное](favourites.md#dashboard-fav), чтобы всегда иметь его под рукой.

{% endnote %}

## Как ограничить доступ к дашборду {#section_k2z_1nk_pz}

Если вы хотите ограничить доступ пользователей к вашему дашборду, вы можете изменить настройки доступа:

1. Откройте страницу дашборда.

1. Выберите ![](../../_assets/tracker/icon-settings.png) → **Настроить права доступа** слева от названия дашборда.

1. Настройте разрешения на **Чтение**, **Запись** и **Настройку прав** для различных категорий пользователей:
    - **Люди и роботы** — индивидуальные настройки для отдельных пользователей. Чтобы добавить пользователя в эту категорию, введите его логин или имя в строку поиска в верхней части таблицы.
    - **Группы** — настройки для всех сотрудников.
    
        {% note tip %}

        Чтобы добавить всех сотрудников организации, укажите группу `Все сотрудники`.

        {% endnote %}

    - **Роли** — настройки для пользователей, обладающих определенными ролями по отношению к дашборду:
        - **Владелец** — владелец дашборда. По умолчанию владелец — это создатель дашборда и только у него есть права на **Запись** и **Настройку прав**.
            
            {% note alert %}

            Старайтесь без необходимости не изменять настройки доступа для владельца дашборда. Если вы случайно запретили владельцу настройку прав доступа, обратитесь к администратору вашей организации.

            {% endnote %}

1. Нажмите кнопку **Применить**.


## Как добавить дашборд в избранное {#section_gnx_s3l_pz}

Если с вами поделились дашбордом, вы можете добавить его в избранное, чтобы всегда иметь его под рукой:

1. Откройте дашборд.

1. Нажмите на значок ![](../../_assets/tracker/add-to-favorites.png) справа от названия дашборда.


Избранные дашборды доступны в меню **Дашборды** в верхней панели {{ tracker-name }}.
