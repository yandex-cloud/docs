# Общий доступ к дашборду

Вы можете поделиться своим дашбордом с коллегами:

1. Откройте страницу дашборда.

1. Скопируйте адрес дашборда и отправьте его пользователю, с которым хотите поделиться.

По умолчанию любой дашборд доступен для чтения всем пользователям{% if audience == "draft" %}, но вы можете ограничить доступ к дашборду{% endif %}.

{% note tip %}

Если с вами поделились дашбордом, добавьте его в избранное, чтобы всегда иметь его под рукой. Для этого справа от названия дашборда нажмите ![](../../_assets/tracker/svg/favourites.svg). Избранные дашборды доступны в меню **Дашборды** в верхней панели {{ tracker-name }}.

{% endnote %}

## Как ограничить доступ к дашборду {#section_k2z_1nk_pz}

Если вы хотите ограничить доступ пользователей к вашему дашборду, вы можете изменить настройки доступа:

1. Откройте страницу дашборда.

1. Слева от названия дашборда выберите ![](../../_assets/tracker/icon-settings.png) → **Настроить права доступа**.

1. Настройте разрешения на **Чтение**, **Запись** и **Настройку прав** для различных категорий пользователей:
    - **Люди и роботы** — индивидуальные настройки для отдельных пользователей. Чтобы добавить пользователя в эту категорию, введите его логин или имя в строку поиска в верхней части таблицы.
    - **Группы** — настройки для {% if audience == "external" %}всех сотрудников.{% else %} отделов и команд. Чтобы добавить новый отдел, введите его название в строку поиска в верхней части таблицы.{% endif %}
    
        {% note tip %}

        Чтобы добавить всех сотрудников {% if audience == "external" %}организации{% else %}Яндекса{% endif %}, укажите группу {% if audience == "external" %}`Все сотрудники`{% else %}`Яндекс`{% endif %}.

        {% endnote %}

    - **Роли** — настройки для пользователей, обладающих определенными ролями по отношению к дашборду:
        - **Владелец** — владелец дашборда. По умолчанию владелец — это создатель дашборда и только у него есть права на **Запись** и **Настройку прав**.
            
            {% note alert %}

            Старайтесь без необходимости не изменять настройки доступа для владельца дашборда. Если вы случайно запретили владельцу настройку прав доступа, обратитесь к администратору {% if audience == "external" %}вашей организации{% endif %}.

            {% endnote %}

1. Нажмите кнопку **Применить**.

{% endif %}



## Как изменить владельца дашборда {#change-owner}

{% note alert %}

Изменить владельца дашборда может только текущий владелец. После смены владельца у вас не будет прав на редактирование дашборда.

{% endnote %}

Если вы хотите изменить владельца дашборда:

1. Откройте страницу дашборда.

1. Слева от названия дашборда выберите ![](../../_assets/tracker/icon-settings.png) → **Настроить права доступа**.

1. В блоке **Сменить владельца** укажите имя нового владельца. 

1. Нажмите кнопку **Применить**.
