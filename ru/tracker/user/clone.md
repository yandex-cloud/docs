# Как клонировать задачу

С помощью клонирования вы можете создавать новые задачи на основе уже существующих. Задачи можно клонировать в любую доступную вам очередь.

{% note tip %}

Если вам часто приходится создавать похожие задачи, воспользуйтесь [шаблонами](ticket-template.md).

{% endnote %}

Чтобы клонировать задачу:

1. Откройте страницу задачи.

1. Выберите **Действия** → **Клонировать задачу**.

1. Заполните поля:
    - **Очередь** — выберите очередь, в которой вы хотите создать задачу-клон.
    - **Скопировать параметры задачи** — если опция выбрана, параметры задачи сохраняются при клонировании. Если каких-то параметров нет в целевой очереди (например версий или компонентов), они остаются пустыми.

        {% note warning %}

        Если в задаче указаны значения [локальных полей](../local-fields.md), при клонировании в другую очередь они будут удалены.

        {% endnote %}

    - **Связать задачи после клонирования** — если опция выбрана, задача и ее клон будут связаны.
    - **Название задачи** — по умолчанию название задачи сохраняется, но вы можете его изменить.
    - **Тип** — выберите тип задачи. Доступные типы задач зависят от настроек целевой очереди.
    - **Статус** — выберите статус задачи. Доступные статусы задач зависят от настроек целевой очереди для выбранного типа задач.
    - **Исполнитель** и **Наблюдатели** — по умолчанию значения этих полей сохраняются, но вы можете их изменить.

1. Нажмите кнопку **Клонировать**.
