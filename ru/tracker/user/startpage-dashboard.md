# Как сделать дашборд начальной страницей

Вы можете {% if audience == "external" %}сделать [начальной страницей](startpage.md){% else %}выводить на вкладке [**Моя страница**](startpage.md){% endif %} любой из доступных дашбордов:

1. Нажмите на значок ![](../../_assets/tracker/tracker-settings.png) в верхней панели {{ tracker-name }} или перейдите по ссылке: {% if audience == "internal" %}[https://st.yandex-team.ru/settings]({{ link-settings-ya }}){% else %}[https://tracker.yandex.ru/settings]({{ link-settings }}){% endif %}.

1. В блоке **Моя страница** выберите из списка опцию **Дашборд** и укажите, какой дашборд вы хотите использовать в качестве начальной страницы.

    {% note tip %}

    Если нужного вам дашборда не оказалось в списке, убедитесь, что у вас [есть к нему доступ](share-dashboard.md#section_k2z_1nk_pz), и [добавьте его в избранное](share-dashboard.md#section_gnx_s3l_pz).

    {% endnote %}

1. Нажмите кнопку **Готово**.
