# Как сделать дашборд стартовой страницей

Вы можете сделать [стартовой страницей](startpage.md) любой из дашбордов, к которому у вас есть доступ. Для этого на странице с дашбордом, в правом верхнем углу, нажмите ![](../../_assets/tracker/svg/actions.svg) → **{{ ui-key.startrek.ui_components_dashboard_DashboardActionsMenu.set-start-page-action }}**.
