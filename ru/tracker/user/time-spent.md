# Отметить затраченное время

{{ tracker-name }} позволяет вести учет времени, которое вы затратили на выполнение задачи.

Для формирования корректной статистики по задаче не вносите данные за другого сотрудника. Важно, чтобы каждый исполнитель самостоятельно регистрировал затраченное время.

## Учет времени {#section_y1t_gkh_vcb}

Чтобы зарегистрировать затраченное время:

1. Откройте страницу задачи.

1. Выберите **Действия** → **Указать затраченное время**.

1. Заполните поле **Затрачено времени**. Учет времени ведется в рабочих неделях (5 рабочих дней), рабочих днях (8 часов), часах, минутах и секундах. Например:
    - 1 неделя и 3 дня — `1н 3д`, `1w 3d`.
    - 3 часа 20 минут 30 секунд — `3ч 20м 30с`, `3h 20m 30s`.

1. При необходимости введите время начала работы и оставьте комментарий. Комментарий не будет виден в задаче, но сохранится в [Отчете по затратам времени](../manager/statistics.md#section_uxt_3ft_xz).

1. Нажмите кнопку **Сохранить**.

Регистрировать затраченное время может любой пользователь, а не только исполнитель задачи. В [отчете по затратам времени](../manager/statistics.md#section_uxt_3ft_xz) будут указаны затраты для всех пользователей, отметивших время в задаче.

## Просмотр затраченного времени {#section_cqc_hkh_vcb}

Просмотреть затраченное на задачу время можно на панели параметров задачи (справа) или в [Отчете по затратам времени](../manager/statistics.md#section_uxt_3ft_xz).

Если на панели параметров задачи не отображается блок **Учет времени**, добавьте его:

1. Нажмите кнопку **![](../../_assets/tracker/task-params-btn.png) Выбрать поля** на панели параметров задачи.

1. В группе **Учет Времени** включите опции **Оценка** и **Затрачено времени**.

С помощью поля **Оценка** вы можете оценивать время, оставшееся на решение задачи. Заполните его в начале работы над задачей. Каждый раз, когда вы будете фиксировать время, значение оценки будет уменьшаться на соответствующую величину.
