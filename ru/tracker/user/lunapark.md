# Лунапарк

Вы можете поставить тикет на нагрузочное тестирование прямо из {{ tracker-name }}:

1. Откройте страницу задачи.

1. Выберите **Действия** → **Отправить на нагрузочное тестирование**.

1. Подтвердите создание тикета.

{% note info %}

Прежде чем ставить тикет, не забудьте договориться с нагрузочными тестировщиками.

{% endnote %}
