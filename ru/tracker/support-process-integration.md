# Настроить интеграцию со сторонними системами 

Если нужно интегрировать {{ tracker-name }} со сторонними системами, например CRM-системой или программным обеспечением для колл-центров, используйте API {{ tracker-name }}. Например, через API можно создавать задачи в {{ tracker-name }} и добавлять в них информацию из сторонней системы. Подробнее о возможностях API {{ tracker-name }} читайте в [Справочнике API](about-api.md).

Если нужно передавать информацию из {{ tracker-name }} в другую систему, например оповещения в мессенджер, можно [использовать триггеры](user/trigger.md). Примеры настройки триггеров для отправки уведомлений в Slack и Telegram приведены в разделе [{#T}](messenger.md).
