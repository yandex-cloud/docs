# Цели

## Добавить связь с целью {#sec_link_goal}

Если задача направлена на достижение определенной цели в сервисе [Goals](https://goals.yandex-team.ru), свяжите задачу с этой целью:

1. Откройте страницу задачи.

1. На панели справа нажмите ссылку **Цели**.

1. Введите номер цели, затем нажмите название цели во всплывающей подсказке.

1. Нажмите кнопку **OK**.


Чтобы перейти в сервис Goals и просмотреть подробную информацию о цели, нажмите значок ![](../../_assets/tracker/goals.png).

## Просмотреть цели, связанные с задачами очереди {#section_vkt_1wp_3bb}

На странице очереди вы можете просмотреть список целей, связанных с задачами очереди. Для этого перейдите на вкладку **Цели**.

#### Фильтровать список целей

Чтобы отфильтровать список целей, воспользуйтесь кнопками:

- **Все** — все цели.

- **Открытые** — только цели со статусами `Новая`, `По плану`, `Есть риски`, `Заблокирована`.

- **Закрытые** — только цели с статусами `Достигнута` и `Отменена`.

#### Просмотреть список задач цели

Чтобы просмотреть список задач, связанных с определенной целью, нажмите на название цели.

#### Подписаться на цель

Если вы хотите получать уведомления об изменениях задач, связанных с определенной целью, вы можете подписаться на нее. Для этого нажмите на значок ![](../../_assets/tracker/subscribe-goals.png) возле нужной цели.

По умолчанию {{ tracker-name }} отправляет подписчикам уведомления о любых изменениях, но вы можете [изменить настройки](../user/notifications.md).

#### Просмотреть цель

Чтобы перейти в сервис Goals и просмотреть подробную информацию о цели, нажмите значок ![](../../_assets/tracker/goals.png) возле нужной цели.
