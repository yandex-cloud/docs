# Шаблоны рабочих процессов

Шаблоны рабочих процессов помогут вам, если вы пока не знаете, как настроить {{ tracker-name }} для своей сферы деятельности. У нас есть несколько вариантов рабочих процессов:

* Управление проектами — подойдет, если у вас несколько команд работает над одним и тем же проектом. Все самое необходимое для планирования и оценки задач.
* Разработка — отличный шаблон для команды разработки, которая активно применяет гибкие методолгии. Настроим для вас доски с бэклогом и спринтами.
* Поддержка — подготовим инструменты для сбора обращений, шаблоны ответов и автоответы, а таймеры SLA в задачах помогут вам контролировать время обработки сообщений.
* Базовый шаблон — подходит для знакомства с возможностями {{ tracker-name }}, а в дальнейшем вы всегда сможете развить его в по-настоящему удобный для вас процесс.

Выберите подходящий шаблон, а мы создадим все необходимые сущности в {{ tracker-name }}.
