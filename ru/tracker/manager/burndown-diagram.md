# Диаграмма сгорания задач

{% note warning %}

Работа со спринтами по умолчанию подключена на досках **С бэклогом и спринтами**. Если вы хотите добавить диаграмму на **Простую доску**, [включите на ней бэклог](backlog.md#add-backlog).

{% endnote %}

Диаграмма сгорания задач показывает скорость выполнения задач и оставшийся объем работы в течение [спринта](create-agile-sprint.md). По диаграмме сгорания можно спрогнозировать, успеет ли команда выполнить запланированные задачи.

На диаграмме по вертикальной оси отмечен суммарный объем оставшейся работы в единицах **Story Point**, по горизонтальной оси отмечено время. Идеальная диаграмма сгорания должна быть близка к линии плановой сходимости, которая начинается в точке с максимальным объемом работы (общий объем задач в спринте) в первый день спринта и заканчивается в нулевой точке в последний день спринта.

Чтобы посмотреть диаграмму, на доске перейдите на вкладку **Диаграмма сгорания** — доступна диаграмма для текущего спринта. При наведении курсора на график во всплывающем окне отображается информация о состоянии спринта и изменении объема работ в **Story points** по конкретным задачам в конкретный день. При нажатии на точку на диаграмме всплывающее окно будет зафиксировано, и из него можно будет открывать задачи, кликая по их названиям.
