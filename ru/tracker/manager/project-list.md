# Список задач проекта

На вкладке **Список задач** отображаются все задачи, которые [добавлены](create-project.md#add-tickets) в проект. Список отображает структуру проекта: вы можете выстроить иерархию работ в виде дерева подзадач, ранжировать задачи вручную или отсортировать их по одному из полей.

Чтобы открыть список задач проекта:

1. На панели слева в разделе ![](../../_assets/tracker/svg/project.svg)&nbsp;**Проекты** выберите проект.

   {% note info %}

   Раздел **Проекты** доступен в новом интерфейсе {{ tracker-name }}. О том, как переключиться на новый интерфейс, читайте в разделе [{#T}](project-new.md).

   {% endnote %}

1. Перейдите на вкладку **Список задач**. 

1. Настройте отображение списка:

    * Чтобы выбрать поля, которые вы хотите видеть в списке задач, нажмите на значок ![](../../_assets/tracker/svg/settings.svg). Отметьте нужные поля и нажмите **Применить**.

    * Чтобы найти задачи в списке, в поле **Найти задачу по названию** введите название задачи.

    * Чтобы изменить вид списка задач, выберите ![](../../_assets/tracker/svg/flat-list.svg)&nbsp;**Одноуровневый список** или ![](../../_assets/tracker/svg/tree-list.svg)&nbsp;**Древовидный список**. 
      * В режиме древовидного списка можно создавать новые подзадачи или изменять связи для подзадач. Чтобы изменить связь, слева от подзадачи нажмите ![](../../_assets/tracker/svg/range.svg) и перетащите ее на ту задачу, которую хотите назначить родительской.
      
        {% note info %}

        В режиме древовидного списка вложенность одних задач в другие обозначает [связи](../user/links.md) типа <q>Родительская задача — Подзадача</q>.

        {% endnote %}

      * В режиме одноуровневого списка можно ранжировать (перемещать) задачи относительно других. Для этого слева от задачи нажмите ![](../../_assets/tracker/svg/range.svg) и перетащите ее на нужную позицию.

    * Чтобы сгруппировать задачи по статусу, исполнителю или другим параметрам, нажмите кнопку ![](../../_assets/tracker/svg/group.svg) и выберите подходящий вариант из списка.

    * Чтобы отфильтровать список задач, используйте конструктор быстрых фильтров. Для этого на панели ![](../../_assets/tracker/svg/filter.svg)&nbsp;**Фильтры** выберите параметры и укажите значения, по которым будет сформирован список задач.

    * Чтобы отсортировать список в порядке убывания или возрастания значений, указанных в столбцах списка, нажмите на заголовок столбца.
