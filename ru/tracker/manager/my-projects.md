# Список моих проектов

Откройте список всех доступных вам проектов:

1. На панели слева выберите ![](../../_assets/tracker/svg/project.svg)&nbsp;**{{ ui-key.startrek.blocks-desktop_b-queues-info.projects }}** или перейдите по [прямой ссылке]({{ link-tracker }}pages/projects) и откройте страницу проекта.

1. Чтобы найти проекты, выберите фильтр:

    * **{{ ui-key.startrek.ui_components_projects_list_FilterPane.preset-mine }}** для всех проектов, в которых вы являетесь автором, ответственным, участником или заказчиком.

    * **{{ ui-key.startrek.ui_components_projects_list_FilterPane.preset-author }}**, **{{ ui-key.startrek.ui_components_projects_list_FilterPane.preset-responsible }}**, **{{ ui-key.startrek.ui_components_projects_list_FilterPane.preset-participant }}** или **{{ ui-key.startrek.ui_components_projects_list_FilterPane.preset-client }}** для проектов, в которых вы выполняете соответствующую роль.

    * **{{ ui-key.startrek.ui_components_projects_list_FilterPane.preset-favorite }}** для проектов, которые были добавлены в <q>{{ ui-key.startrek.blocks-desktop_b-filter.favorite }}</q>.

    * **{{ ui-key.startrek.blocks-desktop_b-head.menu-group--recent }}** для проектов, которые вы открывали последними.

1. Если отобразились не все нужные проекты, откройте полный список:

    * На панели слева в нижней части страницы нажмите кнопку **Полный список проектов**.

    * Чтобы открыть все проекты, которые доступны вам в {{ tracker-name }}, в нижней части окна нажмите кнопку **{{ ui-key.startrek.ui_components_NavigationBar_ProjectsPanel.all-projects-link }}** или перейдите по [прямой ссылке]({{ link-tracker }}pages/projects).

1. Для поиска проектов в списке используйте фильтры:

    * Чтобы найти проект по названию, введите его в поле **{{ ui-key.startrek.ui_components_projects_list_FilterPane.search-placeholder }}**.

    * Чтобы отобразить список проектов в виде календарного плана, используйте [**Диаграмму Ганта**](../gantt/list-of-projects.md). В календарном плане вы можете указать сроки работы над проектом и назначить ответственного.

    * Чтобы выбрать параметр поиска по полю проекта, нажмите на его название над списком. Для выбора дополнительных параметров нажмите кнопку ![](../../_assets/tracker/svg/add-filter.svg) **{{ ui-key.startrek.ui_components_filter-pane_FieldsSelectButton.add-filter-field-button-title }}** и выберите поле.

    * Чтобы настроить порядок отображения, нажмите кнопку ![](../../_assets/tracker/svg/sorting.svg) **{{ ui-key.startrek.ui_components_SortButton.default-title }}** и выберите параметр.

    Настроенный фильтр сохранится и будет отображаться после обновления страницы или при повторном переходе в список проектов.

    {% note tip %}

    Чтобы поделиться ссылкой на отфильтрованный список проектов, скопируйте ссылку из адресной строки браузера.

    {% endnote %}

1. Для просмотра карточки проекта нажмите на его название. Информация о проекте отобразится на боковой панели справа, если в настройках интерфейса {{ tracker-name }} включена [функция **Открывать задачи на боковой панели**](../user/personal.md#choose-theme).
