# Диаграмма Ганта в проектах {{ tracker-name }}

Диаграмма Ганта для проекта в {{ tracker-name }} — это инструмент для составления календарного плана работ над задачами проекта. На диаграмме Ганта отображаются все задачи проекта и связи между ними.

Чтобы перейти к диаграмме Ганта:

1. На панели слева выберите ![](../../_assets/tracker/svg/project.svg)&nbsp;**Проекты** или перейдите по [прямой ссылке]({{ link-tracker }}pages/projects) и откройте страницу проекта.

1. Перейдите на вкладку **Диаграмма Ганта**.

   * Слева от диаграммы расположен список задач проекта: приоритет, тип и название для каждой задачи. Вы можете [настроить отображение списка](#tasks-view).
   * Сроки выполнения задач отображаются на диаграмме в виде полос, направленных вдоль оси времени. Начало каждой полосы соответствует дате начала работы над задачей, а ее конец — дедлайну, то есть дате, к которой задача должна быть выполнена. 
   * Стрелки между задачами обозначают связи типа <q>Зависимая задача — Блокирующая задача</q>.

1. Чтобы настроить масштаб временной шкалы для отображения плана проекта, в нижнем правом углу диаграммы выберите **Неделя**, **Месяцы** или **Квартал**. Настройки масштаба сохранятся после обновления страницы или при повторном переходе в диаграмму.

{% note tip %}

Вы также можете открыть календарный план в виде [диаграммы Ганта для всех проектов](./my-projects.md#gantt-projects), которые вам доступны.

{% endnote %}

## Изменить связи между задачами {#links}

* Чтобы добавить [связь](../user/links.md) <q>Зависимая задача — Блокирующая задача</q>, наведите указатель мыши на полосу блокирующей задачи — справа и слева от полосы появятся точки. 
   
   * Чтобы сделать выбранную задачу зависимой, нажмите на точку слева и перетащите ее на блокирующую задачу. 

   * Чтобы сделать выбранную задачу блокирующей, нажмите на точку справа и перетащите ее на зависимую задачу.

   {% note info %}

   Связи между задачами отображаются в виде стрелки от зависимой задаче к блокеру, если указаны дата начала и дедлайн и настроена связь <q>Зависимая задача — Блокирующая задача</q>.

   Если дедлайн зависимой задачи раньше, чем дедлайн блокирующей, стрелка отображается красным цветом. Вы можете включить [автоматическое изменение сроков](#timing) зависимых задач.

   {% endnote %}

* Чтобы удалить связь между задачами, нажмите на стрелку связи и выберите ![](../../_assets/tracker/svg/del-link.svg) **Удалить связь**.

* Если у задачи есть блокеры, которые не входят в текущий проект, слева от полосы задачи на диаграмме отображается значок ![](../../_assets/tracker/svg/blocker.svg) с количеством блокирующих задач. Чтобы удалить связь между такими задачами, нажмите на значок ![](../../_assets/tracker/svg/blocker.svg) и выберите ![](../../_assets/tracker/svg/del-link.svg) **Удалить связь**.

## Изменить сроки задач {#timing}

* Если сроки задачи не отображаются на диаграмме (ее дата начала и дедлайн не заданы), наведите указатель мыши на диаграмму напротив названия задачи, затем расположите полосу задачи в нужном месте и нажмите левую кнопку мыши.

* Чтобы сдвинуть одновременно дату начала и дедлайн задачи, перетащите полосу вдоль оси времени. 

* Чтобы изменить длительность задачи, потяните за левый край полосы (дата начала) или за правый край (дедлайн). 

* Чтобы настроить автоматическое изменение сроков для зависимых задач, перейдите в ![](../../_assets/tracker/svg/gantt-settings-button.svg)&nbsp;**Настройки диаграммы** и включите опцию **Автоматически сдвигать сроки зависимой задачи при переносе дедлайна блокера**. Если дедлайн блокера позже, чем дата начала работ по зависимой задаче, дата начала работ по зависимой задаче будет сдвигаться в соответствии с новым дедлайном блокера.

* Чтобы настроить автоматическое изменение сроков родительской задачи, перейдите в ![](../../_assets/tracker/svg/gantt-settings-button.svg)&nbsp;**Настройки диаграммы** и включите опцию **Автоматически продлять сроки задачи-родителя по срокам подзадач**. Если дедлайн у подзадачи позже, чем у задачи-родителя, он будет сдвигаться в соответствии с новым дедлайном подзадачи.

* Если для родительской задачи не указаны сроки, напротив нее на диаграмме отображается подсказка в виде голубой полосы в соответствии со сроками всех подзадач.

## Назначить исполнителя {#assigned}

На диаграмме Ганта вы можете назначить или изменить исполнителя задачи. 

Для этого на полосе задачи нажмите значок пользователя и выберите имя сотрудника, который будет назначен исполнителем.

{% note info %}

Если вы не видите значок пользователя, перейдите в ![](../../_assets/tracker/svg/gantt-settings-button.svg)&nbsp;**Настройки диаграммы** и в блоке **Показывать на полосе задачи** укажите пункт **исполнителя**.

{% endnote %}

## Настроить диаграмму {#settings}

Над временной шкалой справа нажмите ![](../../_assets/tracker/svg/gantt-settings-button.svg)&nbsp;**Настройки диаграммы** и в блоке **Показывать на полосе задачи** выберите настройку: 

* Чтобы на полосе задачи отображалось ее название, укажите пункт **название**. 

* Чтобы на полосе задачи отображался значок исполнителя, укажите пункт **исполнителя**. 

* Чтобы отметить на диаграмме красным цветом признак просроченной задачи (разницу между дедлайном и текущей датой), укажите пункт **просроченный дедлайн**.

* Чтобы на диаграмме не отображались стрелки связей, укажите пункт **скрыть связи**.

## Настроить отображение блока со списком задач {#tasks-view}

* Чтобы выбрать поля, которые вы хотите видеть в таблице списка задач слева от диаграммы, нажмите на значок ![](../../_assets/tracker/svg/settings-old.svg). Отметьте нужные поля и нажмите **Применить**.

* Чтобы изменить размер столбца в блоке, зажмите его край и измените ширину.

* Чтобы скрыть блок со списком задач, нажмите на значок ![](../../_assets/tracker/svg/close-block.svg).
