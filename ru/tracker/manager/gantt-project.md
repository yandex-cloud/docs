# Диаграмма Ганта в проектах {{ tracker-name }}

Диаграмма Ганта для проекта в {{ tracker-name }} — это инструмент для составления календарного плана работ над задачами проекта. На диаграмме Ганта отображаются все задачи проекта и связи между ними.

Чтобы перейти к диаграмме Ганта:

1. На панели слева выберите ![](../../_assets/tracker/svg/project.svg)&nbsp;**Проекты** или перейдите по [прямой ссылке]({{ link-tracker }}pages/projects) и откройте страницу проекта.

1. {% include [go-to-gantt](../../_includes/tracker/go-to-gantt.md) %}

1. {% include [gantt-time-scale](../../_includes/tracker/gantt-time-scale.md) %}

{% note tip %}

Вы также можете открыть календарный план в виде [диаграммы Ганта для всех проектов](./my-projects.md#gantt-projects), которые вам доступны.

{% endnote %}

## Изменить связи между задачами {#links}

{% include [gantt-links](../../_includes/tracker/gantt-links.md) %}

* Если у задачи есть блокеры, которые не входят в текущий проект, на диаграмме слева от полосы задачи появится значок ![](../../_assets/tracker/svg/blocker.svg) с их количеством. 

   * Вы можете включить отображение внешних блокеров в списке задач: для этого перейдите в ![](../../_assets/tracker/svg/gantt-settings-button.svg)&nbsp;**Настройки диаграммы** и выберите опцию **Показывать внешние блокеры**.

   * В списке задач внешние блокеры отображаются серым цветом под зависимой задачей. Если включен [иерархический тип отображения](#trees), блокеры отобразятся под веткой задач. Внешние блокеры нельзя перемещать в списке, но их можно открыть и отредактировать как любую задачу.

   * Сроки начала работ и дедлайн для внешних блокеров можно [изменять](#timing).

   * Чтобы удалить связь между внешним блокером и зависимой задачей, нажмите на значок ![](../../_assets/tracker/svg/blocker.svg) и выберите ![](../../_assets/tracker/svg/del-link.svg) **Удалить связь**.

## Изменить сроки задач {#timing}

{% include [gantt-timing](../../_includes/tracker/gantt-timing.md) %}

* Чтобы настроить автоматическое изменение сроков родительской задачи, перейдите в ![](../../_assets/tracker/svg/gantt-settings-button.svg)&nbsp;**Настройки диаграммы** и включите опцию **Автоматически продлять сроки задачи-родителя по срокам подзадач**. Если дедлайн у подзадачи наступает после дедлайна родительской задачи, дедлайн родительской задачи будет сдвигаться. Дата начала подзадачи также изменяется в соответствии с датой начала родительской задачи.

* Если для родительской задачи не указаны сроки, напротив нее на диаграмме отображается подсказка в виде голубой полосы в соответствии со сроками всех подзадач.
