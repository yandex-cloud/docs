# Изменить основные параметры очереди

{% note warning %}

По умолчанию настраивать очередь может [только ее владелец](queue-access.md).

{% endnote %}

Чтобы настроить основные параметры очереди:

1. Откройте [страницу очереди](../user/queue.md).

1. Слева от имени очереди выберите ![](../../_assets/tracker/icon-settings.png) → **Администрирование**.

1. Перейдите на вкладку **Основные параметры**

1. Настройте параметры очереди.
    * Основные параметры:
 
        * **Название очереди**.
        * **Описание очереди**. Описание отображается на странице очереди на вкладке **Описание**.
        * **Приоритет задач по умолчанию**.
        * **Голосование в задачах** — разрешить или запретить пользователям [голосовать за задачи](../user/votes.md) очереди.
    * В блоке **Расширенные настройки**:
        * **Обязательные для отображения поля** — параметры, которые будут отображаться на странице задачи вне зависимости от [настроек пользователя](../user/edit-ticket.md#section_jqw_ppn_jz).
        * **Поля в уведомлениях** — параметры задачи, которые будут присутствовать во всех уведомлениях о задачах очереди.

        {% if audience == "internal" %}

        * **Статус при добавлении внешней связи** — статус, в который задачи очереди автоматически переводятся при создании связи с коммитами или пулл-реквестами на Crusible, Review Board, Stash или Git Hub. Список возможных статусов задач определяется настройками [воркфлоу](workflow.md).
        
        {% endif %}

        * **Отправка писем** — пользователи смогут [отправлять письма](../user/comments.md#send-comment) на любые адреса прямо со страницы задачи. Текст письма прикрепляется к задаче в виде комментария.
        * **Добавлять номер задачи в тему письма** — указывать ключ задачи в теме [писем, отправленных со страницы задачи](../user/comments.md#send-comment).
      
        * **Не связывать с задачами других очередей автоматически** — не добавлять связь, если в комментарий или в описание задачи вставлен ключ задачи из другой очереди.
    * В блоке **Настройки интеграций**: {#integration}
        * **Отображать вкладку Коммиты** — включить или выключить вкладку **Коммиты** на странице очереди и на страницах задач. На вкладке отображаются все [привязанные к задачам коммиты](../user/ticket-links.md#section_commit).

        {% if audience == "internal" %}

        * **Отображать вкладку «Цели»** — включить или выключить вкладку [Цели](goals.md) на странице очереди. На вкладке отображается список целей, к которым привязаны задачи очереди.

        {% endif %}

        * **Отображать вкладку «Проекты»** — включить или выключить вкладку [Проекты](projects.md) на странице очереди.

        {% if audience == "internal" %}

        * **Отображать вкладку «Транзакции OEBS»** — включить или выключить вкладку **Транзакции OEBS** на странице очереди. На вкладке отображается список транзакций, к которым привязаны задачи очереди.
        * **Отображать ссылку на связь с сервисом** — если опции включены, на странице задачи будут отображаться разделы со связями задачи с внешними сервисами.
        * **Отключить автоматическую связь с Кондуктором** — по умолчанию {{ tracker-name }} автоматически привязывает задачу к тикету в Кондукторе, если в комментариях к тикету указан ключзадачи.
        * **Показывать стандартную форму создания задачи** — выберите, нужно ли на странице создания задачи показывать стандартную форму, если у вас [подключена своя форма](attach-form.md) из {{ forms-full-name }}.

        {% endif %}

1. Нажмите кнопку **Сохранить**.
