После успешной аутентификации на сервере федерации в профиле сохранится [IAM-токен](../../iam/concepts/authorization/iam-token.md). При каждой операции аутентификация будет происходить с помощью этого токена, пока не истечет [время жизни токена](../../iam/concepts/authorization/iam-token.md). После этого CLI снова выведет сообщение о необходимости пройти аутентификацию в браузере.
