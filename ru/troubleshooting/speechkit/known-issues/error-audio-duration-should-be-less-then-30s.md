# Устранение ошибки audio duration should be less than 30s в SpeechKit

## Описание проблемы {#issue-description}
При загрузке в API SpeechKit для потокового распознавания файла с аудиозаписью в формате OGG рвозникает ошибка:

```text
{'error_code': 'BAD_REQUEST', 'error_message': 'audio duration should be less than 30s'}
```
Однако длительность загружаемого в SpeechKit аудиофайла может составлять менее 30 секунд.

## Решение {#issue-resolution}
Ошибка может возникать из-за неподдерживаемого кодека файла, отправленного на распознание. 
Например, для формата медиаконтейнера OGG сервис поддерживает только [кодек Opus](../../../speechkit/formats.md#OggOpus).

Для корректного распознавания используйте данные, закодированные с помощью аудиокодека Opus.
