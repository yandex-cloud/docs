# Как восстановить доступ к виртуальной машине

## Описание задачи {#case-description}

Потерян пароль или SSH-ключ для доступа к виртуальной машине. Необходимо восстановить доступ, задав новый пароль пользователю.

## Решение {#case-resolution}

Вы можете восстановить доступ к виртуальной машине, изменив её метаданные. 
Для этого выполните следующие действия:

1. [Установите](https://yandex.cloud/ru/docs/cli/quickstart#install) утилиту командной строки YC CLI и [настройте профиль](https://yandex.cloud/ru/docs/cli/operations/profile/profile-create).

2. Создайте файл `set-passwd.yaml` со следующим содержимым:

   ```yaml
   #cloud-config
   bootcmd:
     - [sh, -xc, "sudo echo user:password| chpasswd"]
   ```

   Где:
   * `user` — имя пользователя.
   * `password` — новый пароль, который вы хотите задать.

3. Передайте созданный файл в метаданные виртуальной машины, включив при этом доступ к серийной консоли:

   ```bash
   yc compute instance update \
     --id <id_vm> \
     --metadata-from-file user-data=set-passwd.yaml \
     --metadata serial-port-enable=1
   ```

   Где `<id_vm>` — идентификатор виртуальной машины. Его можно узнать в [консоли управления](https://yandex.cloud/ru/) на странице виртуальной машины.

4. [Перезапустите](../../../compute/operations/vm-control/vm-stop-and-start#restart) виртуальную машину.

5. Подключитесь к ВМ через [серийную консоль](../../../compute/operations/serial-console/) или по SSH, используя новый пароль.

   Рекомендуется сразу сменить пароль на более надежный. В ОС Linux это можно сделать командой:

   ```bash
   sudo passwd <user_name>
   ```

6. После успешного подключения очистите метаданные, чтобы пароль не сбрасывался при каждой перезагрузке ВМ. Для этого выполните на вашем локальном компьютере команду:

   ```bash
   yc compute instance update \
     --id <id_vm> \
     --metadata=''
   ```

{% note alert %}

Рекомендуем включать доступ к серийной консоли только на время выполнения операций по восстановлению доступа. После завершения работ отключите доступ в настройках ВМ.

{% endnote %}
