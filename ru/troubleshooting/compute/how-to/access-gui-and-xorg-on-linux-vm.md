# Настройка GUI и графического окружения на ВМ с Linux


## Описание задачи {#case-description}

* Необходимо использовать графические приложения на виртуальных машинах с ОС Linux;

## Решение {#case-resolution}

Cуществует три варианта, позволяющих использовать графическое окружение рабочего стола на ВМ с Linux:

* Использовать VNC или RDP. Для этого:
    1. Установите окружение рабочего стола;
    2. После этого установите xrdp или VNC-сервер (соответственно, для подключения по RDP или VNC);
    Рекомендуем использовать связку VNC-сервера, прослушивающего только localhost, и SSH-туннеля, — в этом случае вы исключите риск подключения по VNC к вашей ВМ злоумышленников, а также зашифруете VNC-трафик. В этом вам может помочь руководство [по этой ссылке](https://habr.com/ru/company/ua-hosting/blog/273201/).

* Пробросить Xorg на локальный компьютер. О том, как это сделать, также есть множество статей в сети. Для Windows используйте [Xming](https://sourceforge.net/projects/xming/):
    1. Установите окружение рабочего стола;
    2. Настройте проброс Xorg в вашем SSH-клиенте и подключитесь к ВМ;
    3. Запустите графическое приложение из терминала.

* Использовать Terraform, для создания ВМ с установленным графическим окружением, например, как [здесь](https://nikolaymatrosov.medium.com/rdp-на-ubuntu-в-yandex-cloud-c9d7870a47cc).
