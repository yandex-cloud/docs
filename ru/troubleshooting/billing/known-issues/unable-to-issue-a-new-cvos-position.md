# Не удается оформить резервируемое потребление для Compute cloud

## Описание проблемы {#case-description}
При попытке оформления нового [резерва потребляемых ресурсов](../../../billing/concepts/cvos.md) (CVoS) для Compute Cloud появляется сообщение с  ошибкой: `Не удалось зарезервировать потребление. Для резервирования ресурсов обратитесь в отдел продаж.`

### Диагностика
Для резервирования ресурсов в Compute Cloud действует ограничение на количество ресурсов в рамках одной покупки.

В рамках одного резерва CVoS для Compute Cloud на текущий момент можно оформить:
* vCPU: не более 20 ядер
* RAM: не более 40 ГБ

## Решение {#case-resolution}

Чтобы обойти ограничение, достаточно разделить на несколько покупок необходимое количество ресурсов Облака. 

{% note info %}

Ограничений на количество покупок нет, поэтому бóльший резерв ресурсов можно разбить на несколько покупок.

{% endnote %}
