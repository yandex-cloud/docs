# Пересчет ассистентов

Чтобы запустить пересчет ассистентов в диалогах за определенный период, создайте задачу на пересчет. Для этого:

1. Откройте [главную страницу]({{ link-speechsense-main }}) {{ speechsense-name }}.
1. Перейдите в нужное пространство, затем выберите нужный проект.
1. Перейдите на вкладку **{{ ui-key.yc-ui-talkanalytics.assistants.assistants }}** и отметьте ассистентов, которые нужно пересчитать в диалогах.

   {% note info %}
   
   Если в указанных ассистентах используется анализ по выбранным диалогам с фильтрами по тегам, то алгоритм дополнительно предложит пересчитать теги. [Смысловые pro-теги](../../concepts/tags.md##sense-pro-tags) пересчитываются за дополнительную плату.
   
   {% endnote %}
   
   
1. Внизу страницы нажмите кнопку **{{ ui-key.yc-ui-talkanalytics.tags.reinit }}**.
1. Далее укажите период пересчета диалогов и нажмите кнопку **{{ ui-key.yc-ui-talkanalytics.common.create }}**.


## Отменить задачу на пересчет ассистентов {#cancel-recalc}

Отменить пересчет ассистентов можно только для задач в статусах **В очереди**, **Пересчитывается**. Для этого:

1. Перейдите на вкладку **{{ ui-key.yc-ui-talkanalytics.reinit.tab.title }}**.
1. Нажмите ![image](../../../_assets/options-grey.svg) справа от нужной задачи и выберите пункт **{{ ui-key.yc-ui-talkanalytics.reinit.action.cancel.title }}**.
