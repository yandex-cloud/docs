# Пересчет тегов

Чтобы вручную запустить пересчет тегов в диалогах за определенный период, создайте задачу на пересчет тегов. Для этого:

1. Откройте [главную страницу]({{ link-speechsense-main }}) {{ speechsense-name }}.
1. Перейдите в нужное пространство, затем выберите нужный проект.
1. Перейдите на вкладку **{{ ui-key.yc-ui-talkanalytics.tags.tags }}** и отметьте теги, которые нужно вручную пересчитать в диалогах.
   
   {% note info  %}
   
   Если у тега есть [зависимые теги](../../concepts/tags.md#dependent-tags), то они тоже будут пересчитаны.
   
   {% endnote %}
   
   
1. Внизу страницы нажмите кнопку **{{ ui-key.yc-ui-talkanalytics.tags.reinit }}**.
1. Далее укажите период пересчета диалогов и нажмите кнопку **{{ ui-key.yc-ui-talkanalytics.common.create }}**.


## Отменить задачу на пересчет тегов {#cancel-recalc}

Отменить пересчет тегов можно только для задач в статусах **В очереди**, **Пересчитывается**. Для этого:

1. Перейдите на вкладку **{{ ui-key.yc-ui-talkanalytics.reinit.tab.title }}**.
1. Нажмите ![image](../../../_assets/options-grey.svg) справа от нужной задачи и выберите пункт **{{ ui-key.yc-ui-talkanalytics.reinit.action.cancel.title }}**.
