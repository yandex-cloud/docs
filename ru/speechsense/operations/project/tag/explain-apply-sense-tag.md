# Объяснить применение смыслового тега

Чтобы получить объяснение, почему сработал [смысловой тег](../../../concepts/tags.md#sense-tags):

1. Откройте [главную страницу]({{ link-speechsense-main }}) {{ speechsense-name }}.
1. Перейдите в нужное пространство, затем выберите проект.
1. Перейдите на вкладку **{{ ui-key.yc-ui-talkanalytics.dialogs.dialogs }}** и найдите нужный диалог.
1. Откройте диалог одним из способов:

    * Чтобы открыть превью диалога, нажмите на строку диалога.
    * Чтобы открыть диалог в новом окне, нажмите в строке диалога справа ![image](../../../../_assets/console-icons/arrow-up-right-from-square.svg).

1. На странице найдите присвоенные диалогу смысловые теги.
1. Справа напротив нужного тега нажмите ![image](../../../../_assets/console-icons/sense-tag-explain.svg).
1. Запустится расчет объяснения, почему сработал смысловой тег. Это займет некоторое время.
   
   {% note info %}
   
   При необходимости можно остановить расчет объяснения.
   
   {% endnote %}

1. После завершения расчета напротив тега появится значок ![image](../../../../_assets/console-icons/sense-tag-explain.svg) с красной точкой.
1. Нажмите на значок, чтобы просмотреть объяснение.
