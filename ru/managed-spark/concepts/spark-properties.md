# Свойства {{ SPRK }}

Свойства {{ SPRK }} определяют конфигурацию заданий, например ресурсы запуска, параметры динамической аллокации и параллелизма, SQL-параметры и т. д. Чтобы переопределить конфигурацию, вы можете изменить свойства {{ SPRK }} на уровне отдельного задания при его создании.

Свойства {{ SPRK }} задаются и хранятся в формате «ключ-значение». Вы можете задать любые свойства, указанные в [официальной документации {{ SPRK }}](https://spark.apache.org/docs/latest/configuration.html), кроме:

* `spark.kubernetes.*` — свойства {{ k8s }} нельзя переопределить, так как это может привести к некорректной работе кластера {{ msp-full-name }}.
* `spark.yarn.*` — {{ msp-full-name }} не использует YARN.
