# Программно-ускоренная сеть в {{ vpc-name }}

Если ваше приложение нагружает ядра ВМ на предельных уровнях и одновременно с этим требует передачи больших объемов сетевого трафика, пользовательские и сетевые процессы могут начать борьбу за аппаратные ресурсы. Чтобы исключить такую ситуацию, рекомендуется использовать в {{ vpc-short-name }} программно-ускоренную сеть.
