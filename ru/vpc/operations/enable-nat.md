# Включить NAT в интернет

{% note info %}

Функция находится на стадии [Preview](../../overview/concepts/launch-stages.md).

{% endnote %}

С помощью NAT в интернет можно дать облачным ресурсам, подключенным к подсети, возможность выходить в интернет без использования публичных IP-адресов. 

Чтобы включить NAT в интернет:

{% list tabs %}

- Консоль управления

  1. В [консоли управления]({{ link-console-main }}) перейдите в каталог, где находится подсеть, облачным ресурсам из которой нужен доступ в интернет.
  1. В списке сервисов выберите **{{ vpc-name }}**.
  1. Выберите сеть, в которой находится нужная подсеть.
  1. В блоке **Подсети** нажмите значок ![options](../../_assets/options.svg) в строке нужной подсети.
  1. В открывшемся меню выберите пункт **Включить NAT в интернет**.

{% endlist %}
