# Включить программно-ускоренную сеть

{% note warning %}

Функциональность доступна только по согласованию с вашим аккаунт-менеджером.

{% endnote %}

Вы можете включить [программно-ускоренную сеть](../concepts/software-accelerated-network.md) при создании или изменении виртуальной машины.

## Включение программно-ускоренной сети {#enable-on-vm-creation}

{% list tabs %}

- Консоль управления

  При [создании](../../compute/operations/vm-create/create-linux-vm.md) или [изменении](../../compute/operations/vm-control/vm-update-resources#enable-software-accelerated-network.md) ВМ установите флаг **Программно-ускоренная сеть** в блоке **Вычислительные ресурсы**.

{% endlist %}
