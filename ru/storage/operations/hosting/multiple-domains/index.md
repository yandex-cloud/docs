# Поддержка нескольких доменных имен

Инструкция описывает, как настроить поддержку основного доменного имени сайта и его дополнительного домена, например: `example.com` и `example2.com`. Вы создадите [бакеты](../../../concepts/backup.md) для основного и дополнительного домена, в первом бакете настроите [хостинг статического сайта](../../../concepts/hosting.md) на основном доменном имени, а во втором бакете включите переадресацию с дополнительного доменного имени на первый бакет. Для управления доменами вы можете воспользоваться сервисом [{{ dns-full-name }}](../../../../dns/).

Вы можете настроить поддержку нескольких доменных имен для статического сайта в {{ objstorage-name }} с помощью одного из инструментов:
* [Консоль управления, CLI, API](console.md) — используйте этот способ, чтобы пошагово создать инфраструктуру в консоли управления {{ yandex-cloud }}, CLI или API.
* [{{ TF }}](terraform.md) — используйте этот способ, чтобы упростить создание ресурсов и управление ими, используя подход «инфраструктура как код» (IaC). Скачайте пример конфигурации {{ TF }} с репозитория GitHub, а затем разверните инфраструктуру с помощью [{{ TF }}-провайдера {{ yandex-cloud }}]({{ tf-docs-link }}).
