# Настройка HTTPS

Если бакет используется для [хостинга статического сайта](../../hosting/index.md), то для доступа к сайту по протоколу [HTTPS](https://ru.wikipedia.org/wiki/HTTPS) необходимо загрузить собственный сертификат безопасности и соответствующий ему секретный ключ.

{{ objstorage-name }} поддерживает только [PEM](https://en.wikipedia.org/wiki/Privacy-Enhanced_Mail)-encoded сертификаты.

{% list tabs %}

- Консоль управления
  
    1. Выберите каталог.
    1. Выберите сервис **Object Storage**.
    1. Нажмите на имя необходимого бакета.
    1. Перейдите на вкладку **HTTPS**.
    1. В отобразившейся панели справа нажмите кнопку **Настроить**.
    1. Добавьте сертификат и секретный ключ.
    1. Нажмите кнопку **Сохранить**.
  
{% endlist %}
