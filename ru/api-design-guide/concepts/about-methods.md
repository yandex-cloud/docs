# Методы в API

Для управления ресурсами в [API {{ yandex-cloud }}](https://github.com/yandex-cloud/cloudapi) определен набор методов. Каждому gRPC-вызову ставится в соответствие HTTP-метод. Например, вызову `List` соответствует метод `GET`,  а вызову `Create` — метод `POST`. Соответствия HTTP-запросам задаются в описании методов, в аннотации [google.api.http](https://github.com/googleapis/googleapis/blob/master/google/api/http.proto).
