#### Описание

[Аудитный лог событий уровня сервисов](https://yandex.cloud/ru/docs/audit-trails/concepts/format-data-plane) — это запись в виде JSON-объекта о событиях, которые произошли с ресурсами Yandex Cloud. Отслеживание событий уровня сервисов упрощает сбор дополнительных событий с облачных сервисов, что позволяет эффективнее реагировать на инциденты безопасности в облаках. Кроме того, отслеживание событий уровня сервисов помогает обеспечить соответствие вашей облачной инфраструктуры нормативным правовым актам и отраслевым стандартам. Например, вы можете отслеживать получение сотрудниками доступа к конфиденциальным данным, хранящимся в [бакетах](https://yandex.cloud/ru/docs/storage/concepts/bucket).

Включать сбор аудитных логов уровня сервисов нужно отдельно для каждого из [поддерживаемых сервисов](https://yandex.cloud/ru/docs/audit-trails/concepts/control-plane-vs-data-plane#data-plane-events).
