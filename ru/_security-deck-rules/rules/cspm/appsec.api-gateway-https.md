#### Описание

Сервис [Yandex API Gateway](https://yandex.cloud/ru/docs/api-gateway/) обеспечивает безопасное подключение по протоколу HTTPS. Вы можете привязать собственный домен и загрузить собственный [сертификат безопасности](https://yandex.cloud/ru/docs/certificate-manager/concepts/#types) для доступа к вашему [API-шлюзу](https://yandex.cloud/ru/docs/api-gateway/concepts/) по протоколу HTTPS.

Без использования протокола HTTPS трафик между клиентом и API-шлюзом передается в незашифрованном виде, что ведет к следующим рискам:

* злоумышленники могут перехватывать данные, например, через атаки типа MITM (Man-in-the-Middle);
* при передаче в открытом виде могут оказаться раскрыты конфиденциальные данные: персональные данные, платежная информация, токены аутентификации, пароли и т.п.

#### Инструкции и решения по выполнению

1. В [консоли управления](https://console.yandex.cloud/) выберите каталог, в котором находится нужный API-шлюз.
1. [Перейдите](https://yandex.cloud/ru/docs/console/operations/select-service#select-service) в сервис API Gateway и в открывшемся окне нажмите на строку с нужным API-шлюзом.
1. В меню слева выберите **Домены** и нажмите кнопку **Подключить**.
1. В открывшемся окне выберите TLS-сертификат и укажите соответствующее этому сертификату доменное имя.
1. Нажмите кнопку **Подключить**.
