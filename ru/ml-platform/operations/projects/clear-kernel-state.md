# Очистка состояния интерпретатора

{% note alert %}

Это действие сбросит все результаты и очистит состояние переменных.

{% endnote %}

Чтобы сбросить состояние интерпретатора, на верхней панели в окне проекта выберите **Kernel** → **Clear kernel state**.
