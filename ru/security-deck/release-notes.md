# История изменений в {{ sd-full-name }}

## III квартал 2025 {#q3-2025}

* Запущены модули [KSPM](./concepts/kspm.md) и [CSPM](./concepts/cspm.md).
* Модули KSPM и CSPM теперь доступны в качестве источников данных для [алертов](./concepts/alerts.md). {{ tag-con }}
* Добавлена возможность [создать окружение {{ sd-name }}](./operations/workspaces/index.md) — контейнер, который содержит настройки и ресурсы модулей, перечень контролируемых ресурсов, параметры контроля и другие настройки. {{ tag-con }}
* Для модуля [DSPM](./concepts/dspm.md) появились возможности частичного сканирования и [сканирования](./operations/dspm/create-scan.md) по изображениям.  {{ tag-con }}
