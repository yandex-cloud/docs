# Создание эндпоинта приемника

Чтобы создать эндпоинт приемника:

{% list tabs %}

- Консоль управления

  1. В [консоли управления]({{ link-console-main }}) выберите каталог, в котором будет создан эндпоинт.
  1. В списке сервисов выберите **{{ data-transfer-name }}**.
  1. Нажмите кнопку **Создать эндпоинт**.
  1. В открывшемся окне в поле **Направление** выберите **Приёмник**.
  1. Укажите **Имя** эндпоинта. Используйте строчные латинские буквы и цифры.
  1. (опционально) Укажите **Описание** эндпоинта.
  1. В поле **База данных** выберите СУБД, в которую вы хотите передавать данные:
  
      {% cut "{{mmy-short-name}}" %}

      В блоке **Параметры эндпоинта**:
      - Укажите **Идентификатор кластера**, в который необходимо передавать данные.
      - Укажите **Имя пользователя репликации**.
      - Введите **Пароль** пользователя для доступа к базе данных.
      - Укажите **Имя базы данных** в кластере.
      - (опционально) В поле **PEM-сертификат** нажмите кнопку **Загрузить файл**:
          - Загрузите файл или добавьте тело сертификата в текстовом виде.
      - (опционально) Укажите **Сетевой интерфейс для эндпоинта**, через который можно подключиться к кластеру БД.
      - (опционально) Установите **Режим sql_mode**.
      - (опционально) При необходимости выберите **Отключение проверки внешних ключей на приёмнике. Используется настройка FOREIGN_KEY_CHECKS=0**.

      {% endcut %}

      {% cut "{{mpg-short-name}}" %}

      В блоке **Параметры эндпоинта** укажите:
      - **Идентификатор кластера**, в который необходимо передавать данные.
      - **Имя пользователя** базы данных.
      - **Пароль** пользователя для доступа к базе данных.
      - **Имя базы данных** в кластере.
      - (опционально) **Сетевой интерфейс для эндпоинта**, через который можно подключиться к кластеру БД.

      {% endcut %}

      {% cut "{{MY}}" %}

      В блоке **Параметры эндпоинта**:
      - Укажите **Хост базы данных**, в который необходимо передавать данные.
      - В поле **Порт базы данных** укажите порт для подключения к СУБД.
      - Укажите **Имя пользователя**.
      - Введите **Пароль** пользователя для доступа к базе данных.
      - Укажите **Имя базы данных** в кластере.
      - (опционально) В поле **PEM-сертификат** нажмите кнопку **Загрузить файл**:
          - Загрузите файл или добавьте тело сертификата в текстовом виде.
      - (опционально) Укажите **Сетевой интерфейс для эндпоинта**, через который можно подключиться к кластеру БД.
      - (опционально) Установите **Режим sql_mode**.
      - (опционально) Выберите значение параметра **Отключение проверки внешних ключей на приёмнике. Используется настройка FOREIGN_KEY_CHECKS=0**.

      {% endcut %}

      {% cut "{{PG}}" %}

      В блоке **Параметры эндпоинта** укажите:
      - **Хост базы данных**, в который необходимо передавать данные.
      - **Порт базы данных** для подключения к СУБД.
      - **Имя пользователя** базы данных.
      - **Пароль** пользователя для доступа к базе данных.
      - **Имя базы данных** в кластере.
      - (опционально) **Сетевой интерфейс для эндпоинта**, через который можно подключиться к кластеру БД.

      {% endcut %}

  1. Нажмите кнопку **Создать**.
