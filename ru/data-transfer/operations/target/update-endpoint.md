# Изменение эндпоинта приемника

Чтобы изменить эндпоинт приемника:

{% list tabs %}

- Консоль управления

  1. В [консоли управления]({{ link-console-main }}) выберите каталог, в котором нужно изменить эндпоинт.
  1. В списке сервисов выберите **{{ data-transfer-name }}**.
  1. Выберите эндпоинт, который нужно изменить.
  1. В верхней правой части окна нажмите кнопку ![pencil](../../../_assets/pencil.svg) **Редактировать**.
  1. Отредактируйте параметры эндпоинта:

      {% cut "{{mmy-short-name}}" %}

      - Укажите **Имя пользователя репликации** базы данных.
      - Введите **Пароль** пользователя для доступа к базе данных.
      - В поле **PEM-сертификат** нажмите кнопку **Загрузить файл**:
          - Загрузите файл или добавьте тело сертификата в текстовом виде.
      - Выберите значение опции **Отключение проверки внешних ключей на приёмнике. Используется настройка FOREIGN_KEY_CHECKS=0**.

      {% endcut %}

      {% cut "{{mpg-short-name}}" %}

      - Введите **Пароль** пользователя для доступа к базе данных.

      {% endcut %}

      {% cut "{{MY}}" %}

      - Введите **Пароль** пользователя для доступа к базе данных.
      - В поле **PEM-сертификат** нажмите кнопку **Загрузить файл**:
          - Загрузите файл или добавьте тело сертификата в текстовом виде.
      - Выберите значение опции **Отключение проверки внешних ключей на приёмнике. Используется настройка FOREIGN_KEY_CHECKS=0**.

      {% endcut %}

      {% cut "{{PG}}" %}

      - Введите **Пароль** пользователя для доступа к базе данных.

      {% endcut %}
  
  1. Нажмите кнопку **Применить**.
  
{% endlist %}
