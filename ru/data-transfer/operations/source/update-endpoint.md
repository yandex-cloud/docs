# Изменение эндпоинта источника

Чтобы изменить эндпоинт источника:

{% list tabs %}

- Консоль управления

  1. В [консоли управления]({{ link-console-main }}) выберите каталог, в котором нужно изменить эндпоинт.
  1. В списке сервисов выберите **{{ data-transfer-name }}**.
  1. Выберите эндпоинт, который нужно изменить.
  1. В верхней правой части окна нажмите кнопку ![pencil](../../../_assets/pencil.svg) **Редактировать**.
  1. Отредактируйте параметры эндпоинта:

      {% cut "{{mmy-short-name}}" %}

      - Введите **Пароль** пользователя для доступа к базе данных.
      - Измените **Белый список таблиц**. Будут передаваться данные только из таблиц белого списка.
      - Измените **Черный список таблиц**. Данные из таблиц черного списка передаваться не будут.
      - **Часовой пояс базы** в формате UTC.

      {% endcut %}

      {% cut "{{mpg-short-name}}" %}

      - **Пароль** пользователя для доступа к базе данных.
      - **Белый список таблиц**. Будут передаваться данные только из таблиц белого списка.
      - **Черный список таблиц**. Данные из таблиц черного списка передаваться не будут.

      {% endcut %}

      {% cut "{{MY}}" %}

      - **Пароль** пользователя для доступа к базе данных.
      - **Белый список таблиц**. Будут передаваться данные только из таблиц белого списка.
      - **Черный список таблиц**. Данные из таблиц черного списка передаваться не будут.
      - **PEM-сертификат** для доступа к базе.
      - **Часовой пояс базы** в формате UTC.

      {% endcut %}

      {% cut "{{PG}}" %}

      - **Пароль** пользователя для доступа к базе данных.
      - **Белый список таблиц**. Будут передаваться данные только из таблиц белого списка.
      - **Черный список таблиц**. Данные из таблиц черного списка передаваться не будут.

      {% endcut %}

  1. Нажмите кнопку **Применить**.
  
{% endlist %}
