# Создание трансфера

Чтобы создать трансфер:

{% list tabs %}

- Консоль управления

  1. Нажмите кнопку **Создать трансфер**.
  1. Выберите эндпоинт **Источник** или [создайте](../source/create-endpoint.md) новый.
  1. Выберите эндпоинт **Приемник** или [создайте](../target/create-endpoint.md) новый.
  1. В блоке **Параметры трансфера**:
      - Введите **Имя** трансфера.
      - Выберите **Тип** трансфера:
        - **Копировать и реплицировать** для переноса и поддержки текущего состояния базы источника в базу приемника.
        - **Копировать** для переноса снапшота базы данных.
        - **Реплицировать** для непрерывного получения изменений из базы данных источника и применения их к базе данных приемнику.
      - Введите **Описание** трансфера.
  1. Нажмите кнопку **Создать**.

      Дождитесь создания трансфера.
  1. В строке с созданным трансфером нажмите ![horizontal-ellipsis](../../../_assets/horizontal-ellipsis.svg) и выберите **Активировать**.

      Дождитесь активации трансфера.

{% endlist %}
