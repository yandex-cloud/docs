# Настройка эндпоинта-приемника Logbroker

При [создании](../index.md#create) или [изменении](../index.md#update) эндпоинта задайте настройки подключения вы можете задать настройки подключения:

## Настройки {#settings}

{% list tabs %}

- Консоль управления

  * **Инстанс** — выберите один из списка.

  * **Настройки топика** — выберите `Полное имя топика` или `Префиск топика`.

    * **Полное имя топика** — имя топика, в который будут отправляться сообщения.
    
    * **Префиск топика** — префикс топика, аналог настройки `database.server.name Debezium`. Сообщения будут отправляться в топик c именем `<префикс_топика>.<схема>.<имя_таблицы>`.
 
{% endlist %}
