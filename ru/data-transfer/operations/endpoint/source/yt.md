# Настройка эндпоинта-источника YT

При [создании](../index.md#create) или [изменении](../index.md#update) эндпоинта задайте настройки доступа к YT.

* Настройки подключения к [кластеру YT](#cluster). Эти параметры обязательные.
* [Расширенные настройки](#additional-settings).

## Кластер {#cluster}

{% list tabs %}

- Консоль управления

    * **Кластер YT** — имя кластера источника YT, например, "HAHN", "ARNOLD", или "FREUD".

    * **Токен** — YT-токен для чтения таблиц.

    * **Таблицы и директории** — пути к таблицам с данными. Для добавления нового пути нажмите кнопку **+ Путь** и введите значение.

{% endlist %}

## Расширенные настройки {#additional-settings}

{% list tabs %}

- Консоль управления

     * **Название колонки для индекса строки** — номер строки из исходной таблицы YT. Используется для сверки данных и исключения дупликации.

{% endlist %}
