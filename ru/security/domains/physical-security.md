# Физическая безопасность

Платформа {{ yandex-cloud }} обеспечивает физическую безопасность своих дата-центров, см. [подробное описание мер физической безопасности](../standarts.md#physic-sec).

Если критичные данные передаются за пределы {{ yandex-cloud }}, то клиент отвечает за управление физическим доступом для всех мест обработки данных.
