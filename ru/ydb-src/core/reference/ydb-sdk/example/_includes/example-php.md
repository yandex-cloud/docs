Фрагмент кода приложения для инициализации драйвера:

```php
<?php

use YandexCloud\Ydb\Ydb;

$config = [
    // ...
];

$ydb = new Ydb($config);

```

Фрагмент кода приложения для создания сессии:

```php
// obtaining the Table service
$table = $ydb->table();

// obtaining a session
$session = $table->session();
```
