### Как найти id объекта (подключения, датасета, чарта, дашборда)? {#how-to-find-object-id}

Id указан в URL-адресе объекта. Например, {% if audience != "internal" %}`https://datalens.yandex.ru/datasets/scuxhh1jrrlql-new-dataset`{% else %}`https://datalens.yandex-team.ru/datasets/scuxhh1jrrlql-new-dataset`{% endif %}, где `scuxhh1jrrlql` — это id.
