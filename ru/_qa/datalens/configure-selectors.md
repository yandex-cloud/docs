### Как настроить селекторы так, чтобы в списке появлялись только элементы, которые есть внутри конкретной выборки датасета? {#how-to-configure-selectors}

Это можно сделать несколькими способами:

* В качестве типа селектора выберите **Ручной ввод** и перечислите нужные значения. После создания селектора настройте связи между селектором и чартами на дашборде. Такой способ подойдет, если значения не меняются в зависимости от выборки.
* Создайте новый селектор и с помощью связи отфильтруйте значения в нужном селекторе.
* С помощью вычисляемого поля:
  * Создайте вычисляемое поле, определяющее список значений. Пример: `CASE ([country], "BY", "СНГ", "KZ", "СНГ", "RU", "СНГ", "Другие страны”)`.
  * Выведите это поле как дополнительный селектор и установите для него значение по умолчанию `"СНГ"`.
    В таком случае во втором селекторе значениями по умолчанию будут: `"BY"`, `"KZ"` и `"RU"`.
* Если другие значения не используются, создайте датасет с заданными значениями и на основе этого датасета сделайте селектор. Свяжите селектор с чартом.
