# Работа с внешними таблицами

#### Каким образом при работе с внешними таблицами передаются учетные данные пользователей? {#pxf-users}

При работе с [внешними таблицами по протоколу PXF](../../managed-greenplum/operations/external-tables.md) учетные данные пользователей передаются в открытом виде. Поэтому учетные данные доступны только [пользователю-администратору с ролью MDB_ADMIN](../../managed-greenplum/concepts/cluster-users.md), остальным пользователям учетные данные недоступны из соображений безопасности.
