#### Как следить за потреблением vRAM через мониторинг? {#track-vRAM-use}

Сервис {{ compute-name }} не может измерить потребление vRAM внутри гостевой операционной системы: для сервиса потребление памяти виртуальной машиной всегда одинаковое — то, которое выделено в момент ее запуска.

Чтобы отслеживать состояние vRAM, используйте сервис {{ monitoring-full-name }}. Он позволяет записывать собственные метрики, смотрите инструкцию [{#T}](../../monitoring/operations/metric/add.md). Просто выгружайте данные о состоянии vRAM заданием по расписанию, а {{ monitoring-name }} будет отображать их.
