# Пробный период для новых клиентов

{% include [free-trial-conditions](../../_includes/free-trial-conditions.md) %}

Пробный период позволяет бесплатно использовать ресурсы {{ yandex-cloud }} в ограниченном режиме. При этом потребленные ресурсы оплачиваются из средств [стартового гранта](bonus-account.md#start). После завершения пробного периода ресурсы будут сохранены, но остановлены. Для возобновления работы с {{ yandex-cloud }} в полном объеме потребуется перейти на платную версию.

Остаток стартового гранта и количество дней до завершения пробного периода отображаются в платежном аккаунте на странице [Биллинг]({{ link-console-billing }}). Также во время пробного периода вы можете [посмотреть статистику](../operations/check-charges.md) о потреблении ресурсов и их стоимости в соответствии с тарифами.

{% note warning %}

Включение пробного периода происходит только на этапе [создания первого платежного аккаунта](../quickstart/index.md) в консоли управления {{ yandex-cloud }}.

{% endnote %}
