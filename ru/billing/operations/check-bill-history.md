# Посмотреть историю платежей

Всю историю зачислений на [лицевой счет](../concepts/personal-account.md#balance) вы можете отслеживать на странице **История платежей**{% if product == "yandex-cloud" %} либо в сервисе Яндекс Баланс. Способ отслеживания зависит от вашего юридического статуса ([типа платежного аккаунта](../concepts/billing-account.md#ba-types)){% endif %}.

{% if product == "yandex-cloud" %}

{% list tabs %}
        
- Организациям и ИП
  
  Организации и ИП могут отслеживать всю историю зачислений в сервисе Яндекс Баланс на странице [Счета](https://balance.yandex.ru/invoices.xml).
  
- Физическим лицам
  
  Физические лица могут посмотреть всю историю зачислений на странице **История платежей**.
  <br/>Все платежи расположены в хронологическом порядке (самый старый платеж находится в самом низу).
  <br/>Чтобы отфильтровать платежи, задайте необходимый период. Результаты отобразятся в таблице.
  
  {% if region == "ru" %}
  
  Название  | Описание

  Дата платежа | Дата зачисления средств на баланс лицевого счета.
  Статус | Статус платежа.
  Сумма платежа | Фактически зачисленные средства на баланс лицевого счета, в рублях.
