Привязка банковской карты необходима только для создания платежного аккаунта. Средства с привязанной карты могут быть списаны, только если [активна платная версия](../operations/activate-commercial.md) и вы пользовались сервисами {{ yandex-cloud }}. В любой момент после создания платежного аккаунта вы можете [изменить способ оплаты](../operations/change-payment-method.md).
