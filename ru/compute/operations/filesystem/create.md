# Создать файловое хранилище

Чтобы создать [файловое хранилище](../../concepts/filesystem.md):

{% list tabs %}

- Консоль управления

  1. В консоли управления выберите каталог, в котором нужно создать файловое хранилище.
  1. Выберите сервис **{{ compute-name }}**.
  1. Перейдите на вкладку **Файловые хранилища**.
  1. Нажмите кнопку **Создать файловое хранилище**.
  1. Введите имя хранилища.
    
     {% include [name-format](../../../_includes/name-format.md) %}
    
  1. Если требуется, добавьте произвольное описание хранилища.
  1. Выберите [зону доступности](../../../overview/concepts/geo-scope.md) хранилища. Его можно будет подключить только к виртуальным машинам из той же зоны доступности.
  1. Выберите [тип хранилища](../../concepts/filesystem.md#types).
  
     {% note warning %}
     
     Зону доступности и тип файлового хранилища нельзя будет изменить после его создания.
     
     {% endnote %}
     
  1. Укажите нужный размер хранилища.
  1. Нажмите кнопку **Создать файловое хранилище**.
  
- API

  Используйте метод [FilesystemService/Create](../../api-ref/grpc/filesystem_service.md#Create) gRPC API или метод [create](../../api-ref/Filesystem/create.md) ресурса Filesystem REST API.

{% endlist %}
