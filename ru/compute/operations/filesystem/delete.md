# Удалить файловое хранилище

1. [Отключите файловое хранилище](detach-from-vm.md) от всех ВМ, к которым оно подключено.
1. Удалите файловое хранилище в {{ compute-name }}:

   {% list tabs %}

   - Консоль управления

     1. В консоли управления выберите каталог, в котором находится файловое хранилище.
     1. Выберите сервис **{{ compute-name }}**.
     1. Перейдите на вкладку **Файловые хранилища**.
     1. В строке нужного файлового хранилище нажмите ![image](../../../_assets/dots.svg) и выберите пункт **Удалить**.
     1. В открывшемся окне подтвердите удаление.
     
   - API
      
     Используйте метод [FilesystemService/Delete](../../api-ref/grpc/filesystem_service.md#Delete) gRPC API или метод [delete](../../api-ref/Filesystem/delete.md) ресурса Filesystem REST API.
  
   {% endlist %}
