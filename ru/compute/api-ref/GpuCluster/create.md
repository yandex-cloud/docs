# Compute Cloud API, REST: GpuCluster.create
Creates a GPU cluster in the specified folder.
 

 
## HTTP request {#https-request}
```
POST https://compute.{{ api-host }}/compute/v1/gpuClusters
```
 
## Body parameters {#body_params}
 
```json 
{
  "folderId": "string",
  "name": "string",
  "description": "string",
  "labels": "object",
  "zoneId": "string",
  "interconnectType": "string"
}
```

 
Field | Description
--- | ---
folderId | **string**<br><p>ID of the folder to create a GPU cluster in.</p> <p>To get a folder ID, make a <a href="/docs/resource-manager/api-ref/Folder/list">list</a> request.</p> 
name | **string**<br><p>Name of the GPU cluster.</p> <p>The name must be unique within the folder.</p> 
description | **string**<br><p>Description of the GPU cluster.</p> 
labels | **object**<br><p>GPU cluster labels as ``key:value`` pairs.</p> 
zoneId | **string**<br><p>ID of the availability zone where the GPU cluster resides. To get a list of available zones use the <a href="/docs/compute/api-ref/Zone/list">list</a> request.</p> 
interconnectType | **string**<br><p>Type of interconnect to use for this GPU cluster.</p> <ul> <li>INFINIBAND: InfiniBand interconnect.</li> </ul>
