# Создать аккаунт

Чтобы создать аккаунт клиента:

{% list tabs group=instructions %}

- Партнерский портал {#partner}

  1. Перейдите на [партнерский портал]({{ link-cloud-partners }}).
  1. На панели слева выберите ![icon](../../_assets/console-icons/persons.svg) **Аккаунты**.
  1. В правом верхнем углу нажмите ![image](../../_assets/console-icons/plus.svg) **Создать аккаунт**.
  1. В открывшемся окне:
     1. В списке **Страна** выберите страну, в которой зарегистрирована компания.
     1. В поле **ИНН/БИН/TIN** введите идентификационный номер компании.

        {% note info %}

        Если компания зарегистрирована в Российской Федерации, выберите название компании из списка. Для компаний из других стран заполните поля **Наименование компании**, **Юридический адрес** и **Почтовый адрес**.

        {% endnote %}

     1. (опционально) В поле **Описание** введите дополнительные сведения о клиенте. Нажмите ![icon](../../_assets/console-icons/ellipsis.svg), чтобы увидеть доступные инструменты форматирования.
  1. Нажмите **Создать**. На странице **Аккаунты** появится аккаунт клиента.

{% endlist %}
