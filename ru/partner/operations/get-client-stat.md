# Посмотреть детализацию сабаккаунтов

Вы можете [посмотреть](#get-client-stat) детальную информацию о том, как клиенты используют сервисы {{ yandex-cloud }} в виде [диаграмм](#graph).


## Посмотреть детализацию на партнерском портале {#get-client-stat}

1. Перейдите на [партнерский портал]({{ link-cloud-partners }}).
1. На панели слева выберите ![image](../../_assets/console-icons/scales-balanced.svg) **Финансы**.
1. Перейдите на вкладку ![image](../../_assets/console-icons/chart-column.svg) **Детализация сабаккаунтов**. Посмотрите детализацию:

    {% note info %}

    По умолчанию в детализации отображается стоимость потребленных ресурсов. Детализация в единицах потребления доступна на вкладке **По продуктам**.

    {% endnote %}

    {% list tabs group=details %}

    - По облакам {#clouds}

      Детализацию можно посмотреть по одному или нескольким облакам. По умолчанию детализация отображается по всем привязанным облакам.

      1. В списке **Облака** выберите облако.
      1. Настройте фильтрацию данных:
         * **Группировка** — по дням, неделям или месяцам.
         * **Продукты** — по определенным продуктам.
         * **Метки** — по определенным меткам.
         * **Сабаккаунты** — по определенным сабаккаунтам.

      {% note info %}

      Если облако ранее было привязано к другому платежному аккаунту, при просмотре детализации учитывайте период использования сервисов: до момента переноса детализация доступна для предыдущего платежного аккаунта, после момента переноса — для того, к которому теперь привязано облако.

      {% endnote %}

    - По каталогам {#folders}

      Детализацию можно посмотреть по одному или нескольким каталогам. По умолчанию детализация отображается для всех каталогов выбранного облака.

      1. В списке **Облака** выберите облако.
      1. В списке **Каталоги** выберите каталог.
      1. Настройте фильтрацию данных:
         * **Группировка** — по дням, неделям или месяцам.
         * **Продукты** — по определенным продуктам.
         * **Метки** — по определенным меткам.
         * **Сабаккаунты** — по определенным сабаккаунтам.

    - По сервисам {#services}

      Детализацию можно посмотреть по одному или нескольким сервисам в разных облаках. Также можно посмотреть детализацию по одному или нескольким сервисам в разных каталогах выбранного облака. По умолчанию детализация отображается для всех сервисов всех облаков и каталогов.

      1. В списке **Облака** выберите облако.
      1. В списке **Каталоги** выберите каталог. 
      1. В списке **Сервисы** выберите сервис.
      1. Настройте фильтрацию данных:
         * **Группировка** — по дням, неделям или месяцам.
         * **Метки** — по определенным меткам.
         * **Сабаккаунты** — по определенным сабаккаунтам.

    - По продуктам {#products}

      Детализацию можно посмотреть по одному или нескольким продуктам в разных облаках. Также можно посмотреть детализацию по одному или нескольким продуктам в разных каталогах выбранного облака. По умолчанию детализация отображается для всех продуктов во всех облаках.

      1. В списке **Облака** выберите облако.
      1. В списке **Каталоги** выберите каталог.
      1. В списке **Продукты** выберите продукт.
      1. Настройте фильтрацию данных:
         * **Группировка** — по дням, неделям или месяцам.
         * **Метки** — по определенным меткам.
         * **Сабаккаунты** — по определенным сабаккаунтам.

    - По меткам {#labels}

      Детализацию можно посмотреть по [меткам ресурсов сервисов](../../resource-manager/concepts/labels.md). Вы можете [добавить метку](../../resource-manager/operations/manage-labels.md) на любой ресурс. При наличии детализации по выбранной метке информация отобразится на графике, а под графиком в таблице будут указаны все значения, соответствующие выбранной метке.

      1. В списке **Облака** выберите облако.
      1. В списке **Каталоги** выберите каталог.
      1. В списке **Ключ метки** выберите метку.
      1. Настройте фильтрацию данных:
         * **Группировка** — по дням, неделям или месяцам.
         * **Продукты** — по определенным продуктам.
         * **Метки** — по определенным меткам.
         * **Сабаккаунты** — по определенным сабаккаунтам.

    - По сабаккаунтам {#subacc}

      Детализацию можно посмотреть по одному или нескольким сабаккаунтам в разных облаках. По умолчанию детализация отображается для всех сабаккаунтов во всех облаках и каталогах.

      1. В списке **Облака** выберите облако.
      1. В списке **Каталоги** выберите каталог. 
      1. В списке **Сабаккаунты** выберите сабаккаунт.
      1. Настройте фильтрацию данных:
         * **Группировка** — по дням, неделям или месяцам.
         * **Продукты** — по определенным продуктам.
         * **Метки** — по определенным меткам.

    {% endlist %}

    По умолчанию загружаются данные за текущий месяц. Чтобы сменить период детализации, нажмите ![image](../../_assets/console-icons/calendar.svg), укажите даты в полях **From** и **To** или выберите временной интервал из списка. Дата определяется как диапазон с 0:00 до 23:59 по московскому времени (UTC +3). Данные могут обновляться с небольшой задержкой (до нескольких часов).


## Расшифровать диаграмму {#graph}

На диаграмме отображается изменение показателей в динамике. Количество и значение показателей зависят от выбранной глубины детализации, вида диаграммы, а также от фильтров:

  * **Потребление** — стоимость потребленных услуг без учета скидки.
  * **К оплате** — стоимость потребленных услуг с учетом скидки.

Столбцы под диаграммой показывают:

  * **Стоимость потребления** — стоимость потребленных услуг сабаккаунтами без учета скидки.
  * **Скидка** — сумму клиентской скидки за потребление услуг сабаккаунтами.
  * **К оплате клиентом** — стоимость потребленных услуг сабаккаунтами с учетом скидки.
  * **Премия** — сумму партнерской премии за потребление услуг сабаккаутами с учетом клиентской скидки (без учета рибейта).
  * **Корректировка** — сумму корректировки по итоговой стоимости потребления услуг сабаккаунтами с учетом клиентской скидки и партнерской премии.
  * **К оплате партнером** — стоимость потребления услуг сабаккаунтами с учетом клиентской скидки, премии партнера и корректировки.
