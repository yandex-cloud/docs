# Посмотреть потребление клиентов

Вы можете посмотреть детальную информацию о том, как клиенты используют сервисы: 
* через консоль управления;
* в партнерском портале.

{% list tabs %}

- Консоль управления

  Чтобы посмотреть сведения по использованию сервисов {{ yandex-cloud }} в виде графиков и таблиц: 
  1. В [консоли управления]({{ link-console-billing }}) нажмите значок ![image](../../_assets/ugly-sandwich.svg) и перейдите в раздел **Биллинг**.
  2. Выберите аккаунт на странице **Список аккаунтов**.
  3. Перейдите на страницу **Детализация**.

  Подробнее о настройках страницы **Детализация** см. в [документации сервиса {{ billing-name }}](../../billing/operations/check-charges.md).

- Партнерский портал

  Войдите в [партнерский портал](https://partners.cloud.yandex.ru/) с Яндекс ID, к которому привязан аккаунт партнера в {{ yandex-cloud }}. Вам будут доступны несколько способов проверить потребление клиентов:

  * Вкладка **Заказчики**
    1. Перейдите на вкладку **Заказчики** партнерского портала. 
    1. Выберите нужного клиента.
    1. Перейдите на вкладку **Потребление услуг**.

  * Вкладка **Вознаграждения**

    Перейдите на вкладку **Вознаграждения** партнерского портала. На ней вы увидите общую сумму потребления всеми клиентами.

    Детальная статистика по клиентам доступна в виде файла `.csv`. Выберите период, за который будет построен отчет, и нажмите **Скачать файл**. Период не должен превышать 92 дней.
