# Удалить ноутбук или проект

## Удалить ноутбук {#delete-notebook}

1. {% include [find project](../../../_includes/datasphere/ui-before-begin.md) %}
1. Нажмите правой кнопкой мыши на имя ноутбука, который вы хотите удалить.
1. Выберите ![image](../../../_assets/cross.svg) **Delete** и подтвердите удаление.

## Удалить проект {#delete-project}

{% note alert %}

После удаления проекта весь код ноутбука и состояние интерпретатора будут потеряны.

{% endnote %}

1. {% include [find project](../../../_includes/datasphere/ui-find-project.md) %}
1. В правом верхнем углу страницы проекта нажмите **Еще** и выберите **Удалить**.
1. Подтвердите удаление проекта.
