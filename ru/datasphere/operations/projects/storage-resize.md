# Изменение размера хранилища проекта

Чтобы изменить размер хранилища проекта:

1. {% include [include](../../../_includes/datasphere/first-step.md) %}
1. Перейдите во вкладку **Проекты**.
1. Выберите проект, который хотите изменить, и нажмите значок ![image](../../../_assets/datalens/horizontal-ellipsis.svg).
1. Выберите **Открыть**.
1. В левом верхнем углу нажмите **File** → **Change project disk size**. Во всплывающем окне вы увидите текущий, минимальный и максимальный размеры хранилища.
1. Укажите желаемый размер — целое число.
1. Нажмите **Resize**.
1. После этого проект перезагрузится, и вы автоматически перейдете на вкладку **Проекты**. Проект снова станет доступен после завершения изменения размера хранилища.

{% note warning %}

Хранилище проекта используется не только для хранения файлов, но и для нужд файловой системы, данных версионирования и другого технического обеспечения.

{% endnote %}
