# Удаление ноды 

{% include [pricing nodes](../../_includes/datasphere/nodes-pricing-warn.md) %}

Если развернутый сервис вам больше не нужен, удалите ноду. 

1. {% include [first-step](../../_includes/datasphere/first-step.md) %}
1. Откройте проект, содержащий ноду, и дождитесь окончания загрузки.
1. Перейдите на вкладку ![Node](../../_assets/datasphere/node.svg) **Nodes** и выберите ноду, которую вы хотите удалить.
1. В правом нижнем углу нажмите кнопку **Delete**.
