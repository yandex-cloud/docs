# Очистка состояния интерпретатора

{% include [link-on-new-ui-datasphere-in-old-ui](../../../_includes/datasphere/datasphere-old-note.md) %}

{% note alert %}

Очистка состояния интерпретатора сбросит все результаты и состояние переменных.

{% endnote %}

Чтобы сбросить состояние интерпретатора:

1. На верхней панели в окне проекта нажмите кнопку **Kernel**.
1. В открывшемся меню нажмите кнопку **Reset kernel state (variables)**.
