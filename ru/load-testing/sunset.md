# Сервис {{ load-testing-full-name }} закрыт

{% note warning %}

Сервис {{ load-testing-full-name }} прекратил работу 1 июля 2026 года. 

{% endnote %}

## Что будет с вашими данными {#data}

Резервные копии конфигураций тестов и результатов нагрузочных испытаний можно запросить через [техническую поддержку](https://center.yandex.cloud/support/).

Если в ваших облаках существуют виртуальные машины, использовавшиеся как агенты нагрузочного тестирования, они продолжают работу после закрытия сервиса {{ load-testing-name }}. Удаление или изменение бывших агентов нагрузочного тестирования останется в вашей зоне ответственности.

## Миграция {#migration}

В качестве альтернативы перенесите свои сценарии нагрузочного тестирования на [k6](https://k6.io/) — инструмент нагрузочного тестирования с открытым исходным кодом.

Мы подготовили руководство, которое поможет адаптировать существующие тесты: [Миграция из {{ load-testing-name }} в k6](tutorials/loadtesting-migration-to-k6.md).
