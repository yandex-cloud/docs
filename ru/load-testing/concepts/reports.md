# Отчеты

Отчет о результатах нагрузочного тестирования нужен содержит сведения о проблемах, возникших в процессе тестирования. {{ load-testing-name }} предоставляет централизованное хранилище тестов, привязанное к ролевой модели {{ yandex-cloud }}. Во время проведения нагрузочного тестирования генератор нагрузки собирает статистику, которую агрегирует и передает в хранилище Яндекс.Танк. 

Если вам не нужно сохранять отчеты, вы можете отключить эту функцию при конфигурации теста.
