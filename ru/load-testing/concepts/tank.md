# Яндекс.Танк

На агент устанавливается Яндекс.Танк — инструмент для управления нагрузочным тестированием. Он позволяет использовать разные [генераторы нагрузки](load-generator.md) (сейчас доступны Phantom и Pandora), управлять ходом теста и тонко его конфигурировать. Код Яндекс.Танка открыт — в случае необходимости можно провести аудит и проверить работу кода.
