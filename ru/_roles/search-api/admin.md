Роль `search-api.admin` позволяет управлять зарегистрированными IP-адресами, а также выполнять поисковые запросы в сервисе Yandex Search API с использованием API v1 и API v2.

Пользователи с этой ролью могут:
* просматривать информацию о [зарегистрированных IP-адресах]({{ link-docs-ai }}search-api/operations/workaround#registration), изменять и удалять их, а также регистрировать новые IP-адреса;
* выполнять поисковые запросы с использованием [API v1]({{ link-docs-ai }}search-api/operations/searching) и [API v2]({{ link-docs-ai }}search-api/operations/web-search);
* просматривать информацию о [квотах]({{ link-docs-ai }}search-api/concepts/limits#search-api-quotas) сервиса Yandex Search API;
* просматривать информацию об [облаках]({{ link-docs }}/resource-manager/concepts/resources-hierarchy#cloud) и [каталогах]({{ link-docs }}/resource-manager/concepts/resources-hierarchy#folder).

Включает разрешения, предоставляемые ролью `search-api.editor`.
