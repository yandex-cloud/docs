Роль `datalens.sharedEntries.limitedEntryBindingCreator` назначается на общий объект и позволяет переиспользовать его в воркбуках без делегации прав доступа. В интерфейсе {{ datalens-name }} эта роль называется `Привязки без делегаций`. Рекомендуем выдавать эту роль только через интерфейс {{ datalens-name }}.
