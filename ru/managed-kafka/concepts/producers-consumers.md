# Производители и потребители

{{ KF }} — это распределенная система для обмена информацией между производителями и потребителями данных.

* **Производитель данных** — приложение-источник данных. Оно подключается к кластеру {{ KF }} и записывает сообщения с данными в определенные [топики](topics.md) и их разделы. Чтобы иметь возможность записывать собщения в топик и раздел, у приложения должна быть [учетная запись](../operations/cluster-accounts.md) с нужным набором прав.
* **Потребитель данных** — приложение, которое принимает и обрабатывает данные от производителей. Вместо того, чтобы получать данные напрямую от производителя, оно подключается к кластеру {{ KF }} и подписывается на определенные [топики](topics.md). Чтобы иметь возможность подписаться на топик, у приложения должна быть [учетная запись](../operations/cluster-accounts.md) с нужным набором прав.


Подробнее см. в документации {{ KF }} для [производителя](https://kafka.apache.org/documentation/#theproducer) и [потребителя](https://kafka.apache.org/documentation/#theconsumer).
