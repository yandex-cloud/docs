# Информация об имеющихся кластерах

Вы можете запросить детальную информацию о каждом созданном вами кластере {{ mkf-name }}.

## Получить список кластеров в каталоге {#list-clusters}

{% list tabs %}

- API
  
  Чтобы получить список кластеров в каталоге, воспользуйтесь методом API [list](../api-ref/Cluster/list.md): передав значение идентификатора требуемого каталога в параметре `folder_id` запроса.

{% endlist %}

## Получить детальную информацию о кластере {#get-cluster}

{% list tabs %}

- API
  
  Чтобы получить детальную информацию о кластере, воспользуйтесь методом API [get](../api-ref/Cluster/get.md), передав значение идентификатора требуемого кластера в параметре `cluster_id` запроса.

  Чтобы узнать идентификатор кластера, [получите список кластеров в каталоге](#list-clusters).

{% endlist %}
