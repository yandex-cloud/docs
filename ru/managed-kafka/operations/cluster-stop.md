# Остановка и запуск кластера

При необходимости вы можете остановить кластер {{ KF }} и запустить его заново. Время работы остановленного кластера не тарифицируется: вы продолжаете платить только за объем хранилища в соответствии с [тарифом](../pricing.md#prices-storage).

{% include [pricing-status-warning.md](../../_includes/mdb/pricing-status-warning.md) %}

## Остановить кластер {#stop-cluster}

Вы можете остановить работу кластера, если вы не планируете использовать {{ KF }} какое-то время.

{% list tabs %}

- API
  
  Чтобы остановить работу всех хостов кластера:
  1. [Получите идентификатор](cluster-list.md#get-cluster) требуемого кластера.
  1. Воспользуйтесь методом API [stop](../api-ref/Cluster/stop.md): передайте значение этого идентификатора в параметре `cluster_id` запроса.

{% endlist %}

## Запустить кластер {#start-cluster}

Кластер в статусе `STOPPED` можно запустить заново.

{% list tabs %}

- API
  
  Чтобы запустить кластер:
  1. [Получите идентификатор](cluster-list.md#get-cluster) требуемого кластера.
  1. Воспользуйтесь методом API [start](../api-ref/Cluster/start.md): передайте значение этого идентификатора в параметре `cluster_id` запроса.

{% endlist %}
