# Изменение настроек кластера

После создания кластера {{ KF }} вы можете:
- [Изменить конфигурацию кластера](#update-cluster).
- [Переместить кластер](#move-cluster) из текущего каталога в другой каталог.

## Изменить конфигурацию {#update-cluster}

{% list tabs %}

- API
  
  Воспользуйтесь методом API [update](../api-ref/Cluster/update.md) и передайте в запросе:
  - Идентификатор кластера в параметре `cluster_id`. Чтобы узнать идентификатор, [получите список кластеров в каталоге](cluster-list.md#list-clusters).
  - Новую конфигурацию кластера в параметре `config_spec`. Чтобы изменить только часть конфигурации кластера, задайте значение параметра `update_mask`.

{% endlist %}

## Переместить кластер {#move-cluster}

{% list tabs %}

- API
  
  Чтобы переместить кластер из текущего каталога в другой, воспользуйтесь методом API [move](../api-ref/Cluster/move.md) и передайте в запросе:
  - Идентификатор кластера в параметре `cluster_id`. Чтобы узнать идентификатор, [получите список кластеров в каталоге](cluster-list.md#list-clusters).
  - Идентификатор каталога назначения в параметре `destination_folder_id`.

{% endlist %}
