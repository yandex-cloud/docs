# Создание кластера

Кластер Kafka — это один или несколько хостов-брокеров, на которых размещены топики и соответствующие топикам разделы. Производители и потребители могут работать с этими топиками, подключившись к кластеру.

{% list tabs %}

- API
  
  Чтобы создать кластер, воспользуйтесь методом API [create](../api-ref/Cluster/create.md) и передайте в параметрах запроса:
  - Идентификатор каталога, в котором должен быть размещен кластер (параметр `folder_id`).
  - Имя кластера (параметр `name`).
