# cli

Yandex Object Storage control tool

## Options

```
  -c, --config-dir string   path to configuration directory
      --debug               enable debug mode
  -h, --help                help for cli
      --insecure            use if console has self-signed certificate
  -p, --profile string      configuration profile
```
