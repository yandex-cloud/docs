# Метод start
Запускает указанный кластер Redis.
 

 
## HTTP-запрос {#https-request}
```
POST https://mdb.api.cloud.yandex.net/managed-redis/v1/clusters/{clusterId}:start
```
 
## Path-параметры {#path_params}
 
Параметр | Описание
--- | ---
clusterId | Обязательное поле. Идентификатор кластера Redis, который следует запустить.  Максимальная длина строки в символах — 50.
