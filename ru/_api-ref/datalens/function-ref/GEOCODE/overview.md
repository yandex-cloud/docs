# GEOCODE

_Географические функции_

#### Синтаксис


```
GEOCODE( address )
```

#### Описание
Преобразует адрес `address` в координаты.

**Типы аргументов:**
- `address` — `Строка`


**Возвращаемый тип**: `Геоточка`

#### Примеры

```
GEOCODE("посёлок Свободный Серп") = [53.650704, 37.224461]
```


#### Поддержка источников данных

`Материализованный датасет`, `ClickHouse 1.1`
