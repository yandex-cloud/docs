# LEN

_Строковые функции_

#### Синтаксис


```
LEN( string )
```

#### Описание
Возвращает количество символов в строке `string`.

**Типы аргументов:**
- `string` — `Строка`


**Возвращаемый тип**: `Целое число`

#### Примеры

```
LEN("Computer") = 9
```


#### Поддержка источников данных

`Материализованный датасет`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`.
