# ANY

_Агрегатные функции_

#### Синтаксис {#syntax}


```
ANY( value )
```

#### Описание {#description}
Возвращает произвольное `value` значение из группы. Результат может меняться от запроса к запросу

**Типы аргументов:**
- `value` — `Любой`


**Возвращаемый тип**: Совпадает с типом аргументов (`value`)

#### Примеры {#examples}

```
ANY([Profit])
```


#### Поддержка источников данных {#data-source-support}

`Материализованный датасет`, `ClickHouse 1.1`, `MySQL 5.7`.
