# Деление (/)

_Операторы_

#### Синтаксис


```
number_1 / number_2
```

#### Описание
Делит число `number_1` на число `number_2`.

**Типы аргументов:**
- `number_1` — `Число`
- `number_2` — `Число`


**Возвращаемый тип**: `Дробное число`

#### Примеры

```
4 / 2 = 2.0
```

```
5 / 2 = 2.5
```

```
5.0 / 2 = 2.5
```

```
5 / 2.0 = 2.5
```


#### Поддержка источников данных

`Материализованный датасет`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`.
