# ATAN



#### Синтаксис {#syntax}


```
ATAN( number )
```

#### Описание {#description}
Возвращает выраженный в радианах арктангенс числа `number`.

**Типы аргументов:**
- `number` — `Дробное число | Целое число`


**Возвращаемый тип**: `Дробное число`

#### Пример {#examples}



| **[n]**   | **ATAN([n])**   |
|:----------|:----------------|
| `-1.00`   | `-0.79`         |
| `-0.87`   | `-0.71`         |
| `-0.71`   | `-0.62`         |
| `-0.50`   | `-0.46`         |
| `0.00`    | `0.00`          |
| `0.50`    | `0.46`          |
| `0.71`    | `0.62`          |
| `0.87`    | `0.71`          |
| `1.00`    | `0.79`          |




#### Поддержка источников данных {#data-source-support}

`Материализованный датасет`, `ClickHouse 19.13`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `Oracle Database 12c (12.1)`, `PostgreSQL 9.3`, `YDB`.
