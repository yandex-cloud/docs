# GEOCODE

_Географические функции_

#### Синтаксис {#syntax}


```
GEOCODE( address )
```

#### Описание {#description}
Преобразует адрес `address` в координаты.

Результаты выполнения функции кэшируются, поэтому при повторном запросе {{ datalens-short-name }} не выполнит геокодирование. Время хранения кэша — 30 дней.
После 30 дней {{ datalens-short-name }} автоматически вызовет функцию `GEOCODE` для преобразования адреса.

**Типы аргументов:**
- `address` — `Строка`


**Возвращаемый тип**: `Геоточка`

{% note warning %}

- Квота на геокодирование — не более 25 000 уникальных записей в сутки. Не вошедшие в ограничение записи будут обработаны на следующий день.
- Функция доступна только в тарифе Стандарт.

{% endnote %}


#### Примеры {#examples}

```
GEOCODE("посёлок Свободный Серп") = [53.650704, 37.224461]
```


#### Поддержка источников данных {#data-source-support}

`Материализованный датасет`.
