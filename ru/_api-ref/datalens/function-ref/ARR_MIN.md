# ARR_MIN



#### Синтаксис {#syntax}


```
ARR_MIN( array )
```

#### Описание {#description}
Возвращает значение минимального элемента в массиве `array`.

**Типы аргументов:**
- `array` — `Массив дробных чисел | Массив целых числел`


**Возвращаемый тип**: Зависит от типов аргументов

{% note info %}

Функция не работает для массивов с `Nullable` элементами. Чтобы удалить из массива элементы, равные `NULL`, используйте [REPLACE](REPLACE_ARRAY.md).

{% endnote %}


#### Пример {#examples}



| **[int_array]**   | **[float_array]**   | **ARR_MIN([int_array])**   | **ARR_MIN([float_array])**   |
|:------------------|:--------------------|:---------------------------|:-----------------------------|
| `'[21,12,0]'`     | `'[14.3,0.42,15]'`  | `0`                        | `0.42`                       |
| `'[-4,12,0]'`     | `'[0,-3,12]'`       | `-4`                       | `-3.00`                      |
| `'[5,7,9]'`       | `'[3.2,2.3,3.2]'`   | `5`                        | `2.30`                       |




#### Поддержка источников данных {#data-source-support}

`ClickHouse 21.8`.
