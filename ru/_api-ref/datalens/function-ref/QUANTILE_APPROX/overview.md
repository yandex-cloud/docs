# QUANTILE_APPROX

_Агрегатные функции_

#### Синтаксис


```
QUANTILE_APPROX( value, quant )
```

#### Описание
Возвращает примерный квантиль уровня `quant` (значение от 0 до 1).

**Типы аргументов:**
- `value` — `Число`
- `quant` — `Число`


**Возвращаемый тип**: Совпадает с типом аргументов (`value`, `quant`)

{% note info %}

Тип аргументов (`value`, `quant`) должен совпадать.

{% endnote %}


#### Поддержка источников данных

`Материализованный датасет`, `ClickHouse 1.1`.
