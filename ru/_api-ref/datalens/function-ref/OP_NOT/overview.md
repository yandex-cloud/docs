# NOT

_Операторы_

#### Синтаксис


```
NOT value
```

#### Описание
Инвертирует логическое значение.

**Типы аргументов:**
- `value` — `Логический | Дата | Дата и время | Геоточка | Геополигон | Число | Строка`


**Возвращаемый тип**: `Логический`

#### Примеры

```
NOT FALSE = TRUE
```

```
NOT TRUE = FALSE
```

```
NOT "" = TRUE
```

```
NOT "text" = FALSE
```

```
NOT 0 = TRUE
```

```
NOT 1 = FALSE
```

```
NOT #2019-01-01# = FALSE
```

```
NOT #2019-01-01 03:00:00# = FALSE
```


#### Поддержка источников данных

`Материализованный датасет`, `ClickHouse 1.1`, `Yandex.Metrica`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`.
