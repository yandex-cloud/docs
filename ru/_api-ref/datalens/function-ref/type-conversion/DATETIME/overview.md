## DATETIME

_Функции преобразования типов_

#### Синтаксис


```
DATETIME( value )
```

#### Описание
Переводит  [выражение] в формат даты и времени. При переводе `Date` в `DateTime` время определяется, как `00:00:00`.
Дата должна быть определена в формате `YYYY-MM-DDThh:mm:ss` или `YYYY-MM-DD hh:mm:ss`.

**Типы аргументов:**
- `value` - `Дата | Дата со временем | Число | Строка`


**Возвращаемый тип**: `Дата со временем`

#### Примеры

```
DATETIME("2019-01-23 15:07:47") = #2019-01-23 15:07:47#
```


#### Поддержка источников данных

`Материализованный датасет`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`
