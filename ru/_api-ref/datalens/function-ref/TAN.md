# TAN



#### Синтаксис {#syntax}


```
TAN( number )
```

#### Описание {#description}
Возвращает тангенс числа `number`, заданного в радианах.

**Типы аргументов:**
- `number` — `Дробное число | Целое число`


**Возвращаемый тип**: `Дробное число`

#### Примеры {#examples}

```
TAN(0) = 0.0
```

```
TAN(PI()/4) = 1.0
```
