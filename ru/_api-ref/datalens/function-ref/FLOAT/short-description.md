**Синтаксис:**`FLOAT( expression )`

Переводит выражение `expression` в формат дробного числа. по следующим правилам:

| Тип                                       | Значение                                                                                                                                                                                                                                       |
|:------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `Число`                                   | Исходное значение                                                                                                                                                                                                                              |
| <code>Дата &#124; Дата со временем</code> | [Unix-время](https://ru.wikipedia.org/wiki/Unix-время) соответствующее дате и времени. Если значение содержит в себе информацию о временной зоне, то она учитывается при вычислении. Если же временная зона неизвестна, то время считается UTC |
| `Строка`                                  | Число из строки в десятичной записи                                                                                                                                                                                                            |
| `Логический`                              | `TRUE` — 1.0, `FALSE` — 0.0                                                                                                                                                                                                                    |
