# SPACE

_Строковые функции_

#### Синтаксис


```
SPACE( value )
```

#### Описание
Возвращает строку с указанным количеством пробелов.

**Типы аргументов:**
- `value` — `Целое число`


**Возвращаемый тип**: `Строка`

#### Примеры

```
SPACE(5) = "     "
```


#### Поддержка источников данных

`Материализованный датасет`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`.
