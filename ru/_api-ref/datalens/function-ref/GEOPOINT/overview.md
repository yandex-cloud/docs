# GEOPOINT

_Функции преобразования типов_

#### Синтаксис


```
GEOPOINT( latitude [ , longitude ] )
```

#### Описание
Формирует значение типа геоточка. Принимает на вход строку, либо значение типа "геоточка", либо координаты — широту `latitude` и долготу `longitude`. Если на вход подается одна строка, в ней должен содержаться список из двух чисел, координат (широты и долготы) в json-синтаксисе.

**Типы аргументов:**
- `latitude` — `Геоточка | Число | Строка`
- `longitude` — `Число | Строка`


**Возвращаемый тип**: `Геоточка`

#### Примеры

```
GEOPOINT("[55.75222,37.61556]") = "[55.75222,37.61556]"
```

```
GEOPOINT(55.75222, 37.61556) = "[55.75222,37.61556]"
```


#### Поддержка источников данных

`Материализованный датасет`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`
