# COUNT_IF

_Функция `COUNT_IF` также доступна как [оконная](COUNT_IF_WINDOW.md)._

#### Синтаксис {#syntax}


```
COUNT_IF( condition )
```

#### Описание {#description}
Возвращает количество элементов в группе, которые удовлетворяют условию `condition`.

**Типы аргументов:**
- `condition` — `Логический`


**Возвращаемый тип**: `Целое число`

#### Примеры {#examples}

```
COUNT_IF([Profit] > 5)
```


#### Поддержка источников данных {#data-source-support}

`Материализованный датасет`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `Oracle Database 12c (12.1)`, `PostgreSQL 9.3`.
