# SymmetricCrypto
Набор методов, выполняющих симметричное шифрование и дешифрование.

## Методы {#methods}
Метод | Описание
--- | ---
[decrypt](decrypt.md) | Расшифровывает указанный шифртекст с помощью указанного ключа.
[encrypt](encrypt.md) | Шифрует заданный текст с помощью указанного ключа.
[generateDataKey](generateDataKey.md) | Создает новый симметричный ключ шифрования данных и возвращает этот ключ данных в виде текста, зашифрованного с помощью указанного ключа KMS.
[reEncrypt](reEncrypt.md) | Перешифрует ранее зашифрованный текст с помощью указанного ключа.
