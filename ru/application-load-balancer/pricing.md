# Правила тарификации для {{ alb-full-name }}



## Из чего складывается стоимость использования {{ alb-name }} {#rules}

При работе с сервисом {{ alb-name }} вы платите за фактическое использование вычислительных ресурсов каждого активного балансировщика. Использование сервиса тарифицируется почасно.

### Пример расчета стоимости {#example}

Балансировщик, размещенный в одной зоне доступности, работал в течение часа со следующими показателями:

{% include [lcu-example](../_includes/application-load-balancer/lcu-example.md) %}

Расчет количества ресурсных единиц:
>Максимум (6000 / 1000, 30 000 / 4000, 500 / 200, 20 / 22, 2) = максимум (6, 8, 3, 1, 2) = 8
>
>Итого: 8 ресурсных единиц.

Где:
* 6000 / 1000 = 6 — количество ресурсных единиц, которые содержат 6000 RPS.
* 30 000 / 4000 = 7,5 ~ 8 — количество ресурсных единиц, которые содержат 30 000 активных соединений.
* 500 / 200 = 2,5 ~ 3 — количество ресурсных единиц, которые содержат 500 новых соединений.
* 20 / 22 = 0,9090... ~ 1 — количество ресурсных единиц, которые содержат 20 МБ трафика в секунду.
* 2 — минимальное количество ресурсных единиц в зоне.

Расчет стоимости за час: 
>8 × 2,22 ₽ = 17,76 ₽
>
>
>
>Итого: 17,76 ₽ — стоимость часа использования балансировщика.

Где:
* 8 — количество ресурсных единиц.
* 2,22 ₽ — цена ресурсной единицы.

Расчет стоимости за месяц (30 дней) с той же почасовой нагрузкой:
>17,76 ₽ × 720 = 12 787,20 ₽
>
>
>
>Итого: 12 787,20 ₽ — стоимость использования балансировщика за месяц.

Где:
* 17,76 ₽ — цена ресурсной единицы.
* 720 — количество часов в месяц.
