# yc managed-opensearch backup

Backups of a managed OpenSearch cluster.

#### Command Usage

Syntax:

`yc managed-opensearch backup <command>`

Aliases:

- `backups`

#### Command Tree

- [yc managed-opensearch backup get](get.md) — Get information about an OpenSearch cluster backup.

- [yc managed-opensearch backup list](list.md) — List available OpenSearch backups in a folder.
