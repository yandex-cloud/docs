# Управление пользователями {{ OS }}

Вы можете управлять пользователями кластера с помощью специального пользователя `admin`. Задать пароль этого пользователя можно при [создании](cluster-create.md#create-cluster) или [изменении](update.md) кластера.

О том, как управлять пользователями, и доступных для этого интерфейсах см. в [документации {{ OS }}]({{ os.docs }}/security-plugin/access-control/users-roles).

Для выполнения повседневных задач создайте отдельных пользователей с минимально необходимым [набором прав доступа]({{ os.docs }}/security-plugin/access-control/permissions/). Не рекомендуется применять для этих целей учетную запись `admin`, поскольку она имеет полный доступ к кластеру. Отправленная от ее имени ошибочная команда может привести к неработоспособности кластера.
