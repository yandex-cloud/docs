> (400 - 15) × {% calc [currency=RUB] 490 %} = {% calc 400 - 15 %} × {% calc [currency=RUB] 490 %} = {% calc [currency=RUB] (400 - 15) × 490 %}
> Итого: {% calc [currency=RUB] (400 - 15) × 490 %} — в месяц.

Где:

* 400 — новый лимит платных пользователей.
* 15 — нетарифицируемое количество пользователей.
* {% calc [currency=RUB] 490 %} — стоимость за пользователя в месяц для количества от 101 до 500 пользователей.

Текущий месяц уже оплачен, поэтому после уменьшения лимита не будет списания.

В начале каждого следующего месяца будет списываться {% calc [currency=RUB] (400 - 15) × 490 %}.
