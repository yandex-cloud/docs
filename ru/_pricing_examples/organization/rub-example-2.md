> (130 - 15) × {% calc [currency=RUB] 490 %} = {% calc 130 - 15 %} × {% calc [currency=RUB] 490 %} = {% calc [currency=RUB] (130 - 15) × 490 %}
> Итого: {% calc [currency=RUB] (130 - 15) × 490 %}.

Где:

* 130 — лимит платных пользователей.
* 15 — нетарифицируемое количество пользователей.
* {% calc [currency=RUB] 490 %} — стоимость за пользователя в месяц для количества от 101 до 500 пользователей.


Сразу после увеличения лимита будет списано {% calc [currency=RUB] (130 - 15) × 490 %}.

В начале каждого следующего месяца будет списываться так же {% calc [currency=RUB] (130 - 15) × 490 %}.
