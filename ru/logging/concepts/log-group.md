# Лог-группа в {{ cloud-logging-name }}

_Лог-группа_ — это способ группировки логов сервисов и пользовательских сообщений. Группы бывают двух видов:
* по умолчанию. Лог-группа по умолчанию создается для каждого каталога. В нее автоматически пишутся логи сервисов {{ sf-name }}, {{ api-gw-name }} и {{ serverless-containers-name }}. Вы можете самостоятельно писать в лог-группу по умолчанию.
* пользовательские. Вы можете самостоятельно создать лог-группу и писать в нее. [Как создать пользовательскую лог-группу.](../operations/create-group.md)
