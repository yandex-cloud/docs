# Журналирование

Сервис {{ sf-name }} автоматически захватывает потоки стандартного вывода приложения и отправляет их в централизованную [систему журналирования](../../operations/function/function-logs.md), доступную в {{ yandex-cloud }}. Помимо журналов выполнения приложения записываются системные записи о событиях выполнения запроса.

Для записи дополнительных сообщений используются стандартные языковые конструкции:
1. `print` — выводит сообщение в стандартный поток вывода `stdout`.
1. `logging` — выводит сообщение в установленном формате в выбранный поток вывода.
