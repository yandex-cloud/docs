# Журналирование выполнения функции на .NET Core

Сервис {{ sf-name }} автоматически захватывает потоки стандартного вывода приложения на .NET Core и отправляет их в централизованную систему журналирования, доступную в {{ yandex-cloud }}. Помимо журналов выполнения приложения записываются системные записи о событиях выполнения запроса.

Для записи дополнительных сообщений используются стандартные языковые конструкции:
1. `Console.Write` — выводит сообщение в стандартный поток вывода `stdout`.
