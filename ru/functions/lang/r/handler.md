# Обработчик запросов

_Обработчик запросов_ — это функция R, которая используется для обработки каждого вызова функции. При создании версии функции необходимо указать точку входа. Она состоит из имени файла и имени функции R, в которую будет приходить запрос (пример для обработчика в файле `function.R`, `function.handler`).

{% note info %}

В каждый момент времени один экземпляр функции обрабатывает только один запрос.

{% endnote %}

Обработчик может принимать и возвращать различные данные. Подробнее об этом, а так же о доступных способах задать обработчик запросов, можно ознакомиться в [соответствующем разделе](model).
