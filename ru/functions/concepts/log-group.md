# Лог-группа

_Лог-группа_ — это способ группировки сообщений из одного и того же источника. Таким источником может быть, например, [функция](function.md) или [реестр](../../iot-core/concepts/index.md#registry) IoT-core.

Узнать идентификатор лог-группы по ее источнику можно при помощи CLI. Чтобы узнать лог-группу функции, выполните команду:

```    
$ yc serverless function get <идентификатор функции> | grep log_group_id
```
Результат:
```
log_group_id: eol6cgr0************
```
