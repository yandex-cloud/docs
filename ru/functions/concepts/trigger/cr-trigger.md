# Триггер для {{ container-registry-name }}

[Триггер](index.md) для {{ container-registry-name }} запускает функцию, когда с [Docker-образом](../../../container-registry/concepts/docker-image.md) {{ container-registry-full-name }} происходят определенные события. Триггер должен находиться в одном облаке с [реестром](../../../container-registry/concepts/registry.md), на события которого он подписан.
