# Окружение

## Переменные окружения {#env}

Список переменных окружения, которые определены в среде выполнения {{ sf-name }} и доступны для функций.

Ключ | Значение
---- | ----
`_HANDLER` | Расположение обработчика, указанное для функции. 
`AWS_LAMBDA_RUNTIME_API` | Хост и порт API среды выполнения.
`LAMBDA_RUNTIME_DIR` | Путь до библиотек среды выполнения.
`LAMBDA_TASK_ROOT` | Путь до кода вашей функции.
`PATH` | Набор каталогов, в которых расположены исполняемые файлы.

Вы можете [добавить переменные окружения](../../operations/function/version-manage.md#version-env) в консоли управления.

## Пользовательские файлы {#files}

Хранение пользовательских файлов реализовано в двух директориях: 

- `/function/code` — рабочая директория пользователя. Содержит весь код вашей функции и все файлы, которые были загружены как ZIP-архив. Доступна по относительному пути. 
- `/tmp` — директория для хранения временных файлов, которые используются для оптимизации повторного вызова экземпляра функции. [Лимит](../limits.md#limits) на максимальный объем временных файлов — 512 МБ.
