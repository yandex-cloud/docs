# Переменные окружения

Список переменных окружения, которые определены в среде выполнения {{ sf-name }} и доступны для функций.

Ключ | Значение
---- | ----
`_HANDLER` | Расположение обработчика, указанное для функции. 
`AWS_LAMBDA_RUNTIME_API` | Хост и порт API среды выполнения.
`LAMBDA_RUNTIME_DIR` | Путь до библиотек среды выполнения.
`LAMBDA_TASK_ROOT` | Путь до кода вашей функции.
`PATH` | Набор каталогов, в которых расположены исполняемые файлы.
