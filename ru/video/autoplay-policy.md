# Политика автозапуска браузеров

Автоматический запуск видео, размещенного на HTML-странице с помощью [SDK видеоплеера для IFrame](./iframe-sdk.md), может не работать из-за политики автозапуска медиаконтента браузера.

Браузеры могут ограничивать автоматический запуск видео по следующим причинам:

* **Потребление трафика** — автоматическое воспроизведение видео может привести к нежелательным расходам трафика пользователя.
* **Потребление ресурсов** — воспроизведение видео требует значительных ресурсов устройства, что может негативно сказаться на производительности.
* **Пользовательский опыт** — автоматическое воспроизведение видео с включенным звуком может быть раздражающим для пользователей.


## Рекомендации {#recommendations}

Для обеспечения корректной работы автоматического запуска видео:

* Проверьте актуальную информацию о политиках автозапуска браузеров.
* Предоставьте пользователю возможность вручную включить звук после начала воспроизведения. Для этого используйте параметр `mute=true` вместе с `autoplay=1`.
