# VideoType

```swift
public enum VideoType: String
```

Тип видеоконтента.

## Contents
- [Cases](#cases)

## Discussion

Определяет режим воспроизведения и доступные возможности плеера. Актуальное значение доступно через `YaPlayer/videoType`.
