# Обновление полей в датасете

Обновление полей необходимо при изменении схемы таблицы (или таблиц) в источнике данных.

{% note info %}

Обновить поля можно для всех источников, кроме CSV.

{% endnote %}

Чтобы обновить поля в датасете:
1. Откройте датасет. Если у вас нет датасета, [создайте его](create.md).
1. В левом верхнем углу переключитесь на вкладку **Поля**.
1. В верхней части экрана нажмите  **Обновить поля**. 

Поля в датасете обновятся, если:

* схема таблицы в источнике изменилась;
* поля были удалены из датасета вручную.
