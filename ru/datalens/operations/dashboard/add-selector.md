# Добавление селектора на дашборд

Чтобы добавить селектор на дашборд:

1. Откройте дашборд. Если у вас нет дашборда, [создайте его](create.md).
1. В верхней части страницы нажмите кнопку **Редактировать**.
1. Нажмите кнопку **Добавить** и выберите **Селектор**.
1. Заполните параметры виджета. Обратите внимание на следующие поля:
	1. **Заголовок**. Задает имя виджета. По умолчанию не отображается на дашборде. Для отображения включите опцию **Показывать**, тогда заголовок отобразится слева от селектора.
	1. **Тип селектора**. Есть два типа селекторов:
		* **На основе датасета**. Значения селектора отображаются из поля датасета.
		* **Ручной ввод**. Значения задаются вручную в настройках селектора.
1. Заполните параметры селектора:

	{% list tabs %}

	- На основе датасета

		* **Датасет**. Определяет датасет с данными для селектора.
		* **Поле**. Определяет поле датасета со значениями селектора.
		* **Тип элемента**. Определяет тип селектора: выпадающий список, поле ввода или календарь.
		* **Множественный выбор**. Определяет возможность выбора нескольких значений.
		* **Значение по умолчанию**. Отображается изначально при открытии дашборда.

	- Ручной ввод

		* **Тип элемента**. Определяет тип селектора: выпадающий список, поле ввода или календарь.
		* **Имя поля**. Задает имя для связи с другими виджетами.
		* **Возможные значения**. Список значений селектора.
		* **Множественный выбор**. Определяет возможность выбора нескольких значений.
		* **Описание**. Задает описание виджета. Отображается на нижней части виджета.
		* **Значение по умолчанию**. Отображается изначально при открытии дашборда.

	{% endlist %}

1. Нажмите кнопку **Добавить**. Виджет отобразится на дашборде.
