# Создание дашборда в {{ datalens-name }}

Для создания дашборда выполните следующее:

1. {% if product == "yandex-cloud" %}На главной странице сервиса {{ datalens-full-name }} нажмите **Создать дашборд**.{% endif %}{% if product == "cloud-il" %}Откройте [главную страницу]({{ link-datalens-main }}) {{ datalens-short-name }}.{% endif %}
{% if product == "cloud-il" %}1. Перейдите в [воркбук](../../concepts/index.md#workbooks-and-collections), внутри которого вы хотите создать дашборд.{% endif %}
{% if product == "cloud-il" %}1. На странице воркбука, в правом верхнем углу, нажмите **Создать** → **Дашборд**.{% endif %}
1. В открывшемся окне введите название дашборда. Дашборд появится в списке на странице навигации.
