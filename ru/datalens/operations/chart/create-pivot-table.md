# Создание сводной таблицы

Чтобы создать сводную таблицу:
1. На главной странице сервиса {{ datalens-full-name }} нажмите **Создать чарт**.
1. В разделе **Датасет** выберите датасет для визуализации. Если у вас нет датасета, [создайте его](../dataset/create.md).
1. Выберите тип чарта **Сводная таблица**.
1. Перетащите измерение из датасета в секцию **Столбцы**.
1. Перетащите измерение из датасета в секцию **Строки**.
1. Перетащите показатель из датасета в секцию **Показатели**. Значения отобразятся в ячейках таблицы.

Подробное описание секций чарта см. в разделе [{#T}](../../concepts/chart/types.md#pivot-table).
