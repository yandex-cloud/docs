# Создание точечной картограммы

Чтобы создать точечную карту:
1. На главной странице сервиса {{ datalens-full-name }} нажмите **Создать чарт**.
1. В разделе **Датасет** выберите датасет для визуализации. Если у вас нет датасета, [создайте его](../dataset/create.md).
1. Выберите тип чарта **Точечная карта**.
1. Перетащите измерение c типом `Геоточка` из датасета в секцию **Геоточки**. Значения отобразятся в виде закрашенной области на карте.
1. Задайте размер отображаемых точек. Для этого нажмите значок в секции **Размер**.

Дополнительно вы можете:
 * применить фильтр;
 * выбрать показатель для интенсивности закраски точек;
 * добавить тултипы из значений измерения и показателя, которые отображаются при наведении на точку.
