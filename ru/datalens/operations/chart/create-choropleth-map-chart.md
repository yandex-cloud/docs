# Создание фоновой картограммы

Чтобы создать фоновую карту:
1. На главной странице сервиса {{ datalens-full-name }} нажмите **Создать чарт**.
1. В разделе **Датасет** выберите датасет для визуализации. Если у вас нет датасета, [создайте его](../dataset/create.md).
1. Выберите тип чарта **Фоновая карта**.
1. Перетащите измерение c типом `Геополигон` из датасета в секцию **Геополигон**. Значения отобразятся в виде закрашенной области на карте.

Дополнительно вы можете:
 * применить фильтр;
 * выбрать показатель для интенсивности закраски полигона.
