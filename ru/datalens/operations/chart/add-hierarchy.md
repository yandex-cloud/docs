# Добавление иерархии

Чтобы добавить иерархию:
1. {% if product == "yandex-cloud" %}На [главной странице]({{ link-datalens-main }}) сервиса {{ datalens-full-name }} нажмите **Создать чарт**.{% endif %}{% if product == "cloud-il" %}Откройте [главную страницу]({{ link-datalens-main }}) {{ datalens-short-name }}.{% endif%}
1. {% if product == "yandex-cloud" %}В разделе **Датасет** выберите датасет для визуализации. Если у вас нет датасета, [создайте его](../dataset/create.md).{% endif %}{% if product == "cloud-il" %}Перейдите в [воркбук](../../concepts/index.md#workbooks-and-collections), внутри которого вы хотите создать чарт с иерархией.{% endif %}
{% if product == "cloud-il" %}1. На странице воркбука, в правом верхнем углу, нажмите **Создать** → **Чарт в Wizard**.{% endif %}
{% if product == "cloud-il" %}1. В поле **Select dataset** выберите из списка датасет. Если у вас нет датасета, [создайте его](../dataset/create.md).{% endif %}
1. Выберите любой тип чарта, кроме **Карта**, **Индикатор**, **Древовидная диаграмма** и **Сводная таблица**.
1. В левой части интерфейса визарда нажмите значок **![image](../../../_assets/plus-sign.svg)** и выберите **Добавить иерархию**.
1. Введите название иерархии, выберите измерения и расположите их в нужном порядке.
1. В правом нижнем углу нажмите кнопку **Сохранить**.
1. Перетащите созданную иерархию в любую доступную секцию. Например, для чарта с типом **Таблица** доступна секция **Столбцы**.
1. Перетащите другие измерения и показатели в нужные секции.

{% note info %}

В визуализацию можно добавить только одну иерархию.

{% endnote %}
