## Подключение к Oracle Database {#oracle-database-connection}

Чтобы создать подключение к Oracle Database:

{% if audience == "internal" %}

1. Перейдите на [страницу подключений](https://datalens.yandex-team.ru/connections).

{% else %}

1. Перейдите на [страницу подключений](https://datalens.yandex.ru/connections).

{% endif %}

1. Нажмите кнопку **Создать подключение**.
1. Выберите подключение **Oracle Database**.
1. Укажите параметры подключения:

    - **Название подключения**. Задайте имя подключения. Оно может быть произвольным.
    - **Имя хоста или IP**. Укажите путь до хоста-мастера или IP-адрес хоста-мастера Oracle Database. Вы можете указать несколько хостов через запятую. Если к первому хосту подключиться не получится, {{ datalens-short-name }} выберет следующий из списка. 
    - **Порт**. Укажите порт подключения к Oracle Database.
    - **Имя базы данных**. Укажите Service Name или SID подключаемой базы данных.
    - **Имя пользователя**. Укажите имя пользователя для подключения к Oracle Database.
    - **Пароль**. Укажите пароль для указанного пользователя.
    - **Время жизни кэша в секундах**. Укажите время жизни кеша или оставьте значение по умолчанию. Рекомендованное значение — 300 секунд (5 минут).
    - **Уровень доступа SQL запросов**. Позволяет использовать произвольный SQL-запрос для [формирования датасета](../../concepts/dataset/settings.md#sql-request-in-datatset).   
1. Нажмите **Создать**. Подключение появится в списке.
