# COT



#### Синтаксис {#syntax}


```
COT( number )
```

#### Описание {#description}
Возвращает котангенс числа `number`, заданного в радианах.

**Типы аргументов:**
- `number` — `Дробное число | Целое число`


**Возвращаемый тип**: `Дробное число`

#### Пример {#examples}



| **[n]**   | **[n] &ast; PI()**   | **COT([n]&ast;PI())**   |
|:----------|:---------------------|:------------------------|
| `-1.00`   | `-3.14`              | `8165619676597685.00`   |
| `-0.50`   | `-1.57`              | `-0.00`                 |
| `-0.25`   | `-0.79`              | `-1.00`                 |
| `0.00`    | `0.00`               | `inf`                   |
| `0.25`    | `0.79`               | `1.00`                  |
| `0.50`    | `1.57`               | `0.00`                  |
| `1.00`    | `3.14`               | `-8165619676597685.00`  |
