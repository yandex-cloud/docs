# GEOINFO



#### Синтаксис {#syntax}


```
GEOINFO( address, scale )
```

#### Описание {#description}

{% note warning %}

- Квота на геокодирование — не более 25 000 уникальных записей в сутки. Не вошедшие в ограничение записи будут обработаны на следующий день.
- Функция доступна только в тарифе Стандарт.

{% endnote %}

Преобразует `address` в название географического объекта (топоним), который соответствует указанному масштабу `scale`.

Допустимые значения параметра`scale`:
- `"country"`;
- `"country_code"`;
- `"region"`;
- `"locality"`.

**Типы аргументов:**
- `address` — `Строка`
- `scale` — `Строка`


**Возвращаемый тип**: `Строка`

{% note info %}

Значения аргументов (`scale`) должны быть константами.

{% endnote %}


#### Примеры {#examples}

```
GEOINFO("посёлок Свободный Серп", "country") = "Россия"
```


#### Поддержка источников данных {#data-source-support}

`Материализованный датасет`.
