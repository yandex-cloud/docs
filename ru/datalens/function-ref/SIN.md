# SIN



#### Синтаксис {#syntax}


```
SIN( number )
```

#### Описание {#description}
Возвращает синус числа `number`, заданного в радианах.

**Типы аргументов:**
- `number` — `Дробное число | Целое число`


**Возвращаемый тип**: `Дробное число`

#### Примеры {#examples}

```
SIN(0) = 0.0
```

```
SIN(PI()/2) = 1.0
```


#### Поддержка источников данных {#data-source-support}

`Материализованный датасет`, `ClickHouse 19.13`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `Oracle Database 12c (12.1)`, `PostgreSQL 9.3`.
