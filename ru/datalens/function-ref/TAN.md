# TAN



#### Синтаксис {#syntax}


```
TAN( number )
```

#### Описание {#description}
Возвращает тангенс числа `number`, заданного в радианах.

**Типы аргументов:**
- `number` — `Дробное число | Целое число`


**Возвращаемый тип**: `Дробное число`

#### Пример {#examples}



| **[n]**   | **[n] &ast; PI()**   | **tan([n]&ast;PI())**   |
|:----------|:---------------------|:------------------------|
| `-1.00`   | `-3.14`              | `0.00`                  |
| `-0.50`   | `-1.57`              | `-16331239353195370.00` |
| `-0.25`   | `-0.79`              | `-1.00`                 |
| `0.00`    | `0.00`               | `0.00`                  |
| `0.25`    | `0.79`               | `1.00`                  |
| `0.50`    | `1.57`               | `16331239353195370.00`  |
| `1.00`    | `3.14`               | `-0.00`                 |
