# POWER



#### Синтаксис {#syntax}


```
POWER( base, power )
```

#### Описание {#description}
Возводит число `base` в степень `power`.

**Типы аргументов:**
- `base` — `Дробное число | Целое число`
- `power` — `Дробное число | Целое число`


**Возвращаемый тип**: `Дробное число`

#### Примеры {#examples}

```
POWER(2.3, 4.5) = 42.43998894277659
```

```
POWER(6, 2) = 36.0
```
