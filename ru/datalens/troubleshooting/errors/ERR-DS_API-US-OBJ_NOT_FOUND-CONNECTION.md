# [{{ datalens-full-name }}] Connection not found

`ERR.DS_API.US.OBJ_NOT_FOUND.CONNECTION`

Запрашиваемое подключение не найдено.

Возможно, запрашиваемое подключение было удалено или перемещено. Укажите существующее подключение или создайте новое.
