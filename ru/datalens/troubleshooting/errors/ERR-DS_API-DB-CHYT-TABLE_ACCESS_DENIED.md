# Access to table was denied

`ERR.DS_API.DB.CHYT.TABLE_ACCESS_DENIED`

Нет доступа к таблице на YT.

Если в подключении к CHYT выбран метод авторизации от имени пользователя, то пользователю {{ datalens-name }}, просматривающему чарт и получающему такую ошибку, необходимо получить права на просмотр таблицы в YT.

Если в подключении к CHYT выбран метод авторизации по OAuth-токену, рекомендуем использовать другой токен (например, от своего пользователя) в подключении, если у вас есть права, или предоставить доступ к таблице пользователю, токен которого используется в подключении. Если вы не знаете, чей токен в используемом подключении, то создайте отдельное подключение.
