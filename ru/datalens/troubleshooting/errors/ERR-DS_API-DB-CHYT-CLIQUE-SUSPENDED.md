# Clique <название клики> is suspended

`ERR.DS_API.DB.CHYT.CLIQUE.SUSPENDED`

Клика, указанная в этом сообщении, приостановлена. Для перезапуска клики:

* Обратитесь к ответственному за клику.

  {% cut "Как найти ответственного за клику" %}
   
  1. Откройте кластер и нажмите на **Operations**, например, [в кластере `hahn`](https://yt.yandex-team.ru/hahn/operations). 
  1. Введите имя клики со звездочкой, например, `*ch_datalens`.
  1. Сбросьте фильтр по текущему пользователю: удалите его логин из поля ниже.
  1. Нажмите **Go to operation**.

  {% endcut %}

* Если владельца клики нет или сложно его найти, обратитесь к администратору подключения, в котором использована эта клика.
* Если ничего не помогло, обратитесь к команде CHYT. Для этого [напишите в чат](https://t.me/joinchat/DPeS0xFdpaSZoCWTYnhnnw).
