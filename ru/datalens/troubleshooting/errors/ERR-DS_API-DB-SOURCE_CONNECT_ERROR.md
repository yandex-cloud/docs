# [{{ datalens-full-name }}] Data source refused connection

`ERR.DS_API.DB.SOURCE_CONNECT_ERROR`

Ошибка загрузки данных.

Ошибка означает, что не удалось установить соединение с БД.

Чтобы исправить ошибку, проверьте настройки [подключения](../../concepts/connection/index.md):

* Проверьте корректность данных, указанных в подключении: **Имя хоста**, **Порт HTTP-интерфейса**, **Имя пользователя** и **Пароль**. При необходимости уточните их у администратора.
* Если указанный порт работает только по HTTP, выключите опцию **HTTPS** в подключении.
* Если в списке хостов указан только один хост, укажите через запятую все хосты кластера в поле **Имя хоста**.
* Если подключаетесь к БД, не являющейся ресурсом {{ yandex-cloud }}, проверьте корректность настройки на стороне БД по [инструкции](../../concepts/connection/index.md#external-db-connection).
* Если подключаетесь к БД, являющейся ресурсом {{ yandex-cloud }}, в режиме `Выбрать в организации`, проверьте, что на кластере управляемой базы данных активирована опция **{{ ui-key.yacloud.mdb.forms.additional-field-datalens }}**. При использовании режима `Указать вручную` проверьте, что используемые хосты имеют публичный адрес.
