# Data source is unavailable

`ERR.DS_API.DATABASE_UNAVAILABLE`

Источник данных (БД) недоступен.

Возможные причины:
- перебои в работе сети;
- нет доступа от сервера {{ datalens-short-name }} до сети источника;
- источник отключен;
- требуется донастройка источника.
