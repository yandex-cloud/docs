# [{{ datalens-full-name }}] Inconsistent aggregation status among SELECT items

`ERR.DS_API.INVALID_QUERY_STRUCTURE.INCONSISTENT_AGGREGATION`

Происходит, когда в одном выражении используются данные с разным [уровнем детализации LOD](../../../datalens/function-ref/aggregation-functions.md#syntax-lod).

При вычислении агрегированного значения происходит преобразование большого набора строк в единственное значение. Для этого применяются специальные [агрегатные функции](../../../datalens/function-ref/aggregation-functions.md). Наиболее часто применяются функции `SUM`, `MIN`, `MAX`, `AVG` и `COUNT`. Агрегатные функции рассчитывают и возвращают одно результирующее значение для всех строк запроса. Если используется группировка, то рассчитываются и возвращаются значения отдельно для каждой группы, на которые разбивается результат запроса.

В {{ datalens-short-name }} вы можете использовать LOD-выражения для создания вложенных агрегаций и агрегаций над всеми данными или группами, которые отличаются от группировки, заданной на уровне чарта. При этом в одном выражении нельзя использовать данные с разным уровнем детализации.

Чтобы исправить ошибку, используйте в одном выражении данные с одним уровнем детализации.
