# Filter invalid value

`ERR.DS_API.FILTER.INVALID_VALUE`

Некорректное значение фильтра.

Ошибка возникает, когда в чарт передается некорректный параметр в качестве фильтра. Например, из дашборда передается параметр с типом `Строка`, который должен фильтровать данные по полю с числовым типом в чарте.

Чтобы исправить ошибку:

1. На дашборде нажмите **Редактировать**.
1. Нажмите значок ![image](../../../_assets/datalens/gear.svg) в правом верхнем углу виджета, который выдает такую ошибку.
1. В настройках виджета в списке параметров удалите некорректный параметр.
1. Сохраните виджет и дашборд.
