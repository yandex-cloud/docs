# You have no access to read this operation

Нет прав на просмотр данных.

Ошибка возникает, когда чарт пытается отобразить результат YQL-запроса, доступ к которому закрыт.

Чтобы исправить ошибку, перейдите в [интерфейс YQL](https://yql.yandex-team.ru) и нажмите кнопку **Share** на странице YQL-запроса. Затем скопируйте и используйте полученный идентификатор.
