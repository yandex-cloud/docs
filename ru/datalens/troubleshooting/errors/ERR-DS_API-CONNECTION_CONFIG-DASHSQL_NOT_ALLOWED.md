# [{{ datalens-full-name }}] Использование QL-чартов запрещено на уровне подключения

`ERR.DS_API.CONNECTION_CONFIG.DASHSQL_NOT_ALLOWED`

Ошибка возникает, когда использование [QL-чартов](../../concepts/chart/ql-charts.md) запрещено на уровне подключения.

Чтобы исправить ошибку, включите опцию **Разрешить подзапросы в датасетах, параметризацию источников и QL-чарты** в настройках подключения.
