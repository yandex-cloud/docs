# [{{ datalens-full-name }}] Field references an unknown source avatar

`ERR.DS_API.AVATAR.NOT_FOUND.FIELD_REF`

Ошибка возникает, когда поле датасета ссылается на ранее удаленную из него таблицу. [Замените источник](../../dataset/create-dataset.md#replace-field) поля в датасете: нажмите на **Источник поля** и укажите корректную таблицу или удалите некорректное поле.
