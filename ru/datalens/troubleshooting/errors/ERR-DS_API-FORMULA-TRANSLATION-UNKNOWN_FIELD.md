# Unknown referenced column

`ERR.DS_API.FORMULA.TRANSLATION.UNKNOWN_FIELD`

Ошибка возникает в вычисляемых полях и в полях с заданным типом агрегации в следующих случаях:

* замена подключения в датасете;
* замена датасета в чарте;
* удаление поля из датасета (при условии, что удаленное поле используется в вычисляемых полях или в агрегациях). 

Это происходит из-за того, что в формулах используются ID (GUID) полей, а не их названия. При замене подключения или датасета у поля с таким же названием будет другой ID.

Чтобы исправить ошибку, создайте заново вычисляемые поля.
