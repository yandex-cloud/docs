# [{{ datalens-full-name }}] Internal Server Error

`ERR.DS_API.ACTION_NOT_ALLOWED`

Ошибка может возникнуть при попытке получить данные через API.

Ошибка означает, что указанное действие с датасетом не разрешено в данном контексте. Например, при запросе данных по датасету разрешено только добавление и редактирование полей, но нельзя подменить подключение или добавить новую таблицу к запросу.

Чтобы избежать такой ошибки, при запросе данных по датасету используйте только действия, относящиеся к полям: `add_field`, `update_field`, `delete_field`.
