# Резервное копирование в {{ datalens-name }}

{{ datalens-short-name }} обеспечивает:
- Ежедневное автоматическое резервирование данных, загруженных в [БД материализации](dataset/settings.md#materializaton). Данные хранятся неделю. 
- Ежедневное автоматическое резервирование метаданных (подключения, датасеты, чарты, дашборды). Резервная копия хранится неделю.

Чтобы получить резервную копию или восстановить данные, [обратитесь в службу технической поддержки]({{ link-console-support }}).

Ответственность за хранение исходных данных для подключения (например, CSV-файлы) и за резервное копирование внешних к {{ datalens-short-name }} данных лежит на самом пользователе экземпляра {{ datalens-short-name }}.
