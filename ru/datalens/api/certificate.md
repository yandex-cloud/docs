# TLS/SSL сертификат

В `API` используется внутренний сертификат [YandexInternalRootCA.crt](https://crls.yandex.net/YandexInternalRootCA.crt).
Необходимо добавить данный сертификат в список доверенных сертификатов операционной системы.

Подробности можно прочитать на [Wiki](https://wiki.yandex-team.ru/security/ssl/sslclientfix).
