# API

{{ datalens-short-name }} предоставляет пользователям cледующие API:

* [API воркбуков](workbooks/) расширяет функциональность API дашбордов и позволяет управлять подключениями, датасетами, чартами и дашбордами. API воркбуков находится в статусе `beta`, поэтому оно менее стабильно по сравнению с API дашбордов. Если при использовании API воркбуков вам не хватает фичей, [сообщите](https://st.yandex-team.ru/createTicket?queue=dlfr) команде разработки.
* [API данных](dashboards/data.md). Позволяет выгрузить сами данные и информацию для отрисовки чарта. Выгрузить по API только данные нельзя.
* [API комментариев](dashboards/comments/index.md). Позволяет создать, отредактировать и удалить комментарий, удалить канал комментариев, а также получить список каналов комментариев.
* API объектов. Позволяет работать с [чартами](dashboards/charts.md) и [дашбордами](dashboards/dash/index.md). Рекомендуем использовать API воркбуков, так как API объектов более низкоуровневое по сравнению с API воркбуков.
* [API скриншотов](dashboards/scr.md). Позволяют создавать скриншоты чартов через API.

API недоступен во внешней инсталляции (в рамках {{ yandex-cloud }}).
