# API

В интранете мы предоставляем ряд API, которыми можно воспользоваться для выгрузки данных и скриншотов, а также создания и изменения чартов, комментариев и дашбордов.

Возможность программного взаимодействия с соединениями и датасетами пока отсутствует, также наши API пока недоступны во внешней инсталляции (в рамках Яндекс.Облака).


## Доступные API {#available-api}

- [Выгрузка данных из чартов](data.md)
- [Создание скриншотов чартов](scr.md)
- [Добавление комментариев на чарты](comments/index.md)
- [Создание и изменение конфигураций чартов](charts/index.md)
- [Создание и изменение конфигураций дашбордов](dash/index.md)
