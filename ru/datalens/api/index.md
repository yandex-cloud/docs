# API

{{ datalens-short-name }} предоставляет пользователям cледующие API:

- [API дашбордов](#dashboards)
- [API воркбуков](#workbooks)

API недоступен во внешней инсталляции (в рамках {{ yandex-cloud }}).

## API дашбордов {#dashboards}

[API дашбордов](dashboards/auth.md) позволяет работать с чартами и дашбордами.


## API воркбуков {#workbooks}

[API воркбуков](workbooks/) расширяет функциональность API дашбордов и позволяет управлять подключениями, датасетами, чартами и дашбордами.
API воркбуков находится в статусе `beta`, поэтому оно менее стабильно по сравнению с API дашбордов,

Используйте API воркбуков, если каких-то фичей не хватает, сообщите команде разработки.
