
# Миграция дашбордов Статистики

**[Yandex Dash](https://dash.yandex-team.ru)** - это сервис для создания дашбордов, позволяющий выводить данные из десятков различных источников в одном месте.
Сервис является заменой [дашбордов Статистики](https://stat.yandex-team.ru/Dashboards), которые [будут закрыты](https://clubs.at.yandex-team.ru/statistics/1804) в 2020 году.

## Миграция {#migration}

Для миграции своего дашборда вы можете воспользоваться [специальной страницей](https://dash.yandex-team.ru/migration).
По схеме дашборда Статистики будет создан новый дашборд в Dash.
Оригинальный дашборд Статистики при этом **не удаляется** и **доступен по прежней ссылке**.

В виду архитектурных особенностей с дашборда Статистики не будут перенесены:
* старые управляторы/менеджеры (вместо них следует использовать [селекторы](../editor/widgets/selector/index.md))
* браузерные скрипты на ChartEditor (помеченные в редакторе как *deprecated*, которые можно отдельно [мигрировать на Node.js](../editor/node-migration.md))
* текстовые чарты с вики-разметкой (вместо них доступны [чарты с поддержкой markdown](../editor/widgets/markdown/index.md#about-markdown))
* права доступа на дашборд (новый дашборд при создании унаследует права родительской папки)

## Редирект {#redirect}

Доступна возможность настроить редирект с дашбордов Статистики на Dash.
Для этого заведите тикет в очереди CHARTS и укажите откуда и куда нужно перенаправлять пользователей.
Дашборд Статистики при этом не удаляется, но попасть на него по прямой ссылке уже не получится,
поэтому рекомендуем сделать копию в другом месте.
