# Партнерская аналитика

Партнерская аналитика — возможность делиться с партнерами дашбордами с разграничением прав доступа на платформе DataLens. Это позволяет реализовать один дашборд над одним кубом данных и, каждый партнер будет видеть только доступные ему данные.

См. подробнее в [статье](https://wiki.yandex-team.ru/DataLens/Partnerskaja-analitika/) на Вики.
