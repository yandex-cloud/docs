## Общая информация {#common-map}

Для отрисовки карт используется библиотека [Highmaps](https://www.highcharts.com/products/highmaps/).
Доступны следующие типы карт:
* Географические
  * Мир
  * Россия
  * Белоруссия
  * Казахстан
  * Украина
  * Турция
  * Франция
  * Бразилия
* Тепловые
* Tile map
* Bubble map
