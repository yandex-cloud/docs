
# Модули

Модуль - это фрагмент кода, который экпортирует определенные объекты/функции/переменные/константы, которые затем можно использовать в скриптах.

Модули можно условно разделить на две группы — пользовательские модули и внешние библиотеки вроде [moment.js](https://momentjs.com).

## Пользовательские модули {#user-modules}

Вы можете выделять полезные фрагменты кода из ваших скриптов в модули, а также использовать модули уже написанные
другими пользователями.
Для этого их необходимо подключить в табе в котором вы хотите их использовать с помощью функции `require`:

```js
const ClickHouse = require('Statbox/ClickHouse');
```

Про создание своих модулей более подробно читайте на странице виджета [Модуль](widgets/module/index.md).

## Внешние библиотеки {#external-libraries}

Мы поддерживаем ряд внешних библиотек, доступных в папке `vendor` и добавляем новые по запросу.

### moment.js {#moment-js}

Библиотека для работы с временем, датами, часовыми поясами.
[Официальный сайт](https://momentjs.com/).

Актуальная версия: `v2.24`

```js
const moment = require('vendor/moment/v2.24');
```

Архивные версии: `v2.21`, `v2.19`, `v2.17`

### lodash {#loadsh}

Упрощает код JavaScript, избавляя от хлопот при работе с массивами, числами, объектами, строками и т.д. [Официальный сайт](https://lodash.com/).

Актуальная версия: `v4.17`

```js
const lodash = require('vendor/lodash/v4.17');
```

### underscore {#underscore}

В состав Underscore входит более 100 функций, часть из которых решает стандартные задачи, например упрощает работу с
коллекциями, а часть позволяет проще делать более экзотические вещи, такие как: функциональное связывание,
javascript-шаблонизация, проверки на равенство (напр. объектов), и т.д. [Официальный сайт](http://underscorejs.org/).

Актуальная версия: `v1.9.1`

```js
const underscore = require('vendor/underscore/v1.9.1');
```

### tinyColor {#tinycolor}

Это небольшая, быстрая библиотека для управления цветом и преобразованием цветов в JavaScript. [Официальный сайт](https://bgrins.github.io/TinyColor/).

Актуальная версия: `v1.4.1`

```js
const tinyColor = require('vendor/tinyColor/v1.4.1');
```

### regression {#regression}

Модуль для расчёта трендов. [Официальный сайт](http://tom-alexander.github.io/regression-js/).

Актуальная версия: `v1.4`

```js
const regression = require('vendor/regression/v1.4');
```
