# История изменений в {{ datalens-full-name }} в январе 2026

* [Изменения базовых возможностей](#base)

## Изменения базовых возможностей {#base}


* В списках [платежных аккаунтов](../settings/service-plan.md#billing-account) теперь отображаются платежные аккаунты, прикрепленные к другим [организациям](../concepts/organizations.md), если пользователю выданы права на них.


* Подготовили первую версию [публичного API](../operations/api-start.md).

  
  {{ datalens-name }} предоставляет публичное API [https://api.datalens.tech](https://api.datalens.tech) для автоматизации операций с дашбордами, чартами, датасетами и подключениями.

  [{{ datalens-name }} Public API](../operations/api-start.md) — это набор методов, аналогичный тем, что используются в веб-интерфейсе {{ datalens-name }}. API описывается OpenAPI-спецификацией и аутентифицируется через IAM-токены Yandex Cloud.

  Для работы с API требуется [IAM-токен](../../iam/operations/iam-token/create) и [идентификатор организации](../settings/index.md#service-settings).




## Исправления и улучшения {#fixes}


* В [{{ datalens-gallery }}]({{ link-datalens-main }}/gallery) форма **Связаться с автором** теперь открывается не в новой вкладке, а в диалоговом окне.
* Добавили возможность [передавать](../settings/seats.md#reassign) рабочее место с истекающим сроком.
* Исправили переопределение цвета фона пункта меню при применении [стилизации интерфейса](../settings/appearance.md#ui-customization) в некоторых темах.


* В окне [настройки связи](../dataset/create-dataset.md#links) в датасетах добавили всплывающую подсказку и поиск по имени поля. Раньше при большом количестве длинных и похожих имен полей было сложно найти и выбрать нужное.
* Теперь в редакторе формул для [создания вычисляемого поля](../concepts/calculations/index.md#how-to-create-calculated-field) в датасетах и чартах, если в списке слева не выбрана функция, в окне описания отображается подсказка и ссылки на документацию и дашборд с примерами.
* Исправили проблему, при которой на некоторых страницах в буфер обмена копировался текст из интерфейса.

### Исправления на дашборде {#dashboard-fixes}


* Добавили возможность управлять тем, на каких вкладках дашборда отображается [селектор](../dashboard/selector.md). При переключении между этими вкладками значение селектора сохраняется.

  Чтобы определить, как селектор будет отображаться на дашборде, в настройке **Показать на вкладках** выберите:

  * `Текущая вкладка <название_вкладки>` (по умолчанию для одиночного селектора) — селектор будет отображаться на указанной вкладке.
  * `Выбранные вкладки` — отметьте одну или несколько вкладок, на которых нужно отображать селектор. По умолчанию отмечена текущая вкладка.
  * `Все вкладки` — селектор будет отображаться на всех вкладках. Доступно для одиночного селектора.
  * `Из настроек группы <значение_для_группы>` (по умолчанию для селектора в группе) — селектор будет отображаться так, как указано в расширенных настройках виджета с группой селекторов. Доступно только для селектора в группе.

  Для виджета с несколькими селекторами можно настроить отображение на вкладке **Настройки группы**.

  Подробнее читайте в разделе [{#T}](../operations/dashboard/add-selector.md).

* Реализовали возможность сохранения значений селекторов при переключении вкладок. Теперь, если выбрать значение в селекторе и переключиться на другую вкладку, а затем вернуться, в селекторе сохранится выбранное значение.


* Теперь на дашборде доступна [настройка связей](../operations/dashboard/dashboard-links.md) для чартов на неоткрытых вкладках виджета:

  * Выберите чарт вверху окна настройки связей — данные чарта дозагрузятся, и можно будет настроить с ним связь.
  * Выберите в окне настройки связей какой-либо селектор, затем в списке виджетов нажмите в колонке **Тип связи** на строке с нужным чартом — данные чарта дозагрузятся, и можно будет настроить с ним связь.

* Исправили стабильную отрисовку таблицы при первоначальной загрузке дашборда.
* Исправили проблему, при которой цвет графиков [Mermaid](../dashboard/markdown.md#mermaid) в визуальном редакторе в текстовых виджетах дашборда отличался от цвета графиков на дашборде.
* Исправили ошибку, из-за которой при настройке [алиасов](../dashboard/link.md#alias) с некоторыми QL-чартами в выборе параметров не отображались параметры, настроенные в интерфейсе QL-чарта.
* В режиме редактирования дашборда добавили подсветку кнопки ![image](../../_assets/console-icons/gear.svg) для селекторов с настройками показа группы `Все вкладки` и `Выбранные вкладки`. 


### Исправления в чартах {#chart-fixes}

* В [линейной диаграмме](../visualization-ref/line-chart.md) исправили проблему настройки формы линий для нескольких показателей, при которой корректно отображалась форма линии только для одного показателя.
* В [картах](../visualization-ref/map-chart.md) исправили добавление поля в секцию `Фильтры слоя` перетаскиванием.
* Исправили автоматическое применение настроек показателя в [QL-чарте](../concepts/chart/ql-charts.md).
