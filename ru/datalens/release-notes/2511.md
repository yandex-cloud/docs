# История изменений в {{ datalens-full-name }} в ноябре 2025

* [Изменения базовых возможностей](#base)
* [Исправления и улучшения](#fixes)

## Изменения базовых возможностей {#base}


* С 1 декабря в {{ datalens-full-name }} изменятся правила тарификации. В рамках единого нового тарифа всем пользователям будут доступны расширенные возможности сервиса, например: [Нейроаналитик](../concepts/neuroanalyst.md), [отчеты](../reports/index.md), JavaScript-кастомизации в [Editor](../charts/editor/index.md), [фоновый экспорт данных](../concepts/chart/data-export.md#background-export) из чартов, [стилизация UI](../settings/appearance.md#ui-customization) и др. Новый тариф будет доступен бесплатно для индивидуального использования, платно — для командной работы.

  Подробнее см. [Изменения в правилах тарификации](../pricing-changes.md).

* Добавили новый раздел с информацией про ежегодный [фестиваль {{ datalens-short-name }}]({{ link-datalens-main }}/festival).
* В подключениях к [{{ TR }}](../operations/connection/create-trino.md), [{{ PG }}](../operations/connection/create-postgresql.md) и [{{ GP }}](../operations/connection/create-greenplum.md) снизили нагрузку на источник при выборе таблиц. Теперь:

  * Фильтрация и пагинация списка таблиц происходит на стороне БД.
  * Для [{{ TR }}](../operations/connection/create-trino.md#create-dataset) при создании датасета поиск таблиц ограничивается выбранным каталогом.

* Чат с [Нейроаналитиком отчета](../reports/insights.md) теперь не закрывается при переходе между страницами отчета.



* Обновили дизайн настроек виджета [чарта](../dashboard/widget.md#chart).

## Исправления и улучшения {#fixes}


* Исправили проблему, при которой не отображался раздел [{{ datalens-gallery }}](https://datalens.ru/gallery), если не была выбрана организация на [стартовой странице]({{ link-datalens-main }}/init) {{ datalens-short-name }}.
* Добавили обновление меню виджета в отчетах после загрузки данных.
* Исправили загрузку чартов при прокрутке дашборда. Ранее периодически появлялись пустые виджеты.
* Исправили проблему в непубличном встраивании, при которой [подписанные параметры](../security/private-embedded-objects.md#signed-parameters) не влияли на селекторы на основе датасета.


* В [настройках поля датасета](../dataset/create-dataset.md#setup-fields) исправили ошибку, при которой было невозможно назначить один и тот же цвет разным полям.
* В отчетах исправили проблему отображения сводных таблиц после смены [формата страницы](../reports/report-operations.md#page-settings).

### Исправления в чартах {#chart-fixes}

* Исправили отображение [таблицы](../visualization-ref/table-chart.md) с пустыми (`null`) значениями в полях с типом `Разметка` (использующих [функции разметки](../function-ref/markup-functions.md)).
* Исправили работу настройки цвета шрифта заголовка чарта в [Gravity UI Charts](../charts/editor/widgets/chart.md).


### Исправления в Нейроаналитике {#neuroanalyst-fixes}

* На дашбордах чат с [Нейроаналитиком](../concepts/neuroanalyst.md) теперь может отображаться не только поверх дашборда, но и сбоку, сжимая его по ширине (по умолчанию). Чтобы изменить положение чата, нажмите:
  
  * ![image](../../_assets/console-icons/pin-slash.svg) — чат будет отображаться поверх дашборда;
  * ![image](../../_assets/console-icons/pin.svg) — чат будет отображаться сбоку от дашборда.

* Теперь при переводе дашборда в режим редактирования чат с Нейроаналитиком закрывается.
* Скрыли кнопку открытия чата с Нейроаналитиком в режиме редактирования или создания дашборда.
* Теперь при закрытии чата с Нейроаналитиком не удаляется его содержимое. При открытии чата для того же виджета или дашборда отображается предыдущая переписка. При открытии чата для нового виджета переписка очищается.
* Исправили позиционирование чата с Нейроаналитиком в свернутом состоянии.
* Исправили ошибки чата с Нейроаналитиком, при которых:

  * после закрытия чата с Нейроаналитиком продолжали выполняться связанные с ним запросы;
  * чат с Нейроаналитиком отображался даже при попытке его скрыть;
  * после нажатия кнопки **Начать заново** Нейроаналитик зависал или появлялся белый экран в чате;
  * кнопки скрытия ![image](../../_assets/console-icons/chevrons-up.svg) и показа ![image](../../_assets/console-icons/chevrons-down.svg) области [закрепленных виджетов](../dashboard/settings.md#widget-fixation) отображались поверх открытого чата с Нейроаналитиком.

### Изменения в мобильной версии {#mobile-version-changes}

* [Легенда](../concepts/chart/settings.md#common-settings) чартов, открытых на весь экран, теперь не перекрывается панелью браузера с нестандартным расположением панели адресной строки (например, Яндекс Браузер для мобильных устройств с панелью снизу).
