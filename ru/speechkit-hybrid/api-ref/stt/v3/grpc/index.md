# Yandex Speech To Text Service API

The service does not operate with resources. Actions are performed by sending HTTP requests to resource URLs or making RPC calls. For more information about {{ yandex-cloud }} API architecture, see [API Concepts](../../../../../api-design-guide/index.yaml).

See the interface definitions of {{ yandex-cloud }} API on [GitHub](https://github.com/yandex-cloud/cloudapi).

Service | Description
--- | ---
[Recognizer](./stt_service.md) | A set of methods for voice recognition.
