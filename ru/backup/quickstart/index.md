# Как начать работать с {{ backup-full-name }}


{{ backup-name }} — сервис для создания резервных копий и восстановления ресурсов {{ yandex-cloud }} и данных на них.

Подключить к {{ backup-name }} вы можете как новую [виртуальную машину](../../compute/concepts/vm.md) {{ compute-full-name }} непосредственно при ее создании, так и уже существующую виртуальную машину с запущенными и настроенными приложениями, ресурсами, данными и т.д.

Чтобы {{ backup-name }} мог создавать резервные копии ВМ или восстанавливать ВМ из резервных копий, виртуальная машина должна быть также привязана к [политике резервного копирования](../concepts/policy.md).

Подробнее о минимальных характеристиках ВМ и поддерживаемых операционных системах см. в разделе [{#T}](../concepts/vm-connection.md).

В зависимости от ваших потребностей вы можете:

* [Начать работать с {{ backup-name }} на виртуальной машине при ее создании](./new-vm.md)
* [Начать работать с {{ backup-name }} на действующей виртуальной машине](./existing-vm.md)
