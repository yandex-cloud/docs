# yc beta managed-sqlserver cluster update

Modifies the specified SQL Server cluster.

#### Command Usage

Syntax: 

`yc beta managed-sqlserver cluster update <CLUSTER-ID>`

|`--description`|<b>`string`</b><br/>New description of the SQL Server cluster.|
|`--labels`|<b>`stringToString`</b><br/>Custom labels for the SQL Server cluster as 'key:value' pairs.<br/><br/>For example, '"project":"mvp"' or '"source":"dictionary"'.<br/><br/>The new set of labels completely replaces the old one.<br/><br/>To add a label, request the current set with the [ClusterService.Get] method, then send an [ClusterService.Update] request with the new label added to the set.|
|`--name`|<b>`string`</b><br/>New name for the SQL Server cluster.|
|`--security-group-ids`|<b>`strings`</b><br/>User security groups.|
|`--service-account-id`|<b>`string`</b><br/>ID of the service account used for access to Object Storage.|
|`--async`|Display information about the operation in progress, without waiting for the operation to complete.|
