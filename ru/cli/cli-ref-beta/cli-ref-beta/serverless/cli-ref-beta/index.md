# yc beta serverless

Manage Serverless resources

#### Command Usage

Syntax:

`yc beta serverless <group>`

#### Command Tree

- [yc beta serverless api-gateway](api-gateway/index.md) — A set of methods for managing API gateways.

  - [yc beta serverless api-gateway add-domain](api-gateway/add-domain.md) — Attaches domain to the specified API gateway.

  - [yc beta serverless api-gateway create](api-gateway/create.md) — Creates an API gateway in the specified folder.

  - [yc beta serverless api-gateway delete](api-gateway/delete.md) — Deletes the specified API gateway.

  - [yc beta serverless api-gateway get](api-gateway/get.md) — Returns the specified API gateway. Note that only API gateway basic attributes are returned.

  - [yc beta serverless api-gateway get-openapi-spec](api-gateway/get-openapi-spec.md) — Returns the OpenAPI specification of specified API gateway.

  - [yc beta serverless api-gateway list](api-gateway/list.md) — Retrieves the list of API gateways in the specified folder.

  - [yc beta serverless api-gateway list-access-bindings](api-gateway/list-access-bindings.md) — Lists existing access bindings for the specified API gateway.

  - [yc beta serverless api-gateway list-operations](api-gateway/list-operations.md) — Lists operations for the specified API gateway.

  - [yc beta serverless api-gateway remove-domain](api-gateway/remove-domain.md) — Detaches domain from the specified API gateway.

  - [yc beta serverless api-gateway set-access-bindings](api-gateway/set-access-bindings.md) — Sets access bindings for the specified API gateway.
