# yc beta compute maintenance

A set of methods for managing maintenances in a service.

#### Command Usage

Syntax:

`yc beta compute maintenance <command>`

#### Command Tree

- [yc beta compute maintenance get](get.md) — Retrieves the specific Maintenance by ID.

- [yc beta compute maintenance list](list.md) — Lists maintenances by conditions, specified in request.

- [yc beta compute maintenance reschedule](reschedule.md) — Allows user to reschedule Maintenance to another date and time.
