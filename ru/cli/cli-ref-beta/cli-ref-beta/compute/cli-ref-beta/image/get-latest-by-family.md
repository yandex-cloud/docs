# yc beta compute image get-latest-by-family

Returns the latest image that is part of an image family.

#### Command Usage

Syntax:

`yc beta compute image get-latest-by-family <FOLDER-ID>`

#### Flags

#|
||Flag | Description ||
|| `--family` | `string`

Name of the image family to search for. ||
|| `--folder-id` | `string`

ID of the folder to get the image from. To get the folder ID, use a [yandex.cloud.resourcemanager.v1.FolderService.List] request. ||
