# yc beta datatransfer transfer



#### Command Usage

Syntax: 

`yc beta datatransfer transfer <command>`

#### Command Tree

- [yc beta datatransfer transfer activate](activate.md) — Activates the specified transfer.
- [yc beta datatransfer transfer create](create.md) — Creates a transfer in the specified folder.
- [yc beta datatransfer transfer deactivate](deactivate.md) — Deactivates the specified transfer.
- [yc beta datatransfer transfer delete](delete.md) — Deletes the specified transfer.
- [yc beta datatransfer transfer get](get.md) — Returns the specified transfer.
- [yc beta datatransfer transfer list](list.md) — Lists transfers in the specified folder.
- [yc beta datatransfer transfer update](update.md) — Updates the specified transfer.
