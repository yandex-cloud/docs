# yc beta datatransfer endpoint get

Returns the specified endpoint.    To get the list of all available endpoints, make a [List] request.

#### Command Usage

Syntax: 

`yc beta datatransfer endpoint get <ENDPOINT-ID>`

#### Flags

| Flag | Description |
|----|----|
|`--endpoint-id`|<b>`string`</b><br/>Identifier of the endpoint to return.<br/><br/>To get the endpoint ID, make an [EndpointService.List] request.|
