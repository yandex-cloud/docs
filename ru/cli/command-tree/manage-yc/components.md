# yc components

Управление установленными компонентами.

#### Использование команды

Синтаксис: 

`yc components <command>`

#### Команды

- `yc components update` — проверить, что установлена последняя версия всех установленных компонентов.
- `yc components restore` — восстановить предыдущее состояние установленных компонентов.
