# yc config

Группа команд 'yc config' позволит вам установить, посмотреть и сбросить параметры YC CLI.

#### Использование команды

Синтаксис: 

`yc config <group|command>`

#### Команды

- `yc config list` — показать значения конфигурации.
- `yc config get` — получить значение указанного конфигурационного параметра. 
- `yc config set` — установить значение указанного конфигурационного параметра. 
- `yc config unset` — сбросить значение указанного конфигурационного параметра.  

#### Группы

- `yc config profile` — управление профилями конфигурациями CLI.
    - `activate`  — активировать указанный профиль конфигураций.
    - `create` — создать профиль конфигураций.
    - `delete` — удалить указанный профиль конфигураций.
    - `get` — получить значения указанного профиля конфигураций. 
    - `list` — получить список профилей конфигураций.
