# yc managed-greenplum resource-preset list

List available Greenplum resource presets.

#### Command Usage

Syntax: 

`yc managed-greenplum resource-preset list <command>`

#### Command Tree

- [yc managed-greenplum resource-preset list master](master.md) — List available Greenplum resource presets for master hosts.
- [yc managed-greenplum resource-preset list segment](segment.md) — List available Greenplum resource presets for segment hosts.
