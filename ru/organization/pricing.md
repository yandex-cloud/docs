# Правила тарификации в {{ org-full-name }}

{% note info %}

Цены действуют с 1 декабря 2025 года.

{% endnote %}

Стоимость использования сервиса {{ org-full-name }} входит в стоимость оплачиваемых сервисов. Дополнительно тарифицируется только подключение к сторонним приложениям.

## Из чего складывается стоимость использования {{ org-name }} {#rules}

Cредства за выбранное количество пользователей списываются в начале каждого месяца.

При покупке лимита на определенное количество пользователей средства списываются сразу.
