# Создать организацию

Чтобы сотрудники вашей компании могли использовать сервисы {{yandex-cloud}}, создайте организацию и добавьте в нее сотрудников вашей компании.{% if product == "yandex-cloud" %} Создать организацию в {{org-full-name}} может любой пользователь Яндекса.{% endif %}

Чтобы создать организацию:

1. [Войдите]({{link-passport}}) в ваш аккаунт {% if product == "yandex-cloud" %}на Яндексе{% endif %}{% if product == "cloud-il" %}Google{% endif %}.{% if product == "yandex-cloud" %} Если у вас еще нет аккаунта, [создайте]({{ support-passport-create }}) его.{% endif %}

1. Перейдите в сервис [{{org-full-name}}]({{link-org-main}}).

1. Ознакомьтесь с условиями использования {{yandex-cloud}} и нажмите кнопку **Войти**.

1. Введите название и описание организации.

1. Нажмите кнопку **Создать организацию**.

После регистрации вы станете владельцем организации и сможете управлять ее настройками.

{% endif %}
