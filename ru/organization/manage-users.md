# Управление аккаунтами пользователей

С помощью {{org-full-name}} вы можете управлять доступом сотрудников вашей организации к сервисам {{yandex-cloud}}.

Вы можете [подключить сотрудников с помощью аккаунтов {% if product == "yandex-cloud" %}на Яндексе{% endif %}{% if product == "cloud-il" %}Google{% endif %}](add-account.md) — тогда они получат доступ к сервисам организации. Если в компании уже используется другая система управления учетными данными (например, Active Directory, Google Workspace и так далее), вы можете [настроить федерацию удостоверений](add-federation.md), чтобы сотрудники использовали свои рабочие аккаунты для доступа к сервисам {{yandex-cloud}}.

В {{org-name}} можно создать федерацию удостоверений с любым сервисом управления учетными данными (поставщиком удостоверений), который поддерживает протокол [SAML]({{link-saml}}).

* [Добавьте пользователя {% if product == "yandex-cloud" %}Яндекса{% endif %}{% if product == "cloud-il" %}Google{% endif %}](add-account.md).
* [Настройте федерацию удостоверений](add-federation.md).
* [Измените или удалите аккаунт сотрудника](edit-account.md).
