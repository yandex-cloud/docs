# SAML Application API, REST: Application

A set of methods for managing SAML applications in the Identity Provider system.

## Methods

#|
||Method | Description ||
|| [Get](get.md) | Returns the specified SAML application. ||
|| [List](list.md) | Retrieves the list of SAML applications in the specified organization. ||
|| [Create](create.md) | Creates a SAML application in the specified organization. ||
|| [Update](update.md) | Updates the specified SAML application. ||
|| [Delete](delete.md) | Deletes the specified SAML application. ||
|| [Suspend](suspend.md) | Suspends the specified SAML application. ||
|| [Reactivate](reactivate.md) | Reactivates a previously suspended SAML application. ||
|| [ListSupportedAttributeValues](listSupportedAttributeValues.md) | Returns the list of supported attribute values for SAML applications. ||
|| [ListOperations](listOperations.md) | Lists operations for the specified SAML application. ||
|| [ListAccessBindings](listAccessBindings.md) | Lists access bindings for the specified SAML application. ||
|| [SetAccessBindings](setAccessBindings.md) | Sets access bindings for the specified SAML application. ||
|| [UpdateAccessBindings](updateAccessBindings.md) | Updates access bindings for the specified SAML application. ||
|| [ListAssignments](listAssignments.md) | Lists assignments for the specified SAML application. ||
|| [UpdateAssignments](updateAssignments.md) | Updates assignments for the specified SAML application. ||
|#
