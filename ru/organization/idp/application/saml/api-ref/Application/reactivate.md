# SAML Application API, REST: Application.Reactivate

Reactivates a previously suspended SAML application.

## HTTP request

```
POST https://organization-manager.{{ api-host }}/organization-manager/v1/idp/application/saml/applications/{applicationId}:reactivate
```

## Path parameters

Request to reactivate a suspended SAML application.

#|
||Field | Description ||
|| applicationId | **string**

Required field. ID of the SAML application to reactivate. ||
|#
