# API {{org-full-name}}

У Яндекс.Коннекта есть два интерфейса REST API для работы с сервисами Коннекта с помощью HTTP-запросов: Directory API и Tracker API.

Используйте эти интерфейсы, чтобы интегрировать возможности Яндекс.Коннекта в ваши веб-сервисы и приложения. Благодаря этому вы автоматизируете рабочие процессы, а пользователи смогут администрировать организацию и работать с задачами Трекера через сторонние приложения.

## Directory API {#directpry-api}

Directory API — интерфейс для управления организацией в Коннекте. С помощью Directory API вы можете:

- управлять структурой организации — создавать и изменять отделы и команды;
- создавать и изменять аккаунты сотрудников;
- подключать и удалять домены, управлять DNS-записями доменов.

## Tracker API {#tracker-api}

Tracker API — интерфейс для работы с Трекером. С помощью Tracker API вы можете:

- интегрировать Трекер с другими сервисами — например, управлять задачами с помощью чат-бота или связать Трекер с CRM-системой;
- автоматизировать процессы, связанные с созданием, массовым изменением и поиском задач по параметрам;
- задавать специфические правила обработки определенных действий — например, обновлять статус задачи по таймеру;
- создавать браузерные расширения для работы с Трекером.
