# Создать дашборд

Чтобы создать новый дашборд и добавить на него метрики:

* Откройте Grafana по ссылке вида `https://grafana.sys.{{ cluster-domain }}` и войдите с вашим именем пользователя и паролем.
* Откройте вкладку **Dashboards**.
* Создайте новый дашборд: нажмите кнопку **New** и в открывшемся меню выберите **New dashboard**.
* В открывшемся окне нажмите кнопку **Add visualization**.
* Выберите источник данных: метрики Prometheus или логи Loki.
* В поле **Metric** укажите метрику, метки и значения. Для проверки визуализации запроса нажмите кнопку **Run queries** и убедитесь, что на графике отображаются нужные данные.
* Нажмите кнопку **Save dashboard**, чтобы сохранить дашборд.

Также вы можете создавать дашборды с помощью кастомного ресурса. Подробнее см. в [документации grafana-operator](https://grafana.github.io/grafana-operator/docs/dashboards/).
