# Удалить кластер PostgreSQL

Если у вас есть кластер [PostgreSQL](../../concepts/components/postgresql.md) в [проекте](../projects/create-project.md)), вы можете удалить его.

{% note warning %}

Если включена **защита от удаления** (`spec.deletionProtection: true`), кластер удалить нельзя. Сначала отключите защиту: установите `spec.deletionProtection: false` в манифесте и примените изменения (`kubectl apply`), либо отключите защиту в консоли управления на странице редактирования кластера. После этого можно удалить кластер.

{% endnote %}

{% list tabs group=instructions %}

- CLI {#cli}

  1. Найдите название нужного кластера в списке кластеров проекта: `kubectl get PostgresqlCluster -n <название проекта>`.
  1. Удалите ресурс кластера: `kubectl delete PostgresqlCluster <название кластера> -n <название проекта>`.


- Консоль управления {#console}

  1. Если вы еще не открыли проект, выберите проект.
  1. В левом меню выберите **PostgreSQL Clusters**.
  1. Выберите кластер.
  1. Нажмите **Удалить**.
  1. Подтвердите удаление.

{% endlist %}

Готово, кластер успешно удален.
