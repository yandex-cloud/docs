# Удалить кластер {{ KF }}

Если у вас есть кластер [{{ KF }}](../../concepts/components/kafka.md) в [проекте](../projects/create-project.md)), вы можете удалить его.

{% list tabs group=instructions %}

- CLI {#cli}

  1. Найдите название нужного кластера в списке кластеров проекта: `kubectl get KafkaCluster -n <название проекта>`.
  1. Удалите ресурс кластера: `kubectl delete KafkaCluster <название кластера> -n <название проекта>`.


- Консоль управления {#console}

  1. Если вы еще не открыли проект, выберите проект.
  1. В левом меню выберите **Kafka Clusters**.
  1. Выберите кластер.
  1. Нажмите **Удалить**.
  1. Подтвердите удаление.

{% endlist %}
