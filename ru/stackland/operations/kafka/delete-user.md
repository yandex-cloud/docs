# Удалить пользователя

Если у вас есть пользователь [{{ KF }}](../../concepts/components/kafka.md) в [проекте](../projects/create-project.md), вы можете удалить его.

{% list tabs group=instructions %}

- CLI {#cli}

  1. Найдите название нужного пользователя в списке пользователей проекта: `kubectl get KafkaUserAccess -n <название проекта>`.
  1. Удалите ресурс пользователя: `kubectl delete KafkaUserAccess <название пользователя> -n <название проекта>`.


- Консоль управления {#console}

  1. Если вы еще не открыли проект, выберите проект.
  1. В левом меню выберите **Kafka Clusters**.
  1. Выберите кластер.
  1. Выберите вкладку **Пользователи**.
  1. Выберите пользователя.
  1. Нажмите **Удалить**.
  1. Подтвердите удаление.

{% endlist %}
