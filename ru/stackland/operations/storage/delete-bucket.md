# Удалить бакет

Если у вас есть [бакет](../../concepts/components/storage.md) и он пуст, вы можете его удалить.

{% note warning %}

Перед удалением бакета убедитесь, что он не содержит объектов. Бакет с объектами удалить нельзя.

{% endnote %}

{% list tabs group=instructions %}

- CLI {#cli}

  1. Получите список бакетов в проекте:

      ```bash
      kubectl get bucket -n <название_проекта>
      ```

  1. Убедитесь, что бакет пуст. При необходимости удалите все объекты через S3 API.

  1. Удалите ресурс бакета:

      ```bash
      kubectl delete bucket <название_бакета> -n <название_проекта>
      ```

  1. Убедитесь, что бакет удален:

      ```bash
      kubectl get bucket -n <название_проекта>
      ```


- Консоль управления {#console}

  1. Если вы еще не открыли проект, выберите проект.
  1. В левом меню выберите **Object Storage**.
  1. Выберите бакет.
  1. Нажмите **Удалить бакет**.
  1. Подтвердите удаление.

{% endlist %}
