# Удалить ключ доступа

Если у вас есть ключ доступа к {{ objstorage-name }}, вы можете его удалить. После удаления ключа приложения, использующие его, потеряют доступ к хранилищу.

{% note warning %}

Удаление ключа доступа необратимо. Убедитесь, что ключ больше не используется приложениями.

{% endnote %}

{% list tabs group=instructions %}

- CLI {#cli}

  1. Получите список ключей доступа в проекте:

      ```bash
      kubectl get apikey -n <название_проекта>
      ```

  1. Удалите ключ доступа:

      ```bash
      kubectl delete apikey <имя_ключа> -n <название_проекта>
      ```

  1. При необходимости удалите связанный секрет:

      ```bash
      kubectl delete secret <имя_секрета> -n <название_проекта>
      ```

  1. Если сервисный аккаунт больше не нужен, удалите его:

      ```bash
      kubectl delete serviceaccount <имя_сервисного_аккаунта> -n <название_проекта>
      ```

  1. Удалите назначение роли:

      ```bash
      kubectl delete accessbinding <имя_назначения> -n <название_проекта>
      ```


- Консоль управления {#console}

  1. Если вы еще не открыли проект, выберите проект.
  1. В левом меню выберите **Сервисные аккаунты**.
  1. Выберите сервисный аккаунт.
  1. Перейдите на вкладку **Ключи доступа**.
  1. Найдите нужный ключ и нажмите **Удалить**.
  1. Подтвердите удаление.

{% endlist %}
