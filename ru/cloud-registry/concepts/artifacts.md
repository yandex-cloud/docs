# Артефакты

_Артефакты_ — это пакеты программного обеспечения и другие файлы, используемые в процессе разработки и развертывания приложений. 

Артефакты {{ cloud-registry-name }} хранятся в [реестрах](registry.md) и организованы в иерархическую структуру путей. Каждый артефакт имеет уникальный путь, который определяет его местоположение в реестре. Путь может состоять из нескольких уровней, что позволяет логически группировать артефакты по проектам, командам или функциональности.

Например, артефакт может иметь путь `project/team/component/artifact`, где каждый уровень пути помогает организовать и найти нужный артефакт.

**Особенности организации артефактов:**
- Артефакты организованы в структуру путей, которая отражает логическую организацию ваших проектов
- Путь к артефакту определяет его уникальное местоположение в реестре
