# Продукт

Продукт {{ marketplace-short-name }} — это форма, которая отражает тип программного продукта и содержит его [версии](version.md).

За использование продукта пользователь платит по назначенному тарифу. Разные версии продукта могут иметь разные тарифы. О том, как создать тариф, см. [{#T}](../operations/create-tariff.md).
