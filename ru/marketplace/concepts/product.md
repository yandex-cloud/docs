# Продукт

Продукт Marketplace — это образ диска c предустановленным ПО, который пользователь может выбрать при создании виртуальной машины в {{ compute-name }}.

За использование продукта пользователь платит по назначенному продукту [тарифу](tariff.md).

Продукты можно версионировать, чтобы выпускать обновления продукта или предлагать новые версии продукта для разных ОС. Разным версиям продукта можно назначать разные тарифы.
