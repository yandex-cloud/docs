# История запросов

{% include [preview-by-request](../../../_includes/note-preview-by-request.md) %}

В этом разделе описано, как работать с историей запросов.

## Просмотр предыдущих запусков {#view-history}

Чтобы просмотреть детали запуска:

{% list tabs group=instructions %}

- Интерфейс {{ sd-name }} {#cloud-sd}

  1. Откройте историю запросов.
  1. Выберите запуск из списка.
  1. Просмотрите информацию о запуске.

{% endlist %}

В истории запросов отображается список предыдущих запусков с информацией:

* дата и время запуска;
* текст запроса;
* временной период;
* статус выполнения;
* количество результатов.

## Запуск запроса из истории {#run-from-history}

Чтобы запустить запрос из истории:

{% list tabs group=instructions %}

- Интерфейс {{ sd-name }} {#cloud-sd}

  1. Откройте историю запросов.
  1. Найдите нужный запуск.
  1. Нажмите **{{ ui-key.yacloud_org.security.siem.cell-result-action-launch_v9v5f }}**.

  Запрос будет выполнен с теми же параметрами, что и в выбранном запуске.

{% endlist %}

## Изменение запроса из истории {#edit-from-history}

Чтобы изменить запрос из истории:

{% list tabs group=instructions %}

- Интерфейс {{ sd-name }} {#cloud-sd}

  1. Откройте историю запросов.
  1. Найдите нужный запуск.
  1. Нажмите **Редактировать**.

  Текст запроса будет скопирован в редактор, где вы сможете внести изменения и запустить запрос с новыми параметрами.

{% endlist %}

## Поделиться запросом из истории {#share-from-history}

Чтобы поделиться запросом из истории:

{% list tabs group=instructions %}

- Интерфейс {{ sd-name }} {#cloud-sd}

  1. Откройте историю запросов.
  1. Найдите нужный запуск.
  1. Нажмите **{{ ui-key.yacloud_org.security.siem.cell-result-action-share_i1khy }}**.
  1. Скопируйте ссылку.

  Ссылка содержит текст запроса и временной период из выбранного запуска.

{% endlist %}
