# Обзор

Чтобы начать работу с {{ yds-full-name }}:
1. [Настройте](aws.md) окружение.
1. [Создайте](create-stream.md) поток данных.
1. Настройте поставку данных из внешних систем в {{ yds-short-name }} с помощью:
   * [Fluentd](fluentd.md);
   * [Logstash](logstash.md);
   * [AWS CLI](aws-cli.md).
1. [Настройте](send-to-clickhouse.md) сохранение данных из {{ yds-short-name }} в [{{ CH }}](https://clickhouse.tech/) с помощью [{{ data-transfer-full-name }}](../../data-transfer/quickstart.md).
