# UpdateStream

Обновляет параметры [потока данных](../../concepts/glossary.md#stream-concepts).

## Запрос {#request}

Запрос содержит данные в формате JSON.

```json
{
  "StreamName": "string",
  "ShardCount": number,
  "WriteQuotaKbPerSec": number,
  "RetentionPeriodHours": number
}
```

### Параметры запроса {#request-options}

Параметр | Описание
----- | -----
`StreamName` | Имя создаваемого потока данных.<br/><br/>**Тип**: Строка<br/>**Размер**: `1`-`128` символов.<br/>**Возможные значения**: `[a-zA-Z][a-zA-Z0-9-]+*(?<!-)$`<br/>**Обязательный**: Да
`ShardCount` | Количество [сегментов](../../concepts/glossary.md#shard) в потоке.<br/>Доступно только увеличение числа сегментов.<br/><br/>**Тип**: Целое число<br/>**Минимальное значение**: `1`.<br/>**Обязательный**: Нет
`WriteQuotaKbPerSec` | Ограничение на скорость записи в поток данных, КБ/с.<br/><br/>**Тип**: Целое число<br/>**Возможные значения**: `128`, `512`, `1024`.<br/>**Обязательный**: Нет
`RetentionPeriodHours` | Время хранения сообщений.<br/><br/>**Тип**: Целое число<br/>**Возможные значения** `4`, `12`, `24`.<br/>**Обязательный**: Нет

## Ответ {#response}

В случае успеха возвращается HTTP-ответ с кодом 200 и данные в формате JSON.

```json
{
  "StreamName": "string",
  "ShardCount": number,
  "WriteQuotaKbPerSec": number,
  "RetentionPeriodHours": number
}
```

### Параметры ответа {#response-options}

Параметр | Описание
----- | -----
`StreamName` | Имя создаваемого потока данных.<br/><br/>**Тип**: Строка<br/>**Размер**: `1`-`128` символов.<br/>**Возможные значения**: `[a-zA-Z][a-zA-Z0-9-]+*(?<!-)$`<br/>**Обязательный**: Да
`ShardCount` | Количество [сегментов](../../concepts/glossary.md#shard) в потоке.<br/><br/>**Тип**: Целое число<br/>**Минимальное значение**: `1`.<br/>**Обязательный**: Нет
`WriteQuotaKbPerSec` | Ограничение на запись в поток данных.<br/><br/>**Тип**: Целое число<br/>**Возможные значения**: `128`, `512`, `1024`.<br/>**Обязательный**: Нет
`RetentionPeriodHours` | Время хранения сообщений.<br/><br/>**Тип**: Целое число<br/>**Возможные значения** `4`, `12`, `24`.<br/>**Обязательный**: Нет

## Ошибки {#errors}

Параметр | Описание | Код HTTP
----- | ----- | -----
