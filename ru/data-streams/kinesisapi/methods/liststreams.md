# ListStreams

Выводит список [потоков данных](../../concepts/glossary.md#stream-concepts).

Параметр ответа `HasMoreStreams` указывает, что остались невыведенные потоки. Чтобы запросить их, укажите имя последнего возвращенного потока в качестве значения `ExclusiveStartStreamName` следующего запроса.

## Запрос {#request}

Запрос содержит данные в формате JSON.

```json
{
  "ExclusiveStartStreamName": "string",
  "Limit": number
}
```

### Параметры запроса {#request-options}

Параметр | Описание
----- | -----
`ExclusiveStartStreamName` | Имя потока, с которого начнется вывод.<br/><br/>**Тип**: Строка<br/>**Размер**: `1`-`128` символов.<br/>**Возможные значения**: `[a-zA-Z0-9_.-]+`<br/>**Обязательный**: Нет
`Limit` | Максимальное количество потоков в списке. Если вы укажете значение больше `100`, то будет выведено не более `100` потоков.<br/><br/>**Тип**: Целое число<br/>**Возможные значения**: `1`-`10000`.<br/>**Значение по умолчанию**: `100`<br/>**Обязательный**: Нет

## Ответ {#response}

В случае успеха возвращаются HTTP-ответ с кодом 200 и данные в формате JSON.

```json
{
  "HasMoreStreams": boolean,
  "StreamNames": ["string"]
}
```

### Параметры ответа {#response-options}

Параметр | Описание
----- | -----
`HasMoreStreams` | Если значение `true`, в списке остались невыведенные потоки.<br/><br/>**Тип**: Булев
`StreamNames` | Имя потока данных.<br/><br/>**Тип**: Строка<br/>**Размер**: `1`-`128` символов.<br/>**Возможные значения**: `[a-zA-Z0-9_.-]+`

## Ошибки {#errors}

Параметр | Описание
----- | -----
`LimitExceededException` | Превышено максимальное число запросов. | 400

Могут возникать [ошибки](../common-errors.md), общие для всех методов.
