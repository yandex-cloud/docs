# IncreaseStreamRetentionPeriod

Увеличивает [время хранения сообщений](../../concepts/glossary.md#retention-time) потока данных.

Доступные значения времени хранения: 4 часа, 12 часов, 24 часа.

## Запрос {#request}

Запрос содержит данные в формате JSON.

```json
{
  "RetentionPeriodHours": number,
  "StreamName": "string"
}
```

### Параметры запроса {#request-options}

Параметр | Описание
----- | -----
`RetentionPeriodHours` | Времени хранения сообщений, часы.<br/>Должно быть больше текущего значения времени.<br/><br/>**Тип**: Целое число<br/>**Возможные значения** `12`, `24`.<br/>**Обязательный**: Да
`StreamName` | Имя потока данных.<br/><br/>**Тип**: Строка<br/>**Размер**: `1`-`128` символов.<br/>**Возможные значения**: `[a-zA-Z0-9_.-]+`<br/>**Обязательный**: Да

## Ответ {#response}

В случае успеха возвращается ответ с кодом HTTP 200 и пустым телом.
