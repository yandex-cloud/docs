# {{ brand-voice-full-name }}

Технология {{ brand-voice-name }} позволяет создавать уникальные голоса для модели синтеза речи. В зависимости от целей использования обученной модели {{ yandex-cloud }} предлагает два направления {{ brand-voice-name }}.

## {{ brand-voice-name }} Full {#full}

{{ brand-voice-name }} Full подойдет для любых бизнес-задач:
* голосовые ассистенты;
* роботы-операторы колл-центра;
* озвучивание произвольных текстов.

Создание полноценной модели с уникальным голосом на основе голоса вашего диктора требует большого количества записанных аудио. Подробная информация о технологии {{ brand-voice-name }} Full и стоимость создания модели предоставляется по запросу.

## {{ brand-voice-name }} Adaptive {#adaptive}

{{ brand-voice-name }} Adaptive разработан для создания диалоговых роботов различного применения:
* телемаркетинг;
* прием звонков первой линии;
* опросы;
* автоматизация колл-центров.

Технология {{ brand-voice-name }} Adaptive позволяет озвучивать текст по _шаблонам_ — специально заготовленным фразам, в которых изменяются отдельные ключевые части — _переменные_. Синтезируемый голос обучается на записях диктора, голос которого должна имитировать результирующая модель. Благодаря этому все слова, и записанные, и синтезированные, звучат как произнесенные одним и тем же говорящим.

### Преимущества {{ brand-voice-name }} Adaptive

* Естественное звучание голоса и интонации.
* Для синтеза фраз требуется гораздо меньше данных для обучения, что экономит затраты.

### Ограничения {{ brand-voice-name }} Adaptive

* Помимо текста, потребуется предзаписанное аудио с шаблоном фразы для синтеза. Подробнее о требованиях к аудио и текстам см. на странице [{#T}](income-data-format.md).
* Применение ограничено фразами, характерными для телефонного звонка: достаточно короткие тексты, ограниченная интонационная выразительность. Для озвучивания художественных текстов такой голос не подойдет.
