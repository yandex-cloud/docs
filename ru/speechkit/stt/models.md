# Модели распознавания

Модель распознавания — модель, которая обучена распознавать речь на определенном языке. Для обучения моделей используются массивы данных из сервисов и приложений Яндекса. Это позволяет постоянно улучшать качество распознавания. 

Основная поддерживаемая модель для каждого типа распознавания — модель `general`. Она распознает речь на любую тему на заданном языке: короткие и длинные фразы, а также имена, адреса, даты и числа.

## Теги версий {#tags}

Одновременно могут быть доступны три версии модели `general`. Выбрать нужную версию можно по тегам:

* `general` — основная версия модели.
* `general:rc` — версия-кандидат для релиза, которую вы можете тестировать.
* `general:deprecated` — предыдущая версия модели.

{% note info %}

Версии, доступные по тегу `general:deprecated`, с выходом новых моделей перестают поддерживаться: {{ speechkit-name }} гарантирует две недели поддержки предыдущей версии со дня обновления версии по тегу `general`.
 
{% endnote %}

{% list tabs %}

- Потоковое распознавание

  В модели `general` доступно распознавание для следующих языков:
  * `ru-RU` (по умолчанию) — русский язык;
  * `kk-KK` — казахский язык;
  * `en-US` — английский язык;
  * `de-DE` — немецкий язык;
  * `fr-FR` — французский язык;
  * `fi-FI` — финский язык;
  * `sv-SE` — шведский язык;
  * `nl-NL` — голландский язык;
  * `pl-PL` — польский язык;
  * `it-IT` — итальянский язык;
  * `es-ES` — испанский язык;
  * `pt-PT` — португальский язык;
  * `tr-TR` — турецкий язык;
  * `auto` — автоматическое распознавание языка.

  Доступные значения не чувствительны к регистру. 
   
  Подробнее о настройке запросов API для потокового распознавания читайте на странице [Потоковое распознавание](streaming.md#specification-msg).
    
- Синхронное распознавание
   
  Модель `general` доступна для русского языка. В запросе API укажите значение параметра `lang`:
  * `ru-RU` (по умолчанию) — русский язык.

  Доступные значения не чувствительны к регистру.

  Подробнее о настройке запросов API для распознавания коротких аудио читайте на странице [Распознавание коротких аудио](request.md#query_params).

- Асинхронное распознавание

  Для [асинхронного распознавания](transcribation.md) кроме модели `general` также доступна модель `deferred-general`. Это модель отложенного распознавания по [специальным тарифам](../pricing.md#prices-stt). Во время отложенного распознавания аудио попадает в очередь с более низким приоритетом и обрабатывается в наименее нагруженное время, но не более 24 часов.

  Для распознавания по тегу `deferred-general` используется та же версия модели, что и по тегу `general`. 

  Для настройки языка в запросе API укажите значение параметра `lang`:
  * `ru-RU` (по умолчанию) — русский язык;
  * `kk-KK` — казахский язык. 

  Доступные значения не чувствительны к регистру.

  Подробнее о настройке запросов API для распознавания длинных аудио читайте на странице [Распознавание длинных аудио](transcribation.md#sendfile-params).
