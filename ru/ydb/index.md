**Yandex Database** (YDB) — это горизонтально масштабируемая распределённая отказоустойчивая СУБД с поддержкой [ACID-транзакций](https://en.wikipedia.org/wiki/ACID_(computer_science)).

## Основные разделы документации

### [Как попробовать](quickstart.md)
В разделе приведена пошаговая инструкция, помогающая быстро познакомиться с системой YDB и совершить ряд элементарных действий — создать базу, создать таблицу в базе, записать строку, прочитать строку, удалить таблицу.

### [Обзор](overview.md)
Раздел содержит общее описание YDB. После прочтения раздела можно получить представление о назначении YDB и ее основных возможностях.

### [Основные понятия](concepts/databases.md)
Раздел дает представление об основных понятиях и принципах, которыми оперирует YDB.
После прочтения раздела можно узнать об устройстве баз данных в YDB, поддерживаемой модели данных и типах, транзакциях и запросах, аутентификации и авторизации.

### [Начало работы](getting_started/create_manage_database.md)
Раздел предоставляет возможность пройти весь путь от создания своей базы данных до запуска приложения, использующего базовые возможности БД.
Доступны примеры приложений на C++, Java и Python.

### Рекомендации
Раздел содержит набор рекомендаций по практическому использованию YDB — как [загружать большие объемы данных](best_practices/batch_upload.md), эмулировать [вторичные индексы](best_practices/secondary_indexes.md) и [постраничный вывод](best_practices/paging.md).
Большинство рекомендаций снабжены примерами приложений на C++, Java и Python.
