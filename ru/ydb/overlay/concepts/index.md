## YDB в {{ yandex-cloud }}

Вы можете создать базу данных в бессерверной (Serverless) конфигурации или с выделенными серверами (Dedicated). Подробнее о различиях в конфигурациях читайте в разделе [Serverless и Dedicated режимы работы](serverless_and_dedicated.md).

Для сервиса действует [соглашение об уровне обслуживания]{% if lang == "ru" %}(https://yandex.ru/legal/cloud_sla?lang=ru){% endif %}{% if lang == "en" %}(https://yandex.ru/legal/cloud_sla?lang=en){% endif %}. Уровень обслуживания сервиса определен в документе [Уровень обслуживания {{ ydb-full-name }}]{% if lang == "ru" %}(https://yandex.ru/legal/cloud_sla_ydb?lang=ru){% endif %}{% if lang == "en" %}(https://yandex.ru/legal/cloud_sla_ydb?lang=en){% endif %}.
