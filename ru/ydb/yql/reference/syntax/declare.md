# DECLARE

Объявляет типизированное [именованное выражение](expressions.md#named-nodes), значение для которого будет передано отдельно от текста запроса.
 
Объявление параметра состоит из четырех частей:

1. Ключевое слово `DECLARE`;
2. Имя именованного выражения, по которому затем можно будет обращаться к переданному значению. Обязано начинаться с символа `$`;
3. Ключевое слово `AS`;
4. Тип данных параметра [в виде строки в принятом формате](../misc/type_string.md), либо для [примитивных типов данных](../types/primitive.md) можно просто его имя без кавычек.

Имеется возможность использовать в качестве типов только сериализуемые типы, а именно:

1. [примитивные типы](../types/primitive.md);
2. [опциональные типы](../types/optional.md) и [контейнеры](../types/containers.md) за исключением `Stream<Type>`;
3. тип `Void` - единственный поддерживаемый [специальный тип](../types/special.md).

**Примеры**

``` yql
DECLARE $x AS String;
DECLARE $y AS "String?";
DECLARE $z AS "List<String>";

SELECT $x, $y, $z;
```
