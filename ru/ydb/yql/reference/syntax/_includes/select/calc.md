## SELECT

Возвращает результат вычисления выражений, указанных после `SELECT`.

Может использоваться в сочетании с другими операциями для получения иного эффекта.

**Примеры:**

``` yql
SELECT "Hello, world!";
```

``` yql
SELECT 2 + 2;
```
