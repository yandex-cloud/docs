# Вторичные индексы

Об использовании вторичных индексов для выборки данных читайте в [документации](https://cloud.yandex.ru/docs/ydb/oss/public/develop/best_practices/secondary_indexes).

В примере используется схема данных, описанная в документе [Начало работы — Работа со схемой](../../getting_started/start_scheme.md).
Чтобы хранить количество просмотров, в таблицу  `series` для каждого сериала добавлено поле `views` типа `Uint64`.

{% note info %}

Создание таблиц с индексом через YQL возможно только при использовании v1 синтаксиса.

{% endnote %}
