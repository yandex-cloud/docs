## Дата и время {#datetime}

| Тип    | Пояснение      | Используется в<br/>запросах и<br/>вычислениях<br/>YQL | Используется в<br/>качестве типа<br/>данных столбца | Используется в<br/>первичных<br/>ключах | Поддерживает<br/>возможность<br/>сравнения |
| ----- | ----- | ----- | ----- | ----- | ----- |
| **Date** | Точность до дней | Да | Да | Да | Да |
| **Datetime** | Точность до секунд | Да | Да | Да | Да |
| **Timestamp** | Точность до микросекунд | Да | Да | Да | Да |
| **Interval** | Точность до микросекунд, допустимы значения интервалов — не более 24 часов | Да | Да | Нет | Да |
| **TzDate** | Точность до дней, содержит метку временной зоны | Да | Да | Нет | Да |
| **TzDatetime** | Содержит метку временной зоны, точность до секунд | Да | Да | Нет | Да |
| **TzTimestamp** | Содержит метку временной зоны, точность до микросекунд | Да | Да | Нет | Да |
