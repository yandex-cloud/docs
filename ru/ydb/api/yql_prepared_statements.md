## Общие сведения {#general-information}
Prepared statements - это возможность базы данных выполнять одни и те же или похожие повторяющиеся запросы с высокой эффективностью. Такие запросы записываются в форме шаблона, в котором определенного вида имена заменяются конкретными параметрами при каждом выполнении запроса. Это позволяет БД выполнить подготовительную работу по разбору и оптимизации запроса один раз для нескольких запросов, отличающихся только значениями параметров.

Внешний пользователь KiKiMR в результате операции prepare получает некий идентификатор своего запроса и в дальнейшем может передавать этот идентификатор на выполнение вместе со значениями параметров для их подстановки.
