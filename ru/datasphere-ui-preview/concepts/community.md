# Сообщества в {{ ml-platform-name }}

_Сообщества_ — это способ организации групповой работы в {{ ml-platform-name }}. Сообщество определяет область видимости проектов и ресурсов и помогает настроить работу в команде.

Каждый пользователь обладает как минимум одним личным сообществом, в котором хранятся созданные им проекты. В любое сообщество вы можете пригласить других пользователей, чтобы вместе работать над проектами. Администратор сообщества может настроить роли для всех участников. Подробнее о ролях в {{ ml-platform-name }} см. [{#T}](../security/index.md).
