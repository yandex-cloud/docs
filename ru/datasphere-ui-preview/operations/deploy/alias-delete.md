# Удалить алиас

1. {% include [find project](../../../_includes/datasphere/ui-find-project.md) %}
1. В блоке **Ресурсы** выберите **Алиас**.
1. Напротив нужного алиаса нажмите ![Options](../../../_assets/options.svg) и выберите ![Delete](../../../_assets/datasphere/delete.svg) **Удалить**.
1. Во всплывающем окне нажмите кнопку **Передать**.
