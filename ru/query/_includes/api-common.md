При использовании HTTP API необходимо [аутентифицироваться](../api/auth.md), указав для этого [обязательные заголовки](../api/index.md#mandatory-headers). Кроме того, можно задать [необязательные заголовки](../api/index.md#optional-headers) для настройки исполнения запросов. При работе с HTTP API могут возвращаться [ошибки](../api/index.md#errors).
