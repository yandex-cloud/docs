При использовании HTTP API необходимо [аутентифицироваться](../api/auth.md), указав для этого [обязательные заголовки](../api/common-information.md#mandatory-headers), кроме этого можно задавать [необязательные заголовки](../api/common-information.md#optional-headers) для настройки исполнения запросов. В процессе работы могут возникать ошибки, работа с ними описана в разделе [Ошибки](../api/common-information.md#errors)
