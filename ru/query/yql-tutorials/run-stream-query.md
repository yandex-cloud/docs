# Выполнить запрос к потоковым данным

Запросы к потоковым данным позволяют обрабатывать записи в процессе их передачи из источника в приемник. При этом сохраняются только результаты вычислений.

Например:

* Получить количество поездок.
* Рассчитать суммарный размер чаевых.
* Группировать данные интервалами по одному месяцу.

```sql
SELECT 
    *
FROM bindings.`tutorial-streaming` LIMIT 10;

SELECT
    HOP_END() AS time,
    COUNT(*) AS ride_count,
    SUM(total_amount) AS total_amount
FROM 
    $data
GROUP BY 
    HOP(CAST(Just(tpep_pickup_datetime) as Timestamp), "PT1M", "PT1M", "PT1M");
```

Рассмотрите пример в блоке справа и нажмите кнопку ![run](../../_assets/query/run.svg) **Выполнить**.
Результат выполнения запроса отобразится на вкладке **Результат** в виде таблицы или схемы.
