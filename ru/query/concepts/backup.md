# Резервное копирование

Настройки соединений, привязок, запросов, запусков {{ yq-name }} надежно хранятся и реплицируются в инфраструктуре {{ yandex-cloud }}.

{{ yq-full-name }} не хранит сами данные, над которыми выполняются запросы.

Максимальная продолжительность хранения результатов исполнения запроса ограничена [лимитом](../concepts/limits.md#yq-limits).
