# create-query 

Создает запрос к данным и начинает его выполнение.

При успехе метод создает запрос к данным, начинает его исполнение, переводя запрос в статус `RUNNING` и возвращает HTTP-ответ с кодом 200. Операция получения результатов возможна только после успешного завершения работы запроса. Чтобы узнать статус работы запроса, используйте метод [get-query-status](get-query-status.md).

{% include [!](../_includes/api-common.md) %}

## Запрос {#request}

`POST`-запрос на адрес `/{folder_id}/queries`, где `{folder_id}`- идентификатор фолдера, а тело запроса содержит данные в формате JSON.

Тело запроса:
```json
{
  "name": "human readable default name",
  "type": "ANALYTICS",
  "text": "string",
  "description": ""
}
```

### Параметры запроса {#request-options}

| Параметр | Описание | Допустимые значения | Примечание |
| ----- | ----- | ----- | ----- |
|`name` | Название запроса |  | Если не указан, будет присвоено имя по умолчанию|
|`type` | Тип запроса, потоковый или аналитический | `STREAMING`, `ANALYTICS` |Если не указан, то запрос считается аналитическим|
|`text` | Текст запроса, обязательный параметр| Строка ||
|`description` | Описание запроса||Если не указан, будет использоваться пустое описание|

## Ответ {#response}

В случае успеха возвращается HTTP-ответ с кодом 200 и идентификатором запроса.

```json
{
  "id": "string"
}
```

| Параметр | Описание | Примечание |
| ----- | ----- | ----- |
|`id`|Идентификатор созданного запроса| Обязательный|

## Пример { #example }

Запрос: 
```
curl -X 'POST' \
  'https://yq/v1/{folder_id}/queries' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -d '{
  "name": "Test query",
  "type": "ANALYTICS",
  "text": "select 1",
  "description": ""
}
```

Ответ: 

```json
{
  "id": "csqugo80f0l35isue09b"
}
```
