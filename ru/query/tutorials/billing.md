# Обработка файлов детализации в сервисе {{ billing-name }}

{{ billing-name }} позволяет получать информацию об объеме использованных ресурсов, проверять расходы денежных средств и оплачивать потребление ресурсов. Детализацию расходов можно отправлять в {{ objstorage-full-name }} и далее обрабатывать с помощью {{ yq-full-name }}.
