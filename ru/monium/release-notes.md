# История изменений в {{ monium-name }}

## Октябрь — декабрь 2025 {#oct-dec-2025}

* Сервис {{ monium-name }} доступен на стадии [Technical Preview](../overview/concepts/launch-stages.md). Для получения доступа обратитесь в [службу поддержки]({{ link-console-support }}).
* Реализован сбор, агрегация, поиск и хранение распределенных [трассировок](traces/index.md) по стандарту [OpenTelemetry (OTLP)](https://opentelemetry.io/docs/). Это позволяет отслеживать полный путь запроса через все микросервисы и системы, что упрощает диагностику проблем и анализ производительности.
* Инструменты телеметрии — метрики, логи и трейсы объединены в одном интерфейсе. Это позволяет исследовать данные, не переключаясь между разными сервисами, и быстрее находить причину проблем. А также проводить сквозную диагностику, быстро переходя между разными видами телеметрии и связывая их между собой для точечного и детального анализа.
* Добавлена возможность отслеживать уровень надежности ваших сервисов в определенные периоды времени — [SLO](slo/index.md) (Service Level Objective).
* Реализована поддержка сбора телеметрии по стандарту OTLP.
* При миграции на платформу {{ monium-name }} будут автоматически перенесены ваши данные, дашборды и алерты без ручного переноса и настройки.
