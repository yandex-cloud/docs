# История изменений в {{ monium-name }}

## Январь — март 2026 {#jan-mar-2026}

* Платформа {{ monium-name }} доступна на стадии [General Availability](../overview/concepts/launch-stages.md).
* В [каналах уведомлений](concepts/alerting/notification-channel.md) и политиках эскалаций добавлена поддержка [Яндекс Мессенджера](https://360.yandex.ru/business/messenger/).

* Обновления в [дашбордах](visualization/index.md):
  * Добавлен виджет состояния алертов. Виджет автоматически отображает все алерты в проекте по указанному запросу, а также добавляет новые по мере их появления.
  * Добавлена поддержка горячих клавиш для базовых действий: сворачивание и разворачивание секций, переход в полноэкранный режим, сохранение дашборда, управление шкалой времени и периодом.
  * Добавлена возможность задать произвольный временной интервал и интервал автообновления для дашборда.
  * Для графиков добавлена возможность отображать несколько подсказок одновременно и переносить их с помощью курсора.
  * В блокнотах добавлена поддержка графиков {{ prometheus-name }} и возможность настраивать графики внутри блокнота.
  * При копировании дашборда добавлена возможность изменить проект.
  * При разбивке графика шкала по умолчанию выравнивается для всех графиков, что позволяет лучше видеть корреляции данных.

* Обновления в [трейсах](traces/index.md):
  * В поиске по спанам и в LLM-мониторинге добавлена настройка отображаемых колонок. Теперь можно вывести нужные атрибуты в колонки и сразу видеть их значения, не открывая трейс.
  * Добавлена кнопка быстрого перехода в интерфейс LLM-мониторинга из полного трейса, если в нем есть спаны с атрибутами LLM-приложений (`gen_ai.*`).
  * Добавлена возможность быстро копировать значение любого атрибута спана: при наведении на атрибут появляется кнопка копирования.
  * Добавлена возможность копировать и передавать ссылку на конкретный спан в LLM-мониторинге.
  * Оптимизирована производительность интерфейса LLM-мониторинга при работе с большими трейсами.


## Октябрь — декабрь 2025 {#oct-dec-2025}

* Платформа {{ monium-name }} доступна на стадии [Technical Preview](../overview/concepts/launch-stages.md). Для получения доступа обратитесь в [службу поддержки]({{ link-console-support }}).
* Реализован сбор, агрегация, поиск и хранение распределенных [трассировок](traces/index.md) по стандарту [OpenTelemetry (OTLP)](https://opentelemetry.io/docs/). Это позволяет отслеживать полный путь запроса через все микросервисы и системы, что упрощает диагностику проблем и анализ производительности.
* Инструменты телеметрии — метрики, логи и трейсы объединены в одном интерфейсе. Это позволяет исследовать данные, не переключаясь между разными сервисами, и быстрее находить причину проблем. А также проводить сквозную диагностику, быстро переходя между разными видами телеметрии и связывая их между собой для точечного и детального анализа.
* Добавлена возможность отслеживать уровень надежности ваших сервисов в определенные периоды времени — [SLO](slo/index.md) (Service Level Objective).
* Реализована поддержка сбора телеметрии по стандарту OTLP.
* При миграции на платформу {{ monium-name }} будут автоматически перенесены ваши данные, дашборды и алерты без ручного переноса и настройки.
