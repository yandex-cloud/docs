# Добавление виджета SLO на дашборд

## Список SLO {#list}

Виджет для отображения списка SLO с актуальными значениями SLI и остатка Error Budget. При нажатии на название SLO откроется страница просмотра.

{% note info %}

Изменение временного интервала дашборда не влияет на виджет SLO, он всегда отображает значения показателей на данный момент.

{% endnote %}
 
## Настройки виджета {#settings}

* **Заголовок** — заголовок виджета.
* **Проект** — проект {{ monium-name }}, в котором будет осуществляться поиск.
* **Селектор** — селектор для поиска SLO в проекте.
* **Сортировать по** — выбор поля и направление сортировки списка SLO, по умолчанию SLO сортируются по возрастанию остатка Error Budget.

Доступные метки для поиска и фильтрации SLO:

* `id` — ID SLO;
* `name` — название SLO;
* `labels.LABEL_NAME` — лейблы SLO;
* `status` — статус вычисления SLO;
* `window` — окно вычисления SLO.

Благодаря использованию селектора достаточно один раз настроить виджет, а при добавлении новых SLO в проект они будут показаны на виджете, если попадут под условие селектора и фильтра.
