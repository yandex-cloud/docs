# Поиск трейсов и спанов

Интерфейс {{ traces-name }} позволяет находить трейсы и спаны по заданным критериям, открывать трейс по идентификатору и просматривать агрегирующие графики по атрибутам спанов.

## Открытие трейса по идентификатору {#open-by-id}

Если вам известен идентификатор трейса (`trace_id`), введите его в поле **{{ ui-key.yacloud_monitoring.traces.trace-id-input.placeholder }}** в правом верхнем углу страницы и нажмите **Перейти**. Откроется страница [просмотра трейса](trace-view.md).

## Временной диапазон {#time-range}

Поиск выполняется в пределах выбранного временного диапазона. По умолчанию — последние 24 часа. Измените диапазон, выбрав нужный интервал на временной шкале.

## Режим поиска трейсов {#search-traces}

В этом режиме результатом поиска являются **трейсы**, содержащие хотя бы один спан, который удовлетворяет условиям запроса.

Введите запрос в строку поиска, используя синтаксис [языка запросов](query-language.md). Строка поиска поддерживает автодополнение: сначала начните вводить имя атрибута — интерфейс предложит подходящие варианты. После выбора атрибута начните вводить значение — автодополнение сработает и для него.

```
{ project="foo", service="bar", span.duration>=1s }
```

Результаты отображаются в виде списка трейсов. Для каждого трейса показаны: идентификатор, корневой сервис, длительность и количество спанов. Нажмите на трейс, чтобы перейти к его [просмотру и анализу](trace-view.md).

## Режим поиска спанов {#search-spans}

В этом режиме результатом являются отдельные **спаны**, которые удовлетворяют всем условиям запроса. Используйте этот режим, чтобы искать конкретные операции по атрибутам, без привязки к целому трейсу.

Поиск по спанам поддерживает те же ключи, что и поиск по трейсам, а также поиск по произвольным атрибутам (лейблам) спанов. Подробнее о поддерживаемых ключах и операторах см. [{#T}](query-language.md).

Помимо списка спанов, на странице отображаются графики по результатам выборки: количество спанов и их длительность в разных процентилях. Данные на графиках группируются по атрибуту, который можно выбрать самостоятельно.
