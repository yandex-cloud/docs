# Текстовый редактор Vim

Вы можете установить плагин редактора [Vim](https://www.vim.org/) для работы с вики-разметкой Яндекса в консольном режиме. [Подробнее о работе плагина](https://wiki.yandex-team.ru/arcadia/ide/vim/).

Исходные файлы плагина вы можете найти по [ссылке](https://github.com/elricbk/vim-yawiki).
