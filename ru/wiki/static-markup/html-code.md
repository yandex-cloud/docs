# Элементы HTML

Чтобы вставить на страницу элементы оформления, которые не предусмотрены разметкой Markdown, вы можете использовать в тексте теги HTML.

Например:

* ```
  <p style="color: gray; font-weight: bold;">Текст серого цвета</p>
  ```
