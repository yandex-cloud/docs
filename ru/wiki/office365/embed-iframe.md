# Вставить документ на вики-страницу

Чтобы вставить на страницу документ Office 365 вместе с другим контентом, оформленным с помощью вики-разметки, можно использовать [динамический блок `iframe`](../actions/iframe.md).

1. На [портале Sharepoint]({{link-sharepoint-common}}) или в [личной папке OneDrive]({{link-sharepoint-personal}}) откройте документ, который нужно вставить на вики-страницу.

1. Получите код для встраивания:
   1. Откройте документ в браузере.
   1. Выберите меню **Файл** → **Поделиться** → **Внедрить**  (в PowerPoint меню **Файл** → **Общий доступ → **Внедрение**).
   1. Настройте параметры вставки документа.
   1. Скопируйте текст из поля **Код внедрения**.

1. Вставьте код в текст вики-страницы.

1. Отредактируйте код, чтобы получился работающий динамический блок: 
   * Удалите текст между тегами `<iframe></iframe>`.
   * Теги `<iframe></iframe>` замените на `not_var{{iframe}}`.
   * Символы `&amp;` замените на `&`.
   
   {% cut "Пример" %}
   
   Код встраивания документа:

   ```html
   <iframe src="https://yandexteam-my.sharepoint.com/personal/login_yandex-team_ru/_layouts/15/Doc.aspx?sourcedoc={95eb752a-0736-4071-b8b2-83792c6a825f}\&amp;action=embedview\&amp;wdStartOn=1" width="476px" height="288px" frameborder="0">Это внедренный файл <a target="_blank" href="https://office.com">Microsoft Office</a> на платформе <a target="_blank" href="https://office.com/webapps">Office</a>.</iframe>
   ```
   
   Блок iframe:
   
   ```
   {{iframe src="https://yandexteam-my.sharepoint.com/personal/login_yandex-team_ru/_layouts/15/Doc.aspx?sourcedoc={95eb752a-0736-4071-b8b2-83792c6a825f}&action=embedview&wdStartOn=1" width="476px" height="288px" frameborder="0"}}
   ```
   {% endcut %}

   {% cut "Дополнительные параметры для вставки документов Excel" %}

   Параметр URL-адреса | Действие | По умолчанию
   --- | --- | ---
   wdHideSheetTabs=True | Скрывает ярлычки, которые отображаются внизу всех листов в книге. | False
   wdAllowInteractivity=True | Позволяет сортировать и фильтровать данные в таблицах и сводных таблицах книги. | True
   wdHideGridlines=True | Скрывает линии сетки на листе. | False
   wdHideHeaders=True | Скрывает заголовки строк и столбцов. | False
   Item=название_элемента | Выводит определенный элемент. Если книга содержит диаграмму, таблицу, сводную таблицу или именованный диапазон и на веб-странице необходимо отобразить только этот элемент, его можно указать с помощью параметра Item. Сведения об именованных диапазонах и именованных элементах в книгах Excel см. в статье [Определение и использование имен в формулах]({{link-excel}}). | Не задано
   ActiveCell=имя_ячейки | Определяет, какая ячейка будет активной (выбранной) во внедренной книге при открытии веб-страницы. Ячейку можно указать путем ссылки на нее (например, A1) или по имени. | Выводится последнее сохраненное представление

   {% endcut %}
   
1. Сохраните страницу.
