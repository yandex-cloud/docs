Чтобы добавить на страницу видео, используйте [динамический блок not_var{{iframe}}](actions/iframe.md). Он позволяет вставлять видео из внешних источников, таких как Vimeo или Youtube.

Если нужно выложить видео только для сотрудников Яндекса, [загрузите](https://wiki.yandex-team.ru/users/askarkamalov/kak-zalivat-nda-video-v-videoxosting/) его на NDA-канал в сервисе Видеохостинг. 

{% endif %}

1. Откройте страницу и нажмите кнопку **Редактировать**.

1. Добавьте на страницу код динамического блока со ссылкой на видео.

Пример:

```
{{iframe src="https://player.vimeo.com/video/207028770" width="640" height="300" frameborder="0" allow="autoplay; fullscreen" allowfullscreen></iframe}}
```

{% note tip %}

При добавлении видео из Youtube используйте ссылку, которая предназначена для размещения на сторонних ресурсах. Такая ссылка имеет вид: 
```
https://www.youtube.com/embed/oCRQj_zyPjk
```
Чтобы получить ссылку, на Youtube под видео выберите **Поделиться** → **Встроить** и скопируйте значение параметра `src`.

{% endnote %}

## Вставить файл с видео или аудио {#post-file}

Медиафайлы можно загрузить на страницу в виде вложений или разместить в тексте при помощи тегов [HTML](static-markup/html-code.md). 

1. Загрузите файл одним из способов:

    * Используйте хостинг, где файл будет доступен другим пользователям. Например, корпоративный файловый хостинг [Jing](https://jing.yandex-team.ru/). В этом случае файл будет доступен всем сотрудникам на любых страницах {{ wiki-name }}.

    * Прикрепите файл к вики-странице: на панели слева нажмите значок ![](../_assets/wiki/attach-btn.png) и перетащите файл. В этом случае он будет доступен пользователям, которые имеют доступ к странице.

1. Скопируйте ссылку на загруженный файл. Чтобы получить ссылку на файл, прикрепленный к вики-странице:

    1. На панели слева наведите курсор мыши на файл.

    1. Щелкните правой кнопкой мыши значок загрузки ![](../_assets/wiki/download-btn.png) и в меню выберите **Копировать адрес ссылки**.

1. Разместите ссылку на странице с помощью HTML-тегов:

    * для видеофайлов используйте тег `<video>`;

    * для аудиофайлов используйте тег `<audio>`.

Пример:

```html
<video src="https://jing.yandex-team.ru/files/mishanikin/afisha.mov"
width="640" height="480" controls="controls">Ваш браузер не поддерживает это видео.
<a href="https://jing.yandex-team.ru/files/mishanikin/afisha.mov"
target="_blank">Скачайте видео</a></video>
```

{% note info %}

В {{ tracker-name }} и Этушке используется старый вики-форматер, поэтому фрагменты HTML-кода в описании тикетов и комментариях нужно окружить символами `<# ... #>`.

{% endnote %}
