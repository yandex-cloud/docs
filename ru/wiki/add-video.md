# Вставить видео или аудио

{% if audience == "internal" %}

Для размещения и хранения видео и аудио рекомендуется использовать корпоративный файловый хостинг [Jing](https://jing.yandex-team.ru/). 

## Ролик с видеохостинга {#hosting}

{% endif %}

Чтобы добавить на вики-страницу видео или аудио, используйте [динамический блок not_var{{iframe}}](actions/iframe.md). Он позволяет вставлять медиафайлы из внешних источников, таких как Яндекс&#160;Музыка, Vimeo или Youtube.

{% if audience == "internal" %}

Если нужно выложить видео только для сотрудников Яндекса, [загрузите](https://wiki.yandex-team.ru/users/askarkamalov/kak-zalivat-nda-video-v-videoxosting/) его на NDA-канал в сервисе Видеохостинг. 

{% endif %}

1. Откройте вики-страницу и нажмите кнопку **Редактировать**.

1. Добавьте на вики-страницу код динамического блока со ссылкой на медиафайл. Используйте ссылку, которая предназначена для размещения на сторонних ресурсах. Чаще всего такие ссылки называются «Код для вставки» или «HTML-код». 

{% note tip %}

* При добавлении видео из Youtube используйте ссылку вида:
    ```
    https://www.youtube.com/embed/oCRQj_zyPjk
    ```
    Чтобы получить такую ссылку, на Youtube под видео выберите **Поделиться** → **Встроить** и скопируйте значение параметра `src`.

* При добавлении трека из Яндекс&#160;Музыки используйте ссылку вида:

    ```
    {{ yandex-music }}/iframe/#track/71263/419460
    ```
    Чтобы получить такую ссылку, в меню трека выберите **Поделиться** → **HTML-код** и скопируйте значение параметра `src`.

{% endnote %}

Пример динамического блока с видео:

```
{{iframe src="https://player.vimeo.com/video/207028770" width="640" height="300" frameborder="0" allow="autoplay; fullscreen" allowfullscreen}}
```

Пример динамического блока с аудио:

```
{{iframe frameborder="0" style="border:none;width:100%;height:180px;" width="100%" height="180" src="{{ yandex-music }}/iframe/#track/71263/419460"}}
```

## Вставить файл с видео или аудио {#post-file}

Медиафайлы можно загрузить на вики-страницу в виде вложений или разместить в тексте при помощи тегов [HTML](static-markup/html-code.md). 

1. Загрузите файл одним из способов:

    * Используйте хостинг, где файл будет доступен другим пользователям. Например, корпоративный файловый хостинг [Jing](https://jing.yandex-team.ru/). В этом случае файл будет доступен всем сотрудникам на любых страницах {{ wiki-name }}.

    * Прикрепите файл к [вики-странице](attach-file.md). В этом случае он будет доступен пользователям, которые имеют доступ к странице.

1. Скопируйте ссылку на загруженный файл. Чтобы получить ссылку на файл, прикрепленный к вики-странице:

    1. В правом верхнем углу страницы нажмите ![](../_assets/wiki/svg/actions-icon.svg) **Действия**, затем выберите ![](../_assets/wiki/svg/attachments.svg) **Файлы**.
	
	1. На панели **Файлы** наведите курсор мыши на файл.

    1. Щелкните правой кнопкой мыши значок ![](../_assets/wiki/svg/download.svg) и в меню выберите **Копировать адрес ссылки**.

1. Разместите ссылку на странице с помощью HTML-тегов:

    * для видеофайлов используйте тег `<video>`;

    * для аудиофайлов используйте тег `<audio>`.

Пример:

```html
<video src="https://jing.yandex-team.ru/files/mishanikin/afisha.mov"
width="640" height="480" controls="controls">Ваш браузер не поддерживает это видео.
<a href="https://jing.yandex-team.ru/files/mishanikin/afisha.mov"
target="_blank">Скачайте видео</a></video>
```

{% note info %}

В {{ tracker-name }} и Этушке используется старый вики-форматер, поэтому фрагменты HTML-кода в описании тикетов и комментариях нужно окружить символами `<# ... #>`.

{% endnote %}
