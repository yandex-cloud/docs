# Вставить видео

Чтобы добавить на страницу видео, используйте [динамический блок not_var{{iframe}}](actions/iframe.md). Он позволяет вставлять видео из внешних источников, таких как Vimeo или Youtube.

1. Откройте страницу и нажмите кнопку **Редактировать**.

1. Добавьте на страницу код динамического блока со ссылкой на видео.

Пример:

```
{{iframe src="https://player.vimeo.com/video/207028770" width="640" height="300" frameborder="0" allow="autoplay; fullscreen" allowfullscreen></iframe}}
```

{% note tip %}

При добавлении видео из Youtube используйте ссылку, которая предназначена для размещения на сторонних ресурсах. Такая ссылка имеет вид: 
```
https://www.youtube.com/embed/oCRQj_zyPjk
```
Чтобы получить ссылку, на Youtube под видео выберите **Поделиться** → **Встроить** и скопируйте значение параметра `src`.

{% endnote %}
