# Информация о команде сервиса ABC

Блок `serviceteam` показывает состав команды, которая работает над сервисом. Блок может автоматически определить сервис из адреса вики-страницы и сведений в [каталоге сервисов Яндекса](https://abc.yandex-team.ru/). Если в карточке ABC-сервиса есть ссылка на страницу, где расположен блок `serviceteam`, то в блоке отобразится информация о команде этого сервиса.

## Вызов блока {#serviceteam-call}

```
{{serviceteam id=287}} 
```

## Параметры блока {#serviceteam-params}

Параметр | Описание
--- | ---
`id`  |  Числовой идентификатор сервиса. Например: {{wiki-name}} = 287.
`slug` |  Текстовый идентификатор сервиса. Например, в ссылке на сервис {{wiki-name}} `slug` = `_wiki_`: <br>https://abc.yandex-team.ru/services/\_wiki_/.
`contacts` | Контакты сервиса.
