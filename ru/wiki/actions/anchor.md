# Закладка на странице

Блок `anchor` содержит *якорь* — метку, с помощью которой можно переходить к указанной области на странице. Якорь нельзя использовать для перехода к строке в [динамической таблице](../create-grid.md).

## Вызов блока {#anchor-call}

```
{{anchor href="якорь"}}
```
или 
```
{{a href="якорь"}}
```

## Параметры блока {#anchor-params}

Параметр | Описание 
--- | --- 
`href` | Обязательный параметр. Содержит имя якоря.
`title` | Содержит заголовок якоря.
`text` | Содержит текст якоря. Отображается на странице.

Чтобы вставить ссылку на якорь, добавьте его имя справа от адреса страницы через символ `#`:

```
{{ wiki-pagename }}/#Якорь
```

Чтобы вставить ссылку на якорь на той же странице, полный адрес указывать необязательно:

```
#Якорь
```
