# Вставка объектов

## Оглавление {#toc}

Создает автоматическое оглавление страницы на основании ее заголовков.

{% note alert %}

Если на странице есть совпадающие заголовки, ссылки в оглавлении могут работать некорректно.

{% endnote %}

### Вызов блока {#toc-call}

```
{{toc from="h2" to="h6"}}
```

Параметры блока `not_var{{toc}}`.

### Параметры блока (необязательные) {#toc-params}

Параметр | Описание
--- | ---
`from` | Самый высокий уровень заголовков, начиная с которого они включаются в оглавление.
`to` | Самый низкий уровень заголовков, заканчивая которым они включаются в оглавление.

## Список задач {#section_task_list}

Блок содержит список задач {{ tracker-name }}, входящих в одну очередь или удовлетворяющих определенному фильтру.

### Вызов блока {#task-call}

```
{{tasks url="адрес фильтра или очереди"}}
```

Дополнительных параметров у блока `not_var{{tasks}}` нет.

Подробнее о списках задач читайте в [документации {{ tracker-name }}](../../tracker/manager/quick-filters.md#section_uy5_hds_3bb).

## Якорь {#anchor}

Блок содержит *якорь* — метку, с помощью которой можно напрямую переходить к этой области страницы.

### Вызов блока {#anchor-call}

```
{{Anchor href="якорь"}}
```

### Параметры блока {#anchor-params}

Параметр | Описание | Обязательность
--- | --- | ---
`href` | Содержит имя якоря. | Обязательный
`title` | Содержит заголовок якоря. | Необязательный
`text` | Содержит текст якоря. | Необязательный

Чтобы вставить ссылку на якорь, добавьте его имя справа от адреса страницы через символ `#`:

```
[Текст ссылки](http://адресСтраницы#Якорь)
```

Если ссылка находится на той же странице, где якорь, адрес страницы указывать не обязательно:

```
[Текст ссылки](#Якорь)
```

## Страница или видео {#iframe-video}

Блок `iframe` позволяет вставлять на страницу любую другую страницу или видеоролики из внешних источников, таких как Youtube, Vimeo, Coub.

### Вызов блока {#iframe-call}

```
{{iframe src="http://адресСтраницы" width=700px height=600px frameborder=0 scrolling=no}}
```

### Параметры блока {#iframe-params}

Параметр | Описание
--- | ---
`src` | Обязательный параметр. Содержит адрес объекта.
`width` | Указывает на ширину фрейма в пикселях или в процентах от ширины страницы.
`height` | Указывает на высоту фрейма в пикселях.
`frameborder` | Указывает на наличие или отсутствие рамки фрейма.
`scrolling` | Указывает на наличие полос прокрутки для окна фрейма.
`allowfullscreen` | Разрешает для фрейма полноэкранный режим. Всегда имеет значение `true`.

### Разрешенные поддомены {#video-domain}

Параметр `src` может содержать адреса, которые принадлежат доменам:

- `vimeo.com`

- `yandex.ru`

- `yandex.ua`

- `yandex.kz`

- `yandex.by`

- `youtube.com`

- `youtube-nocookie.com`

- `player.vimeo.com`

- `player.vimple.ru`

- `vkontakte.ru`

- `vk.com`

- `coub.com`
