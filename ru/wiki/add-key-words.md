# Добавить ключевые слова (теги)

Для страницы можно задать ключевые слова или теги, по которым ее можно будет легко найти с помощью [строки поиска](search.md):

1. В верхней правой части страницы нажмите ![](../_assets/wiki/svg/actions-icon.svg) **Действия** → **Настройки**.

1. Выберите **Ключевые слова**.

1. Введите список ключевых слов через запятую.

1. Нажмите кнопку **Добавить**.

{% note info %}

Новые ключевые слова могут отображаться в поиске по вики-страницам с задержкой до 30 минут.

{% endnote %}
