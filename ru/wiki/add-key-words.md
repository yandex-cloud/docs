# Как добавить ключевые слова

Для страницы можно задать ключевые слова или теги, по которым страницу можно будет легко найти с помощью [строки поиска](search.md):

1. В верхней правой части страницы нажмите ![](../_assets/wiki/ico-actions.png) **Действия** → **Настройки**.

1. Выберите **Ключевые слова**.

1. Введите список ключевых слов через запятую.

1. Нажмите кнопку **Добавить**.

{% note info %}

Новые ключевые слова могут отображаться в поиске по вики-страницам с задержкой до 30 минут.

{% endnote %}
