# Создание страниц

{{ wiki-name }} поддерживает несколько типов страниц:

## Страница {#page}

Страница с текстом, изображениями, таблицами, прикрепленными файлами и так далее.

* [{#T}](create-page.md)

## Заметка {#note}

Страница в вашем разделе заметок (`{{ wiki-notes }}`). По умолчанию заметки доступны для чтения только вам, но вы можете изменить настройки доступа.

* [{#T}](create-note.md)

## Динамическая таблица {#grid}

Таблица с возможностью задать тип данных столбца, отсортировать значения или сделать ячейку обязательной для заполнения. Вы можете использовать таблицу напрямую или [встроить ее в страницу](add-grid.md).

* [{#T}](create-grid.md)

* [{#T}](import-export-grid.md)
