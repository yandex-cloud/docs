# Как назначить автора страницы

У страницы может быть один или несколько авторов. По умолчанию автор страницы — пользователь, который ее создал.

{% note alert %}

Назначить нового автора страницы может только ее действующий автор.

{% endnote %}

Чтобы назначить другого пользователя автором вашей страницы:

1. Откройте страницу.

1. В правом верхнем углу страницы нажмите кнопку **Действия** и выберите **Настройки**.

1. В окне **Настройки страницы** нажмите строку **Авторы страницы** и укажите логин нового автора страницы.

1. Нажмите кнопку **Сохранить**.
