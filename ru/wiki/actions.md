# Динамические блоки

С помощью динамического блока на [вики-страницу](pages-types.md#page) можно вставить фрагмент, содержимое которого формируется автоматически на основе данных о странице или информации из внешнего источника.

С помощью динамических блоков вы можете:

* построить [списки страниц](actions/page-lists.md), связанных с заданной;

* встроить в вики-страницу [динамическую таблицу](actions/grid-reference.md);

* сформировать [оглавление документа](actions/toc.md);

* вставить на вики-страницу [объекты из внешних источников](actions/iframe.md);

* настроить интеграцию с [{{ forms-full-name }}](actions/forms.md) и [{{ tracker-full-name }}](actions/tracker.md).

Общий формат вызова динамических блоков:

```
{{имяБлока параметр1="значение1" параметр2="значение2" ...}}
```
