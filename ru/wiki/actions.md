# Динамические блоки

Динамический блок — фрагмент страницы, содержимое которого формируется автоматически на основе данных о странице или информации из внешнего источника.

С помощью динамических блоков вы можете:

* построить [списки страниц](actions/page-lists.md), связанных с заданной;

* встроить в страницу [динамическую таблицу](actions/grid-reference.md);

* сформировать [оглавление документа](actions/objects.md#toc).

Общий формат вызова динамических блоков:

```
{{имяБлока параметр1="значение1" параметр2="значение2" ...}}
```
