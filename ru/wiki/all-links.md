# Посмотреть все ссылки на страницу

Вы можете посмотреть список всех страниц, ссылающихся на вики-страницу.

Эта функция поможет подготовиться к удалению страницы: найти и убрать все связанные с ней ссылки.

1. Откройте страницу.

1. В правом верхнем углу страницы нажмите ![](../_assets/wiki/svg/actions-icon.svg) **Действия** и выберите ![](../_assets/wiki/svg/delete-page.svg) **Удалить**.

1. Подождите несколько секунд, пока данные загрузятся.

{% note warning %}

Этот метод не найдет ссылку, которая ведет не на саму страницу, а на [якорь](static-markup/links#section-link) на ней. {% if audience == "internal" %}Страницы, в которые данная страница вставлена при помощи [динамического блока `include`](actions/include) тоже не будут найдены.{% endif %}

{% endnote %}
