# Тикет из Лунапарка

[Лунапарк](http://lunapark.yandex-team.ru/) — внутренний сервис для нагрузочного тестирования. 

Чтобы отобразить на странице в {{wiki-name}} ссылку на задачу или другой объект из Лунапарка, скопируйте его адрес из адресной стройки браузера, например:
* `http://lunapark.yandex-team.ru/MAPSRENDER-308` 
* `http://lunapark.yandex-team.ru/100` 

В результате на странице появится магическая ссылка на задачу или объект из Лунапарка:
