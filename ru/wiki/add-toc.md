# Вставить оглавление страницы

Вы можете вставить на страницу оглавление, формируемое автоматически на основе ее заголовков. Для этого:

1. Убедитесь, что заголовки страницы [правильно размечены](basic-markup.md#headers).

1. Создайте оглавление с помощью динамического блока:
   
    ```
    not_var{{toc}}
    ```


По умолчанию в оглавление будут попадать все заголовки текущей страницы. Вы можете настроить вывод оглавления с помощью [дополнительных параметров](actions/toc.md).

> Пример оглавления, в которое включены заголовки от первого (`h1`) до третьего (`h3`) уровня.
> 
> ```
> {{toc from="h1" to="h3"}}
> ```
