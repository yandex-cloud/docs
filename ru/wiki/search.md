# Поиск по {{ wiki-name }}

Для поиска по {% if audience == "external" %}тексту вики-страниц{% else %}{{ wiki-full-name }} и другим сервисам Интранета{% endif %}:

1. На левой панели {{ wiki-name }} нажмите значок ![](../_assets/wiki/svg/search.svg) **Поиск**.

1. Введите запрос в строку поиска. Под строкой отобразятся вики-страницы, в которых встречается указанный текст. 

{% if audience == "external" %}

1. Чтобы уточнить результаты, в строке поиска нажмите клавишу **Enter** и используйте фильтры на панели справа:

   - **Раздел**. Адрес [раздела](structure.md#clusters) относительно главной страницы `{{ wiki-host-name }}`. Например, чтобы найти текст на подстраницах раздела `{{ wiki-username }}`, введите `users/имяПользователя`.
   - **Автор страницы** — имя или логин автора страницы {{ wiki-name }}.
   - **Дата создания** или **Дата изменения** — период, в который страница была создана или на нее были добавлены последние изменения.
   - **Тип документа** — поиск по документам {{ wiki-name }} определенного типа: страница, таблица или файл.

   {% if audience == "draft" %}Чтобы в результаты поиска попали страницы, помеченные как [неактуальные](exp-page.md), выберите **Показывать устаревшие страницы**.{% endif %}

1. Отсортируйте результаты поиска: по релевантности, по дате изменения или по дате создания.

{% endif %}

{% if audience == "internal" %}

1. Под строкой поиска вы можете выбрать имя сервиса, в котором надо выполнить поиск.

1. Для поиска по текущему разделу и его подстраницам, выберите **Вики** и нажмите **Только в текущем разделе**.

{% endif %}

{% note warning %}

Новые изменения на вики-страницах могут появляться в результатах поиска с задержкой, поскольку индексирование страниц выполняется раз в 30 минут.

{% endnote %}
