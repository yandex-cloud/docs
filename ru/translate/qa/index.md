#### Могу ли я пометить фрагмент текста, который не нужно переводить? {#notranslate}

Да, при переводе текста в формате `HTML` фрагмент текста можно исключить из перевода с помощью атрибута `translate="no"` элемента-контейнера:

```html
Это <span translate="no">непереведенный</span> текст.
```
