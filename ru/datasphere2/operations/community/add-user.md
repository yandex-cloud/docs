# Добавить пользователя в сообщество

{% include [include](../../../_includes/datasphere2/organization-users.md) %}

{% list tabs %}

- Консоль управления

  1. Откройте [главную страницу]({{ link-datasphere-main }}) {{ ml-platform-name }}.
  1. Перейдите на вкладку **Мои сообщества**.
  1. Выберите сообщество, в которое вы хотите добавить пользователей.
  1. Перейдите на вкладку **Участники** и выберите способ приглашения:

     * Выбрать из списка

       {% include [include](../../../_includes/datasphere2/select-from-list.md) %}

     * Поделиться ссылкой

       Нажмите кнопку **Создать пригласительную ссылку**. Скопируйте ссылку для приглашения и отправьте ее пользователю.

  После принятия приглашения или перехода по ссылке пользователь станет участником сообщества. Ему будут доступны все проекты и ресурсы этого сообщества.
