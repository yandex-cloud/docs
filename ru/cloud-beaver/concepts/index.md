# Взаимосвязь ресурсов в {{ cloud-beaver.name }}

## Connections {#connections}

Для каждой базы данных в {{ cloud-beaver.name }} создается объект *Connection*, в котором содержатся сведения о параметрах соединения, имени базы и учетных данных пользователя, которые следует использовать для подключения.
