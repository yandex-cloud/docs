# Удалить группу рабочих столов

{% list tabs %}

- Консоль управления

  1. В [консоли управления]({{ link-console-main }}) выберите каталог, в котором расположена группа рабочих столов.
  1. В списке сервисов выберите **Cloud Desktop**.
  1. Выберите группу рабочих столов → нажмите значок ![image](../../../_assets/options.svg) → выберите пункт **Удалить**.

      {% note tip %}

      Чтобы выполнить это действие с несколькими группами рабочих столов, выделите нужные в списке → нажмите кнопку **Удалить** в нижней части экрана.

      {% endnote %}

  1. Подтвердите удаление.

{% endlist %}
