# Создать рабочий стол

Чтобы создать рабочий стол:

{% list tabs %}

- Консоль управления
  
  1. В [консоли управления]({{ link-console-main }}) выберите каталог, в котором будет создан рабочий стол.
  
  1. В списке сервисов выберите **Cloud Desktop**.
  
  1. Перейдите на вкладку **Рабочие столы**.
  
  1. Нажмите кнопку **Создать рабочий стол**.
  
  1. Выберите группу рабочих столов, в котором будет создан рабочий стол.
  
  1. Укажите пользователя, для которого создается рабочий стол.
  
  1. Нажмите кнопку **Создать**.

Рабочий стол появится в списке, после этого вы можете к нему [подключиться](connect.md).

{% endlist %}
