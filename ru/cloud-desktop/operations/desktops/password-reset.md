# Сбросить пароль пользователя для рабочего стола

1. В [консоли управления]({{ link-console-main }}) выберите каталог, в котором расположен рабочий стол.

1. В списке сервисов выберите **Cloud Desktop**.

1. Перейдите на вкладку **Рабочие столы** и выберите рабочий стол.

1. В области **Доступ** напротив пользовательского аккаунта нажмите значок ![image](../../../_assets/options.svg) и выберите **Сбросить пароль**.
    
    После сброса временный пароль будет скопирован в буфер обмена.
