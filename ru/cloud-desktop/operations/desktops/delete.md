# Удалить рабочий стол

{% list tabs %}

- Консоль управления

  1. В [консоли управления]({{ link-console-main }}) выберите каталог, в котором расположен рабочий стол.
  1. В списке сервисов выберите **{{ cloud-desktop-name }}**.
  1. На панели слева выберите ![image](../../../_assets/cloud-desktop/desktops.svg) **Рабочие столы**.
  1. Выберите рабочий стол → нажмите значок ![image](../../../_assets/options.svg) → выберите пункт **Удалить**.

     {% note tip %}

     Чтобы выполнить это действие с несколькими рабочими столами, выделите нужные в списке → нажмите кнопку **Удалить** в нижней части экрана.

     {% endnote %}

  1. В открывшемся окне нажмите кнопку **Удалить**.
