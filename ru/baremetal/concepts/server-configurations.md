# Конфигурации серверов


Вы можете арендовать серверы следующих конфигураций при наличии свободных мощностей в нужных пулах:


<MDX>
  <Calculator services={["baremetal"]} regionConfig={{"ru":{"available": true},"kz":{"available": true},"animated":false,"title":{"text":"Рассчитать стоимость"}}} />
</MDX>


Конфигурации, доступные для аренды в каждом пуле, можно посмотреть при заказе сервера в [консоли управления]({{ link-console-main }}).

Подробнее о том, как арендовать сервер, см. в инструкции [{#T}](../operations/servers/server-lease.md).
