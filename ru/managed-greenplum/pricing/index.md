# Действующие правила тарификации для {{ mgp-name }}

{% note info %}

Указанные тарифы действуют до 13 марта 2022 года включительно. Тарифы с 14 марта 2022 года см. в разделе [Правила тарификации на стадии General Availability](general-availability.md).

{% endnote %}

В зависимости от [типа хоста](../concepts/index.md) стоимость использования {{ mgp-name }} вычисляется по-разному:

* Стандартные хосты

  На стадии [Preview](../../overview/concepts/launch-stages.md) использование сервиса {{ mgp-name }} не тарифицируется.

* Выделенные хосты

  На стадии [Preview](../../overview/concepts/launch-stages.md) оплачиваются только [вычислительные ресурсы {{ compute-full-name }}](../../compute/pricing.md#prices-dedicated-host).
