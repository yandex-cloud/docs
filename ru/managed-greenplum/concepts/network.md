## Имя хоста и FQDN {#hostname}

Имя для каждого хоста в кластере {{ mgp-short-name }} генерирует при его создании. Это имя будет являться доменным именем хоста (FQDN). Имя хоста и, соответственно, FQDN невозможно изменить.
