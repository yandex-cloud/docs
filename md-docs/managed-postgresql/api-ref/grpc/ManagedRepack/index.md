# Managed Service for PostgreSQL API, gRPC: ManagedRepackService

A set of methods to manipulate with Repack in Postgresql cluster

## Methods

#|
||Method | Description ||
|| [ListRunningRepacks](listRunningRepacks.md) | Lists currently running in given cluster Repacks ||
|| [LaunchRepack](launchRepack.md) | Launches new Repack in given cluster ||
|| [TerminateRepack](terminateRepack.md) | Terminates currently running Repack in given cluster ||
|#
