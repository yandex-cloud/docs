# Просмотр информации об отправленных письмах

1. Перейдите в сервис **Cloud Postbox**.
1. Перейдите на вкладку **Отправленные письма**. На странице отобразятся отправленные письма за указанный период.
1. Выберите письмо, чтобы посмотреть историю его событий, например информацию о том, когда письмо доставили получателю или когда он открыл его. Для поиска нужного письма можно воспользоваться фильтром.
