# Ограничение списка отправителей в Yandex Cloud Postbox

Ограничить список отправителей можно во время или после [создания](create-address.md) адреса. Чтобы ограничить список отправителей для уже созданного адреса:

{% list tabs group=instructions %}

- Консоль управления {#console}

    1. В [консоли управления](https://console.yandex.cloud) перейдите в [каталог](../../resource-manager/concepts/resources-hierarchy.md#folder), в котором находится адрес.
    1. Перейдите в сервис **Cloud Postbox**.
    1. В правой части строки адреса, для которого вы хотите ограничить список отправителей, нажмите ![image](../../_assets/console-icons/ellipsis.svg) и выберите **Редактировать**.
    1. Включите опцию **Ограничить список отправителей**.
    1. Настройте список отправителей:
       * Чтобы добавить отправителя, в поле **Разрешённые отправители** укажите имя без домена и нажмите **Добавить отправителя**.
       * Чтобы удалить отправителя, в правой части строки нажмите ![image](../../_assets/console-icons/trash-bin.svg).
    1. Нажмите кнопку **Сохранить**.

{% endlist %}
