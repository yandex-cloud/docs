# История изменений в Yandex Cloud Postbox

## Июнь 2026 {#june-2026}

### Обновления {#updates}

* Добавлена поддержка [массовой отправки писем](aws-compatible-api/api-ref/send-bulk-email.md) через AWS-совместимый HTTP API. Теперь метод `SendBulkEmail` позволяет в одном запросе отправить несколько шаблонных писем — с общим шаблоном и индивидуальными для каждого письма данными для подстановки, адресатами, заголовками и метками.
* Добавлена поддержка вложений в письмах, отправляемых через AWS-совместимый HTTP API. Файлы передаются в параметре `Attachments`:

    * в методе [SendEmail](aws-compatible-api/api-ref/send-email.md) — в содержимом типа `Simple` или `Template`;
    * в методе [SendBulkEmail](aws-compatible-api/api-ref/send-bulk-email.md) — в общем содержимом `DefaultContent.Template`.

    Поддерживаются как обычные вложения, так и вложения, встроенные в тело письма (например, изображения в HTML). Подробнее об [ограничениях на вложения](concepts/restrictions.md).

## Апрель 2026 {#april-2026}

### Обновления {#updates}

* Добавлена поддержка полного цикла обработки жалоб на письма: теперь сервис принимает жалобы от интернет-провайдеров, парсит их, определяет отправителя и отправляет пользователю [уведомление о жалобе](concepts/notification.md#complaint). В [статистике](concepts/statistics.md) по отправленным письмам в консоли управления появилась отдельная метрика жалоб.
* Добавлена поддержка [просмотра](operations/messages.md) информации об отправленных письмах в консоли управления.

## Февраль 2026 {#february-2026}

### Обновления {#updates}

* Добавлена поддержка простого способа настройки подписи DKIM (Easy DKIM).

## Октябрь 2025 {#october-2025}

### Обновления {#updates}

* Добавлена поддержка ограничения списка отправителей для адреса: теперь при [создании адреса](operations/create-address.md) можно добавить список почтовых ящиков, с которых будет разрешена отправка писем.

## Сентябрь 2025 {#september-2025}

### Обновления {#updates}

* Добавлена графическая [статистика](concepts/statistics.md) для отслеживания доставки и открытий писем, переходов по ссылкам и отписок.

## Август 2025 {#august-2025}

### Обновления {#updates}

* Добавлена поддержка [отслеживания переходов по ссылкам](concepts/click-tracking.md) в письмах.

## Июль 2025 {#july-2025}

### Обновления {#updates}

* Добавлена поддержка [аутентификации с помощью IAM-токена](api-ref/authentication.md#iam-ses-example).
* Добавлена поддержка получения уведомлений о том, что:
    * [письмо открыто](concepts/notification.md#open);
    * [доставка письма задерживается](concepts/notification.md#delayed-delivery);
    * [получатель отписался от рассылки](concepts/notification.md#subscription).
* Добавлена поддержка [доставки письма по защищенному соединению](aws-compatible-api/api-ref/put-configuration-set-delivery-options.md).

## Май 2025 {#may-2025}

### Обновления {#updates}

* Добавлена поддержка Terraform — теперь можно создавать адреса и проверять владение пользовательским доменом с помощью Terraform. Подробнее в [Создание адреса Yandex Cloud Postbox и проверка владения доменом с помощью Terraform](tutorials/domain-identity-creating.md)

## Март 2025 {#march-2025}

### Обновления {#updates}

* Добавлена поддержка SMTPS — теперь пользователи, чей почтовый клиент не поддерживает STARTTLS, могут отправлять письма с помощью Yandex Cloud Postbox.
* Добавлена поддержка [шаблонизации писем](operations/send-templated-email.md).

## Январь 2025 {#january-2025}

### Обновления {#updates}

* Добавлена поддержка отправки [уведомлений о доставке писем](concepts/notification.md) в поток данных Yandex Data Streams.
* Добавлена поддержка [аутентификации по API-ключу](operations/send-email.md#smtp-send) при отправке писем из почтового клиента по протоколу SMTP.

## Август 2024 {#august-2024}

### Обновления {#updates}

* Добавлена поддержка [записи логов](operations/logs-write.md) в журнал выполнения.
