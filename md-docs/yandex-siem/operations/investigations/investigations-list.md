# Работа со списком расследований

{% note info %}

Функциональность находится на стадии [Preview](../../../overview/concepts/launch-stages.md). Чтобы получить доступ, обратитесь в [техническую поддержку](https://center.yandex.cloud/support) или к вашему аккаунт-менеджеру.

{% endnote %}

В этом разделе описано, как работать со списком расследований: фильтровать по дате, искать по названию и тегам, сортировать по дате изменения.

## Перед началом работы {#before-you-begin}

Раздел Yandex SIEM появится в интерфейсе Cloud Center как модуль Security Deck после одобрения заявки на доступ.

Для работы с сервисом вам потребуется [роль](../../../iam/roles-reference.md#ycdr-admin) `ycem.editor`.

## Открытие списка расследований {#open-list}

Чтобы открыть список расследований:

{% list tabs group=instructions %}

- Интерфейс Cloud Center {#cloud-center}

  1. Перейдите в сервис [Security Deck](https://center.yandex.cloud/security/).
  1. На панели слева выберите **Yandex SIEM**.
  1. Перейдите на вкладку **Расследования**.

  В списке отображаются все расследования, доступные вам в соответствии с правами доступа.

{% endlist %}

## Фильтрация по дате создания {#filter-date}

Чтобы отфильтровать расследования по дате создания:

{% list tabs group=instructions %}

- Интерфейс Cloud Center {#cloud-center}

  1. Перейдите в раздел **Расследования**.
  1. Нажмите на фильтр даты.
  1. Выберите период:
     * конкретную дату;
     * диапазон дат.
  1. Примените фильтр.

  Список обновится и отобразит только расследования, созданные в выбранный период.

{% endlist %}

## Поиск по имени или тегу {#search}

Чтобы найти расследование:

{% list tabs group=instructions %}

- Интерфейс Cloud Center {#cloud-center}

  1. Перейдите в раздел **Расследования**.
  1. Введите поисковый запрос в поле поиска.
  1. Результаты отобразятся автоматически.

  Поиск выполняется по следующим полям:
  * название расследования;
  * описание расследования;
  * теги.

  {% note tip %}

  Используйте теги для быстрого поиска расследований по категориям. Например, теги `prod`, `security-incident`, `february-2024`.

  {% endnote %}

{% endlist %}

## Сортировка по дате изменения {#sort}

Чтобы отсортировать расследования:

{% list tabs group=instructions %}

- Интерфейс Cloud Center {#cloud-center}

  1. Перейдите в раздел **Расследования**.
  1. Нажмите на заголовок столбца **Изменено**.
  1. Выберите порядок сортировки:
     * по возрастанию — от старых к новым;
     * по убыванию — от новых к старым.

  По умолчанию расследования отсортированы по дате изменения от новых к старым.

{% endlist %}

## Просмотр информации о расследовании {#view-info}

В списке расследований для каждого расследования отображается:

* **Название** — краткое описание цели расследования.
* **Дата создания** — время создания расследования.
* **Дата изменения** — время последнего изменения расследования.
* **Теги** — ключевые слова для группировки и поиска.

Чтобы открыть расследование, нажмите на его название в списке.
