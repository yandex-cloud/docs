# Beta-версия интерфейса командной строки

Beta-версия Yandex Cloud CLI интегрирована в <a href="../index.md">основную версию утилиты</a>.

Чтобы опробовать команды beta-версии, используйте префикс <code>yc beta</code> перед командой, например:

<pre><code> yc beta compute instance list </code></pre>
