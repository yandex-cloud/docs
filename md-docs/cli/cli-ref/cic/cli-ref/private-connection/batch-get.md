# yc cic private-connection batch-get

Get list of PrivateConnections by their IDs

#### Command Usage

Syntax:

`yc cic private-connection batch-get`

#### Flags

#|
||Flag | Description ||
|| `--private-connection-ids` | `[]string`

List of IDs of the PrivateConnection resources to return. ||
