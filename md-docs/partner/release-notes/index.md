# История изменений партнерского портала

## I квартал 2026 {#q1-2026}

1. Обновлена страница организации Яндекс 360. Добавлена новая таблица с отображением аддонов, подключенных к тарифу.

1. Добавлена новая страница просмотра данных о тарифе Яндекс 360.

1. Добавлена возможность подключать и отключать аддоны Яндекс 360.

1. Стало доступно публичное API [Партнёрского портала](https://partners.yandex.cloud/) для работы со следующими модулями:

   {% cut "Аккаунты" %}

     * Создание аккаунта
     * Список аккаунтов
     * Данные аккаунта
     * Обновление аккаунта

   {% endcut %}

   {% cut "Сабаккаунты" %}

     * Создание сабаккаунта
     * Список сабаккаунтов
     * Данные сабаккаунта
     * Обновление сабаккаунта
     * Блокировка сабаккаунта
     * Активация сабаккаунта
     
   {% endcut %}

   {% cut "Тарифы и аддоны Яндекс 360" %}

     * Получение данных об организации
     * Получение списка тарифов и аддонов организации
     * Получение списка доступных тарифов для организации
     * Добавление тарифа для организации
     * Изменение тарифа в организации
     * Отмена тарифа в организации
     * Получение списка доступных аддонов для тарифа
     * Работа с аддонами по тарифу

   {% endcut %}

   Подробнее о доступных методах см. [Gateway API — Таблицы атрибутов методов](https://disk.yandex.ru/d/gsRt3rHTsTkC4w).

1. Добавлен разовый и регулярный экспорт [партнерской детализации](../operations/get-partner-report.md).

1. Исправлены ошибки при работе с профилем партнера и продуктами для партнерского каталога.

## IV квартал 2025 {#q4-2025}

1. Обновлен дизайн партнерского портала.

1. Добавлена новая [партнерская детализация](../operations/get-client-stat.md) потребления по сабаккаунтам и платежному аккаунту партнера. Теперь можно просматривать свое потребление и потребление клиентов с детализацией по сабаккаунтам, включая скидки и премии.

1. Изменен процесс [создания сабаккаунта](../operations/create-account.md). Теперь сабаккаунт создается только после создания аккаунта партнера. Сабаккаунт можно создать из нового или существующего аккаунта партнера.

1. Улучшена [работа со списком сабаккаунтов](../portal.md#subaccounts). Добавлены постраничное отображение, фильтры и поиск.


1. В партнерский портал добавлено управление тарифами Яндекс 360. В разделе **Квоты и инстанс** для сабаккаунта вы можете:

    * отслеживать связанные организации Яндекс 360;
    * просматривать историю тарифов;
    * создавать, изменять и отменять тарифы.

   {% note warning %}

   Доступ к управлению тарифами Яндекс 360 предоставляется только прямым партнерам. Партнеры, работающие через дистрибьютора, используют инструменты, предоставленные дистрибьютором.

   {% endnote %}


## III квартал 2025 {#q3-2025}

Обновлена навигация партнерского портала.

1. Появились новые разделы:

   * **Поиск по документации** — открывает поисковую строку, в которой по запросу будет подготовлен генеративный ответ по документации [партнерской программы](../quickstart.md). Для подготовки ответа используется сервис Yandex Search API.

   * **Аккаунты** — инструмент для ведения клиентской базы. В этом разделе можно вести список всех потенциальных и текущих клиентов, а также связанных с ними сабаккаунтов. Для удобства добавлена возможность фильтрации и сохранения параметров фильтра. Также теперь можно создавать сабаккаунты из аккаунтов без необходимости дублирования данных аккаунта.

   * **Финансы** — раздел для отслеживания вознаграждения, баланса начислений и партнерской премии. Улучшена детализация: добавилось разбиение по сабаккаунтам, продуктам, сервисам, облакам и каталогам.

   * **Профиль партнера** — раздел с данными о мастер-аккаунте и специализациях. Также в разделе вы сможете управлять профилем и продуктами в партнерском каталоге.

1. Обновлено содержимое раздела **Сабаккаунты**: добавлен новый раздел **Квоты и инстансы**. В этом разделе можно просматривать связанные с сабаккаунтом инстансы — облака и обособленные сервисы Yandex Cloud, например DataLens, Трекер и другие. Кроме того, можно управлять доступностью ресурсов через квоты в облаках. Для настройки ограничений в обособленных сервисах и сабаккаунтах обращайтесь в [поддержку](../../support/index.md).

1. Удален раздел **Дашборд**, а его подразделы **Сабаккаунты** и **Управление доступом** вынесены как самостоятельные.
