# Начало работы с Cloud CDN

Yandex Cloud CDN предоставляет разработчикам веб-сервисов _сеть распространения контента_ (Content Delivery Network, CDN). _CDN-серверы_, расположенные в [точках присутствия](../concepts/points-of-presence.md), получают контент из ваших _источников_, кешируют его и доставляют конечным клиентам по их запросам. Это позволяет снизить нагрузку на источники и сократить время доставки контента конечным пользователям.

Чтобы начать работать с сервисом, выберите подходящую инструкцию:
* [Создание CDN-ресурса с источником-бакетом Yandex Object Storage](bucket.md)
* [Создание CDN-ресурса с источником-сервером в Yandex Cloud](server.md)
