# Правила тарификации для {{ cdn-full-name }}


{% note warning %}

С 1 июля 2026 года начинает действовать новая модель тарификации.

При расчете стоимости использования {{ cdn-name }} будут учитываться:

* Количество [CDN-ресурсов](concepts/resource.md), объем исходящего трафика и количество запросов.
* Платные функции, включенные для ресурсов: [экранирование источников](concepts/origins-shielding.md), [выгрузка логов](concepts/logs.md) и [выделенная IP-адресация](concepts/dedicated-ip-addressing.md).

Тарификация платных функций, включенных для ресурсов, не изменится.

За каждый созданный CDN-ресурс будет взиматься ежемесячная плата за пакет, включающий 150 ГБ бесплатного трафика.

При превышении объема стоимость будет рассчитана по существующему тарифу за каждый 1 ГБ исходящего трафика.

При удалении ресурса, оставшийся трафик обнуляется.
Передача трафика между ресурсами невозможна.

Запросы к CDN-ресурсу будут тарифицироваться по следующим правилам:

* 100 000 000 запросов не тарифицируются.
* При превышении 100 000 000 запросов будет взиматься плата за каждые 100 000 запросов.

{% endnote %}




Чтобы рассчитать стоимость использования сервиса, ознакомьтесь с тарифами в этом разделе.

Цены на продукты сервиса также доступны в [Прайс-листе](https://yandex.cloud/ru/price-list).




Все цены в рублях и тенге указаны с НДС, все цены в долларах — без НДС.

При расчете стоимости использования {{ cdn-name }} учитываются:
* Объем исходящего трафика с CDN-серверов, в том числе запрошенного с пользовательских ресурсов сервисов {{ yandex-cloud }}, например с [виртуальных машин](../compute/concepts/vm.md) {{ compute-full-name }}. Входящий трафик, поступающий на CDN-серверы от сервисов и ресурсов {{ yandex-cloud }} и из интернета, не тарифицируется.
* Платные функции, включенные для ресурсов: [экранирование источников](concepts/origins-shielding.md), [выгрузка логов](concepts/logs.md) и [выделенная IP-адресация](concepts/dedicated-ip-addressing.md).




## Цены для региона Россия {#prices}


{% note info %}

Цены на ресурсы {{ yandex-cloud }} в разных регионах различаются. Подробнее о доступных регионах см. [{#T}](../overview/concepts/region.md).

Валюта, которой можно оплачивать ресурсы, зависит от юридического лица, с которым пользователь заключил договор. Подробнее о регистрации аккаунта см. [{#T}](../billing/quickstart/index.md).

{% endnote %}

В день включения [экранирования источников](concepts/origins-shielding.md) или [выгрузки логов](concepts/logs.md) списывается плата за полный месяц. Далее плата взимается первого числа каждого следующего месяца.

При [подключении выделенной IP-адресации](operations/resources/dedicated-ip-addressing.md) вы можете выбрать схему оплаты:

* За каждый [CDN-ресурс](concepts/resource.md) отдельно. Выгодно, если CDN-ресурсов немного или выделенная IP-адресация нужна только для некоторых из них.
* За все CDN-ресурсы [платежного аккаунта](../billing/concepts/billing-account.md). Выгодно, если выделенная IP-адресация нужна для множества CDN-ресурсов.
