# Cloud CDN API, gRPC: CacheService

A set of methods for managing Cache Service resources.

## Methods

#|
||Method | Description ||
|| [Purge](purge.md) | Removes specified files from the cache of the specified resource. For details about purging, see [documentation](../../../concepts/caching.md#purge). ||
|| [Prefetch](prefetch.md) | Uploads specified files from origins to cache of the specified resource. For defails about prefetching, see [documentation](../../../concepts/caching.md#prefetch). ||
|#
