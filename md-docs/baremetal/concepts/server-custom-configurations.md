# Своя конфигурация сервера BareMetal {#custom}

Если ваша задача требует конкретных аппаратных характеристик сервера, вы можете создать свою [конфигурацию сервера](../operations/servers/custom-server-lease.md) и выбрать:

* Нужный тип центрального процессора (CPU).
* Тип и объем оперативной памяти (RAM).
* Тип, количество и емкость дисковых накопителей.
* Скорость соединения сетевых карт и наличие агрегирования [MC-LAG](mc-lag.md).
* Наличие [резервирования питания](servers.md#server-power-supply).

Yandex Cloud гарантирует наличие запчастей, которые можно использовать в конфигурации. Выбор компонентов и заказ конфигурации осуществляется с помощью консоли управления. Подробнее о том, как арендовать сервер в своей конфигурации, см. в инструкции [Арендовать сервер в своей конфигурации](../operations/servers/custom-server-lease.md).

## Шаблоны конфигураций {#templates}

В качестве основы при создании своей конфигурации сервера вы можете использовать _шаблон_ — совокупность аппаратных характеристик, которая соответствует одной из [готовых конфигураций](server-configurations.md), существующих в Yandex BareMetal. При создании своей конфигурации из шаблона вы можете изменять любые аппаратные характеристики, заданные этим шаблоном.

## Сборка сервера {#assembly}

После заказа сервера в своей конфигурации начинается его сборка. На сборку и запуск такого сервера может потребоваться до четырех календарных дней. После того как сборка завершится и сервер перейдет в [статус](servers.md#server-status) `RUNNING`, вы получите уведомление и сможете приступить к использованию сервера.
