# Готовые конфигурации серверов

Вы можете заказать сервер в одной из готовых конфигураций. Например, выбрать конфигурацию с нужным вам процессором, памятью и количеством [дисков](disks/disk-types.md). Помимо этого, можно подобрать сервер под конкретную задачу, например, серверы с мощным процессором для вычислений, требующих высокой производительности, или серверы для хранения данных с большим количеством объемных дисков.

Конфигурации, доступные для аренды в каждом пуле, можно посмотреть при заказе сервера в [консоли управления](https://console.yandex.cloud).

Подробнее о том, как арендовать сервер в готовой конфигурации, см. в инструкции [Арендовать сервер в готовой конфигурации](../operations/servers/server-lease.md).

Если для ваших задач не нашлось подходящих готовых конфигураций, вы можете собрать [свою конфигурацию](../operations/servers/custom-server-lease.md) из готовых к заказу частей или [связаться с нами](https://yandex.cloud/ru/services/baremetal#contact-form) и заказать [индивидуальную конфигурацию](server-individual-configurations.md).
