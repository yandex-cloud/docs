# Готовые конфигурации серверов

Вы можете заказать сервер в одной из готовых конфигураций. Например, выбрать конфигурацию с нужным вам процессором, памятью, количеством [дисков](disks/disk-types.md), типом и количеством сетевых интерфейсов. Помимо этого, можно подобрать сервер под конкретную задачу, например, серверы с мощным процессором для вычислений, требующих высокой производительности, или серверы для хранения данных с большим количеством объемных дисков.

Подробнее о том, как арендовать сервер в готовой конфигурации, см. в инструкции [Арендовать сервер в готовой конфигурации](../operations/servers/server-lease.md).

Если для ваших задач не нашлось подходящих готовых конфигураций, вы можете собрать [свою конфигурацию](../operations/servers/custom-server-lease.md) из готовых к заказу частей или [связаться с нами](https://yandex.cloud/ru/services/baremetal#contact-form) и заказать [индивидуальную конфигурацию](server-individual-configurations.md).

## Состав конфигурации {#components}

Конфигурации, доступные для аренды в каждом пуле, можно посмотреть при заказе сервера в [консоли управления](https://console.yandex.cloud), а также с помощью [Yandex Cloud CLI](../../cli/cli-ref/baremetal/cli-ref/configuration/get.md) и [API](../api-ref/Configuration/get.md).

Информация о конфигурации включает в себя следующие сведения:

* Имя конфигурации. Например: `LA-i107-S-1/10G`.
* Идентификатор конфигурации в стандартном [формате](../../api-design-guide/concepts/resources-identification.md) Yandex Cloud. Идентификатор доступен только при получении конфигурации с помощью Yandex Cloud CLI и API.
* Объем оперативной памяти.
* Данные о центральном процессоре (процессорах).
* Данные о накопителях.
* Данные о [сетевых интерфейсах](servers.md#network-interfaces). В том числе возможность подключения сетевых интерфейсов к разным типам подсетей:

    * к [приватным](private-network.md#private-subnet) подсетям (`PRIVATE`);
    * к [публичным](public-network.md) подсетям (`PUBLIC`).

В Yandex BareMetal вы можете арендовать серверы готовых конфигураций при наличии свободных мощностей в нужных пулах. Список готовых конфигураций доступен на [странице сервиса](https://yandex.cloud/ru/services/baremetal).
