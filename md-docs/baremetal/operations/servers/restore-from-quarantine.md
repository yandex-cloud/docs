# Восстановить сервер из карантина

{% note info %}

Функция восстановления из карантина доступна только в консоли управления Yandex Cloud.

{% endnote %}

Если сервер находится в карантине, вы можете восстановить его для дальнейшего использования в течение 72 часов. При восстановлении период времени, в течение которого сервер находился в карантине, будет включен в оплачиваемый срок аренды. Подробнее о карантине на странице [Карантин](../../concepts/servers.md#quarantine).

{% list tabs group=instructions %}

- Консоль управления {#console}

  1. В [консоли управления](https://console.yandex.cloud) выберите каталог, которому принадлежит сервер.
  1. Перейдите в сервис **BareMetal**.
  1. В строке с нужным сервером нажмите значок ![image](../../../_assets/console-icons/ellipsis.svg) и выберите **Восстановить**.
  1. В открывшемся окне подтверждения ознакомьтесь с сообщением `Время нахождения сервера в карантине оплачивается после восстановления сервера из карантина.`. Нажмите кнопку **Восстановить**.

{% endlist %}
