# yc baremetal v2 extend stackland-cluster delete

Deletes the specified Stackland cluster.

#### Command Usage

Syntax:

`yc baremetal v2 extend stackland-cluster delete <STACKLAND-CLUSTER-ID>`

#### Flags

#|
||Flag | Description ||
|| `--cloud-id` | `string`

ID of the parent cloud. To get the cloud ID, use a [yandex.cloud.resourcemanager.v1.CloudService.List] request. ||
|| `--folder-id` | `string`

ID of the parent folder. To get the folder ID, use a [yandex.cloud.resourcemanager.v1.FolderService.List] request. ||
|| `--id` | `string`

ID of the cluster to delete. To get the cluster ID, use a [StacklandClusterService.List] request. ||
|| `--async` | Display information about the operation in progress, without waiting for the operation to complete. ||
