# Получение списка хранилищ данных

{% note info %}

Эта функциональность находится на стадии [Preview](../../../overview/concepts/launch-stages.md).

{% endnote %}

{% list tabs group=instructions %}

- Консоль управления {#console}

  1. В [консоли управления](https://console.yandex.cloud) выберите [каталог ресурсов](../../../resource-manager/concepts/resources-hierarchy.md#folder), в котором создан каталог метаданных.
  1. Выберите сервис **Yandex MetaData Hub**.
  1. Hа панели слева выберите ![image](../../../_assets/console-icons/folder-magnifier.svg) **Data Catalog**.
  1. В открывшемся списке выберите каталог метаданных, содержащий нужные хранилища данных.
  1. На панели слева выберите ![image](../../../_assets/console-icons/book.svg) **Хранилище данных**.

{% endlist %}

## Настройка списка хранилищ данных {#setting-list-data-store}

### Найти хранилища данных {#search-data-store}

Найдите нужное хранилище с помощью одного из инструментов:

* Используйте поисковую строку. Для этого над списком хранилищ введите имя хранилища, выберите тип кластера и тип подключения кластера.
* Настройте фильтрацию. Для этого нажмите кнопку **Добавить фильтр** над списком хранилищ и выберите фильтр:
  
  * **Домен** — выберите один или несколько доменов и нажмите **Сохранить**. Будут найдены хранилища, размеченные выбранными доменами.
  * **Тег** — выберите один или несколько тегов и нажмите **Сохранить**. Будут найдены хранилища, размеченные выбранными тегами.
  * **Термин** — выберите один или несколько терминов и нажмите **Сохранить**. Будут найдены хранилища, размеченные выбранными терминами.

  Можно задать несколько фильтров. Для этого повторно нажмите кнопку **Добавить фильтр**.

### Отсортировать хранилища данных {#sort-data-store}

Нажмите на заголовок нужной колонки:

* **Имя** — сортировка по названию хранилища в алфавитном порядке;
* **Дата создания** — сортировка по дате создания в порядке возрастания дат.

Второе нажатие на тот же заголовок сменит направление сортировки. Третье нажатие отменит сортировку.

### Изменить отображение колонок {#setting-columns}

1. В строке заголовков колонок нажмите на значок ![image](../../../_assets/console-icons/gear.svg). Откроется список всех колонок.
1. Выполните в списке колонок нужные действия:

    * Чтобы скрыть ненужные колонки, снимите для них отметки.
    * Чтобы отобразить скрытые колонки, отметьте их.
    * Чтобы изменить порядок колонок, слева от названия колонки нажмите ![image](../../../_assets/console-icons/grip.svg) и перетяните колонку вверх или вниз на нужную позицию в списке.

1. Нажмите кнопку **Применить**.
