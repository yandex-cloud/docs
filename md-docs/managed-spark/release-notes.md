# История изменений в Yandex Managed Service for Apache Spark™

## Сентябрь 2025 {#september-2025}

* 22 сентября 2025 Yandex Managed Service for Apache Spark™ перешел на стадию [General Availability](../overview/concepts/launch-stages.md), и за его использование начала взиматься плата в соответствии с [правилами тарификации](pricing.md).
* Добавлена возможность работать с [заданиями SparkConnect](operations/jobs-sparkconnect.md).

## Август 2025 {#august-2025}

* Сервис Yandex Managed Service for Apache Spark™ стал доступен в [регионе](../overview/concepts/region.md) Казахстан.
* Добавлена возможность [изменять настройки кластера Apache Spark™](operations/cluster-update.md) после его создания.

## Май 2025 {#may-2025}

* Сервис Yandex Managed Service for Apache Spark™ вышел в [Public Preview](../overview/concepts/launch-stages.md).
* Добавлена возможность совершать операции над кластером Apache Spark™ с помощью [Terraform](tf-ref.md) и [API](api-ref/authentication.md).
