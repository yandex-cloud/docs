# Начало работы

При создании первого платежного аккаунта новые клиенты Yandex Cloud получают <a href="usage-grant.md">стартовый грант</a> сроком действия 60 дней. Условия использования гранта различаются для личных и бизнес-аккаунтов.

Кроме того, новым клиентам-юридическим лицам доступен пробный период. Пробный период позволяет получить практический опыт использования Yandex Cloud, оплачивая потребленные ресурсы средствами стартового гранта. Перейти на платную версию с сохранением данных можно в любой момент во время пробного периода или в течение 60 дней после его завершения.

Зарегистрироваться в Yandex Cloud могут резиденты и нерезиденты РФ. При этом нерезиденты РФ и РК могут зарегистрироваться только как юридические лица.
