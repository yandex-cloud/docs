# Политики

При создании группы необходимо определить _политики_, которые описывают в каких [зонах](allocation-policy.md), как [развертывать](deploy-policy.md), [масштабировать](scale-policy.md) и [восстанавливать](healing-policy.md) виртуальные машины в группе.

В [CLI](../../../../glossary/cli.md) описание политик передается вместе с описанием [шаблона виртуальной машины](../instance-template.md) в виде YAML-файла при создании или изменении группы ВМ с помощью параметра `--file`. Таким образом, удобно передать значение из нескольких строк. Подробнее читайте в разделе [Создать группу виртуальных машин фиксированного размера](../../../operations/instance-groups/create-fixed-group.md).
