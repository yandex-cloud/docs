# Compute Cloud API, REST: Instance.Relocate

Moves the specified instance to another availability zone
Running instance will be restarted during this operation.

## HTTP request

```
POST https://compute.api.cloud.yandex.net/compute/v1/instances/{instanceId}:relocate
```

## Path parameters

#|
||Field | Description ||
|| instanceId | **string**

Required field. ID of the instance to move.
To get the instance ID, make a [InstanceService.List](list.md#List) request.
The length must be less than or equal to 50.
This field is required. ||
|#
