#### Как следить за потреблением vRAM через мониторинг? {#track-vRAM-use}

Сервис Compute Cloud не может измерить потребление vRAM внутри гостевой операционной системы: для сервиса потребление памяти виртуальной машиной всегда одинаковое — то, которое выделено в момент ее запуска.

Чтобы отслеживать состояние vRAM, используйте сервис Yandex Monitoring. Он позволяет записывать собственные метрики, смотрите инструкцию [Запись пользовательских метрик через API](../../monitoring/operations/metric/add.md). Просто выгружайте данные о состоянии vRAM заданием по расписанию, а Monitoring будет отображать их.
