# Compute Audit Trails Events: REST reference

This is a directory of security events that the service sends to Yandex Audit Trails. The directory contains all the fields that can be passed along with the event.

**Interface definitions** available at [GitHub](https://github.com/yandex-cloud/cloudapi/tree/master/yandex/cloud/audit/compute).

**Service URL**: `https://compute.api.cloud.yandex.net`
