# Установка NVIDIA-драйверов


Для совместимости с [GPU](../../concepts/gpus.md) на виртуальной машине должны быть установлены драйверы NVIDIA. Драйверы можно установить как при [подготовке образа](../image-create/custom-image.md), так и изнутри ВМ после ее создания.

{% note info %}
   
Compute Cloud проверяет работоспособность и рекомендует устанавливать только [LTS версии драйверов](https://docs.nvidia.com/datacenter/tesla/drivers/releases.json).

{% endnote %}

## Драйверы для Tesla® V100 {#install-v100-drivers}

Чтобы установить драйверы для графической карты NVIDIA® Tesla® V100:

1. Перейдите на страницу [формы](https://www.nvidia.com/download/index.aspx?lang=ru).
1. В поле **Тип продукта** выберите `Data Center / Tesla`.
1. В поле **Серия продуктов** выберите `V-Series`.
1. В поле **Семейство продуктов** выберите `Tesla V100`.
1. В поле **Операционная система** выберите нужную ОС.
1. Укажите остальные настройки в соответствии с вашей операционной системой и требованиям к драйверу.
1. Нажмите кнопку **Поиск**. Откроется страница с подходящим драйвером.
1. Нажмите кнопку **Загрузить сейчас**.
1. Запустите загруженный установщик и следуйте инструкциям.

## Драйверы для Ampere® A100 {#install-a100-drivers}

В Compute Cloud доступно две [конфигурации](../../concepts/gpus.md#config) ВМ на платформе AMD EPYC® with NVIDIA® Ampere® A100: с 1 GPU и с 8 GPU.

{% note info %}
   
При установке укажите совместимую версию драйвера — `535`.

Мы рекомендуем использовать именно эту версию драйвера — обновление до других версий не поддерживается и может привести к нестабильной работе GPU.

{% endnote %}

Чтобы установить драйверы для графической карты NVIDIA® Ampere® A100 в графическом интерфейсе:

1. Перейдите на страницу [формы](https://www.nvidia.com/download/index.aspx?lang=ru).
1. В поле **Тип продукта** выберите `Data Center / Tesla`.
1. В поле **Серия продуктов** выберите `A-Series`.
1. В поле **Семейство продуктов** выберите `NVIDIA A100`.
1. В поле **Операционная система** выберите нужную ОС.
1. Укажите остальные настройки в соответствии с вашей операционной системой и требованиям к драйверу.
1. Нажмите кнопку **Поиск**. Откроется страница с подходящим драйвером.
1. Нажмите кнопку **Загрузить сейчас**.
1. Запустите загруженный установщик и следуйте инструкциям.

Чтобы установить драйверы без графического интерфейса:

1. В терминале выполните установку:

   ```bash
   sudo apt install -y nvidia-driver-<версия_драйвера>-server
   ```

1. Проверьте работу Fabric Manager:

   ```bash
   nvidia-smi nvlink -s
   ```
