# Закрытие сервиса {{ load-testing-full-name }}

{% note warning %}

Сервис {{ load-testing-full-name }} прекращает работу. На этой странице описаны сроки, порядок закрытия и рекомендации по миграции.

{% endnote %}

## Что происходит {#what-happens}

Мы приняли решение закрыть сервис {{ load-testing-name }}. Понимаем, что это изменение может затронуть ваши рабочие процессы, и постараемся сделать переход комфортным.

## Ключевые даты {#key-dates}

Закрытие сервиса пройдет в три этапа:

* **13 апреля 2026** — прекращение продаж сервиса {{ load-testing-name }}.
* **1 июня 2026** — сервис переходит в режим read-only. Запуск новых тестов и изменение конфигураций станут недоступны. Вы сможете просматривать результаты ранее проведенных тестов и существующие настройки.
* **1 июля 2026** — сервис полностью прекращает работу. Доступ к интерфейсу сервиса будет закрыт.

## Что будет с вашими данными {#data}

Конфигурации тестов и результаты нагрузочных испытаний не будут потеряны. После закрытия сервиса мы сохраним резервные копии ваших данных — запросить их можно будет через [техническую поддержку](https://center.yandex.cloud/support/).

{% note info %}

Если в ваших облаках существуют виртуальные машины, использовавшиеся как агенты нагрузочного тестирования, они продолжат работу после закрытия сервиса {{ load-testing-name }}. Удаление или изменение бывших агентов нагрузочного тестирования останется в вашей зоне ответственности.

{% endnote %}

## Миграция {#migration}

В качестве альтернативы перенесите свои сценарии нагрузочного тестирования на [k6](https://k6.io/) — инструмент нагрузочного тестирования с открытым исходным кодом.

Мы подготовили руководство, которое поможет адаптировать существующие тесты: [Миграция из {{ load-testing-name }} в k6](tutorials/loadtesting-migration-to-k6.md).

{% note warning %}

Инструмент k6 является продуктом с открытым исходным кодом, разработанным и поддерживаемый третьей стороной (компанией Grafana Labs), не связанной с Яндексом. Предоставленное руководство по миграции носит исключительно информационный характер. Мы не несем ответственности за работу, функциональность, безопасность или любые последствия использования программного обеспечения k6, а также за возможные ошибки при адаптации ваших тестовых сценариев. Внимательно ознакомьтесь со всеми документами, регулирующими использование инструмента k6.

{% endnote %}
