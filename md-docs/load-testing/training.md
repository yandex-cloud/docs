# Обучающие курсы по Yandex Load Testing

Получить больше знаний о сервисах Yandex Cloud и научиться использовать их для решения конкретных практических задач можно с помощью обучающих онлайн-курсов. Эти курсы разработаны архитекторами Yandex Cloud, бесплатны и позволяют глубже изучить нужные вам темы в удобном для вас темпе.

Подробнее узнать об использовании сервиса Yandex Load Testing можно в курсе:
* [**Нагрузочное тестирование**](https://yandex.cloud/ru/training/loadtesting)
  
  Курс о том, как правильно проводить нагрузочное тестирование и анализировать его результаты: вы научитесь разворачивать копии production‑стенда, используя GitOps‑подход, организовывать нагрузочное тестирование в сервисе Load Testing и подводить его итоги.

Информацию о всех доступных и еще готовящихся курсах по Yandex Cloud вы найдете [на этой странице](https://yandex.cloud/ru/training).
