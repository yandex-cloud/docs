# Квоты и лимиты в Load Testing

{% note warning %}

С 1 июля 2026 года сервис Load Testing прекращает работу. Подробнее на странице [Закрытие сервиса Yandex Load Testing](../sunset.md).

{% endnote %}


В сервисе Load Testing нет действующих квот.

Потребление вычислительных ресурсов [агентами](agent.md) учитывается в [квотах](https://console.yandex.cloud/cloud?section=quotas) и ограничено лимитами сервиса [Yandex Compute Cloud](../../compute/concepts/limits.md).
