# Yandex Neurosupport

Сервис Yandex Neurosupport предоставляет умные и своевременные подсказки оператору поддержки на базе YandexGPT. Благодаря подсказкам оператор может быстрее и точнее отвечать на вопросы пользователей, уделять больше времени сложным и креативным задачам. Сервис не заменяет работу оператора, но помогает автоматизировать рутинные задачи и повысить качество обслуживания.

Сервис находится на стадии <a href="../overview/concepts/launch-stages.md">Preview</a> и предоставляется по <a href="https://yandex.cloud/ru/services/neurosupport#contact-form">запросу</a>.
