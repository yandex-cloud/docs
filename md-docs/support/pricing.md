С помощью одного платежного аккаунта можно оплачивать поддержку для разных организаций, в которых могут быть подключены разные тарифы поддержки. Стоимость технической поддержки рассчитывается исходя из [стоимости платно потребленных ресурсов](../billing/pricing.md). Если платежному аккаунту начислен [грант](../billing/concepts/bonus-account.md), то он учитывается при оплате тарифа поддержки.

### Базовый {#base}

Базовый тарифный план предоставляется всем пользователям Yandex Cloud и отдельно не тарифицируется. Тариф подходит для домашних и исследовательских проектов.

### Бизнес {#business}

Тариф подходит для бизнес-проектов, требующих поддержки 24/7.

Стоимость складывается из фиксированной и процентной частей. Процентная часть рассчитывается несколько раз в день из учета объема потребленных ресурсов. Сведения о потреблении ресурсов и начислениях доступны в [детализации](../billing/operations/check-charges.md) сервиса Yandex Cloud Billing.

Цена вычисляется из фиксированной части и из учета потребления ресурсов за текущий отчетный период (календарный месяц). Чтобы рассчитать стоимость фиксированной части, воспользуйтесь [нашим калькулятором](https://yandex.cloud/ru/prices#calculator) или ознакомьтесь со способами расчета в разделах ниже.

#### Стоимость тарифного плана {#business-price}


{% list tabs group=pricing %}

- Цены в рублях {#prices-rub}

  Стоимость поддержки по тарифному плану складывается из фиксированной суммы 5 000 ₽ в месяц и 5% от стоимости платно потребленных ресурсов, находящихся в организации. Процентная часть рассчитывается несколько раз в день из учета объема потребленных ресурсов. Сведения о потреблении ресурсов и начислениях доступны в [детализации](../billing/operations/check-charges.md) сервиса Yandex Cloud Billing.
  
  Стоимость поддержки начисляется на тот же платежный аккаунт, к которому прикреплен платно потребленный ресурс. Счет за предыдущий отчетный период (календарный месяц) выставляется в начале следующего отчетного периода. Процесс оплаты для физических и юридических лиц отличаются. Подробную информацию см. в разделах [Цикл оплаты для физических лиц](../billing/payment/billing-cycle-individual.md) и [Цикл оплаты для организаций и ИП](../billing/payment/billing-cycle-business.md) документации сервиса Yandex Cloud Billing.

Процентная часть рассчитывается несколько раз в день из учета объема потребленных ресурсов. Сведения о потреблении ресурсов и начислениях доступны в [детализации](../billing/operations/check-charges.md) сервиса Yandex Cloud Billing.

Счет за предыдущий отчетный период (календарный месяц) выставляется в начале следующего отчетного периода. Процесс оплаты для физических и юридических лиц отличаются. Подробную информацию см. в разделах [Цикл оплаты для физических лиц](../billing/payment/billing-cycle-individual.md) и [Цикл оплаты для организаций и ИП](../billing/payment/billing-cycle-business.md) документации сервиса Yandex Cloud Billing.

При расчете тарифа поддержки в 5% от стоимости потребляемых ресурсов включаются корректировки, подписки и зарезервированное потребление ресурсов платежного аккаунта, привязанного к организации и оплачивающего поддержку. Из расчета тарифа поддержки в 5% от стоимости потребляемых ресурсов исключается стоимость продуктов Cloud Marketplace. См. [пример расчета](#business-example-one-ba-cvos).

### Премиум {#premium}

[Тариф Премиум](https://yandex.cloud/ru/support) включает в себя все услуги остальных тарифных планов и может быть дополнен, чтобы лучшим образом отвечать вашим потребностям. Он может включать рекомендации по устранению проблем на стыке сервисов Yandex Cloud со сторонним программным обеспечением, консультации выделенного инженера с учетом вашего сценария использования сервисов Yandex Cloud и услуги персонального технического менеджера.

Чтобы получить оценку стоимости тарифного плана Премиум, пожалуйста, обратитесь к вашему менеджеру в Yandex Cloud или в [техническую поддержку](https://center.yandex.cloud/support).


## Как изменить тарифный план {#change-service-plan}

Перед подключением, отключением, изменением тарифного плана технической поддержки, а также сменой платежного аккаунта, с которого будет списываться плата по тарифу, убедитесь, что у вас есть следующие роли:

* [organization-manager.organizations.owner](../organization/security/index.md#organization-manager-organizations-owner) на облачную организацию, для которой необходимо изменить или подключить платный тариф технической поддержки.
* [billing.accounts.admin](../billing/security/index.md#billing-accounts-admin) или выше на ваш платежный аккаунт, если он не привязан к этой облачной организации. Если аккаунт привязан к организации, то роль будет унаследована от нее. Подробнее о ролях и связи облачной организации и платежного аккаунта читайте в разделе [Организация](../billing/concepts/organization.md).

Чтобы изменить тарифный план технической поддержки Yandex Cloud:

1. На [главной странице поддержки](https://center.yandex.cloud/support) в Cloud Center на панели слева выберите раздел **Тарифные планы**.
1. В окне **Тарифные планы** будет указан платежный аккаунт, с которого будут списываться средства платных тарифов технической поддержки.
1. Смените тариф:
    * Чтобы перейти на **Базовый** тариф, нажмите кнопку **Перейти**.
    * Чтобы перейти на **Бизнес** тариф, нажмите кнопку **Перейти**.
    * Чтобы перейти на **Премиум** тариф, нажмите кнопку **Оставить заявку**.

Если в одной организации с включенным тарифом поддержки есть ресурсы, оплачивающиеся разными платежными аккаунтами, то процентная часть тарифа будет взиматься с каждого платежного аккаунта в зависимости от стоимости потребления ресурсов, а фиксированную часть платит только один аккаунт, который был указан для этого при выборе тарифа **Бизнес**. При смене тарифа вы можете выбрать, с какого платежного аккаунта будет списываться фиксированная часть. Это может делать только владелец организации. Чтобы изменить платежный аккаунт:

1. В окне **Тарифные планы** для выбора другого аккаунта нажмите кнопку **Изменить**.
1. В окне **Платёжный аккаунт** укажите альтернативный аккаунт одним из способов:
    * Выберите аккаунт из списка.
    * Создайте новый аккаунт.

Повышение тарифного плана происходит сразу, понижение — первого числа следующего месяца. Действующие тарифные планы и примеры расчета вы можете посмотреть в разделе [Правила тарификации](pricing.md).
