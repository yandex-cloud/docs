# Описание технической поддержки Yandex Cloud

Техническая поддержка Yandex Cloud предоставляется в соответствии с <a href="https://yandex.ru/legal/cloud_technicalregulations">Регламентом технического обслуживания</a> на русском и английском языках.

<a href="https://center.yandex.cloud/support">Yandex Cloud Center </a> является главной точкой входа для решения проблем и ответов на вопросы о функционировании сервисов.

Этот документ рассказывает о работе с Cloud Center для получения технической поддержки, описывает уровни технической поддержки и  действующие тарифные планы.
