# Резервное копирование

Настройки соединений, привязок, запросов, запусков Query надежно хранятся и реплицируются в инфраструктуре Yandex Cloud.

Yandex Query не хранит сами данные, над которыми выполняются запросы.

Максимальная продолжительность хранения результатов исполнения запроса ограничена [лимитом](limits.md#yq-limits).
