# Обучающие курсы по Yandex Cloud Logging

Получить больше знаний о сервисах Yandex Cloud и научиться использовать их для решения конкретных практических задач можно с помощью обучающих онлайн-курсов. Эти курсы разработаны архитекторами Yandex Cloud, бесплатны и позволяют глубже изучить нужные вам темы в удобном для вас темпе.

Подробнее узнать об использовании сервиса Yandex Cloud Logging можно в следующих курсах:
* [**Защита облачной инфраструктуры**](https://yandex.cloud/ru/training/infrastructure-protection)
  
  Курс о том, как обеспечить безопасность вашего облака с помощью сервисов и инструментов Yandex Cloud. Вы узнаете, что и как нужно делать в первую очередь и как поддерживать необходимый уровень безопасности в дальнейшем.

Информацию о всех доступных и еще готовящихся курсах по Yandex Cloud вы найдете [на этой странице](https://yandex.cloud/ru/training).
