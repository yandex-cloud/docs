# Брендирование


{% note info %}

Функциональность находится на стадии [Preview](../../overview/concepts/launch-stages.md).

{% endnote %}

В {{ org-full-name }} вы можете настраивать внешний вид страниц аутентификации и формы входа, чтобы они соответствовали вашему корпоративному стилю и предпочтениям.

Брендирование страниц аутентификации работает только при входе через SSO во внешние [приложения](applications.md). Страницы для входа в другие сервисы {{ yandex-cloud }} не поддерживают брендирование.

{% note info %}

Функциональность доступна только для организаций, в которых [оплачен лимит](../pricing.md) пользователей {{ org-full-name }}, использующих SSO для внешних приложений. Если лимит был оплачен ранее, но в новом месяце оплаты не было, то брендирование прекратит работать.

{% endnote %}

Настроить брендирование могут только пользователи с [ролью](../security/index.md) `organization-manager.admin` или `organization-manager.organizations.owner`. Доступны следующие настройки отдельно для светлой и темной цветовой темы:

* редактирование изображений для логотипов и фавиконок перед загрузкой;
* загрузка собственных изображений в форматах `WebP`, `PNG`, `JPEG`, `JPG`:
  * фоновое изображение (до 3840×2160 px, до 4 МБ);
  * логотип для верхней части формы (от 1920×1920 px, до 512 КБ);
  * фавиконка (до 512×512 px, до 512 КБ);
  
  {% note info %}
  
  Встроенный редактор позволяет подготовить логотип или фавиконку к загрузке: обрезать лишний фон, изменить масштаб и выстроить композицию.
  
  {% endnote %}

* цвет кнопок и рамок для текстовых полей;
* текст приветствия;
* шаблон почты для подсказки пользователю в поле ввода;
* включение и отключение способов входа с помощью Яндекс ID и [SSO](../../glossary/sso.md).
