# Identity Provider API, REST: SynchronizationSession

A set of methods for managing synchronization sessions.

## Methods

#|
||Method | Description ||
|| [OpenSession](openSession.md) | Opens a new synchronization session. ||
|| [CloseSession](closeSession.md) | Closes a synchronization session. ||
|| [ReportSessionProgress](reportSessionProgress.md) | Reports progress for a synchronization session. ||
|| [Heartbeat](heartbeat.md) | Sends a heartbeat for a synchronization session. ||
|| [GetSession](getSession.md) | Returns the specified synchronization session. ||
|| [ListSessions](listSessions.md) | Lists synchronization sessions for a subject container. ||
|#
