# AI-ассистент в {{ sd-name }}

{% note info %}

Функциональность находится на стадии [Preview](../../overview/concepts/launch-stages.md).

{% endnote %}

AI-ассистент в {{ sd-name }} позволяет получить рекомендации по безопасной настройке сервисов {{ yandex-cloud }} и соответствию вашей инфраструктуры требованиям безопасности. AI-ассистент создан на базе дообученной [модели]({{ link-docs-ai }}ai-studio/concepts/generation/models) {{ yagpt-name }} и использует информацию из внутренней базы знаний и [документации](../../index.md) {{ yandex-cloud }}.

## Интеграция с алертами

Запустить ИИ-анализ вы можете на странице [алерта](alerts.md). AI-ассистент поможет исправить проблему, найденную в конфигурации, с учетом контекста алерта и его параметров. В дополнение к рекомендациям AI-ассистент подготовит суммаризацию и объяснение сути алерта.
