# AI-ассистент в Security Deck

{% note info %}

Функциональность находится на стадии [Preview](../../overview/concepts/launch-stages.md).

{% endnote %}

AI-ассистент в Security Deck позволяет получить рекомендации по безопасной настройке сервисов Yandex Cloud и соответствию вашей инфраструктуры требованиям безопасности. AI-ассистент создан на базе дообученной [модели](https://aistudio.yandex.ru/docs/ru/ai-studio/concepts/generation/models) YandexGPT и использует информацию из внутренней базы знаний и [документации](../../index.md) Yandex Cloud.

## Интеграция с алертами

Запустить ИИ-анализ вы можете на странице [алерта](alerts.md). AI-ассистент поможет исправить проблему, найденную в конфигурации, с учетом контекста алерта и его параметров. В дополнение к рекомендациям AI-ассистент подготовит суммаризацию и объяснение сути алерта.
