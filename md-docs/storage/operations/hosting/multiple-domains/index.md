# Поддержка нескольких доменных имен

Инструкция описывает, как настроить поддержку основного доменного имени сайта и его дополнительного домена, например: `example.com` и `example2.com`. Вы создадите [бакеты](../../../concepts/backup.md) для основного и дополнительного домена, в первом бакете настроите [хостинг статического сайта](../../../concepts/hosting.md) на основном доменном имени, а во втором бакете включите переадресацию с дополнительного доменного имени на первый бакет. Для управления доменами вы можете воспользоваться сервисом [Yandex Cloud DNS](../../../../dns/index.md).

Вы можете настроить поддержку нескольких доменных имен для статического сайта в Object Storage с помощью одного из инструментов:
* [Консоль управления, CLI, API](console.md) — используйте этот способ, чтобы пошагово создать инфраструктуру в консоли управления Yandex Cloud, CLI или API.
* [Terraform](terraform.md) — используйте этот способ, чтобы упростить создание ресурсов и управление ими, используя подход «инфраструктура как код» (IaC). Скачайте пример конфигурации Terraform с репозитория GitHub, а затем разверните инфраструктуру с помощью [Terraform-провайдера Yandex Cloud](../../../../terraform/index.md).
