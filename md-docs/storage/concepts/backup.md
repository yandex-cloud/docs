# Резервное копирование объектов

Object Storage обеспечивает надежное хранение загруженных вами объектов в реплицируемом хранилище, но не предоставляет специальных инструментов для резервного копирования.

Если вы хотите иметь резервную копию объектов, вы можете регулярно скачивать нужные объекты из Object Storage и хранить их в собственной инфраструктуре, или в другом облачном хранилище.

Автоматизировать процесс резервного копирования можно с помощью [инструментов, которые поддерживает сервис](../tools/index.md).
