# Вопросы о регулярном экспорте детализации

#### За какой самый ранний период можно выгрузить данные при экспорте за прошлые периоды? {#earliest-period}
Можно выгрузить данные начиная с даты создания [платежного аккаунта](../concepts/billing-account.md).

#### Что произойдет, если при создании экспорта за прошлые периоды указать дату, за которую нет данных? {#no-data-period}
Система выгрузит данные начиная с даты создания [платежного аккаунта](../concepts/billing-account.md). Ошибки не возникнет.

#### Обновляются ли CSV-файлы за прошлые периоды после их первоначальной выгрузки? {#csv-files-update}
Файлы за прошлые периоды обновляются в случае изменения данных о потреблении в ранее выгруженный период.

#### Какие именно типы шифрования бакетов поддерживаются для регулярного экспорта? {#encryption-types}
Регулярный экспорт поддерживает бакеты Yandex Object Storage с использованием ключей шифрования сервиса [Yandex Key Management Service](../../kms/index.md).

Подробнее в разделе [Шифрование в Object Storage](../../storage/concepts/encryption.md) документации сервиса Yandex Object Storage.

#### Как проверить, что выгрузка в бакет с шифрованием прошла успешно? {#check-encrypted-bucket}
Перейдите в бакет Yandex Object Storage, указанный при настройке [регулярного экспорта](../operations/get-folder-report.md#regular-download). Убедитесь в наличии CSV-файлов. Файлы появятся через 5–10 минут после создания регулярной выгрузки. Экспорт за длительные периоды (более года) может занять больше времени.

#### Можно ли изменить параметры регулярного экспорта после его создания? {#change-parameters}
Нет. Чтобы изменить параметры (язык, тип детализации, бакет), [прекратите](../operations/get-folder-report.md#stop-export) существующий регулярный экспорт и [настройте новый](../operations/get-folder-report.md#set-up-regular-download).

#### Можно ли довыгрузить данные за прошлые периоды в рамках ранее созданного экспорта? {#additional-export}
Да. Вы можете [довыгрузить](../operations/get-folder-report.md#stop-export.md#additional-export) данные в ранее созданный регулярный экспорт. Они будут доступны после следующего обновления регулярного экспорта.

#### Можно ли создать экспорт за прошлые периоды с использованием зашифрованного бакета? {#past-period-encrypted}
Да. Вы можете создать экспорт за прошлые периоды с использованием зашифрованного бакета.

#### Что произойдет, если выгрузка в бакет завершилась с ошибкой? {#export-failed}
Статус регулярной выгрузки сменится на `Failed`. Владелец платежного аккаунта получит письмо на почту.
