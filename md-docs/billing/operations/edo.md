# Подключиться к электронному документообороту


Юридические лица и индивидуальные предприниматели, которые являются резидентами РФ, могут воспользоваться услугой электронного документооборота (ЭДО) в Yandex Cloud с помощью сторонних операторов ЭДО.

С помощью ЭДО можно получать [закрывающие отчетные документы](../concepts/edo.md#document).

{% note warning %}

При подключении к ЭДО отправка электронных версий закрывающих документов на email-адрес и оригиналов закрывающих документов на почтовый адрес прекращается.
Но скачивание закрывающих документов будет доступно в сервисе Yandex Cloud Billing. Подробнее в инструкции [Посмотреть историю расходов и скачать отчетные документы в сервисе Yandex Cloud Billing](download-reporting-docs.md).

{% endnote %}


## Подключитесь к ЭДО {#connect}

Чтобы начать сотрудничество по ЭДО с Yandex Cloud:

1. Подключитесь к [оператору](../concepts/edo.md#operator) ЭДО — [Контур.Диадок (СКБ Контур)](https://promo.diadoc.ru/yandexfd?p=z05983&utm_abtest=order-lightbox) или [СБИС (Тензор)](https://sbis.ru/edo/telecoms/yandex). Инструкцию по подключению можно найти на сайте оператора.

1. Подпишите соглашение от Yandex Cloud. Перейдите в ваш личный кабинет [Контур.Диадок (СКБ Контур)](https://promo.diadoc.ru/yandexfd?p=z05983&utm_abtest=order-lightbox) или [СБИС (Тензор)](https://sbis.ru/edo/telecoms/yandex) и подпишите соглашение от ООО «Яндекс.Облако». Соглашение об ЭДО приходит вместе с приглашением как вложение с требованием подписи.

    Yandex Cloud автоматически направляет соглашения в одностороннем порядке двум операторам ЭДО, если:
    * регистрируется новый плательщик;
    * выставляется счет;
    * заключается договор.

{% note info %}

Подключить роуминг можно только в Контур.Диадок (СКБ Контур). Для приглашения в роуминг нужно направить запрос на электронную почту [docs-project@yandex-team.ru](mailto:docs-project@yandex-team.ru).

Как это работает:
1. Вы направляете запрос на подключение роуминга.
1. Yandex Cloud отправляет приглашение в ваш оператор через роуминг Контур.Диадок (СКБ Контур) (без вложений).
1. После принятия приглашения Yandex Cloud направляет вам соглашение на подпись и отслеживает процесс его подписания.

После настройки электронного документооборота (ЭДО) работа с роумингом ничем не отличается от работы без него.

{% endnote %}

После подписания соглашения вы начнете получать документы от Yandex Cloud. Это документы, сформированные не ранее даты доставки подписанного соглашения во внутренний ИТ-сервис ЭДО. Они будут автоматически отправляться через ЭДО в сроки, которые указаны в договоре на оказание услуг.

Оригиналы документов, сформированных до подписания соглашения, отправляются только Почтой России на физический адрес.

Информацию о подписантах соглашения с принимающей стороны можно найти в личном кабинете оператора. Если информацию найти не удается, обратитесь в техническую поддержку оператора.

Yandex Cloud не обрабатывает вопросы из личного кабинета оператора. Если у вас есть вопрос или проблемы с подключением, обратитесь в поддержку по электронной почте [docs-project@yandex-team.ru](mailto:docs-project@yandex-team.ru).
