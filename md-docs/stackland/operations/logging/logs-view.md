# Посмотреть логи

Чтобы посмотреть логи кластера:

1. Добавьте **Query**.
1. На вкладке **Explore** выберите **Logs**.
1. Выберите нужный сервис поставки логов (по умолчанию `fluentbit`) и нажмите кнопку **Show logs**. Откроется страница логов.

На странице с логами можно настроить фильтры для отображения логов на нужном уровне, требуемых контейнеров, пространств имен и т.д.
