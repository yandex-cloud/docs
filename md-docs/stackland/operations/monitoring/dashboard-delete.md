# Удалить дашборд

Чтобы удалить дашборд:

1. Откройте Grafana по ссылке вида `https://grafana.sys.<домен кластера>` и войдите с вашим именем пользователя и паролем.
1. Откройте раздел **Dashboards**.
1. Отметьте дашборд, который нужно удалить.
1. Нажмите кнопку **Delete**.
