# [Yandex DataLens] Source avatar title conflicts with another avatar

`ERR.DS_API.AVATAR.TITLE.CONFLICT`

Ошибка возникает при совпадении названий таблиц, добавленных в датасет.

Чтобы исправить ошибку:

* Если ошибка указывает на источник-таблицу, попробуйте удалить ее из датасета и добавить снова.
* Если ошибка указывает на источник в виде подзапроса, измените заголовок подзапроса таким образом, чтобы названия источников в датасете не совпадали.
