# [Yandex DataLens] Ошибка при обработке ответа

Не удалось корректно обработать ошибку. Пожалуйста, попробуйте повторить ваши действия. Если ошибка сохраняется, обратитесь в [техническую поддержку](https://center.yandex.cloud/support) Yandex Cloud. В сообщении опишите действия, при которых возникает ошибка.
