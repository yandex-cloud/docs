# [Yandex DataLens] Использование QL-чартов запрещено на уровне подключения

`ERR.DS_API.CONNECTION_CONFIG.DASHSQL_NOT_ALLOWED`

Ошибка возникает, когда использование [QL-чартов](../../concepts/chart/ql-charts.md) запрещено на уровне подключения.

Чтобы исправить ошибку, включите опцию **SQL на чтение** в настройках подключения.
