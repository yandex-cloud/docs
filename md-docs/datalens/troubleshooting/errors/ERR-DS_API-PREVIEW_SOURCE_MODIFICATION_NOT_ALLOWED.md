# [Yandex DataLens] Изменение источника данных недоступно

`ERR.DS_API.PREVIEW_SOURCE_MODIFICATION_NOT_ALLOWED`

Ошибка означает, что у пользователя нет права `Просмотр` на подключение, используемое в источнике, при попытке изменить параметры источника в запросе к API.

Например, в [Public API](../../operations/api-start.md), когда у пользователя нет права доступа `Просмотр` на подключение, при попытке изменить источник в запросе на предпросмотр датасета.

Чтобы исправить ошибку, воспользуйтесь одним из способов:

* Используйте в запросе подключение, ранее сохраненное в датасете.
* Если вам требуется изменить параметры источника, запросите право доступа у администратора подключения. Подробнее о правах доступа в разделе [Управление доступом к Yandex DataLens](../../security/index.md).
