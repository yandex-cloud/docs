# [Yandex DataLens] Invalid connection ID

`ERR.DS_API.US.VALIDATION_FAILED.CONNECTION`

Ошибка валидации идентификатора запрашиваемого подключения. Частный случай ошибки [ERR.DS_API.US.VALIDATION_FAILED](ERR-DS_API-US-VALIDATION_FAILED.md) — когда запрашивается подключение с некорректным ID.

Проверьте и исправьте ID запрашиваемого подключения.

Идентификаторы объектов можно скопировать из меню соответствующего объекта или в навигации, нажав **Копировать ID**. Идентификатор будет сохранен в буфере обмена.
