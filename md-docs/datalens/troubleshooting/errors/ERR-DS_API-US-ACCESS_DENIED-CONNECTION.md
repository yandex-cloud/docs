# [Yandex DataLens] Access to the connection was denied

`ERR.DS_API.US.ACCESS_DENIED.CONNECTION`

Ошибка означает, что у пользователя нет прав на доступ к запрашиваемому подключению.

Чтобы исправить ошибку, запросите у администратора подключения право доступа `Просмотр`. Подробнее о правах доступа в разделе [Управление доступом к Yandex DataLens](../../security/index.md).
