# [{{ datalens-full-name }}] Access to the connection was denied

`ERR.DS_API.US.ACCESS_DENIED.CONNECTION`

Ошибка означает, что у пользователя нет прав на доступ к запрашиваемому подключению.

Чтобы исправить ошибку, запросите у администратора подключения право доступа `{{ permission-read }}`. Подробнее о правах доступа см. раздел [{#T}](../../security/index.md).
