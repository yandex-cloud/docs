# [Yandex DataLens] There are too many fields in the dataset

`ERR.DS_API.DATASET.TOO_MANY_FIELDS`

Ошибка возникает при попытке добавления в датасет новых полей, когда это приводит к превышению [лимита](../../concepts/limits.md#datalens-common-limits).

Для исправления ошибки удалите поля из датасета, чтобы количество полей соответствовало установленным лимитам.
