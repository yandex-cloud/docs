# [{{ datalens-full-name }}] Source avatar not found

`ERR.DS_API.AVATAR.NOT_FOUND`

Указанная таблица не найдена в датасете.

Ошибка возникает при попытке доступа к таблице, ранее удаленной из датасета.

Чтобы исправить ошибку, удалите таблицы из рабочей области датасета, перетащите их заново и нажмите **Обновить поля**.


Если ошибка сохраняется, обратитесь в [техническую поддержку]({{ link-console-support }}) {{ yandex-cloud }}. В сообщении опишите, когда возникает ошибка.
