# [Yandex DataLens] Unknown field found in formula

`ERR.DS_API.FORMULA.UNKNOWN_FIELD_IN_FORMULA`

Ошибка возникает, когда в формуле указано несуществующее поле.

Возможно, поле было удалено из чарта или датасета.

Чтобы исправить ошибку, перепроверьте и скорректируйте формулы.
