# [Yandex DataLens] Access denied

`ERR.DS_API.US.ACCESS_DENIED`

Ошибка означает, что у пользователя нет прав на доступ к объекту. Например, при отсутствии права на `Просмотр` датасета или попытке создать подключение в папке, к которой у пользователя нет доступа.

Чтобы исправить ошибку, запросите у администратора объекта необходимые права. Подробнее о правах доступа в разделе [Управление доступом к Yandex DataLens](../../security/index.md).
