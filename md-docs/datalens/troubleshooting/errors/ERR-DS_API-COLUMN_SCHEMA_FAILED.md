# [Yandex DataLens] Failed to load description of table columns

`ERR.DS_API.COLUMN_SCHEMA_FAILED`

Возникает в случае, когда не удается получить схему таблицы.


Проверьте, что схему таблицы можно прочитать стандартными для вашего источника методами, и обратитесь в [техническую поддержку](https://center.yandex.cloud/support) Yandex Cloud. В сообщении опишите, когда возникает ошибка.
