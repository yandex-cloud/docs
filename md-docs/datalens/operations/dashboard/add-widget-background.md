# Настроить фон и прозрачность для виджетов на дашборде в Yandex DataLens

Настройка задается для всех виджетов на дашборде отдельно для светлой и темной темы.

Чтобы настроить фон и прозрачность для виджетов на дашборде:


{% note warning %}

Если вы работаете с новой объектной моделью DataLens, включающей [воркбуки и коллекции](../../workbooks-collections/index.md):

1. На панели слева выберите ![collections](../../../_assets/console-icons/rectangles-4.svg) **Коллекции и воркбуки**.
1. Откройте нужный воркбук и выберите объект.

Продолжайте инструкцию с шага 2.

{% endnote %}


1. На панели слева нажмите ![image](../../../_assets/console-icons/layout-cells-large.svg) **Дашборды** и выберите нужный дашборд. Если у вас нет дашборда, [создайте его](create.md).
1. В правом верхнем углу нажмите кнопку **Редактировать**.
1. В верхней части экрана нажмите значок ![image](../../../_assets/console-icons/gear.svg). Справа откроется окно настроек.
1. В блоке **Внешний вид** выберите тему оформления дашборда, для которой нужно задать настройку:

   * ![image](../../../_assets/console-icons/sun.svg) — для светлой темы;
   * ![image](../../../_assets/console-icons/moon.svg) — для темной темы.

1. Задайте фон для виджетов. Для этого нажмите на поле ввода **Фон виджетов** и выберите цвет из предустановленных вариантов или задайте его в шестнадцатеричном формате.
1. Задайте прозрачность для виджетов — укажите ее в процентах справа от поля ввода цвета виджетов.
1. В правом верхнем углу редактирования дашборда нажмите **Сохранить**.

Настройки применятся ко всем виджетам дашборда, у которых эти настройки не определены в настройках виджета.
