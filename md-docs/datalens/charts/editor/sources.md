# Источники данных в Editor Yandex DataLens

## Виды источников {#source-types}

Для получения данных используйте:

* [датасет](tabs.md#sources-dataset);
* [подключение к базам данных через SQL-запрос](tabs.md#sources-database);
* [подключение через API Connector](tabs.md#sources-api-connector).

Подробнее о вкладке [Sources](tabs.md#sources).


Editor может работать с несколькими источниками данных.

![multi-dataset-chart](../../../_assets/datalens/editor/multi-dataset-chart.svg)

## Ограничения {#limits}

Размер и время загрузки данных ограничены:

* 100 Мбайт — суммарный размер данных из всех источников за один запрос;
* 50 Мбайт — размер данных из одного источника;
* 95 сек — суммарное время загрузки данных из всех источников за один запрос;
* 95 сек — время загрузки данных из одного источника;
* 10 сек — выполнение вкладки [Prepare](tabs.md#prepare).

Ограничения обеспечивают стабильную работу сервиса и источников данных.
