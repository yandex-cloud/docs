# Пользователи

Есть несколько способов, как добавить пользователя в DataLens:

1. [Через DataLens](#dl-user) — если пользователь не планирует пользоваться другими сервисами Yandex Cloud. Так можно приглашать пользователей с аккаунтами на Яндексе. Если у пользователя появится необходимость в использовании других сервисов Yandex Cloud, вы сможете [назначить ему дополнительную роль](../../organization/security/index.md#add-role) в организации с текущим экземпляром DataLens.

1. [Через Yandex Identity Hub](#org-user) — если пользователь планирует пользоваться и другими сервисами Yandex Cloud. В этом случае можно пригласить в организацию [пользователей с аккаунтом на Яндексе](#passport-user) или добавить [федеративных](#federated-user) и [локальных](../../organization/operations/add-account.md#local) пользователей.

{% note info %}

Добавлять и приглашать пользователей может администратор (роль `organization-manager.admin`) или владелец (роль `organization-manager.organizations.owner`) организации.

{% endnote %}

### Добавить пользователя через DataLens {#dl-user}

Чтобы добавить пользователя в DataLens:

1. Перейдите на [главную страницу](https://datalens.ru/?skipPromo=true) DataLens.
1. Откройте [настройки](https://datalens.ru/settings) сервиса: на панели навигации слева нажмите ![image](../../_assets/console-icons/sliders.svg).
1. В разделе **Управление пользователями** нажмите кнопку **Пригласить пользователей**.
1. Введите почтовые адреса пользователей, которых вы хотите пригласить (например, `login@yandex.ru`).
1. Нажмите кнопку **Отправить приглашения**.

Пользователь получит на почту приглашение в DataLens. Приняв приглашение и выбрав аккаунт для входа, пользователь сможет войти в вашу организацию. Пользователь получит роль `datalens.creator` в организации с текущим экземпляром DataLens. Для входа в DataLens пользователю потребуется [рабочее место](../settings/seats.md).

![image](../../_assets/datalens/settings/users-settings.png)

### Добавить пользователя через Yandex Identity Hub {#org-user}

#### Пригласить пользователя с аккаунтом на Яндексе {#passport-user}

Чтобы пригласить в организацию пользователя с аккаунтом на Яндексе, воспользуйтесь [инструкцией](../../organization/operations/add-account.md#useraccount) из документации Yandex Identity Hub.

{% note warning %}

Пользователь сможет открыть DataLens только после получения роли для доступа к сервису. Поэтому дождитесь, когда он примет приглашение в организацию и [назначьте](../../organization/security/index.md#add-role) ему роль `datalens.visitor`.

{% endnote %}

#### Добавить федеративного пользователя {#federated-user}

Если у вас настроена [федерация удостоверений](../../organization/concepts/add-federation.md), добавьте федеративного пользователя по [инструкции](../../organization/operations/add-account.md#add-user-sso). После добавления [назначьте](../../organization/security/index.md#add-role) ему роль `datalens.visitor`.

Федеративный пользователь может сразу перейти к нужному дашборду, используя для аутентификации ссылку вида:

```html
https://datalens.ru/<идентификатор_дашборда>?__yc_federation_id=<идентификатор_федерации>
```
