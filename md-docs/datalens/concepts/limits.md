# Квоты и лимиты в Yandex DataLens



В Yandex Cloud для сервисов действуют следующие виды ограничений:

* [_Квоты_](https://console.yandex.cloud/cloud?section=quotas) — организационные ограничения, которые можно изменить по запросу в техническую поддержку.
* _Лимиты_ — технические ограничения, обусловленные особенностями архитектуры Yandex Cloud. Изменить лимиты невозможно.


#### Квоты {#datalens-quotas}

Квоты для сервиса DataLens не определены.

#### Лимиты данных в чартах {#datalens-chart-data-limits}

Чарты в DataLens могут отображать лишь ограниченное количество данных. Если количество строк (столбцов и ячеек для сводных таблиц) в датасете превышает установленный лимит, чарт не будет построен. Поэтому, чтобы построить чарт по данным из объемного датасета, с помощью [фильтров](../dataset/settings.md#default-filters) выберите для отображения часть данных в соответствии с лимитами:
