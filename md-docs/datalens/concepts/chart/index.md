# Чарт в Yandex DataLens

_Чарт_ — это визуализация данных из датасета в виде таблицы, диаграммы или картограммы.

В DataLens несколько видов чартов:

* [Чарты в визарде](dataset-based-charts.md) — создаются в визарде на основе данных из одного или нескольких [датасетов](../../dataset/index.md).
* [QL-чарты](ql-charts.md) — для построения таких чартов используется прямой запрос в источник.
* [Чарты в Editor](../../charts/editor/index.md) — визуализации и селекторы создаются с помощью кода на JavaScript.

В DataLens поддерживаются различные типы визуализаций: диаграммы, таблицы, географические карты и другие средства. Подробнее в [Справочнике визуализаций](../../visualization-ref/index.md).
