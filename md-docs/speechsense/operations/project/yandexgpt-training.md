# Обучение YandexGPT

[YandexGPT](https://aistudio.yandex.ru/docs/ru/ai-studio/concepts/generation/models) используется в [диалогах](../../concepts/dialogs.md) SpeechSense для анализа данных. Чтобы анализ был более качественным (например, для разметки смысловыми тегами или для распознавания аудиоданных), модели YandexGPT могут обучаться на данных вашего проекта.

С 1 сентября 2025 года при создании проекта обучение YandexGPT включено по умолчанию. При необходимости его можно [отключить](#disable-yandexgpt-training). Для проектов, созданных до 1 сентября 2025 года, обучение YandexGPT нужно [включить](#enable-yandexgpt-training) вручную.

Сбор данных начинается сразу, как только обучение было включено, а обновление модели YandexGPT на основе собранных данных происходит каждые три месяца. Результат обучения модели внедряется только в ваш проект.

Для включения или отключения обучения YandexGPT нужна роль `Admin` в пространстве.

## Включить обучение YandexGPT {#enable-yandexgpt-training}

{% note warning %}

Подключая к проекту обучение [YandexGPT](https://aistudio.yandex.ru/docs/ru/ai-studio/concepts/generation/models), вы разрешаете использовать данные вашего проекта для обучения модели. [Подробнее об использовании данных](https://yandex.ru/legal/cloud_terms_speechsense/).

{% endnote %}

Чтобы включить обучение YandexGPT для одного или нескольких проектов:

1. Откройте [главную страницу](https://speechsense.yandex.cloud/) SpeechSense.
1. Перейдите в нужное пространство.
1. Перейдите на вкладку **Настройки**.
1. В блоке **Обучение YandexGPT** нажмите ![image](../../../_assets/console-icons/pencil-to-line.svg) **Настроить**.
1. Отметьте один или несколько проектов, к которым вы хотите подключить обучение YandexGPT, и нажмите **Сохранить**.

С этого момента все новые данные, добавляемые в проект, будут собираться для обучения YandexGPT. Старые данные не используются для обучения.

## Отключить обучение YandexGPT {#disable-yandexgpt-training}

Чтобы отключить обучение YandexGPT для одного или нескольких проектов:

1. Откройте [главную страницу](https://speechsense.yandex.cloud/) SpeechSense.
1. Перейдите в нужное пространство.
1. Перейдите на вкладку **Настройки**.
1. В блоке **Обучение YandexGPT** нажмите ![image](../../../_assets/console-icons/pencil-to-line.svg) **Настроить**.
1. Снимите отметку у тех проектов, для которых вы хотите отключить обучение YandexGPT, и нажмите **Сохранить**.
