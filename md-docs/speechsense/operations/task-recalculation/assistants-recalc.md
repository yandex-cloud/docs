# Пересчет ассистентов

Чтобы запустить пересчет ассистентов в диалогах за определенный период, создайте задачу на пересчет. Для этого:

1. Откройте [главную страницу](https://speechsense.yandex.cloud/) SpeechSense.
1. Перейдите в нужное пространство, затем выберите нужный проект.
1. Перейдите на вкладку **Ассистенты** и отметьте ассистентов, которые нужно пересчитать в диалогах.

   {% note info %}
   
   Если в указанных ассистентах используется анализ по выбранным диалогам с фильтрами по тегам, то алгоритм дополнительно предложит пересчитать теги. [Смысловые pro-теги](../../concepts/tags.md#sense-pro-tags) пересчитываются за дополнительную плату.
   
   {% endnote %}
   
   
1. Внизу страницы нажмите кнопку **Пересчитать**.
1. Далее укажите период пересчета диалогов и нажмите кнопку **Создать**.

## Отменить задачу на пересчет ассистентов {#cancel-recalc}

Отменить пересчет ассистентов можно только для задач в статусах **В очереди**, **Пересчитывается**. Для этого:

1. Перейдите на вкладку **Задачи на пересчет**.
1. Нажмите ![image](../../../_assets/options-grey.svg) справа от нужной задачи и выберите пункт **Отменить пересчет**.
