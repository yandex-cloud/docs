# Пересчет тегов

Чтобы вручную запустить пересчет тегов в диалогах за определенный период, создайте задачу на пересчет тегов. Для этого:

1. Откройте [главную страницу](https://speechsense.yandex.cloud/) SpeechSense.
1. Перейдите в нужное пространство, затем выберите нужный проект.
1. Перейдите на вкладку **Теги** и отметьте теги, которые нужно вручную пересчитать в диалогах.
   
   {% note info  %}
   
   Если у тега есть [зависимые теги](../../concepts/tags.md#dependent-tags), то они тоже будут пересчитаны.
   
   {% endnote %}
   
   
1. Внизу страницы нажмите кнопку **Пересчитать**.
1. Далее укажите период пересчета диалогов и нажмите кнопку **Создать**.

## Отменить задачу на пересчет тегов {#cancel-recalc}

Отменить пересчет тегов можно только для задач в статусах **В очереди**, **Пересчитывается**. Для этого:

1. Перейдите на вкладку **Задачи на пересчет**.
1. Нажмите ![image](../../../_assets/options-grey.svg) справа от нужной задачи и выберите пункт **Отменить пересчет**.
