# Удалить пространство

Для удаления пространства нужна роль `Admin` в пространстве.

Чтобы удалить пространство:

1. Откройте [главную страницу](https://speechsense.yandex.cloud/) SpeechSense.
1. Перейдите в нужное пространство.
1. В правом верхнем углу нажмите кнопку **Еще** → ![delete](../../../_assets/console-icons/trash-bin.svg) **Удалить**.
1. Подтвердите удаление пространства.

{% note warning %}

Удаленное пространство можно восстановить в течение 7 дней, обратившись в [техническую поддержку](https://center.yandex.cloud/support). В этот период создать новое пространство с тем же именем невозможно.

{% endnote %}
