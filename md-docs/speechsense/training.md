# Обучающие курсы по Yandex SpeechSense

Получить больше знаний о сервисах Yandex Cloud и научиться использовать их для решения конкретных практических задач можно с помощью обучающих онлайн-курсов. Эти курсы разработаны архитекторами Yandex Cloud, бесплатны и позволяют глубже изучить нужные вам темы в удобном для вас темпе.

Подробнее узнать об использовании сервиса SpeechSense и получить дополнительные навыки, чтобы раскрыть весь его потенциал, можно в следующих курсах:

* [**Техники работы с промптами**](https://yandex.cloud/ru/training/prompting)

  Курс предлагает бесплатное обучение по техникам работы с промптами для генеративных моделей. Вы узнаете, как писать инструкции для языковых моделей и создавать промпты. Навыки работы с промптами помогут вам эффективнее использовать в работе с SpeechSense [смысловые Pro-теги](concepts/tags.md#sense-pro-tags) и [анализ диалогов](concepts/dialogs.md#yandexgpt). Смысловые Pro-теги — сложный экспертный инструмент SpeechSense. Качество его работы зависит от правильности составления промптов. Рекомендуем обязательно пройти этот курс.
  
* [**Основы работы с Yandex SpeechSense**](https://yandex.cloud/ru/training/speechsense)

  Бесплатный курс для тех, кто только осваивает сервис. Вы узнаете, как использовать SpeechSense для автоматизации процессов, улучшения качества обслуживания клиентов и оптимизации работы операторов. Курс состоит из четырех тем с теорией и практикой, после его завершения вы получите свидетельство об окончании.

* [**DataLens: анализ и визуализация данных**](https://yandex.cloud/ru/training/datalens)

  Бесплатный курс будет полезен для тех, кто хочет изучить визуализацию и анализ данных. Вы научитесь работать со сложными формулами, создавать графики и дашборды, анализировать сезонные данные и оптимизировать скидки. Курс подойдет не только менеджерам и аналитикам, но и тем, кто хочет научиться работать с данными SpeechSense.

* [**Инженер облачных сервисов**](https://practicum.yandex.ru/ycloud/)
  
  Бесплатный курс для тех, кто планирует использовать облачные сервисы или хочет системно познакомиться с возможностями и технологиями облачной платформы.
  
  В рамках курса вы научитесь создавать отказоустойчивые облачные системы, работать с управляемыми базами данных и контейнерами, освоите основы бессерверных технологий и подходы к созданию защищенной от внешних атак облачной инфраструктуры.

Информацию о всех доступных и еще готовящихся курсах по Yandex Cloud вы найдете [на этой странице](https://yandex.cloud/ru/training).
