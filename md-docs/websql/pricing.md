# Правила тарификации для Yandex WebSQL



Использование сервиса Yandex WebSQL не тарифицируется.

За потребление других ресурсов Yandex Cloud, которые вы будете использовать в процессе работы с сервисом (например, сервисов управляемых баз данных или [Yandex Lockbox](../lockbox/pricing.md)), будет взиматься плата в соответствии с их [тарифами](https://yandex.cloud/ru/prices#calculator).
