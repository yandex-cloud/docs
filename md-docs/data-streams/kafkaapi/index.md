# Kafka API

Data Streams в ограниченном режиме поддерживает работу с потоками данных по протоколу [Apache Kafka®](https://kafka.apache.org/) (поддерживается Kafka API до версии [3.4.0](https://kafka.apache.org/34/apis/) включительно). 

Основную документацию по работе с Data Streams через Kafka API и примеры читайте в [документации YDB](https://ydb.tech/docs/ru//reference/kafka-api).
