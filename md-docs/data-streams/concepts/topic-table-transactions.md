# Транзакции с участием потоков данных и таблиц

Data Streams поддерживает транзакции с участием потоков данных и таблиц YDB. Участвующие в транзакции потоки данных и таблицы должны быть расположены в одной БД YDB. Таким образом, можно транзакционно перекладывать данные из таблиц в потоки данных и в обратном направлении, а также между потоками данных, чтобы данные не терялись и не дублировались даже в случае непредвиденных обстоятельств.

Подробнее о транзакционных операциях при работе с потоками данных смотрите в [документации YDB](https://ydb.tech/docs/ru//concepts/transactions#topic-table-transactions).
