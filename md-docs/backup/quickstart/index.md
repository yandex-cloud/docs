# Как начать работать с Yandex Cloud Backup


Cloud Backup — сервис для создания резервных копий и восстановления ресурсов Yandex Cloud и данных на них.

Подключить к Cloud Backup вы можете как новую [виртуальную машину](../../compute/concepts/vm.md) Yandex Compute Cloud непосредственно при ее создании, так и уже существующую виртуальную машину с запущенными и настроенными приложениями, ресурсами, данными и т.д.

Чтобы Cloud Backup мог создавать резервные копии ВМ или восстанавливать ВМ из резервных копий, виртуальная машина должна быть также привязана к [политике резервного копирования](../concepts/policy.md).

Подробнее о минимальных характеристиках ВМ и поддерживаемых операционных системах см. в разделе [Подключение виртуальных машин Compute Cloud и серверов Yandex BareMetal к Cloud Backup](../concepts/vm-connection.md).

* [Начать работать с Cloud Backup на действующей виртуальной машине](existing-vm.md)
