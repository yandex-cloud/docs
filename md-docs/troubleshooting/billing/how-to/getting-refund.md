# Как вернуть деньги


## Описание задачи {#case-description}

Необходимо вернуть денежные средства, зачисленные на платежный аккаунт, привязанную банковскую карту или расчетный счет.

## Решение {#case-resolution}

Для возврата средств понадобится [создать запрос в техническую поддержку](https://center.yandex.cloud/support). При создании запроса укажите следующую информацию:

1. Идентификатор платежного аккаунта, для которого нужно выполнить возврат средств. Он имеет вид `d2n*****************`. Этот идентификатор можно найти [на странице с данными по платежному аккаунту](https://center.yandex.cloud/support).
1. Сумму средств к возврату.

{% note warning %}

Сумма возврата не может превышать остаток текущего баланса аккаунта.

{% endnote %}
