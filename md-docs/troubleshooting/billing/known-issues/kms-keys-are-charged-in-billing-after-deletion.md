# KMS-ключи продолжают тарифицироваться после удаления


## Описание проблемы {#issue-description}

Исходя из данных детализации платежного аккаунта, ресурсы сервиса Yandex Key Management Service продолжают тарифицироваться после их удаления пользователем.

## Решение {#issue-resolution}

Ключи KMS продолжают тарифицироваться на протяжении 3 дней после [удаления](../../../kms/operations/key.md#delete). Это необходимо для того, чтобы обеспечить возможность их восстановления по запросу в [техническую поддержку](https://center.yandex.cloud/support).

Трехдневная задержка перед фактическим удалением ключей необходима на случай, если удаленным ключом были зашифрованы какие-либо данные, которые не были загружены из Yandex Cloud до удаления KMS-ключа.
