# Устранение ошибки `Такой логин не подойдет`


## Описание проблемы {#issue-description}

При входе в [Консоль управления](https://console.yandex.cloud) через Яндекс ID появляется сообщение `Такой логин не подойдет`.

## Решение {#issue-resolution}

Необходимо пройти полную регистрацию и снова войти в [Консоль управления](https://console.yandex.cloud). Подробную инструкцию можно найти [здесь](https://yandex.ru/support/id/authorization/lite.html#lite__login-yandex).
