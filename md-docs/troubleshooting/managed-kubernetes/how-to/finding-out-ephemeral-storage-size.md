# Как выяснить размер эфемерного хранилища узла Managed Service for Kubernetes


## Описание сценария {#case-description}

Необходимо выяснить размер эфемерного хранилища узла Managed Service for Kubernetes.

## Решение {#case-resolution}

Размер эфемерного хранилища может занимать до 50% от размера диска узла. Часть свободного пространства в эфемерном хранилище может быть зарезервирована под системные нужды.

Подробнее о концепции эфемерных хранилищ вы можете узнать из [официальной документации от разработчиков Kubernetes](https://kubernetes.io/docs/concepts/storage/ephemeral-volumes).
