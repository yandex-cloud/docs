# Словари ClickHouse®

_Словарь_ в ClickHouse® — это хранилище данных типа «ключ-значение», которое полностью или частично хранится в оперативной памяти сервера ClickHouse®.
Доступ к данным словаря осуществляется с помощью [набора функций](https://clickhouse.com/docs/ru/sql-reference/functions), которые выполняются быстрее стандартных SQL-запросов. Доступ к словарям осуществляется только на чтение, записать данные в словарь с помощью ClickHouse® невозможно. 

Основное преимущество словарей — высокая скорость работы по сравнению с JOIN-операциями. Словари будут полезны в тех случаях, когда вам приходится часто обращаться к справочнику для получения набора значений по ключу.

В качестве источников данных словарей могут выступать [встроенные словари ClickHouse®](https://clickhouse.com/docs/ru/sql-reference/statements/create/dictionary/embedded) или [внешние источники](https://clickhouse.com/docs/ru/sql-reference/statements/create/dictionary): HTTP(s) ресурс или другая СУБД: MySQL®, ClickHouse®, Yandex StoreDoc, PostgreSQL.


## Встроенные словари {#internal-dicts}

Managed Service for ClickHouse® содержит встроенный словарь-геобазу и набор функций для работы с ним. Словарь позволяет:

* получить имя региона на нужном языке по идентификатору региона;
* получить идентификатор города, области, федерального округа, страны, континента по идентификатору региона;
* проверить, входит ли один регион в другой;
* получить цепочку родительских регионов.

Подробнее о функциях для работы со встроенными словарями читайте в [документации ClickHouse®](https://clickhouse.com/docs/ru/sql-reference/functions/ym-dict-functions).

Если встроенная геобаза вам не подходит, подключите к ClickHouse® [собственную геобазу](../operations/geobase.md).


## Внешние словари {#external-dicts}

Источниками данных для внешних словарей могут быть HTTP(s) ресурс или другая СУБД: MySQL®, ClickHouse®, Yandex StoreDoc, PostgreSQL.

В зависимости от настроек словаря, ClickHouse® полностью или частично хранит словари в оперативной памяти, периодически их обновляет и динамически подгружает отсутствующие значения.

О том, как подключить и настроить словарь с помощью Yandex Managed Service for ClickHouse®, читайте в разделе [Подключение внешних словарей](../operations/dictionaries.md).
