# Обучающие курсы по Yandex MPP Analytics for PostgreSQL

Получить больше знаний о сервисах Yandex Cloud и научиться использовать их для решения конкретных практических задач можно с помощью обучающих онлайн-курсов. Эти курсы разработаны архитекторами Yandex Cloud, бесплатны и позволяют глубже изучить нужные вам темы в удобном для вас темпе.

Подробнее узнать об использовании сервиса Yandex MPP Analytics for PostgreSQL можно в следующих курсах:
* [**Yandex MPP Analytics for PostgreSQL**](https://yandex.cloud/ru/training/greenplum)
  
  Курс о том, как создать, настроить и эффективно использовать облачную инсталляцию Greenplum®. После прохождения курса вы научитесь учитывать особенности СУБД для продуктивной работы с Yandex MPP Analytics for PostgreSQL, писать и оптимизировать запросы под специфику Greenplum®, разбираться в архитектуре Greenplum®, проектировать оптимальную физическую модель данных.

Информацию о всех доступных и еще готовящихся курсах по Yandex Cloud вы найдете [на этой странице](https://yandex.cloud/ru/training).

_Greenplum® и Greenplum Database® являются зарегистрированными товарными знаками или товарными знаками Broadcom Inc в США и/или других странах._
