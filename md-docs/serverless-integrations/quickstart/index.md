# Начало работы с Yandex Serverless Integrations

Yandex Serverless Integrations позволяет настраивать интеграции и управлять ими с помощью serverless-технологий в Yandex Cloud.

Начните работу с функциональностями Serverless Integrations:
* [Yandex Workflows](workflows.md) — создайте рабочий процесс, который вызывает функцию Cloud Functions, и запустите его. Подробнее о возможностях Workflows в [Концепциях](../concepts/workflows/workflow.md).

    {% note info %}
    
    Workflows находится на стадии [Preview](../../overview/concepts/launch-stages.md).
    
    {% endnote %}

* [Yandex EventRouter](eventrouter.md) — создайте шину и передайте через нее данные. Подробнее о возможностях EventRouter в [Концепциях](../concepts/eventrouter/bus.md).

    {% note info %}
    
    EventRouter находится на стадии [Preview](../../overview/concepts/launch-stages.md).
    
    {% endnote %}

* Yandex API Gateway — сконфигурируйте API-шлюз для получения статического ответа, а затем добавьте интеграцию для вызова функции Cloud Functions. Подробнее о возможностях в [документации API Gateway](../../api-gateway/quickstart/index.md).
