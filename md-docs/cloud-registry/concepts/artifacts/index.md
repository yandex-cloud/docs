# Артефакты

_Артефакты_ — это пакеты программного обеспечения и другие файлы, используемые в процессе разработки и развертывания приложений. 

Артефакты {{ cloud-registry-name }} хранятся в [реестрах](../registry.md).

Сервис поддерживает:
* [Docker-образы](docker.md)
* Helm-чарты
* [Java-артефакты](java.md)
* [Node.js-артефакты](nodejs.md)
* [NuGet-артефакты](nuget.md)
* [Python-артефакты](python.md)
* [Binary-артефакты](binary.md)
