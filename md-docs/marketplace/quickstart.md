# Как разместить свой продукт в Yandex Cloud Marketplace

Cloud Marketplace дает возможность размещать свои продукты в каталоге готовых решений Yandex Cloud и зарабатывать на них. Вы можете разместить различные типы продуктов:
* готовые образы для сервиса Yandex Compute Cloud;
* приложения для Managed Service for Kubernetes;
* решения для сервиса Yandex DataLens.

![image](../_assets/marketplace/scheme-ru.svg)

## Подайте заявку {#send-application}

Чтобы стать партнером Marketplace, заполните анкету на [странице Marketplace](https://yandex.cloud/ru/marketplace) и опишите ваш продукт. Подробное описание полезных для пользователей функций и особенностей продукта поможет быстрее рассмотреть вашу заявку. Отправка заявки означает принятие оферты на предоставление доступа к программному продукту в Marketplace:
* [Оферта для партнеров, которые ведут расчеты с Yandex Cloud в рублях](https://yandex.ru/legal/marketplace_offer/?lang=ru)
* [Оферта для партнеров, которые ведут расчеты с Yandex Cloud в тенге](https://yandex.com/legal/marketplace_offer_kz/?lang=ru)

Если у вашего юридического лица уже есть платежный аккаунт, укажите его при подаче заявки. После прохождения модерации вы получите письмо с подтверждением на электронный адрес, указанный в анкете. 

## Зарегистрируйте аккаунт {#registration}

Для доступа в кабинет партнера Marketplace вам потребуется платежный аккаунт юридического лица — резидента Российской Федерации или Республики Казахстан. Если у вас еще нет платежного аккаунта, [создайте его](operations/registration.md).

{% note warning %}

Реквизиты, указанные в платежном аккаунте, должны быть актуальны. Они потребуются для создания договора-оферты. 
Yandex Cloud не списывает деньги с карты и не выставляет счета за использование вашего программного продукта пользователями Marketplace (за исключением продуктов для DataLens).

{% endnote %}

Данные платежного аккаунта будут использованы для предоставления доступа в [кабинет партнера](https://partners.yandex.cloud/).

## Создайте продукт в кабинете партнера {#create-product}

[Создайте продукт](operations/create-product.md). Дальнейшие действия будут доступны после одобрения заявки. Вы можете уточнить заявку и связаться с менеджером Marketplace в разделе **Тикет заявки**.

## Создайте тариф {#create-tariff}

{% note warning %}

Yandex Cloud взимает 20% комиссии от стоимости использования программного продукта.

{% endnote %}

Тариф определяет стоимость использования вашего продукта.

[Создайте тариф](operations/create-tariff.md), чтобы назначить его версии продукта.

## Создайте и загрузите продукт {#create-image}

В Marketplace можно размещать образы ВМ из публичных образов Yandex Cloud и приложения для Managed Service for Kubernetes. Требования к образам и приложениям и рекомендации по их созданию описаны в разделах:

* [Создание образа продукта на базе Linux для загрузки в Cloud Marketplace](concepts/compute-product.md)
* [Создание продукта для Managed Service for Kubernetes](concepts/kubernetes-product.md)

## Создайте первую версию продукта {#create-version}

О том, как создать первую версию продукта в кабинете партнера Cloud Marketplace, читайте на странице [Создание новой версии продукта](operations/create-new-version.md).

## Пройдите модерацию версии {#moderate-version}

Все версии продуктов, размещенные в Marketplace, проходят техническую модерацию и модерацию контента с учетом требований к продуктам Marketplace для [Compute Cloud](concepts/compute-product.md) и [Managed Service for Kubernetes](concepts/kubernetes-product.md). После успешного прохождения всего процесса модерации вы получите уведомление и сможете сделать версию вашего продукта доступной для пользователей.
