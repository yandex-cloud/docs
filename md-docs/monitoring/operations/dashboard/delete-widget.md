# Удаление виджета с дашборда

Чтобы удалить виджет:

1. Откройте [главную страницу сервиса](https://monitoring.yandex.cloud).
1. В левой части окна выберите раздел **Дашборды**.
1. Напротив нужного дашборда нажмите ![image](../../../_assets/console-icons/ellipsis.svg) и выберите пункт **Редактировать**.
1. В правом верхнем углу удаляемого виджета нажмите значок ![image](../../../_assets/console-icons/xmark.svg).
1. Нажмите кнопку **Сохранить**.
