# Удаление дашборда

Чтобы удалить дашборд:

1. Откройте [главную страницу сервиса](https://monitoring.yandex.cloud).
1. В левой части окна выберите раздел **Дашборды**.
1. Справа от имени удаляемого дашборда нажмите значок ![image](../../../_assets/console-icons/ellipsis.svg) и выберите пункт **Удалить**.
1. В открывшемся окне нажмите кнопку **Удалить**.
