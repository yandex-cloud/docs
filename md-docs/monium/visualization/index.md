# Визуализация данных телеметрии

Инструменты визуализации представляют метрики и логи на графиках и диаграммах, которые помогают анализировать состояние системы и находить проблемы.

Визуализация решает три основные задачи:

* Мониторинг текущего состояния — отслеживание ключевых показателей в реальном времени.
* Анализ инцидентов — поиск причин сбоев через корреляцию данных из разных источников.
* Планирование ресурсов — оценка трендов потребления и прогнозирование нагрузки.

Инструменты визуализации:

* [Дашборды](../operations/dashboard/create.md) — объединяют несколько виджетов на одном экране для комплексного представления данных.
* Страница обзора — просмотр данных в реальном времени. Страницы обзора есть для каждого типа данных: [метрики](../metrics/metric-explorer.md), [логи](../logs/logs-explorer.md) и [трейсы](../traces/operations/traces-explorer.md).

Данные на графиках можно фильтровать по временным интервалам, сервисам, окружениям и другим параметрам. А также разбивать по определенных параметрам для более детального исследования.

Дашборды можно создавать вручную или воспользоваться готовыми дашбордами сервисов Yandex Cloud.

## Сервисные дашборды {#service-dashboards}

Чтобы посмотреть дашборд с наиболее распространенными метриками для ресурса или сервиса Yandex Cloud:

{% list tabs group=instructions %}

- Интерфейс Monium {#console}

  1. На главной странице [Monium](https://monium.yandex.cloud) слева выберите **Дашборды**.
  1. Перейдите на вкладку **Сервисные дашборды**.
  1. Найдите нужный сервис и выберите его в списке.

     Например, **Compute** (для поиска дашбордов сервиса Compute Cloud) или **Logs Overview** (для сервиса Monium Logs).
  
     Описание дашбордов и метрик см. в документации сервиса.
  
  1. Чтобы составить собственный дашборд из виджетов сервисного дашборда, [скопируйте виджеты](../operations/dashboard/widget-management.md) на свой дашборд или [дашборд целиком](../operations/dashboard/copy.md).

{% endlist %}
