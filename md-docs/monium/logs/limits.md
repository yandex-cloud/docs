# Ограничения

**Лимиты**

### Лимиты {#loggs-limits}

Вид ограничения | Значение
--------------- | --------
Максимальный размер одного батча с логами | 8 МБ
Максимальное время начала лога в прошлом (`time_unix_nano`) | 24 часа назад
Максимальное время лога в будущем (`time_unix_nano`) | 5 минут вперед
Максимальная длина имени [метки (label)](../concepts/data-model.md#label) | 64 байта
Максимальная длина значения метки | 256 байт
Максимальная длина имени [метаинформации (meta)](../concepts/data-model.md#meta) | 400 байт
Максимальная длина значения метаинформации | 1 МБ

{% note info %}

Допустимы батчи с логами размером до 16 МБ, но такой прием не гарантируется.

{% endnote %}

Перед записью каждый батч логов проходит валидацию. Сервис проверяет временные метки, соответствие [лимитам](limits.md#logs-limits) и корректность форматов.

Если значение метаинформации превышает лимит, сервис обрезает его до допустимого размера.

Если часть логов не появляется в интерфейсе, проверьте параметры отправки и соответствие данных указанным ограничениям.
