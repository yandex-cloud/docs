# Начало работы с метриками приложений

Monium позволяет работать с метриками сервисов Yandex Cloud, а также с любыми другими метриками, загруженными в Monium. Например, из ваших приложений, вашей инфраструктуры или других облаков.

Эта инструкция поможет настроить передачу метрик из вашего приложения в Monium и просмотреть их в интерфейсе.

Если вы еще не работали с Yandex Cloud или у вас нет приложения с настроенной OTLP-телеметрией, начните с раздела [Как начать работать с Monium](../quickstart.md). Там описана базовая настройка окружения и поставка данных с помощью OTel Collector.

## Поставка метрик {#data-send}

### Параметры подключения {#main-parameters}

* Формат поставки данных в Monium — [OpenTelemetry (OTLP)](https://opentelemetry.io/docs/).

* Адрес для записи метрик: `ingest.monium.yandex.cloud:443`.

* Для записи метрик потребуется [сервисный аккаунт](../../iam/operations/sa/create.md) с ролью `monium.metrics.writer` и [API-ключ](../../iam/operations/iam-token/create-for-sa.md) с областью действия `yc.monium.metrics.write`.

* Интерфейс просмотра метрик — [главная страница Monium](https://monium.yandex.cloud) > **Метрики**.

Поставка метрик настраивается так же, как и поставка других видов телеметрии. Правила поставки и распределения данных, настройка агентов и готовые примеры приведены в общем разделе [Поставка данных](../collector/index.md).

## Просмотр метрик {#view-metrics}

1. На главной странице сервиса [Monium](https://monium.yandex.cloud) слева выберите **Метрики**.
1. Вверху укажите промежуток поиска с помощью временной шкалы, предустановленного промежутка или введите нужное значение времени.
1. В строке поиска введите запрос для поиска метрик приложения:
   1. Выберите нужный проект `folder__<идентификатор_каталога>`. По умолчанию выбран проект в текущем каталоге.
   1. Последовательно выберите `cluster`, `service` и нужные метрики.
   1. Нажмите **Выполнить запрос**.
1. Чтобы посмотреть детали метрики, наведите курсор на график.

Дополнительные возможности работы с метриками см. в разделе [Посмотреть метрики в Monium Metrics](metric-explorer.md).
