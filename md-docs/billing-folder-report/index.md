# Billing с детализацией расходов по каталогам

Billing позволяет получать информацию об объеме использованных ресурсов, проверять расходы денежных средств и оплачивать потребление ресурсов.

В Yandex Cloud вы платите только за потребленные ресурсы и время их фактического использования.
