# Yandex Cloud Apps

<em>Сервис Yandex Cloud Apps находится на <a href="../overview/concepts/launch-stages.md">стадии Preview</a></em>.

Yandex Cloud Apps — сервис, который позволяет в один клик развернуть приложение на базе Yandex Cloud.

Приложение состоит из интегрированных друг с другом ресурсов Yandex Cloud. Настройки ресурсов пользователь может менять в зависимости от того, как планирует использовать приложение.

Партнеры могут размещать собственные приложения Cloud Apps в Yandex Cloud Marketplace. Подробнее см. инструкцию <a href="../marketplace/operations/create-new-version.md#cloud-apps">Создание новой версии продукта</a>.
