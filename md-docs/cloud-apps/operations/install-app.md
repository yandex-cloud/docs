# Установка приложения Cloud Apps

{% list tabs group=instructions %}

- Консоль управления {#console}

    1. В [консоли управления](https://console.yandex.cloud) выберите каталог, в котором хотите развернуть приложение.
    1. Перейдите в сервис **Cloud Apps**.
    1. На панели слева выберите **Магазин приложений**.
    1. Выберите нужное приложение и нажмите кнопку **Использовать**.
    1. Укажите имя приложения и выберите сервисный аккаунт. От имени выбранного сервисного аккаунта будут развертываться все ресурсы приложения, поэтому у него должны быть права на управление этими ресурсами. Например, права, которые позволяют создавать функции Cloud Functions или назначать права доступа другому сервисному аккаунту на каталог. Чтобы cоздать сервисный аккаунт с ролью `admin` на каталог, выберите `Автоматически`.
    1. Задайте настройки приложения.
    1. Нажмите кнопку **Установить**.

    В выбранном каталоге автоматически создадутся ресурсы Yandex Cloud, которые необходимы для работы приложения.
