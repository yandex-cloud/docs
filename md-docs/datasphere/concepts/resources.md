# Ресурсы проектов DataSphere

{% note warning %}

После 20.04.2026 создание новых [датасетов](dataset.md) станет невозможным. Для хранения данных используйте [файловые хранилища](filestores.md). Датасеты, созданные до 20.04.2026, продолжат работать без изменений.

{% endnote %}

В проектах DataSphere можно создавать ресурсы — вспомогательные объекты, которые помогают повысить скорость работы с данными, автоматизировать рутинные действия и выполнять специальные операции.

Ресурсы, созданные или доступные в проекте, можно посмотреть на странице проекта в блоке **Ресурсы проекта**.

Вы можете использовать ресурсы в нескольких проектах, не создавая их повторно. Чтобы ресурс стал доступен для всех проектов [сообщества](community.md), поделитесь им на вкладке **Доступ**. Для этого вам минимально необходимы роли `Editor` в проекте и `Developer` в сообществе, в котором вы хотите опубликовать ресурс. Ресурсы сообщества доступны на странице сообщества в блоке **Ресурсы сообщества**.
