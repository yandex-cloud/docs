# Задать ограничения для проекта

В DataSphere вы можете настроить ограничения потребления для проекта. Пороги потребления для проекта задаются в [юнитах](../../pricing.md#unit).

1. Выберите нужный проект в своем сообществе или на [главной странице](https://datasphere.yandex.cloud) DataSphere во вкладке **Недавние проекты**.
1. Перейдите на вкладку **Настройки** и в блоке **Ограничения** нажмите кнопку ![pencil](../../../_assets/console-icons/pencil-to-line.svg) **Редактировать**.
1. Задайте **Баланс проекта** — общее количество юнитов, доступных для проекта. После запуска вычислений баланс проекта будет уменьшаться на количество юнитов, необходимых для работы выбранной [конфигурации](../../concepts/configurations.md) до тех пор, пока не выполнится одно из условий:
   * баланс станет меньше или равен нулю. Тогда все запущенные вычисления будут остановлены с предупреждением, что баланса проекта недостаточно.
   * истечет время, указанное в разделе **Общие настройки**.
   * вы самостоятельно остановите ВМ.

1. Нажмите кнопку **Сохранить**.
