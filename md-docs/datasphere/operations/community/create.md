# Создать сообщество

1. Откройте [главную страницу](https://datasphere.yandex.cloud) DataSphere. На панели слева выберите ![community-panel](../../../_assets/console-icons/circles-concentric.svg) **Сообщества**.
1. Нажмите кнопку **Создать сообщество**.
1. Введите имя сообщества.
1. (Опционально) Введите описание сообщества.
1. Выберите [платежный аккаунт](../../../billing/concepts/billing-account.md) Yandex Cloud.
1. Выберите [**Зону доступности**](../../../overview/concepts/geo-scope.md).
1. Нажмите кнопку **Создать**.
