# Удалить алиас

1. Выберите нужный проект в своем сообществе или на [главной странице](https://datasphere.yandex.cloud) DataSphere во вкладке **Недавние проекты**.
1. В блоке **Ресурсы проекта** выберите **Алиас**.
1. Напротив нужного [алиаса](../../concepts/deploy/index.md#alias) нажмите ![Options](../../../_assets/console-icons/ellipsis.svg) и выберите ![Delete](../../../_assets/console-icons/trash-bin.svg) **Удалить**.
1. Во всплывающем окне нажмите кнопку **Подтвердить**.

{% note warning %}

Фактически [удаление ресурсов](../../../overview/concepts/data-deletion.md) может занимать до 72 часов.

{% endnote %}
