# Как начать работать с Yandex Cloud Video

Yandex Cloud Video — сервис для организации потоковых трансляций и видеохостинга. На [каналах](concepts/index.md#channels) сервиса вы можете загружать, хранить и обрабатывать видео для дальнейшей публикации на внешних ресурсах, а также запускать онлайн-трансляции. Трансляции и видео воспроизводятся в интегрированном [видеоплеере](concepts/player.md).

Начните работу с [хостингом видео](hosting.md) или [видеотрансляциями](streaming.md). Об элементах сервиса читайте в разделе [Обзор сервиса Yandex Cloud Video](concepts/index.md).

О том, как работать с сервисом с помощью API, описано на странице [Как начать работать с API Yandex Cloud Video](api-ref/quickstart.md).
