# Как начать работать с {{ video-full-name }}

{{ video-full-name }} — сервис для организации потоковых трансляций и видеохостинга. На [каналах](concepts/index.md#channels) сервиса вы можете загружать, хранить и обрабатывать видео для дальнейшей публикации на внешних ресурсах, а также запускать онлайн-трансляции. Трансляции и видео воспроизводятся в интегрированном [видеоплеере](concepts/player.md).

Начните работу с [хостингом видео](hosting.md) или [видеотрансляциями](streaming.md). Об элементах сервиса читайте в разделе [{#T}](concepts/index.md).

О том, как работать с сервисом с помощью API, см. на странице [{#T}](api-ref/quickstart.md).
