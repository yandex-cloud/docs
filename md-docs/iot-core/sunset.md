# Закрытие сервиса Yandex IoT Core

{% note warning %}

Сервис Yandex IoT Core прекращает работу. На этой странице описаны сроки, порядок закрытия и рекомендации по миграции.

{% endnote %}

## Что происходит {#what-happens}

Мы приняли решение закрыть сервис Yandex IoT Core. Понимаем, что это изменение может затронуть ваши рабочие процессы, и постараемся сделать переход комфортным.

## Ключевые даты {#key-dates}

Закрытие сервиса пройдет в три этапа:

* **27 апреля 2026** — прекращение новых продаж сервиса Yandex IoT Core. Клиенты, подключившие сервис ранее, смогут пользоваться им до 1 ноября 2026 года.

* **1 ноября 2026** — сервис переходит в режим read-only. Создание новых ресурсов и запись данных в реестры и брокеры станут недоступны. Текущие пользователи Yandex IoT Core смогут только просматривать ранее созданные ресурсы.

* **1 декабря 2026** — сервис полностью прекращает работу. Доступ к интерфейсу сервиса будет закрыт для всех пользователей.

## Что будет с вашими данными {#data}

Ваши данные (списки устройств, алиасов устройств, брокеров, а также сертификаты устройств и брокеров) не будут потеряны. После закрытия сервиса резервные копии ваших данных будут храниться до 31 января 2027 года — их можно будет запросить через [техническую поддержку](https://center.yandex.cloud/support).
