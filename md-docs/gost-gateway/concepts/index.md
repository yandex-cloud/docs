Сервис GOST Gateway предоставляет выделенный шлюз в собственном ЦОД под управлением Yandex Cloud. Сервис позволяет организовать защищенное взаимодействие между географически распределенными объектами в любых регионах и выполнить требования российского законодательства, касающиеся конфиденциальности и целостности данных.

В сервисе GOST Gateway используются средства криптографической защиты информации вплоть до класса КС3, сертифицированные регуляторами. Используемое оборудование указано [на сайте ИнфоТеКС](https://infotecs.ru/products/vipnet-coordinator-hw-4/?ysclid=mh4r8p4rc2195807053).
