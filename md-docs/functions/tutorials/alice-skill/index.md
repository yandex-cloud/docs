# Создание навыка Алисы


В этом руководстве вы создадите навык Алисы на основе [функции](*functions) Yandex Cloud Functions. В качестве примера будет создан навык <q>Попугай</q>, который повторяет все, что ему написал или сказал пользователь. Пример может быть реализован на двух языках программирования: Python и Node.js.

Вы можете подготовить инфраструктуру для создания навыка Алисы с помощью следующих инструментов:

[*functions]: Функции Yandex Cloud Functions позволяют размещать в Yandex Cloud код, который можно запускать по запросу или триггеру. Подробнее читайте в разделе [Функция](../../concepts/function.md).

* [Консоль управления, Yandex Cloud CLI, API](console.md) — используйте этот способ, чтобы пошагово создать навык Алисы в консоли управления Yandex Cloud, с помощью CLI или API.
* [Terraform](terraform.md) — используйте этот способ, чтобы упростить создание ресурсов и управление ими, используя подход «инфраструктура как код» (IaC). Скачайте пример конфигурации Terraform из репозитория на GitHub, а затем разверните инфраструктуру с помощью [Terraform-провайдера Yandex Cloud](../../../terraform/index.md).
