# История изменений Data Transfer в октябре 2023

## Новые возможности {#new-features}

1. Поддержаны [инкрементальные трансферы](../concepts/regular-incremental-copy.md) для источника [YDB](../operations/endpoint/source/ydb.md).
1. Задокументированы [стратегии поставки данных](../concepts/delivery-configuration.md), реализованные в Data Transfer.  
1. Добавлен парсер Debezium для источников [Apache Kafka®](../operations/endpoint/source/kafka.md) и [YDS](../operations/endpoint/source/data-streams.md).

## Решенные проблемы {#problems-solved}

1. Исправлено подключение к Greenplum® для выполнения запросов и извлечения схемы данных.
1. Устранена проблема потери данных для шардированного источника Greenplum®.
1. Добавлена возможность указывать часовой пояс при копировании в приемник MySQL®.
1. Исправлено создание трансфера для пользовательской инсталляции приемника ClickHouse® с указанной подсетью.
1. Устранена ошибка показа исключенных таблиц для приемника ClickHouse®.

_ClickHouse® является зарегистрированным товарным знаком [ClickHouse, Inc](https://clickhouse.com)._
