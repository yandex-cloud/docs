# История изменений Data Transfer в мае 2024

## Новые возможности {#new-features}

1. Закрыта возможность создания эндпоинтов Managed Service for Elasticsearch из-за прекращения поддержки сервиса в Yandex Cloud. 


## Решенные проблемы {#problems-solved}

1. Расширена функциональность сопоставления типов `Date` и `DateTime` для [приемника Managed Service for YDB](../operations/endpoint/target/yandex-database.md), которая позволяет создать трансфер с требуемым типом без необходимости выполнения дополнительного преобразования.

1. Улучшено поведение репликации из [PostgreSQL](../operations/endpoint/source/postgresql.md) при удалении колонок из таблиц в базе данных источника.
