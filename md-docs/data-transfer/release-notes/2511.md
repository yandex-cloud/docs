# История изменений Data Transfer в ноябре 2025

## Новые возможности {#new-features}


1. Поддержан трансфер данных из [Managed Service for YDB](../operations/endpoint/source/ydb.md) в [PostgreSQL](../operations/endpoint/target/postgresql.md).


1. Реализована принудительная остановка регулярных трансферов, если в течение двух месяцев попытки запустить трансфер завершаются с ошибкой.

## Решенные проблемы и улучшения {#problems-solved}

1. Исправлено поведение трансфера данных из [PostgreSQL](../operations/endpoint/source/postgresql.md) в [PostgreSQL](../operations/endpoint/target/postgresql.md) с `gpfdist` при попытке переноса пустой таблицы.


1. Повышена устойчивость работы трансферов при отключении [зоны доступности](../../overview/concepts/geo-scope.md).
