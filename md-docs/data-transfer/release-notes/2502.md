# История изменений Data Transfer в феврале 2025

## Новые возможности {#new-features}

1. Поддержан перенос многоуровнево партиционированных таблиц из [PostgreSQL](../operations/endpoint/source/postgresql.md).

1. Добавлена возможность выбирать каталог при настройке эндпоинтов, для которых активирована интеграция с [Yandex Connection Manager](../../metadata-hub/concepts/connection-manager.md).


## Решенные проблемы {#problems-solved}

1. Исправлена работа [трансформера фильтра колонок](../concepts/data-transformation.md#columns-filter) при обновлении и удалении строк.

1. Исправлено зависание трансфера в случае недоступности необходимого WAL-лога в источнике [PostgreSQL](../operations/endpoint/source/postgresql.md).


1. Исправлено поведение поля **Метод репликации** для источника [SQL Server](../operations/endpoint/source/mssql.md).

1. Исправлена ссылка на топик [Data Streams](../operations/endpoint/source/data-streams.md) в параметрах эндпоинта.
