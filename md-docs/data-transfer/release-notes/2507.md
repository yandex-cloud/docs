# История изменений Data Transfer в июле 2025

## Новые возможности {#new-features}


1. Реализован запрет на перезапуск трансфера при редактировании метаинформации.

1. Доработан трансформер [Шардирование](../concepts/data-transformation.md#shard) для поставок из очередей.

## Решенные проблемы {#problems-solved}

1. Исправлена проблема переноса данных из [Object Storage](../operations/endpoint/source/object-storage.md) в [ClickHouse®](../operations/endpoint/target/clickhouse.md).

1. Исправлена проблема падения репликации из [PostgreSQL](../operations/endpoint/source/postgresql.md) при переносе большого количества таблиц.

1. Исправлена проблема с переносом партицированных таблиц из [PostgreSQL](../operations/endpoint/source/postgresql.md).

1. Исправлена ошибка при разбиении больших таблиц на части для переноса из [PostgreSQL](../operations/endpoint/source/postgresql.md).

1. Повышена стабильность запуска трансферов.
