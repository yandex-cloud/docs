# История изменений Data Transfer в январе 2024

## Новые возможности {#new-features}

1. Оптимизирована структура документации: консолидирована информация, относящаяся к работе с конкретным эндпоинтом (например, источником [MySQL®](../operations/endpoint/source/mysql.md) или приемником [PostgreSQL](../operations/endpoint/target/postgresql.md)). 

## Решенные проблемы {#problems-solved}

1. Исправлена ошибка подключения к Apache Kafka®.
1. Устранена проблема при редактировании трансформера.
1. Запрещена репликация из эндпоинта MySQL® при отсутствии бинарного лога (binlog).
1. Устранена проблема передачи данных из эндпоинта Metrica.
