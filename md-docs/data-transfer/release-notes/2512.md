# История изменений Data Transfer в декабре 2025

## Новые возможности {#new-features}

* Источник данных [Object Storage](../operations/endpoint/source/object-storage.md) перешел в Public Preview и стал доступен всем пользователям сервиса.

* Появилась возможность отфильтровать список трансферов по доступным параметрам:
  * имя эндпоинта или трансфера;
  * тип базы данных источника и приемника;
  * тип трансфера;
  * стадия, на которой находится трансфер.

* Добавлена расшифровка ошибок трансфера с помощью ИИ.
