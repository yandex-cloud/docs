# История изменений Data Transfer в июне 2024

## Новые возможности {#new-features}

Поддержана [версия 16](https://www.postgresql.org/docs/release/16.0/) для базы данных PostgreSQL.

## Решенные проблемы {#problems-solved}


1. Теперь трансферы с [источниками](../operations/endpoint/source/opensearch.md) и [приемниками](../operations/endpoint/target/opensearch.md) Managed Service for OpenSearch могут работать с любыми кластерами, а не только с теми, у которых есть публичные хосты.


1. Удалены лишние подчеркивания в начале имен файлов при поставке в [Yandex Object Storage](../operations/endpoint/target/object-storage.md).

1. Исправлена ошибка репликации из [MySQL®](../operations/endpoint/source/mysql.md) для совместимых серверов MariaDB.
