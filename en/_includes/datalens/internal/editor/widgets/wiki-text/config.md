## Config {#config}

### General information {#common}

As a result of executing the Config tab, data responsible for additional display configuration should be exported.

### Example {#example}

**Config** tab
```js
module.exports = {
    // CSS styles to be applied to the Wiki text container
    style: {
        color: 'grey',
        'text-align': 'center'
    }
};
```
