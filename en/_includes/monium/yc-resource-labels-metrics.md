* `project = folder__<folder_ID>`: ID of the folder the {{ yandex-cloud }} resource resides in.
* `cluster = default`.
* `service = <__service_name__>`, e.g., `__application-load-balancer__`, `__cdn__`, or `__iam__`.
* Labels of the resource. For example, for {{ iam-name }}, you can specify a service account: `service_account = <service_account_name>`.
