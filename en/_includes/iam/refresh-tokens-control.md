With the {{ yandex-cloud }} CLI and [API](../../iam/api-ref/RefreshToken/index.md), you can [view](../../iam/operations/refresh-token/list.md) the list of refresh tokens issued for a user and [revoke](../../iam/operations/refresh-token/revoke.md) such tokens. If you have a Yandex account, you can view the list of your refresh tokens on the [My account]({{ link-my-account }}) portal.
