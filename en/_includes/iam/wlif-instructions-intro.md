[Workload identity federations](../../iam/concepts/workload-identity.md) allow you to configure a link between external systems and {{ yandex-cloud }} via the [OpenID Connect](https://openid.net/developers/how-connect-works/) (OIDC) protocol. This allows external systems to perform actions with {{ yandex-cloud }} resources under {{ iam-short-name }} [service accounts](../../iam/concepts/users/service-accounts.md) without using [authorized keys](../../iam/concepts/authorization/key.md). This is a more secure method that minimizes the risk of credential leakage and the possibility of unauthorized access.
