{% note tip %}

We recommend [prefetching](../../cdn/operations/resources/prefetch-files.md) large files (over 200 MB) into the CDN server [cache](../../cdn/concepts/caching.md). This is an optional step. If skipped, such content will be automatically cached on [CDN servers](../../cdn/concepts/points-of-presence.md) when first accessed by the end user.

{% endnote %}
