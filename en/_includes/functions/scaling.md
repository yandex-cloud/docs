* `zone_instances_limit`: Number of function instances per [availability zone](../../overview/concepts/geo-scope.md).
* `zone_requests_limit`: Number of concurrent function calls per availability zone.
* `provisioned_instances_count`: Number of [provisioned instances](../../functions/concepts/function.md#provisioned-instances).
* `concurrency`: Maximum number of calls processed concurrently by a single function instance. It is only available in specific runtime environments. For more information, see [{#T}](../../functions/concepts/function.md#concurrency).
