* In the **{{ ui-key.yacloud.serverless-functions.triggers.form.field_retry-interval }}** field, specify how long to wait before retrying the function if it fails. The values may range from 10 to 60 seconds. The default value is 10 seconds.
* In the **{{ ui-key.yacloud.serverless-functions.triggers.form.field_retry-attempts }}** field, specify the number of function retries before the trigger sends a message to the dead letter queue. The values may range from 1 to 5. The default value is 1.
