The L7 load balancer supports [IETF](https://www.ietf.org/) (RFC [2459](https://www.ietf.org/rfc/rfc2459.txt)/[3280](https://www.ietf.org/rfc/rfc3280.txt)/[5280](https://www.ietf.org/rfc/rfc5280.txt)) compliant [X.509 certificates](https://{{ lang }}.wikipedia.org/wiki/X.509). For [ECDSA](https://{{ lang }}.wikipedia.org/wiki/ECDSA) certificates, only the P-256 curve is supported.
