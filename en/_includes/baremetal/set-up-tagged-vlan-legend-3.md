`<server_IP_address_in_secondary_subnet>`: IPv4 address the server will use in the secondary subnet, with the [CIDR](https://en.wikipedia.org/wiki/Classless_Inter-Domain_Routing) prefix, e.g., `192.168.1.10/24`.

This address must be available and belong to the secondary private subnet range.
