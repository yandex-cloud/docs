* `partitions`: Number of partitions, provided as an object with the `value` field.
* `replication_factor`: Replication factor, provided as an object with the `value` field.
* `topic_config_3`: Topic settings for {{ KF }} `3.x`. Specify each setting on a new line, separated by commas.

   See [{#T}](../../../../managed-kafka/concepts/settings-list.md#topic-settings) for the descriptions and possible values of the settings.
