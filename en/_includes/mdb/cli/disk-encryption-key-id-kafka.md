`--disk-encryption-key-id`: ID of the [custom KMS key](../../../kms/concepts/key.md). To encrypt the disk, provide the KMS key ID in this parameter. To learn more about disk encryption, see [Storage](../../../managed-kafka/concepts/storage.md#disk-encryption).

{% note warning %}
   
You can enable disk encryption only when creating a cluster.
   
{% endnote %}
