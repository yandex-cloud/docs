* `--cloud-storage-data-cache`: Enables caching files in the cluster storage. The default value is `true` (enabled).
* `--cloud-storage-data-cache-max-size`: Sets the maximum cache size, in bytes, allocated in the cluster storage. The default value is `1073741824` (1 GB).
* `--cloud-storage-move-factor`: Sets the minimum percentage of free space in the cluster storage. If your free space percentage is below this value, the data will be moved to {{ objstorage-full-name }}. The minimum value is `0`, the maximum value is `1`, and the default value is `0.01`.
* `--cloud-storage-prefer-not-to-merge`: Disables [merging of data parts]({{ ch.docs }}/engines/table-engines/mergetree-family/custom-partitioning-key/) in cluster and object storages. To disable merging, set to `true` or provide this setting without a value. To leave merging enabled, set to `false` or do not provide this setting in your CLI command when creating a cluster.
