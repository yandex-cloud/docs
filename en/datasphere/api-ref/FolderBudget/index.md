# FolderBudget
A set of methods for managing Datasphere folder budgets.

## Methods {#methods}
Method | Description
--- | ---
[get](get.md) | Returns the specified folder budget.
[set](set.md) | Sets the unit balance and the limits of the specified folder budget.
