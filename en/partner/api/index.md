# Partner portal API

You can interact with the {{ yandex-cloud }} Partner portal via the API.

For available commands, see [Gateway API: Method attribute tables](https://disk.yandex.ru/d/gsRt3rHTsTkC4w).

An [IAM token](../../iam/concepts/authorization/iam-token.md) is used for authentication.
