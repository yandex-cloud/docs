# yc serverless container revision

Manage container revisions

#### Command Usage

Syntax:

`yc serverless container revision <command>`

Aliases:

- `revisions`

#### Command Tree

- [yc serverless container revision deploy](deploy.md) — Deploy new container revision

- [yc serverless container revision get](get.md) — Show information about the specified container revision

- [yc serverless container revision list](list.md) — List container revisions
