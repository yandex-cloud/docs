# yc iam access-key delete

Delete the specified access key
Access keys are used for the authentication by Yandex Object Storage API
to provide compatibility with Amazon S3 API.

#### Command Usage

Syntax:

`yc iam access-key delete <ACCESS-KEY-ID> [<ACCESS-KEY-ID>...] [Global Flags...]`

#### Flags

#|
||Flag | Description ||
|| `--id` | `string`

Access key id. This is id of key resource in cloud, but not key id that is used for authentication. ||
|#
