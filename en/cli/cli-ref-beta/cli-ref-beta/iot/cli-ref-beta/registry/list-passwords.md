# yc beta iot registry list-passwords

Retrieves the list of passwords for the specified registry.

#### Command Usage

Syntax: 

`yc beta iot registry list-passwords <REGISTRY-ID>`

#### Flags

| Flag | Description |
|----|----|
|`--registry-id`|<b>`string`</b><br/>ID of the registry to list passwords in.<br/><br/>To get a registry ID make a [RegistryService.List] request.|
