# yc beta serverless function get

Returns the specified function.
To get the list of all available functions, make a [List] request.

#### Command Usage

Syntax:

`yc beta serverless function get <FUNCTION-ID>`

#### Flags

#|
||Flag | Description ||
|| `--function-id` | `string`

ID of the function to return. To get a function ID make a [FunctionService.List] request. ||
|#
