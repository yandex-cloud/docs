# yc beta serverless trigger update

Updates the specified trigger.

#### Command Usage

Syntax: 

`yc beta serverless trigger update <TRIGGER-ID>`

#### Flags

| Flag | Description |
|----|----|
|`--description`|<b>`string`</b><br/>New description of the trigger.|
|`--labels`|<b>`stringToString`</b><br/>Trigger labels as 'key:value' pairs.<br/><br/>Existing set of labels is completely replaced by the provided set, so if you just want<br/>to add or remove a label, request the current set of labels with a [TriggerService.Get] request.|
|`--name`|<b>`string`</b><br/>New name for the trigger.<br/>The name must be unique within the folder.|
