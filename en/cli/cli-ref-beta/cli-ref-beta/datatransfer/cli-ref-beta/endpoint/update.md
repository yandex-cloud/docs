# yc beta datatransfer endpoint update

Updates the specified endpoint.

#### Command Usage

Syntax: 

`yc beta datatransfer endpoint update <ENDPOINT-ID>`

#### Flags

| Flag | Description |
|----|----|
|`--description`|<b>`string`</b><br/>The new description for the endpoint.|
|`--endpoint-id`|<b>`string`</b><br/>Identifier of the endpoint to be updated.|
|`--labels`|<b>`stringToString`</b><br/>Endpoint labels as 'key:value' pairs.<br/><br/>For details about the concept, see [documentation]({{ api-url-prefix<br/>}}/resource-manager/concepts/labels).|
|`--name`|<b>`string`</b><br/>The new endpoint name. Must be unique within the folder.|
