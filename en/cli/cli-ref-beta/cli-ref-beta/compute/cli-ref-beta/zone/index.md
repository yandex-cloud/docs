# yc beta compute zone

A set of methods to retrieve information about availability zones.

#### Command Usage

Syntax:

`yc beta compute zone <command>`

#### Command Tree

- [yc beta compute zone get](get.md) — Returns the information about the specified availability zone.

- [yc beta compute zone list](list.md) — Retrieves the list of availability zones.
