# yc beta compute host-group update-host

Update host

#### Command Usage

Syntax:

`yc beta compute host-group update-host <HOST-GROUP-ID>`

#### Flags

#|
||Flag | Description ||
|| `--deadline-at` | `time`

The date and time when this host will be automatically freed of instances. Timestamp in RFC3339 text format. (RFC3339) ||
|| `--host-group-id` | `string`

ID of the host group to update. ||
|| `--host-id` | `string`

ID of the host to update. ||
|| `--async` | Display information about the operation in progress, without waiting for the operation to complete. ||
|#
