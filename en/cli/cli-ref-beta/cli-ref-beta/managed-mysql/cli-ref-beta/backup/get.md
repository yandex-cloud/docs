# yc beta managed-mysql backup get

Retrieves information about the specified backup.

#### Command Usage

Syntax:

`yc beta managed-mysql backup get <BACKUP-ID>`

#### Flags

#|
||Flag | Description ||
|| `--backup-id` | `string`

ID of the backup to return information about. To get this ID, make a [BackupService.List] request (lists all backups in a folder) or a [ClusterService.ListBackups] request (lists all backups for an existing cluster). ||
|#

#### Global Flags

#|
||Flag | Description ||
|| `--profile` | `string`

Set the custom profile. ||
|| `--region` | `string`

Set the region. ||
