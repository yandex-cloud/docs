# yc beta application-load-balancer load-balancer update

Updates the specified application load balancer.

#### Command Usage

Syntax: 

`yc beta application-load-balancer load-balancer update <LOAD-BALANCER-ID>`

#### Flags

| Flag | Description |
|----|----|

Get the current settings with a [LoadBalancerService.Get] request.<br/>2. Add or remove a policy in this set.<br/>3. Send the new set in this field.<br/><br/>Example:|
|`--allocation-policy`|<b>`locations=[{disable-traffic=true,`</b><br/>subnet-id=value, zonal-shift-active=true, zonal-traffic-disabled=true, zone-id=value}]<br/><br/>Shorthand Syntax:<br/>{locations=[{disable-traffic=bool, subnet-id=str, zonal-shift-active=bool, zonal-traffic-disabled=bool, zone-id=str},...]}<br/>Fields:<br/>locations  []struct  — Availability zones and subnets that the application load balancer resides.<br/>disable-traffic         bool              — Disables the load balancer node in the specified availability zone.<br/><br/>Backends in the availability zone are not directly affected by this setting.<br/>They still may receive traffic from the load balancer nodes in other availability zones,<br/>subject to [LoadBalancingConfig.locality_aware_routing_percent] and [LoadBalancingConfig.strict_locality] settings.<br/>subnet-id               string            — ID of the subnet that the application load balancer belongs to.<br/>zonal-shift-active      bool              — Show zonal shift status for the location.<br/>Deprecated: use [zonal_traffic_disabled] below to track traffic status.<br/>zonal-traffic-disabled  bool              — Computed field: will be set to true if all traffic in zone is disabled<br/>either manually by user or automatically by Cloud infrastructure.<br/>zone-id                 string  required  — ID of the availability zone where the application load balancer resides.<br/><br/>Each availability zone can only be specified once.<br/>|

|`--description`|<b>`string`</b><br/>New description of the application load balancer.|

Get the current set of labels with a [LoadBalancerService.Get] request.<br/>2. Add or remove a label in this set.<br/>3. Send the new set in this field.|
