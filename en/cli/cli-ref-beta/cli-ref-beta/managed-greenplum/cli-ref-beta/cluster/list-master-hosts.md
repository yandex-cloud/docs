# yc beta managed-greenplum cluster list-master-hosts

Retrieves a list of master hosts for the specified cluster.

#### Command Usage

Syntax: 

`yc beta managed-greenplum cluster list-master-hosts <CLUSTER-ID>`

#### Flags

| Flag | Description |
|----|----|
|`--cluster-id`|<b>`string`</b><br/>ID of the Greenplum® cluster. To get the Greenplum® cluster ID use a [ClusterService.List] request.|
|`--page-size`|<b>`int`</b><br/>The maximum number of results per page to return. If the number of available results is larger than [page_size], the service returns a [ListClusterHostsResponse.next_page_token] that can be used to get the next page of results in subsequent list requests.|
|`--page-token`|<b>`string`</b><br/>Page token. To get the next page of results, set [page_token] to the [ListClusterHostsResponse.next_page_token] returned by the previous list request.|
