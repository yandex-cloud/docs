# yc beta cdn provider

A set of methods for managing Provider Service resources.

#### Command Usage

Syntax:

`yc beta cdn provider <command>`

#### Command Tree

- [yc beta cdn provider activate](activate.md) — Deprecated: This functionality is no longer supported - provider activation happens automatically.

- [yc beta cdn provider list-activated](list-activated.md) — List activated providers for specified client.

#### Global Flags

#|
||Flag | Description ||
|| `--profile` | `string`

Set the custom profile. ||
|| `--region` | `string`

Set the region. ||
|| `--debug` | Debug logging. ||

Enable gRPC retries. By default, retries are enabled with maximum 5 attempts.
