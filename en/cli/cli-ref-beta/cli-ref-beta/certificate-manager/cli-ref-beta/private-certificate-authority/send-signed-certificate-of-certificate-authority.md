# yc beta certificate-manager private-certificate-authority send-signed-certificate-of-certificate-authority

Sends a signed certificate chain after a CSR has been generated.  This finalizes the creation process after a signed certificate has been obtained.

#### Command Usage

Syntax: 

`yc beta certificate-manager private-certificate-authority send-signed-certificate-of-certificate-authority <CERTIFICATE-AUTHORITY-ID>`

|`--example-json`|Generates a JSON template of the request.<br/>The template can be customized and used as input for the command.<br/>Usage example:<br/><br/>1.
