# delete method

Deletes an object.

## Request {#request}

```
DELETE /{bucket}/{key} HTTP/1.1
```

### Path parameters {#path-parameters}

| Parameter | Description |
| ----- | ----- |
| `bucket` | Bucket name. |
| `key` | Object key. |

### Headers {#request-headers}

Use only [common request headers](../common-request-headers.md) in requests.
