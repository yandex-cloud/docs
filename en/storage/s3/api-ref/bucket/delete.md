# deleteBucket method

Deletes an empty bucket. If the bucket isn't empty, first delete all the objects inside the bucket.

## Request {#request}

```
DELETE /{bucket} HTTP/1.1
```

### Path parameters {#path-parameters}

| Parameter | Description |
| ----- | ----- |
| `bucket` | Bucket name. |
