# Managing access to dashboards

You can limit dashboard access to individual users or user groups at the dashboard folder level or for individual dashboards. Individual dashboards inherit access permissions from their folder.

1. Open Grafana at `https://grafana.sys.{{ cluster-domain }}` and log in with your username and password.
1. Open the **Dashboards** section.
1. Select the folder for which you want to change dashboard access.
1. Click **Folder actions** and select **Manage permissions**.
1. In the window that opens, you can change access permissions for existing roles and users or add a new permission for a user, role, group, or service account.
