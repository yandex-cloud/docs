# Limits {{ mrd-name }}

{{ mrd-name }} allows you to run [any Redis commands](https://redis.io/commands), except the following:

- `BGREWRITEAOF`
- `BGSAVE`
- `COMMAND`
- `CONFIG`
- `DEBUG`
- `LASTSAVE`
- `MIGRATE`
- `MODULE`
- `MONITOR`
- `MOVE`
- `OBJECT`
- `REPLICAOF`
- `SAVE`
- `SHUTDOWN`
- `SLAVEOF`
