# [{{ datalens-full-name }}] Failed to load description of table columns

`ERR.DS_API.COLUMN_SCHEMA_FAILED`

This error occurs when the table schema cannot be retrieved.


Make sure you can read the table schema using the standard methods for your source, and contact the {{ yandex-cloud }} [support]({{ link-console-support }}). In your message, describe when the error occurs.
