# [{{ datalens-full-name }}] Invalid connection ID

`ERR.DS_API.US.VALIDATION_FAILED.CONNECTION`

Error validating the requested connection ID. A special case of the [ERR.DS_API.US.VALIDATION_FAILED](./ERR-DS_API-US-VALIDATION_FAILED.md) error when requesting a connection with an incorrect ID.

Check and correct the requested connection ID.

You can copy object IDs from the relevant object's menu or in navigation by clicking **Copy ID**. The ID will be saved to the clipboard.
