# [{{ datalens-full-name }}] Field references an unknown source avatar

`ERR.DS_API.AVATAR.NOT_FOUND.FIELD_REF`

This error occurs when a dataset field references a table that was previously deleted. [Replace the field source](../../dataset/create-dataset.md#replace-field) in the dataset: click the **Field source** and specify a valid table or delete the incorrect field.
