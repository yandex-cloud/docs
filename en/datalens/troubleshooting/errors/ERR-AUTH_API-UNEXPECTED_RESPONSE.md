# [{{ datalens-full-name }}] Got an unexpected response from an external API

`ERR.AUTH_API.UNEXPECTED_RESPONSE`

The error occurs when an invalid response is received during an attempt to authorize a connection in an external system, e.g., Yandex ID, Google, or Snowflake.

To fix the error, retry the authorization after a while.


If the error persists, contact {{ yandex-cloud }} [support]({{ link-console-support }}). In your message, describe when the error occurs.
