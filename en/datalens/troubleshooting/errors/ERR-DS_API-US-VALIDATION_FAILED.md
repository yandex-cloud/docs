# [{{ datalens-full-name }}] Invalid input data: validation failed

`ERR.DS_API.US.VALIDATION_FAILED`

Error validating the requested object ID, e.g., when requesting a connection or dataset with an invalid ID.

Check and correct the requested object ID.

You can copy object IDs from the relevant object's menu or in navigation by clicking **Copy ID**. The ID will be saved to the clipboard.
