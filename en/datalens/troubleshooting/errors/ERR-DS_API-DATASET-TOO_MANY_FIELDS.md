# [{{ datalens-full-name }}] There are too many fields in the dataset

`ERR.DS_API.DATASET.TOO_MANY_FIELDS`

This error occurs when attempting to add new fields to a dataset and this results in exceeding the [limit](../../concepts/limits.md#datalens-common-limits).

To fix this error, delete dataset fields so that their number remains within the limits.
