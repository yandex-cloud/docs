# [{{ datalens-full-name }}] Source avatar not found

`ERR.DS_API.AVATAR.NOT_FOUND`

The specified table was not found in the dataset.

This error occurs when attempting to access a table that was previously deleted from a dataset.

To fix the error, delete the tables from the dataset workspace, drag them in again, and click **Update fields**.


If the error persists, contact {{ yandex-cloud }} [support]({{ link-console-support }}). In your message, describe when the error occurs.
