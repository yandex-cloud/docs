# Adding dataset info in {{ datalens-full-name }}

To add dataset info, do the following:

1. Open the dataset to add info for.
1. Click ![image](../../_assets/console-icons/circle-info.svg) at the top of the screen.
1. Enter the information and click **Save**.

   {% cut "Adding dataset info" %}

   ![add-description](../../_assets/datalens/dataset/dataset-add-description.png)

   {% endcut %}

1. In the top-right corner, click **Save**.

Now all users will see the ![image](../../_assets/console-icons/circle-info.svg) icon in the top-right corner of the dataset. Clicking it will open the **Dataset info** window with the information.

{% cut "Viewing dataset info" %}

![add-description](../../_assets/datalens/dataset/dataset-description.png)

{% endcut %}

To remove dataset info, save an empty field in the **Dataset info** window. Users will no longer see the ![image](../../_assets/console-icons/circle-info.svg) icon.
