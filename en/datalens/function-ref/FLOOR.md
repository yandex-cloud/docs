# FLOOR



#### Syntax {#syntax}


```
FLOOR( number )
```

#### Description {#description}
Rounds the value down to the nearest integer.

**Argument types:**
- `number` â€” `Fractional number | Integer`


**Return type**: `Fractional number`

#### Example {#examples}



| **[value]**   | **FLOOR([value])**   |
|:--------------|:---------------------|
| `1.00`        | `1.00`               |
| `0.10`        | `0.00`               |
| `-2.00`       | `-2.00`              |
| `50.50`       | `50.00`              |
| `0.00`        | `0.00`               |
| `-3.50`       | `-4.00`              |
