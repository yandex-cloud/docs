# EXP



#### Syntax {#syntax}


```
EXP( number )
```

#### Description {#description}
Returns the result of raising the number 'e' to the power of `number`.

**Argument types:**
- `number` â€” `Fractional number | Integer`


**Return type**: `Fractional number`

#### Examples {#examples}

```
EXP(0) = 1.0
```

```
EXP(1) = 2.718282
```

```
EXP(3) = 20.08553
```
