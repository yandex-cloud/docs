# LEN



#### Syntax {#syntax}


```
LEN( value )
```

#### Description {#description}
Returns the number of characters in the string or items in array `value`.

**Argument types:**
- `value` â€” `Array of fractional numbers | Array of integers | Array of strings | String`


**Return type**: `Integer`

#### Example {#examples}

```
LEN("Computer") = 8
```
