# ASIN



#### Syntax {#syntax}


```
ASIN( number )
```

#### Description {#description}
Returns the arcsine of `number` in radians.

**Argument types:**
- `number` â€” `Fractional number | Integer`


**Return type**: `Fractional number`

#### Example {#examples}



| **[n]**   | **ASIN([n])**   |
|:----------|:----------------|
| `-1.00`   | `-1.57`         |
| `-0.87`   | `-1.05`         |
| `-0.71`   | `-0.79`         |
| `-0.50`   | `-0.52`         |
| `0.00`    | `0.00`          |
| `0.50`    | `0.52`          |
