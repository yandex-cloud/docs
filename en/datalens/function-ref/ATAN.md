# ATAN



#### Syntax {#syntax}


```
ATAN( number )
```

#### Description {#description}
Returns the arctangent of `number` in radians.

**Argument types:**
- `number` â€” `Fractional number | Integer`


**Return type**: `Fractional number`

#### Example {#examples}



| **[n]**   | **ATAN([n])**   |
|:----------|:----------------|
| `-1.00`   | `-0.79`         |
| `-0.87`   | `-0.71`         |
| `-0.71`   | `-0.62`         |
| `-0.50`   | `-0.46`         |
| `0.00`    | `0.00`          |
| `0.50`    | `0.46`          |
| `0.71`    | `0.62`          |
| `0.87`    | `0.71`          |
