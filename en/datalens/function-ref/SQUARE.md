# SQUARE



#### Syntax {#syntax}


```
SQUARE( number )
```

#### Description {#description}
Returns the number `number` raised to the power of 2.

**Argument types:**
- `number` â€” `Fractional number | Integer`


**Return type**: Same type as (`number`)
