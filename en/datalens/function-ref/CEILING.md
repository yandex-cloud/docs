# CEILING



#### Syntax {#syntax}


```
CEILING( number )
```

#### Description {#description}
Rounds the value up to the nearest integer.

**Argument types:**
- `number` â€” `Fractional number | Integer`


**Return type**: `Fractional number`

| **[value]**   | **CEILING([value])**   |

| `-2.00`       | `-2.00`                |
| `50.50`       | `51.00`                |
| `0.00`        | `0.00`                 |
