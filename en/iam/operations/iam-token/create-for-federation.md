# Getting an IAM token for a federated account

{% note info %}

Lifetime of an [IAM token](../../../iam/concepts/authorization/iam-token.md) cannot exceed {{ iam-token-lifetime }} and is limited by the cookie lifetime for the [federation](../../concepts/users/identity-federations.md).

{% endnote %}

{% list tabs %}

- CLI

  1. [Authenticate with the CLI as a federated user](../../../cli/operations/authentication/federated-user.md).

  2. Get an IAM token:

      ```
      $ yc iam create-token
      ```

{% endlist %}
