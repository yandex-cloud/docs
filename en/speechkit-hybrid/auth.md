# Authentication with the {{ sk-hybrid-name }} API

When using {{ sk-hybrid-name }}, authentication is not required for operations via the API.

However, when using the [Cloud Billing licensing model](pricing.md#billing), you need an [API key]({{ link-docs }}/iam/concepts/authorization/api-key) to authenticate in the cloud.

The {{ sk-hybrid-name }} API and the [{{ speechkit-full-name }} API]({{ link-docs-ai }}/speechkit/concepts/api) are the same.
