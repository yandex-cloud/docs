# {{ data-transfer-name }} release notes: November 2025

## New features {#new-features}


1. Added support for data transfers from [{{ ydb-name }}](../operations/endpoint/source/ydb.md) to [{{ PG }}](../operations/endpoint/target/postgresql.md).


1. Implemented a forced stop of regular transfers if attempts to start a transfer fail for two months.

## Fixed issues and improvements {#problems-solved}

1. Fixed the behavior bug in data transfers from [{{ PG }}](../operations/endpoint/source/postgresql.md) to [{{ PG }}](../operations/endpoint/target/postgresql.md) with `gpfdist` when trying to migrate an empty table.


1. Improved the stability of transfers when the [availability zone](../../overview/concepts/geo-scope.md) is disabled.
