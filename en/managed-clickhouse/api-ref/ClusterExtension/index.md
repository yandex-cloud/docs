# Managed Service for ClickHouse API, REST: ClusterExtension

## Methods

#|
||Method | Description ||
|| [Get](get.md) | Returns the specified cluster extension. ||
|| [List](list.md) | Lists extensions installed in the specified cluster. ||
|| [Create](create.md) | Installs a new extension in the specified cluster. ||
|| [Delete](delete.md) | Deletes the specified extension from the cluster. ||
|| [Update](update.md) | Updates the specified cluster extension. ||
|| [SetExtensions](setExtensions.md) | Set the list of cluster extensions. Not specified extensions will be deleted. ||
|#
