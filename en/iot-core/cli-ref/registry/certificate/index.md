# yc iot registry certificate

Manage IoT device registry certificates

#### Command Usage

Syntax:

`yc iot registry certificate <command>`

Aliases:

- `certificates`

#### Command Tree

- [yc iot registry certificate add](add.md) — Add new certificate to specified registry

- [yc iot registry certificate delete](delete.md) — Delete specified certificate from registry

- [yc iot registry certificate list](list.md) — List certificates associated with specified registry
