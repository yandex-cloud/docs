> {% calc [currency=USD] 7,75 / 120 %} × 2 + {% calc [currency=USD] 18,90 / 120 %} + {% calc [currency=USD] 14,02 / 120 %} + {% calc [currency=USD] 26,76 / 120 %} + {% calc [currency=USD] 7,66 / 120 %} × 10 = {% calc [currency=USD] (7,75 × 2 + 18,90 + 14,02 + 26,76 + 7,66 × 10) / 120 %}
