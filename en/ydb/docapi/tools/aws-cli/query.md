# Data selections

To select data from the `series` table by the `series_id` key:

{% list tabs %}

- AWS CLI

   Run the command by replacing `https://your-database-endpoint` the [previously prepared Document API endpoint](index.md#before-you-begin) of your DB:

   {% note warning %}

   To work with the AWS CLI from Windows, we recommend using the [WSL]({{ ms.docs }}/windows/wsl/).

   {% endnote %}

   ```bash
   endpoint="https://your-database-endpoint"
   aws dynamodb query \
       --table-name series \
       --key-condition-expression "series_id = :name" \
       --expression-attribute-values '{":name":{"N":"2"}}' \
       --endpoint $endpoint
   ```

   Result:

