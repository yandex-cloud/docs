# Updating database parameters

## Serverless

1. In the [management console]({{ link-console-main }}), select the folder where you want to update DB settings.
1. In the list of services, select **{{ ydb-name }}**.
1. Click ![horizontal-ellipsis](../../../_assets/horizontal-ellipsis.svg) in the line with the desired DB and select **Edit**.
1. Configure the DB settings:

   * Under **Computing resources**, select the type and amount of [computing resources](../../concepts/databases.md#compute-units).
   * Under **Storage groups**, select the disk type and number of [storage groups](../../concepts/databases.md#storage-groups) that determines the total amount of storage.
1. Click **Update database**.
