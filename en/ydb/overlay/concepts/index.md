## YDB in {{ yandex-cloud }}

You can create a database in a Serverless configuration or with dedicated servers. For more information about differences in configurations, see [Serverless and Dedicated modes](serverless_and_dedicated.md).

The service is subject to the [service level agreement (SLA)]{% if lang == "ru" %}(https://yandex.ru/legal/cloud_sla?lang=ru){% endif %}{% if lang == "en" %}(https://yandex.ru/legal/cloud_sla?lang=en){% endif %}. The service level is defined in [Service Level for {{ ydb-full-name }}]{% if lang == "ru" %}(https://yandex.ru/legal/cloud_sla_ydb?lang=ru){% endif %}{% if lang == "en" %}(https://yandex.ru/legal/cloud_sla_ydb?lang=en){% endif %}.
