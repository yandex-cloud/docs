{% note info %}

Users can't change database parameters or delete a database. In the current version, only a system administrator can perform these actions.

To modify database parameters or delete a database, submit a request form.

* [Request form](https://forms.yandex-team.ru/surveys/30510/) to modify resources allocated to a database.
* [Request form](https://forms.yandex-team.ru/surveys/30513/) to delete a database.

{% endnote %}
