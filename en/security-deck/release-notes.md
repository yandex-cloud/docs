# {{ sd-full-name }} release notes

## Q3 2025 {#q3-2025}

* Launched [KSPM](./concepts/kspm.md) and [CSPM](./concepts/cspm.md).
* You can now use KSPM and CSPM as data sources for [alerts](./concepts/alerts.md). {{ tag-con }}
* You can now [create a {{ sd-name }} workspace](./operations/workspaces/index.md), which is a container holding module settings and resources, list of controlled resources, control parameters, and other settings. {{ tag-con }}
* [DSPM](./concepts/dspm.md) now supports partial scanning and image [scanning](./operations/dspm/create-scan.md). {{ tag-con }}
