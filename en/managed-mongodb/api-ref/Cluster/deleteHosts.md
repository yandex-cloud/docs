# Method deleteHosts
Deletes the specified hosts for a cluster.
 

 
## HTTP request {#https-request}
```
POST https://mdb.{{ api-host }}/managed-mongodb/v1/clusters/{clusterId}/hosts:batchDelete
```
 
## Path parameters {#path_params}
 
Parameter | Description
--- | ---
clusterId | <p>Required. ID of the MongoDB cluster to remove hosts from. To get the MongoDB cluster ID, use a <a href="/docs/managed-mongodb/api-ref/Cluster/list">list</a> request.</p> <p>The maximum string length in characters is 50.</p> 
 
## Body parameters {#body_params}
 
```json 
{
  "hostNames": [
    "string"
  ]
}
```

 
Field | Description
--- | ---
hostNames[] | **string**<br><p>Required. Names of hosts to delete.</p> <p>Must contain at least one element. The maximum string length in characters for each value is 253.</p>
