# Contacting support to create a new service connection {#pe-ticket}

To create a service connection, open a [support ticket]({{ link-console-support }}).


Where:

* `service_name`: [Service name](../concepts/private-endpoint.md#pe-services).
* `folder_id`: [ID of the folder](../../resource-manager/operations/folder/get-id.md) for the service connection.
* `network_id`: [ID of the cloud network](network-get-info.md#console_1) for the service connection.
* `private_dns_records_enabled`: Whether to create a primary DNS record for the service’s common FQDN to work via the service connection.
