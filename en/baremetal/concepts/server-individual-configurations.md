# On-request server configuration

If you are after an optimal configuration with specific needs in mind, you can have a server assembled based on your individual request. We will accommodate a project of any complexity, advise on optimal server configurations, and help organize the migration of your infrastructure to {{ baremetal-name }}.

## GPU cards and RAID controllers {#gpu-and-raid}

On request, we equip servers with GPU cards and RAID controllers.

For tasks requiring a substantial number of GPUs, we can hook up a JBOG (Just a Bunch of GPUs) module: an additional unit with multiple GPUs onboard. JBOG extends the server's computing capabilities beyond its built-in slots.

## Disk shelves {#disk-shelves}

To expand your disk space, you can opt for a JBOD (Just a Bunch of Disks) disk shelf, which is an additional module with multiple storage drives onboard.

Available disk shelf drive configurations:

* Up to 12 HDDs/SSDs
* Up to 24 NVMe SSDs
* Up to 2 M.2 SSDs

Your specific configuration will depend on the server model and the configuration option you select.

## How to order {#how-to-order}

To order an on-request server configuration:

1. [Contact us](https://yandex.cloud/en/services/baremetal#contact-form) using the feedback form.
1. Describe your server requirements: type and number of processors, memory size, storage drive configuration, need for GPU cards or RAID controllers.
1. Get a tailored offer.

The assembly timeline and terms of rent for an on-request configuration are determined on case-by-case basis.
