The `data-catalog.auditor` role enables viewing info on {{ data-catalog-name }}s resources and quotes.

Users with this role can:
* View info on catalogs in {{ data-catalog-name }} and [access permissions](../../iam/concepts/access-control/index.md) granted for them.
* View info on [domains](../../iam/concepts/access-control/index.md)in {{ data-catalog-name }} and access permissions granted for them.
* View info on [sources and downloads](../../iam/concepts/access-control/index.md) in {{ data-catalog-name }}.
* View info on data and data links in {{ data-catalog-name }}.
* View info on [glossaries and terms](../../metadata-hub/concepts/data-catalog.md#glossaries-and-terms) links in {{ data-catalog-name }}.
* View info on [classifications and tags](../../metadata-hub/concepts/data-catalog.md#glossaries-and-terms) links in {{ data-catalog-name }}.
* View info on the {{ data-catalog-name }} [quotes](../../metadata-hub/concepts/limits.md#data-catalog-quota).

This role includes the `data-catalog.catalogs.auditor`, `data-catalog.domains.auditor`, `data-catalog.ingestionSources.auditor`, `data-catalog.ingestions.auditor`, `data-catalog.assets.auditor`, `data-catalog.lineages.auditor`, `data-catalog.glossaries.auditor`, `data-catalog.glossaryTerms.auditor`, `data-catalog.classifications.auditor`, and `data-catalog.classificationTags.auditor` permissions.
