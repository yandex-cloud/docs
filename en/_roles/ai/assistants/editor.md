The `ai.assistants.editor` role enables using AI agents as well as managing files and Vector Store search indexes.

Users with this role can:
* Use [AI agents]({{ link-docs-ai }}ai-studio/concepts/agents/).
* View info on uploaded [files]({{ link-docs-ai }}ai-studio/concepts/search/vectorstore#file-uploading) as well as create, update, view, and delete them.
* View info on Vector Store [search indexes]({{ link-docs-ai }}ai-studio/concepts/search/vectorstore), create, modify, and delete them, as well as conduct searches using these indexes.
* View info on Yandex AI Studio [quotas]({{ link-docs-ai }}ai-studio/concepts/limits#yandexgpt-quotas).
* View info on the relevant [cloud]({{ link-docs }}/resource-manager/concepts/resources-hierarchy#cloud).
* View info on the relevant [folder]({{ link-docs }}/resource-manager/concepts/resources-hierarchy#folder).

This role includes the `ai.assistants.viewer` permissions.
