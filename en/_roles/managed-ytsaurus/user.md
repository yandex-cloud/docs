The `managed-ytsaurus.user` role enables performing basic operations on {{ ytsaurus-name }} clusters.

Users with this role can:
* Use the {{ ytsaurus-name }} web UI.
* View info on {{ ytsaurus-name }} clusters.
* View info on the [quotas](../../managed-ytsaurus/concepts/limits.md#quotas) for {{ myt-name }}.
* View info on resource operations for {{ myt-name }}.

This role includes the `managed-ytsaurus.viewer` permissions.
