The `marketplace.productInstances.admin` role enables managing installed {{ marketplace-short-name }} products and access to them.

Users with this role can:
* View info on installed {{ marketplace-short-name }} [products](../../../marketplace/concepts/product.md).
* View info on [access permissions](../../../iam/concepts/access-control/index.md) granted for installed {{ marketplace-short-name }} products and modify such permissions.
* Create {{ marketplace-short-name }} products, modify their metadata, as well as activate and deactivate {{ marketplace-short-name }} products.
* View info on the relevant [folder](../../../resource-manager/concepts/resources-hierarchy.md#folder).

This role includes the `marketplace.productInstances.editor` permissions.
