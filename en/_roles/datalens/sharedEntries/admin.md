The `datalens.sharedEntries.admin` role is assigned for a common object and enables viewing it and entirely managing it, including editing, moving, deleting, and configuring its access permissions. In the {{ datalens-name }} UI, this role is referred to as `Admin`. We recommend assigning this role only via the {{ datalens-name }} UI.

This role includes the `datalens.sharedEntries.editor` and `datalens.sharedEntries.entryBindingCreator` permissions.
