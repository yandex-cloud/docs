The `datalens.sharedEntries.limitedViewer` role is assigned for a common object and enables viewing its [charts](../../../datalens/concepts/chart/index.md) and the [dashboards](../../../datalens/concepts/dashboard.md) that use it, without direct access to the common object itself. In the {{ datalens-name }} UI, this role is referred to as `Limited viewer`. We recommend assigning this role only via the {{ datalens-name }} UI.
