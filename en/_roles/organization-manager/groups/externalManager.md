
The `organization-manager.groups.externalManager` role enables managing {{ org-full-name }} user groups associated with user groups in Active Directory or another external source.

Users with this role can:
* Associate {{ org-full-name }} [user groups](../../../organization/concepts/groups.md) with user groups in Active Directory or another external source.
* Modify and delete {{ org-full-name }} user groups associated with user groups in Active Directory or another external source.
* View the list of members belonging to {{ org-full-name }} user groups associated with user groups in Active Directory or another external source, as well as manage membership in such groups.
* View info on [access permissions](../../../iam/concepts/access-control/index.md) granted for {{ org-full-name }} user groups.
