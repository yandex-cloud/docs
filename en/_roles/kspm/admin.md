The `kspm.admin` role allows the user to engage, set up, and disconnect [{{ kspm-name }}](../../security-deck/concepts/kspm.md), create, modify, and delete exceptions from the rules and exceptions from the scope of control, view info on {{ managed-k8s-name }} [clusters](../../managed-kubernetes/concepts/index.md#kubernetes-cluster) connected to {{ kspm-name }}, {{ kspm-name }} users, and {{ kspm-name }} operations.

This role includes the `kspm.editor` permissions.
