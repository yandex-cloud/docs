The `kspm.viewer` role allows the user to view info on [{{ kspm-name }}](../../security-deck/concepts/kspm.md) settings, {{ managed-k8s-name }} [clusters](../../managed-kubernetes/concepts/index.md#kubernetes-cluster) connected to {{ kspm-name }}, exceptions from the rules, exceptions from the scope of control, {{ kspm-name }} users, and {{ kspm-name }} operations.

This role includes the `kspm.auditor` permissions.
