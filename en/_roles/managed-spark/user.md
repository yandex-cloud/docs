
The `managed-spark.user` role enables performing basic operations on {{ SPRK }} clusters and jobs.

Users with this role can:
* Use the {{ SPRK }} web interface.
* View info on [jobs](../../managed-spark/operations/index.md#jobs), as well as create, run, and cancel them.
* View info on [{{ SPRK }} clusters](../../managed-spark/concepts/index.md) and [access permissions](../../iam/concepts/access-control/index.md) granted for them.
* View info on [maintenance](../../managed-spark/concepts/maintenance.md) tasks for {{ SPRK }} clusters.
* View info on [quotas](../../managed-spark/concepts/limits.md#quotas) for {{ msp-name }}.

This role includes the `managed-spark.viewer` permissions.
