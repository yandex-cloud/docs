# yc config profile

Manage configuration profiles

#### Command Usage

Syntax:

`yc config profile <command>`

Aliases:

- `profiles`

#### Command Tree

- [yc config profile activate](activate.md) — Activate the specified configuration profile

- [yc config profile create](create.md) — Create a configuration profile

- [yc config profile delete](delete.md) — Delete the specified configuration profile

- [yc config profile get](get.md) — List values for the specified configuration profile

- [yc config profile list](list.md) — List configuration profiles

#### Global Flags

#|
||Flag | Description ||
|| `--profile` | `string`

Pass 0 to disable retries. Pass any negative value for infinite retries.
