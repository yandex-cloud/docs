#### Description

A [data event audit log](https://yandex.cloud/en/docs/audit-trails/concepts/format-data-plane) is a JSON object with a record of events related to Yandex Cloud resources. Data event monitoring makes it easier for you to collect additional events from cloud services and, as a result, effectively respond to security incidents in clouds. This also helps you ensure your cloud infrastructure meets regulatory requirements and industry standards. For example, you can keep track of your employees' access permissions to sensitive data stored in [buckets](https://yandex.cloud/en/docs/storage/concepts/bucket).

You need to enable collection of data event audit logs individually for each [supported service](https://yandex.cloud/en/docs/audit-trails/concepts/control-plane-vs-data-plane#data-plane-events).
