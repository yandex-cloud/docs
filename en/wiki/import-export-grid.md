# Uploading a table from a file

You can create dynamic tables from CSV or TXT files:

1. Save your table (such as an Excel file) in CSV format. Alternatively, create a TXT file in which the cell values are separated by a comma, semicolon, or tab.

1. [Create](create-grid.md) a new dynamic table or open an existing one.

1. In the lower-left corner of the table, select ![](../_assets/wiki/table-settings-footer.png) â†’ **Import data**.

1. Select a CSV or TXT file and check the delimiter settings.

1. Click **Upload**.

1. Select the table columns that you want to upload from the file.

1. Choose one of the methods for adding the columns you selected:

    - As a new column: Columns are added to the right of existing ones.

    - Into existing column: Columns are added to the end of existing columns.

    Behavior for inserting columns can be set up for each individual column after you import them from the file.

1. Select the data type for the new columns.

1. Click **Upload**.
