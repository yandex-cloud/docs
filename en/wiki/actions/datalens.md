# Charts from {{ datalens-full-name }}

[{{ datalens-full-name }}](https://datalens.yandex-team.ru/) is an analytics and visualization service enabling you to build advanced charts and dashboards based on multi-source data.

Read more about the service features in the [{{ datalens-full-name }} documentation](../../datalens/overview.md).

To add a chart to your Wiki page, copy the embed code from the dashboard:

1. Click ![image](../../_assets/options.svg) in the upper-right corner of the chart.
1. Select **Get link and code** from the drop-down list.
1. Go to the **Wiki** tab.
1. Configure the chart parameters.
1. Copy the code.

Paste the resulting `iframe` in your page:

```
{{iframe frameborder="0" width="350px" height="420px" src="https://charts.yandex-team.ru/preview/editor/fwnyltulelsq9?_embedded=1"}}
```

To learn more about the `iframe` block parameters, see [{#T}](iframe.md).
