# Draw.io diagrams

[Draw.io](https://drawio.yandex-team.ru/) is a service for creating diagrams online.

To add a diagram to a Wiki page:
1. Draw a diagram in draw.io.
1. In the **File** menu, select **Embed** â†’ **Yandex Wiki**.
1. Configure the diagram parameters and click **Create**.
1. Copy the code.

{% note info %}

To edit your diagram, return to draw.io, generate a new code, and embed it in the page. Otherwise, the diagram is not updated on {{ wiki-name }}.

If you edit your diagram on a Wiki page, its copy is created in draw.io. The original diagram remains unchanged.
