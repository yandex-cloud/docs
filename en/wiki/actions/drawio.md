{% note info %}

To edit your diagram, go back to draw.io, generate a new embed code, and paste it into the page. Otherwise, the diagram hosted on {{ wiki-name }} won't update.

If you edit the diagram on the Wiki page, the edited copy will appear on draw.io. However, the changes will not be applied to the original diagram.

{% endnote %}
