# Editing dynamic tables

To change a table, open it and click **Edit** in the lower-right corner.

## Edit the contents of table cells {#edit-cell}

Select a cell and enter your text. Use [Markdown](basic-markup.md) to format the text. To switch to a new row inside the cell, press **Shift** + **Enter**.

To save the changes, click **Done**. Each time you save your changes, a new copy of the table is moved to the [edit history](history.md). In the history, users can view all previous versions of a given table, compare the changes, and roll back to any of them.

## Configuring table column parameters {#column-params}

1. Click **≡** in a column's first cell.

1. Change the column parameters in the window that opens:

    - Enter the column heading in the top field.

    - Specify the column's width in pixels or as a percentage of the table width.

    - If you want the cells to require a value, choose the **Required** option.

    - For columns with the **Checkbox** data type, enable the **Mark as done** option. If the option is enabled, all rows selected with the checkbox will turn gray.

1. Click **Apply** to save your changes.

## Adding and removing columns {#edit-column}

To add a column:

1. Click **+** in the upper-right corner of the table.

1. Configure the column settings:

    - Enter the column heading in the top field.

    - Choose a data type to display in the column from the drop-down list.

    - Specify the column width in pixels or as a percentage of the table width.

    - If you want the cells to require a value, choose the **Required** option.

    - For columns with the **Checkbox** data type, enable the **Mark as done** option. If the option is enabled, all rows selected with the checkbox will turn gray.

1. Click **Add**.

To remove a column:

1. Click **≡** in a column's first cell.

1. Click **Delete column**.

## Adding and removing rows {#edit-line}

- To add a row, select **Add row** at the bottom of the page.

- To delete a row, hover the cursor over it and click ![](../_assets/wiki/del-row.png) to the right of the line.

## Renaming tables {#rename-tab}

1. Open the table.

1. In the panel on the left, click ![](../_assets/wiki/table-settings-sidebar.png).

1. Enter a new name for the table.

1. Click **Save**.

{% note info %}

You don't have to enable editing mode to rename your table.

{% endnote %}

## Sorting table rows {#sort-line}

Rows in dynamic tables can be quickly sorted by column:

1. In the column title, click ![](../_assets/wiki/sort-column.png).

1. Click the icon again to sort the rows in reverse order.

## Sorting tables by multiple columns at the same time {#sort-multi-column}

You can arrange tables using multiple types of sorting. Sorting types are specified in descending order of priority.

To create a new sorting arrangement:

1. In the panel on the left, click ![](../_assets/wiki/table-settings-sidebar.png).

1. Click **Add sorting criteria**.

1. Choose a column and sorting order.
