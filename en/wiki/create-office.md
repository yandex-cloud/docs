# Creating a Microsoft Office 365 document

{{wiki-name}} supports Word, Excel, and PowerPoint documents. Documents can be created in one of the following ways: [inÂ {{ wiki-name }}](office365/create-ms-office.md) or [on the Sharepoint site](office365/create-sharepoint.md).
