# Viewing a page and cluster structure

## Viewing the table of contents of a page {#toc}

To view the table of contents of a page:

1. Open the page.

1. In the panel on the left, click ![](../_assets/wiki/ico-toc.png).

1. To insert a table of contents into the page text, use the [`not_var{{toc}}`](add-toc.md) dynamic block.

## View the list of subpages {#subpages}

A page and all its subpages form a [cluster](structure.md). To view the cluster hierarchy:

1. Open the page.

1. To see the list of pages one level below the current page, click the left panel ![](../_assets/wiki/ico-subpages.png).

1. To see all the subpages of the current page, in the upper-right corner, click **Actions** and select **Cluster structure**.

    To view the whole structure, click **Show all subpages**.

## Embedding a cluster tree into a page {#action-tree}

Using the `not_var{{tree}}` dynamic block, you can create a cluster page tree and place it on any wiki page.

To build a tree for the current page, place the `not_var{{tree}}` block in the text of the page.

You can configure the tree's design with additional parameters.
