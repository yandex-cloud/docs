# Create a copy of a page

You can create a copy of any {{ wiki-name }} page:

1. In the upper-right corner of the page, click ![](../_assets/wiki/svg/actions-icon.svg) **Actions** and select ![](../_assets/wiki/svg/copy-pages.svg) **Creating a copy**.

1. You can change the page title or leave it as is.

1. Select a cluster where you want to create a copy of your page.

1. The **Page address** field is populated automatically.

1. Click **Copy**.
