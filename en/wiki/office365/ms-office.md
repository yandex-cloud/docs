# Working with Microsoft Office 365 documents

MS Office 365 documents that are stored on the Sharepoint  corporate portal or in one's personal OneDrive folder can be embedded in Wiki pages. This makes it easy to share documents, spreadsheets, and presentations and [co-edit them with your colleagues](co-edit.md). In {{wiki-name}}, you can work with Word, Excel, and PowerPoint documents. You can create documents right from the {{wiki-name}} interface and edit them in your browser using Microsoft Office cloud applications.

{% note alert %}

To create and edit embedded cloud documents, [log in to office.com](#office-login). If you don't have an MS Office 365 license, submit a request using the [form](https://help.yandex-team.ru/?form=soft).

{% endnote %}

## Logging in to Office 365 {#office-login}

To edit a document embedded in a Wiki page or create a new cloud document, log in to the office.com portal using your corporate Yandex account:

1. Go to [https://www.office.com/](https://www.office.com/).

1. In the upper-right corner, click **Sign in**.

1. Enter your corporate username such as `login@yandex-team.ru`.

1. If you have a license, you'll be forwarded to the Office 365 homepage.

   If not, you'll see a message saying:
   ![](../../_assets/wiki/login-office.png)

   To get an MS Office 365 license, send an email to `help@yandex-team.ru`.

{% note info %}

Contractors can get a license to view Office 365 documents. A license to create and edit documents is issued to contractors subject to approval by their supervisor.

{% endnote %}
