# Search {{ wiki-name }}

To search the text of {{ wiki-name }} pages and [keywords](add-key-words.md), use the search bar that appears in the upper-right part of each page: enter a query and click ![](../_assets/wiki/search-icon.png) or press **Enter**.

{% note warning %}

Recent changes to {{ wiki-name }} pages may appear in search results after a delay, since the pages are indexed once every 30 minutes.

{% endnote %}
