# How to reassign an IP address to a different availability zone


## Case description {#case-description}

You need to re-assign an IP address to a different availability zone.

## Solution {#case-resolution}

Currently, you cannot re-assign a reserved IP address between availability zones, as each zone has its own pool of addresses.
If your case requires re-assigning an IP address to a different availability zone, [create a support ticket]({{ link-console-support }}).
Provide the following information in your ticket:

1. Public IP address or its ID in {{ vpc-name }}.
1. Description of your use case that requires reassigning an IP address to a different availability zone.

We will recommend the most suitable solution for your needs.
