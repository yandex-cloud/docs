# Closing documents format in EDM



## Case description {#case-description}

You need to know which format the EDM-provided closing documents use.

## Solution {#case-resolution}

{{ yandex-cloud }} provides the following closing documents through EDM:

1. Report as a PDF and XML file, version 5.03, as per Federal Tax Service Order No. 1032 dated November 15, 2024, which amended the version previously approved by Federal Tax Service Order No. 970 dated December 19, 2023.
1. Invoice as an XML file, version 5.03, as per Federal Tax Service Order No. 1032 dated November 15, 2024, which amended the version previously approved by Federal Tax Service Order No. 970 dated December 19, 2023.

For more details, see [this article](../../../billing/concepts/edo).
