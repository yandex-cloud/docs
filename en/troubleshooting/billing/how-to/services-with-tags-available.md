# What services support tag-based billing 



## Case description {#case-description}

You need to find out which {{ yandex-cloud }} services offer tag-based billing.

## Solution {#case-resolution}

Tag-based billing is available for the following services:

* {{ compute-full-name }}: Tags for VMs, disks, snapshots, images, and instance groups.
* {{ vpc-full-name }}: Tags for IP addresses.
* {{ managed-k8s-full-name }}: Tags for node groups.

You can find a detailed list of services supporting tags on [this page](../../../resource-manager/concepts/labels.md#services).

## If your service is not listed here {#if-nothing-worked}

If you need a certain option or improvement, [request it in a support ticket]({{ link-console-support }}). 

In your request, describe your use case that requires tag-based billing.
