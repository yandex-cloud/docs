# Fixing the `Invitation not found` error when joining the partner program



## Issue description {#issue-description}

When trying to join the partner program, you get the `Invitation not found` error, while the client status remains unconfirmed.

## Solution {#issue-resolution}

This might occur when you are logged in with a different account. Make sure you are using a proper account: you can select it in the bottom-left corner of the form.
