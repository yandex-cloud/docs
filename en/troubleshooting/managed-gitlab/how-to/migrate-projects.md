# How to set up project migration



## Case description {#case-description}

You need to set up migration of projects from gitlab.com to {{ mgl-full-name }}.

## Solution {#case-resolution}

We recommend using the [import or export projects option](https://docs.gitlab.com/ee/user/project/settings/import_export.html).

## If nothing helped {#if-nothing-worked}

If the above actions did not help, [create a support ticket]({{ link-console-support }}), and we will migrate the project for you. Provide the following information in your ticket:

1. `Application ID`.
1. `Secret`.
1. Other parameters as needed, e.g., `Scopes`.
1. Preferred date and time for your project migration.
