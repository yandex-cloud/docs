# How to connect to MyWarehouse connector



## Case description {#case-description}

You need to connect to external MyWarehouse connector.

## Solution {#case-resolution}

Check that you have entered a valid token, then establish a connection following [this developer guide](https://www.htmls.ru/learning/course/index.php?COURSE_ID=12&LESSON_ID=384&LESSON_PATH=380.381.384).

## If nothing helped {#if-nothing-worked}

If the actions above did not help you solve the issue, contact the MyWarehouse connector support [via email](mailto:datalens@htmls.ru).
