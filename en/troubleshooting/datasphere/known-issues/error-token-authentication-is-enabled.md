# Resolving the `Token authentication is enabled` error



## Issue description {#issue-description}

When trying to open a project in {{ ml-platform-name }}, you see this error: `Token authentication is enabled`.

## Solution {#issue-resolution}

The issue arises because of your browser's security settings. You can learn how to fix it in this [article](../../../datasphere/qa/index.md#browser).

The possible causes might include the following:

* Your **Google Chrome** blocks third-party cookies.
* Your **Safari** has the **Prevent cross-site tracking** setting on.
* Your **Yandex Browser** blocks cookies.
