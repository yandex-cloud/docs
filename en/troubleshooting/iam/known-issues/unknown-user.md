# Unknown user detected in an organization


## Issue description {#issue-description}

In your organization, there is a user whose name and email address are not displayed.

## Solution {#issue-resolution}

This behavior might be caused by deleting a user on the {{ yandex-360 }} side or your federation server side. You can simply remove such a user from the organization, as this will not cause any problems in your cloud infrastructure.

## If the issue persists {#if-issue-still-persists}

If the above actions did not help, [create a support ticket]({{ link-console-support }}). Provide the following information in your ticket:

1. User ID.
1. Organization ID.
