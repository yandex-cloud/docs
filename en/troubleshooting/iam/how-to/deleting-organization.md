# How to delete an organization


## Case description {#case-description}

You need to delete an organization.

## Solution {#case-resolution}

Before deleting an organization, check the list of clouds in this organization by navigating to [Cloud Center]({{ link-org-main }}). 

Make sure you have selected the organization you need by clicking the profile picture in the bottom-left corner of the page. If the list is not empty, you will need to [delete](../../../resource-manager/operations/cloud/delete.md) the resources or [move them to another organization](../../../resource-manager/operations/cloud/change-organization.md).

Otherwise, you will not be able to delete your organization.

{% note info %}

This article covers {{ yandex-cloud }} organizations. If you need to delete a {{ yandex-360 }} organization, check [this guide](https://yandex.com/support/yandex-360/business/admin/en/delete-organisation.html).

{% endnote %}

Then follow these steps to delete your organization:

1. Go to [Cloud Center](https://center.yandex.cloud/).
1. Make sure you see the details on the organization you want to delete. If you selected the wrong organization, navigate to the one you need by clicking **Account** in the bottom-left corner.
1. Click **{{ ui-key.yacloud.common.delete }}** in the top-right corner of the page.


1. In the window that opens, select the organization deletion deadline. By default, any organization is deleted seven days after you click **{{ ui-key.yacloud.common.delete }}**.

1. Click **{{ ui-key.yacloud.common.delete }}**.
