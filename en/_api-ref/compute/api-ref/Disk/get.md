# Method get
Returns the specified Disk resource.
 
To get the list of available Disk resources, make a [list](/docs/compute/api-ref/Disk/list) request.
 
## HTTP request {#https-request}
```
GET https://compute.{{ api-host }}/compute/v1/disks/{diskId}
```
 
## Path parameters {#path_params}
 
Parameter | Description
--- | ---
diskId | <p>Required. ID of the Disk resource to return. To get the disk ID use a <a href="/docs/compute/api-ref/Disk/list">list</a> request.</p> <p>The maximum string length in characters is 50.</p>
