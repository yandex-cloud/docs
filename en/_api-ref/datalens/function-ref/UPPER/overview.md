# UPPER

_String functions_

#### Syntax


```
UPPER( string )
```

#### Description
Returns the string `string` in uppercase.

**Argument types:**
- `string` — `String`


**Return type**: `String`

#### Examples

```
UPPER("Lorem ipsum") = "LOREM IPSUM"
```

```
UPPER("Карл у Клары") = "КАРЛ У КЛАРЫ"
```


#### Data source support

`Materialized Dataset`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`
