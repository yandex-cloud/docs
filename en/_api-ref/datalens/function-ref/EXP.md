# EXP

_Mathematical functions_

#### Syntax


```
EXP( number )
```

#### Description
Returns the result of raising the number 'e' to the power of `number`.

**Argument types:**
- `number` â€” `Number`


**Return type**: `Number (decimal)`

#### Examples

```
EXP(0) = 1.0
```

```
EXP(1) = 2.718282
```

```
EXP(3) = 20.08553
```
