# COUNTD



#### Syntax {#syntax}


```
COUNTD( value )
```

#### Description {#description}
Returns the number of unique values in the group.

See also [COUNTD_APPROX](COUNTD_APPROX.md).

- `value` â€” `Any`


**Return type**: `Number (whole)`

