# LOWER

_String functions_

#### Syntax


```
LOWER( string )
```

#### Description
Returns the string `string` in lowercase.

**Argument types:**
- `string` — `String`


**Return type**: `String`

#### Examples

```
LOWER("Lorem ipsum") = "lorem ipsum"
```

```
LOWER("Карл у Клары") = "карл у клары"
```


#### Data source support

`Materialized Dataset`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`
