# PI

_Mathematical functions_

#### Syntax


```
PI()
```

#### Description
Returns PI. The accuracy depends on the data source.

**Return type**: `Number (decimal)`

#### Examples

```
PI() = 3.14159
```


#### Data source support

`Materialized Dataset`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`.
