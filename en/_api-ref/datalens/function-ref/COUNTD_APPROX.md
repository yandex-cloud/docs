# COUNTD_APPROX



#### Syntax {#syntax}


```
COUNTD_APPROX( value )
```

#### Description {#description}
Returns the approximate number of unique values in the group. Faster than [COUNTD](COUNTD.md), but doesn't guarantee accuracy.

- `value` â€” `Any`


**Return type**: `Number (whole)`

```

```
