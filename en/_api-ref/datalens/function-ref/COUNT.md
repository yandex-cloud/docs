# COUNT

_Function `COUNT` is also available as a [window function](COUNT_WINDOW.md)._

#### Syntax {#syntax}


```
COUNT(  [ value ] )
```

#### Description {#description}
Returns the number of items in the group.

**Argument types:**
- `value` â€” `Any`


**Return type**: `Number (whole)`

#### Examples {#examples}

```
COUNT()
```

```

```


#### Data source support {#data-source-support}
