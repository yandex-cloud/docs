# Power (^)

_Operators_

#### Syntax


```
base ^ power
```

#### Description
Raises `base` to the power of `power`.

**Argument types:**
- `base` — `Number`
- `power` — `Number`


**Return type**: `Number (decimal)`

#### Examples

```
2 ^ 3 = 8.0
```

```
2.1 ^ -0.3 = 0.80045
```


#### Data source support

`Materialized Dataset`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`.
