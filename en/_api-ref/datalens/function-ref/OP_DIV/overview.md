# Division (/)

_Operators_

#### Syntax


```
number_1 / number_2
```

#### Description
Divides the number `number_1` by the number `number_2`.

**Argument types:**
- `number_1` — `Number`
- `number_2` — `Number`


**Return type**: `Number (decimal)`

#### Examples

```
4 / 2 = 2.0
```

```
5 / 2 = 2.5
```

```
5.0 / 2 = 2.5
```

```
5 / 2.0 = 2.5
```


#### Data source support

`Materialized Dataset`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `PostgreSQL 9.3`
