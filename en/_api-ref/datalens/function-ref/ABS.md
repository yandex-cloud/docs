# ABS



#### Syntax {#syntax}


```
ABS( number )
```

#### Description {#description}
Returns the absolute value of `number`.

**Argument types:**
- `number` â€” `Number`


**Return type**: Same type as (`number`)

#### Examples {#examples}

```

```
