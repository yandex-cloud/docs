# SIN



#### Syntax {#syntax}


```
SIN( number )
```

#### Description {#description}
Returns the sine of `number` in radians.

**Argument types:**
- `number` â€” `Fractional number | Integer`


**Return type**: `Fractional number`

#### Examples {#examples}

```
SIN(0) = 0.0
```

```
SIN(PI()/2) = 1.0
```
