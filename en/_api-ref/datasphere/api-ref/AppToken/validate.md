# Method validate
Validates app token.
 

 
## HTTP request {#https-request}
```
POST https://datasphere.{{ api-host }}/datasphere/v1/appTokens/{token}:validate
```
 
## Path parameters {#path_params}
 
Parameter | Description
--- | ---
token | <p>App token to validate.</p>
