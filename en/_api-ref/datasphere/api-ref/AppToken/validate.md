# Method validate
Validates app token.
 

 
## HTTP request {#https-request}
```
POST https://datasphere.api.cloud.yandex.net/datasphere/v1/appTokens/{token}:validate
```
 
## Path parameters {#path_params}
 
Parameter | Description
--- | ---
token | App token to validate.

**HTTP Code: 200 - OK**

```json 
"object"
```
