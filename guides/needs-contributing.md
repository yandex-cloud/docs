# Важные темы для документирования

В этом файле находится список актуальных для документации Yandex Cloud тем, за материалы по которым мы начисляем повышенные гранты в рамках контент-программы. 

Если вы готовы помочь нам с написанием сценариев использования Yandex Cloud — создайте Issue с планом, обсудите его с техническими редакторами Yandex Cloud и приступайте к работе. 

## Темы

1. **Миграция в Yandex Cloud** — от других провайдеров, с использованием специальных инструментов, перенос баз данных, интернет-магазинов и блогов.
2. **Отказоустойчивая архитектура** — поделитесь опытом в построении высокодоступной архитектуры с помощью сервисов Yandex Cloud.
3. Сценарии использования **Object Storage** — расскажите о способах применения одного из самых важных сервисов.
